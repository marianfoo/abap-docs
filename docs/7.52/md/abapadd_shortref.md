ADD - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapadd.htm)

Syntax

ADD dobj1 TO dobj2.

Effect

Adds the content of a numeric data object dobj1 to the content of a numeric data object dobj2 and assigns the result to dobj2.
