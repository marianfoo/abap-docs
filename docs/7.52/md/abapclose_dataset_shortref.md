CLOSE DATASET - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapclose_dataset.htm)

Syntax

CLOSE DATASET dset.

Effect

Closes a file on the [application server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry") that was specified in dset and opened using OPEN DATASET.
