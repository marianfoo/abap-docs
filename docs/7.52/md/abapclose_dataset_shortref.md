CLOSE DATASET - Quick reference

[Reference](javascript:call_link\('abapclose_dataset.htm'\))

Syntax

CLOSE DATASET dset.

Effect

Closes a file on the [application server](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") that was specified in dset and opened using OPEN DATASET.
