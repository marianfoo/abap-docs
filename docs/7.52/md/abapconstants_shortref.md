CONSTANTS - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapconstants.htm)

Syntax

CONSTANTS const options.

Effect

Declares a constant const.

Addition

-   options
    As in [DATA](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapdata_shortref.htm), except that the addition VALUE must be specified and the addition READ-ONLY cannot be specified.
