ENDAT

[Quick Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapat_shortref.htm)

Syntax

ENDAT.

Effect

The statement ENDAT closes a control structure started using [AT](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapat_itab.htm).
