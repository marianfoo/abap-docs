ENDCLASS

[Quick Reference](javascript:call_link\('abapclass_shortref.htm'\))

Syntax

ENDCLASS.

Effect

The statement ENDCLASS closes a class declaration introduced using the following statement:
[CLASS class DEFINITION](javascript:call_link\('abapclass_definition.htm'\))
It also closes a class implementation introduced using the following formulation:
[CLASS class IMPLEMENTATION](javascript:call_link\('abapclass_implementation.htm'\))
