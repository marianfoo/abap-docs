FIELDS - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapfields.htm)

Syntax

FIELDS dobj.

Effect

Obsolete: Addresses a dobj data object to prevent a warning about the enhanced program check if the data object is not addressed statically.
