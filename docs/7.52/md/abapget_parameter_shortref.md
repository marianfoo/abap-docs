GET PARAMETER - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapget_parameter.htm)

Syntax

GET PARAMETER ID pid FIELD dobj.

Effect

Returns the value of the [SPA/GPA parameters](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenspa_gpa_parameter_1_glosry.htm "Glossary Entry") specified in in pid in dobj.
