LEAVE LIST-PROCESSING - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapleave_list-processing.htm)

Syntax

LEAVE LIST-PROCESSING.

Effect

Exits list processing and continues processing with the [PBO](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpbo_glosry.htm "Glossary Entry") event of the [dynpro](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendynpro_glosry.htm "Glossary Entry") from which the list processor was called using LEAVE TO LIST-PROCESSING or the dynpro that was specified using AND RETURN TO SCREEN.
