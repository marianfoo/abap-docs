LOOP - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abaploop-.htm)

Syntax

LOOP.
  ...
ENDLOOP.

Effect

Obsolete: Reads the [extract data set](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenextract_dataset_glosry.htm "Glossary Entry") of the program in a loop and assigns (for each loop pass) the content of the read row to the data objects that make up the [field group](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfield_group_glosry.htm "Glossary Entry") defined using INSERT INTO field\_group.
