MULTIPLY-CORRESPONDING - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapmultiply-corresponding.htm)

Syntax

MULTIPLY-CORRESPONDING struc1 BY struc2.

Effect

Obsolete: Multiplies the components of a structure struc1 with the identically named components of a structure struc2.
