RANGES - Quick reference

[Reference](javascript:call_link\('abapranges.htm'\))

Syntax

RANGES rtab FOR dobj *\[*OCCURS n*\]*.

Effect

Obsolete: Creates a [ranges table](javascript:call_link\('abenranges_table_glosry.htm'\) "Glossary Entry") with the row type of a [selection table](javascript:call_link\('abenselection_table_glosry.htm'\) "Glossary Entry") and a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry").

Addition

-   OCCURS n
    Specifies the initial memory area n.
