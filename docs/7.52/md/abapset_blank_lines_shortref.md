SET BLANK LINES - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapset_blank_lines.htm)

Syntax

SET BLANK LINES *{*ON*|*OFF*}*.

Effect

Specifies whether blank lines created using WRITE are displayed.
