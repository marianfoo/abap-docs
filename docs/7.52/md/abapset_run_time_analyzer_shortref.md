SET RUN TIME ANALYZER - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapset_run_time_analyzer.htm)

Syntax

SET RUN TIME ANALYZER *{*ON*|*OFF*}*.

Effect

Defines which units of a program are measured with the [runtime analysis](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenruntime_analysis_glosry.htm "Glossary Entry") tool.
