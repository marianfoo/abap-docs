STOP - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapstop.htm)

Syntax

STOP.

Effect

Exits the event blocks AT SELECTION-SCREEN, START-OF-SELECTION, and GET and raises the event END-OF-SELECTION.
