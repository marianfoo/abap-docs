ABAP Compiler

ABAP Compiler creates a [byte code](javascript:call_link\('abenbytecode_glosry.htm'\) "Glossary Entry") as interim code when generating a program from the [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry"). This interim code is stored in the database as a [load](javascript:call_link\('abenload_glosry.htm'\) "Glossary Entry") program and is loaded to [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry") when required.
