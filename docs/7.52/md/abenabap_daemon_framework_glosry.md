ABAP Daemon Framework

ADF for short. A framework for using [ABAP Daemons](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry"), based on [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") and [interfaces](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry"). ADF includes APIs both for communication with ABAP Daemons in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and for the handling of events by ABAP Daemons in [ABAP Daemon sessions](javascript:call_link\('abenabap_daemon_session_glosry.htm'\) "Glossary Entry").
