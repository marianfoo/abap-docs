ABAP Daemon Manager

The class CL\_ABAP\_DAEMON\_CLIENT\_MANAGER. The static methods in ABAP Daemon Manager are used to create and consumer [ABAP Daemons](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_daemon_glosry.htm "Glossary Entry") in [ABAP programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry").
