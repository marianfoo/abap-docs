ABAP dump analysis

This is a tool for listing and analyzing [runtime errors](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenruntime_error_glosry.htm "Glossary Entry") that occur in [AS ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_personality_glosry.htm "Glossary Entry"). These can be used to access the corresponding [short dumps](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenshort_dump_glosry.htm "Glossary Entry"). The tool is called using [transaction code](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentransaction_code_glosry.htm "Glossary Entry") ST22.
