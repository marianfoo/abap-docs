ABAP language element addition

[ABAP word](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_word_glosry.htm "Glossary Entry") or combination of several ABAP words. A language element addition can be used in [ABAP statements](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_statement_glosry.htm "Glossary Entry") as an addition of the associated [ABAP language element](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_language_element_glosry.htm "Glossary Entry"). Examples of different ABAP language element additions are WITH KEY and WITH TABLE KEY.
