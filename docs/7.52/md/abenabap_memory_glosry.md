ABAP memory

Memory area within every [ABAP session](javascript:call_link\('abenmain_session_glosry.htm'\) "Glossary Entry") that can be accessed jointly by the programs of a [call sequence](javascript:call_link\('abencall_sequence_glosry.htm'\) "Glossary Entry") using the statements [EXPORT](javascript:call_link\('abapexport_data_cluster.htm'\)) and [IMPORT](javascript:call_link\('abapimport_data_cluster.htm'\)). The data of this memory area is persisted as long as the [top-level transaction](javascript:call_link\('abentop_level_transaction_glosry.htm'\) "Glossary Entry") of a call sequence.
