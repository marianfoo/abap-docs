Object Services

Object Services is an object-oriented framework for working with relational databases. Instead of working directly with Open SQL statements, [persistent objects](javascript:call_link\('abenpersistent_object_glosry.htm'\) "Glossary Entry") are used whose attributes are mapped to database content. In this way the actual database access is encapsulated by the classes and interfaces of Object Services.

Object Services provide the following services for handling persistent objects on the database:

-   [Persistence service](javascript:call_link\('abenabap_object_services_persist.htm'\))

-   [Transaction service](javascript:call_link\('abenabap_object_services_transact.htm'\))

-   [Query service](javascript:call_link\('abenabap_object_services_query.htm'\))

Persistent objects are based on [persistent classes](javascript:call_link\('abenpersistent_class_glosry.htm'\) "Glossary Entry"). The creation of persistent classes is supported in Class Builder by generating an object-relational mapping using the attributes of the class and automatic creation of associated class agents.
