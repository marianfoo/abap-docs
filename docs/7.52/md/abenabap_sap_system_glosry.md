ABAP-based SAP system

Also known as ABAP system. [SAP system](javascript:call_link\('abensap_system_glosry.htm'\) "Glossary Entry") based on [Application Server ABAP](javascript:call_link\('abenabap_personality_glosry.htm'\) "Glossary Entry") (AS ABAP). An ABAP-based SAP system is identified by an SID (system ID) consisting of three characters.
