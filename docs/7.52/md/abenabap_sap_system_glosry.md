ABAP-based SAP system

Also known as ABAP system. [SAP system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_system_glosry.htm "Glossary Entry") based on [Application Server ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_personality_glosry.htm "Glossary Entry") (AS ABAP). An ABAP-based SAP system is identified by an SID (system ID) consisting of three characters.
