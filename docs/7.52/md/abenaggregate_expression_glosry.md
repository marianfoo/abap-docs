aggregate expression

In [Open SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") or in [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"), an aggregate expression uses an [aggregate function](javascript:call_link\('abenaggregate_function_glosry.htm'\) "Glossary Entry") to determine a value from the values in a column of the results set of a [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry").
