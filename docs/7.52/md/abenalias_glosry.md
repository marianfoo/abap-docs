Alias name

Either a name declared with the statement [ALIASES](javascript:call_link\('abapaliases.htm'\)) for an [interface component](javascript:call_link\('abeninterface_component_glosry.htm'\) "Glossary Entry") in a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") or in a [composite interface](javascript:call_link\('abencompound_interface_glosry.htm'\) "Glossary Entry"), or a name defined with the addition [ALIAS](javascript:call_link\('abaptypes_primary_key.htm'\)) for the [primary table key](javascript:call_link\('abenprimary_key_glosry.htm'\) "Glossary Entry") of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry").
