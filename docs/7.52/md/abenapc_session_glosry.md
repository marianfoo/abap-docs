APC session

[User session](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenuser_session_glosry.htm "Glossary Entry") for [APC processing](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapc_processing_glosry.htm "Glossary Entry"). An APC session is opened by the [APC framework](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapc_glosry.htm "Glossary Entry") when handling a WebSocket query or TCP query in an [APC handler class](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapc_handler_class_glosry.htm "Glossary Entry").
