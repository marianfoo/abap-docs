Application layer

Software layer of an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") in which application programs are executed. The application layer is realized from exactly one [message server](javascript:call_link\('abenmessage_server_glosry.htm'\) "Glossary Entry") and one or more [application servers](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry"), on which [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") are executed.
