Application Table

[Database table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_table_glosry.htm "Glossary Entry") with [client column](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclient_column_glosry.htm "Glossary Entry"). An application table contains client-specific data that is processed by [application programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_program_glosry.htm "Glossary Entry").
