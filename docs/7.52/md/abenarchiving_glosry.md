archiving

Archiving of [spool lists](javascript:call_link\('abenprint_list_glosry.htm'\) "Glossary Entry") using the [ArchiveLink](javascript:call_link\('abensap_archivelink_glosry.htm'\) "Glossary Entry") tool. Any spool lists stored in the [SAP spool system](javascript:call_link\('abensap_spool_system_glosry.htm'\) "Glossary Entry") can be archived. When being archived, spool lists from the spool system are saved in exactly the same format used there.
