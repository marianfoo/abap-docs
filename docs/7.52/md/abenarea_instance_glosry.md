area instance

Set of all [area instance versions](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenarea_instance_version_glosry.htm "Glossary Entry") with the same area instance name. An area instance is a unique variant of an [area](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenarea_glosry.htm "Glossary Entry") and is divided into versions.
