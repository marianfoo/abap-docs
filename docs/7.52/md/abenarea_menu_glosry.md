area menu

Tree structure whose nodes can be associated with [transaction codes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentransaction_code_glosry.htm "Glossary Entry"), [executable programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenexecutable_program_glosry.htm "Glossary Entry"), and other area menus. Area menus are edited using transaction code SE43 and displayed from [SAP Easy Access](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_easy_access_glosry.htm "Glossary Entry").
