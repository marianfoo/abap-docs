arithmetic expression

Formulation of an arithmetic calculation in a [calculation expression](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencalculation_expression_glosry.htm "Glossary Entry"). An arithmetical expression is associated with a [calculation type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencalculation_type_glosry.htm "Glossary Entry"), which is used to perform the calculation. The result of an arithmetic expression is a numerical value, which is in the calculation type. Arithmetic expressions occur in the [reader positions of specific statements](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoperands_expressions.htm).
