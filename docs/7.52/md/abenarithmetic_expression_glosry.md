arithmetic expression

Formulation of an arithmetic calculation in a [calculation expression](javascript:call_link\('abencalculation_expression_glosry.htm'\) "Glossary Entry"). An arithmetical expression is associated with a [calculation type](javascript:call_link\('abencalculation_type_glosry.htm'\) "Glossary Entry"), which is used to perform the calculation. The result of an arithmetic expression is a numerical value, which is in the calculation type. Arithmetic expressions occur in the [reader positions of specific statements](javascript:call_link\('abenoperands_expressions.htm'\)).
