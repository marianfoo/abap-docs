BAdI Builder

[ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") tool used to create and edit [BAdIs](javascript:call_link\('abenbadi_glosry.htm'\) "Glossary Entry"). BAdI Builder is the BAdI-specific access point to [Enhancement Builder](javascript:call_link\('abenenhancement_builder_glosry.htm'\) "Glossary Entry"). Previously, BAdI Builder was merely used for editing classic BAdIs. The builder can be called using the [transaction codes](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SE18 (definitions) and SE19 (implementations).
