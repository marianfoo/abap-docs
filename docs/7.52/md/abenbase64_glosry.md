Base64

Procedure to code any binary data in characters that can be printed. Every byte is converted into a short character string whose characters are obtained from a phrase of 64 [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry")\-independent [ASCII](javascript:call_link\('abenascii_glosry.htm'\) "Glossary Entry") characters.
