background remote function call

Asynchronous remote function call for which the called [RFMs](javascript:call_link\('abenrfm_glosry.htm'\) "Glossary Entry") can be executed in a transactional manner and in the prescribed call order. bgRFC replaces [tRFC](javascript:call_link\('abentrfc_2_glosry.htm'\) "Glossary Entry") and [qRFC](javascript:call_link\('abenqrfc_glosry.htm'\) "Glossary Entry") apart from the no-send scenario of qRFC. The latter is replaced by the [local data queue (LDQ)](javascript:call_link\('abenlocal_data_queue_glosry.htm'\) "Glossary Entry").
