binary file

[File](javascript:call_link\('abenfile_glosry.htm'\) "Glossary Entry") on an [application server](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") opened using the [BINARY](javascript:call_link\('abapopen_dataset_mode.htm'\)) addition of the [OPEN DATASET](javascript:call_link\('abapopen_dataset_mode.htm'\)) statement. The content of a binary file is read or written without special handling.
