binary floating point arithmetic

Calculation with [binary floating point numbers](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbinfloat_glosry.htm "Glossary Entry"). Can lead to rounding errors as floating point numbers are not represented internally using the decimal system.
