Bound Data Type

[Data type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_type_glosry.htm "Glossary Entry") that only exists as a property of a [data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry"). See also [Stand-Alone Data Type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenstand-alone_data_type_glosry.htm "Glossary Entry").
