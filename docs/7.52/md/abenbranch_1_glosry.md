branch

Also known as selection. [Control structure](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencontrol_structure_glosry.htm "Glossary Entry") that can consist of multiple [statement blocks](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenstatement_block_glosry.htm "Glossary Entry") defined using the keywords [IF - ELSEIF - ELSE - ENDIF](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapif.htm), [CASE - WHEN - ENDCASE](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapcase.htm), [TRY - CATCH - CLEANUP - ENDTRY](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abaptry.htm) and whose execution is dependent on conditions. Branches can be implemented in [operand positions](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoperand_position_glosry.htm "Glossary Entry") using [conditional expressions](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenconditional_expression_glosry.htm "Glossary Entry").
