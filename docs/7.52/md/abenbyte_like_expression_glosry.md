byte-like expression

Represents a [byte string](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbyte_string_glosry.htm "Glossary Entry"). A byte-like expression is a [bit expression](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbit_expression_glosry.htm "Glossary Entry") or a [function](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfunction_glosry.htm "Glossary Entry"), whose return value has a [numerical data type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbyte_like_data_typ_glosry.htm "Glossary Entry").
