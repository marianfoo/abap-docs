calculation type

Property of an [arithmetic expression](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenarithmetic_expression_glosry.htm "Glossary Entry") according to the [numeric data types](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abennumeric_data_type_glosry.htm "Glossary Entry") involved. The calculation type determines the calculation rule used to evaluate an arithmetic expression. There are different rules for the calculation types i, int8, p, decfloat34, and f.
