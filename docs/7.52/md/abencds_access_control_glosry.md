CDS access control

Implicit restriction applied to a selection of [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") by [access conditions](javascript:call_link\('abenaccess_condition_glosry.htm'\) "Glossary Entry") defined in [CDS roles](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry") when accessing objects using [Open SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry"). CDS access control can currently be applied to [CDS views](javascript:call_link\('abencds_view_glosry.htm'\) "Glossary Entry") and is enabled by default for every CDS view. It can be disabled for individual views using a [view annotation](javascript:call_link\('abencds_f1_view_entity_annotations.htm'\)).
