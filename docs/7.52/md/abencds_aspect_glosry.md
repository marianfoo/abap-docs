CDS aspect

Object used to define user-specific values and how they are used in the conditions of a [CDS role](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencds_role_glosry.htm "Glossary Entry"). Currently, the predefined aspects pfcg\_auth and user can be used.
