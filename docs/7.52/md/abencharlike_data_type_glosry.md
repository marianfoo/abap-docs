character-like data type

[Data type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_type_glosry.htm "Glossary Entry") whose [data objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry") contain [characters](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencharacter_glosry.htm "Glossary Entry") encoded according to the current [code page](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencodepage_glosry.htm "Glossary Entry"). The real character-like data types are [text field](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentext_field_glosry.htm "Glossary Entry"), [numeric text field](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abennumeric_text_field_glosry.htm "Glossary Entry"), and [text string](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentext_string_glosry.htm "Glossary Entry"). Alongside these, [date type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendate_type_glosry.htm "Glossary Entry") and [time type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentime_type_glosry.htm "Glossary Entry") are special character-like data types. The associated [generic data type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [clike](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbuilt_in_types_generic.htm).
