character-type expression

Represents a [string](javascript:call_link\('abencharacter_string_1_glosry.htm'\) "Glossary Entry"). A character type expression is a [string expression](javascript:call_link\('abenstring_expression_glosry.htm'\) "Glossary Entry") or a [function](javascript:call_link\('abenfunction_glosry.htm'\) "Glossary Entry"), whose return value has a [character-type data type](javascript:call_link\('abencharlike_data_type_glosry.htm'\) "Glossary Entry").
