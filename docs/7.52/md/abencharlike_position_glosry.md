character-like operand position

‎[Reading position](javascript:call_link\('abenreading_position_glosry.htm'\) "Glossary Entry") with character-like [operand type](javascript:call_link\('abenoperand_type_glosry.htm'\) "Glossary Entry"), in which a [character-like data object](javascript:call_link\('abencharlike_data_object_glosry.htm'\) "Glossary Entry") (which can be converted to the operand type) can be specified. See also [character-like expression position](javascript:call_link\('abencharlike_expr_position_glosry.htm'\) "Glossary Entry").
