Open SQL - System Classes

To extend the Open SQL language, the following system classes are available:

-   [CL\_OSQL\_EXTENDED\_RESULT](javascript:call_link\('abencl_osql_extended_result.htm'\)) handles extended results

-   [CL\_OSQL\_REPLACE](javascript:call_link\('abencl_osql_replace.htm'\)) provides a replacement service for executing [module tests](javascript:call_link\('abenunit_test_glosry.htm'\) "Glossary Entry")
