cluster table

[Table category](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenddic_database_tables_tab_type.htm) of a [database table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_table_glosry.htm "Glossary Entry") in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A cluster table does not exist as a variant on the [database](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_glosry.htm "Glossary Entry"). Instead, multiple cluster tables are saved in a [table cluster](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentable_cluster_glosry.htm "Glossary Entry") in the database. Only [Open SQL](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenopen_sql_glosry.htm "Glossary Entry") can be used to access cluster tables. Certain other restrictions also apply.
