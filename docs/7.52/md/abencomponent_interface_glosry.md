Component Interface

[Interface](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry") included in a [compound interface](javascript:call_link\('abencompound_interface_glosry.htm'\) "Glossary Entry") by means of an [INTERFACES](javascript:call_link\('abapinterfaces.htm'\)) statement. A component interface can itself be a compound interface.
