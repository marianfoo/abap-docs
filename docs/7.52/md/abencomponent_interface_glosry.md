Component Interface

[Interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninterface_oo_glosry.htm "Glossary Entry") included in a [compound interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencompound_interface_glosry.htm "Glossary Entry") by means of an [INTERFACES](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapinterfaces.htm) statement. A component interface can itself be a compound interface.
