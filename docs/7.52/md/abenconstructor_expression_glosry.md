constructor expression

A constructor expression uses a [constructor operator](javascript:call_link\('abenconstructor_operator_glosry.htm'\) "Glossary Entry") to create a result for a specified [type](javascript:call_link\('abentype_glosry.htm'\) "Glossary Entry") and constructs its content.
