Control Framework

Abbreviated as CFW. Class hierarchy whose [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") are prefixed with CL\_GUI\_. The common [superclass](javascript:call_link\('abensuperclass_glosry.htm'\) "Glossary Entry") is CL\_GUI\_OBJECT. The classes in Control Framework wrap [GUI controls](javascript:call_link\('abengui_control_glosry.htm'\) "Glossary Entry") and screen containers for these controls.
