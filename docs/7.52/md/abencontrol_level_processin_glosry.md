group level processing

The evaluation of [group levels](javascript:call_link\('abencontrol_level_glosry.htm'\) "Glossary Entry") in [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") or obsolete [extracts](javascript:call_link\('abenextract_glosry.htm'\) "Glossary Entry") . Group level processing is performed in [LOOP](javascript:call_link\('abaploop_at_itab.htm'\))\- [loops](javascript:call_link\('abenloop_glosry.htm'\) "Glossary Entry") using [control structures](javascript:call_link\('abencontrol_structure_glosry.htm'\) "Glossary Entry") defined using the keyword [AT](javascript:call_link\('abapat_itab.htm'\)).
