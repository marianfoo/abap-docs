Control Structure

Means of structuring a [processing block](javascript:call_link\('abenprocessing_block_glosry.htm'\) "Glossary Entry") in [statement blocks](javascript:call_link\('abenstatement_block_glosry.htm'\) "Glossary Entry"). The possible control structures are [sequence](javascript:call_link\('abensequence_glosry.htm'\) "Glossary Entry"), [branch](javascript:call_link\('abenbranch_1_glosry.htm'\) "Glossary Entry"), and [loop](javascript:call_link\('abenloop_glosry.htm'\) "Glossary Entry") as well as special control structures for exception handling. Control structures can be nested, so that closed statement blocks can contain other control structures.
