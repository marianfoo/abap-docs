Control Structure

Means of structuring a [processing block](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenprocessing_block_glosry.htm "Glossary Entry") in [statement blocks](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenstatement_block_glosry.htm "Glossary Entry"). The possible control structures are [sequence](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensequence_glosry.htm "Glossary Entry"), [branch](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbranch_1_glosry.htm "Glossary Entry"), and [loop](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenloop_glosry.htm "Glossary Entry") as well as special control structures for exception handling. Control structures can be nested, so that closed statement blocks can contain other control structures.
