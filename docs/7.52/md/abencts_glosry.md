Change and Transport System

Abbreviated to CTS. The Change and Transport System encompasses all [AS ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") tools used to manage and transport [repository objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrepository_object_glosry.htm "Glossary Entry") between [ABAP systems](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_system_glosry.htm "Glossary Entry"). The interface between a repository object and CTS is defined using the [package](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpackage_glosry.htm "Glossary Entry") of the object.
