Currency Code

Content of a [currency key](javascript:call_link\('abencurrency_key_glosry.htm'\) "Glossary Entry"). Valid currency codes are defined in the TCURC database table. The TCURX database table contains the currency code for currencies whose number of decimal places is not two.
