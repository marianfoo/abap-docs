Currency Code

Content of a [currency key](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencurrency_key_glosry.htm "Glossary Entry"). Valid currency codes are defined in the TCURC database table. The TCURX database table contains the currency code for currencies whose number of decimal places is not two.
