database note

Extension of the SQL standard, which allows it to inform the [database system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_system_glosry.htm "Glossary Entry") how to execute access. Database notes are usually specific to the database system and release. In [Open SQL](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenopen_sql_glosry.htm "Glossary Entry"), database notes can be specified using addition [%\_HINTS](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenosql_db_hints.htm).
