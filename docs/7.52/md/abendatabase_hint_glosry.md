database note

Extension of the SQL standard, which allows it to inform the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") how to execute access. Database notes are usually specific to the database system and release. In [Open SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry"), database notes can be specified using addition [%\_HINTS](javascript:call_link\('abenosql_db_hints.htm'\)).
