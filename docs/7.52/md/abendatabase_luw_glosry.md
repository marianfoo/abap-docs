database LUW

Also called database transaction. Non-divisible sequence of database operations concluded by a [database commit](javascript:call_link\('abendatabase_commit_glosry.htm'\) "Glossary Entry"). The database LUW is either executed completely or not at all by the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry"). If an error is detected within a database LUW, it is possible to revoke all database changes since the start of the database LUW using a [database rollback](javascript:call_link\('abendatabase_rollback_glosry.htm'\) "Glossary Entry").
