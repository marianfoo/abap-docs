database server

Computer in the [database layer](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_layer_glosry.htm "Glossary Entry"), on which the [database system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_system_glosry.htm "Glossary Entry") is installed.
