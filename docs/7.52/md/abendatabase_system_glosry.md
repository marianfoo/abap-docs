database system

System to store extensive amounts of data in a [database](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_glosry.htm "Glossary Entry"). Besides the database, a database system consists of a database management system that administers the database and offers access via a programming interface ([SQL](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensql_glosry.htm "Glossary Entry") for [relational databases](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrelational_database_glosry.htm "Glossary Entry")).
