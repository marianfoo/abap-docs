decimal number

Number in the decimal numbering system. It consists of [places](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenplace_glosry.htm "Glossary Entry"), which can contain the decimal digits 0-9, and can also include a [decimal separator](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendecimal_point_separat_glosry.htm "Glossary Entry") and a sign.
