default value

Value that appears in an [input field](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninput_output_field_glosry.htm "Glossary Entry") when a [screen](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenscreen_glosry.htm "Glossary Entry") is called and can be overwritten by the user. A default value is usually set in the associated [ABAP program](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry") before the screen is called.
