DOM

Document object model. Interface used to access [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") documents. Based on an object-oriented tree-like representation of the documents in the memory, where each node is represented by an object. In ABAP, [iXML Library](javascript:call_link\('abenixml_library_glosry.htm'\) "Glossary Entry") can be used to edit XML documents in DOM representation.
