DOM

Document object model. Interface used to access [XML](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenxml_glosry.htm "Glossary Entry") documents. Based on an object-oriented tree-like representation of the documents in the memory, where each node is represented by an object. In ABAP, [iXML Library](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenixml_library_glosry.htm "Glossary Entry") can be used to edit XML documents in DOM representation.
