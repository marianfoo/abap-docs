domain

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that describes properties of [data elements](javascript:call_link\('abendata_element_glosry.htm'\) "Glossary Entry"), such as the [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") or value range. A domain can be associated with any number of data elements.
