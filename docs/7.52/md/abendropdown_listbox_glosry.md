dropdown list box

A list of values for an [input/output field](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninput_output_field_glosry.htm "Glossary Entry"). Entries can only be selected from this list. The value that is entered is assigned to an entry in the dropdown list box. The program uses the assigned value and not the entry.
