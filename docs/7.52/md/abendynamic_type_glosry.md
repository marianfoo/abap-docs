Dynamic type

In a [reference variable](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"), the [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") or the [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") of the [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry"), currently referred to by the reference variable. The dynamic type is always more specific than or the same as the [static type](javascript:call_link\('abenstatic_type_glosry.htm'\) "Glossary Entry").
