edit mask

Template for formatting the output of a [data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry") in a [list](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlist_glosry.htm "Glossary Entry"). An edit mask is a character string consisting of placeholders for the characters of the data object in the output, and special characters for formatting the output.
