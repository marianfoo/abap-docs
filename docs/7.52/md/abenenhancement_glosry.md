enhancement

Addition to a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") administered according to the [enhancement concept](javascript:call_link\('abenenhancement_concept_glosry.htm'\) "Glossary Entry"), which changes its behavior without modifying it. An enhancement can be made by creating [enhancement implementation elements](javascript:call_link\('abenenhancement_impl_elem_glosry.htm'\) "Glossary Entry") for explicitly defined or implicitly available [enhancement options](javascript:call_link\('abenenhancement_point_glosry.htm'\) "Glossary Entry"). Enhancements are switchable using [Switch Framework](javascript:call_link\('abenswitch_framework_glosry.htm'\) "Glossary Entry").
