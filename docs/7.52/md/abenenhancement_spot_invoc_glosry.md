enhancement spot element invocation

Part of the definition of an explicit [enhancement point](javascript:call_link\('abenenhancement_point_glosry.htm'\) "Glossary Entry"). Position where an [enhancement spot element definition](javascript:call_link\('abenenhancement_spot_def_glosry.htm'\) "Glossary Entry") is invoked. An enhancement spot element definition can be called in multiple places, as well as in [source code enhancements](javascript:call_link\('abensource_code_enhancement_glosry.htm'\) "Glossary Entry").
