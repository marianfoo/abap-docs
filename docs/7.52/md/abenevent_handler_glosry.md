event handler

[Method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") that can handle an [event](javascript:call_link\('abenevent_glosry.htm'\) "Glossary Entry") triggered by [RAISE EVENT](javascript:call_link\('abapraise_event.htm'\)) in a different method or in the same method. To ensure that an event handler handles a triggered event, it must be registered with the statement [SET HANDLER](javascript:call_link\('abapset_handler.htm'\)).
