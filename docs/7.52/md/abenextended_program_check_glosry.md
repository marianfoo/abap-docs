extended program check

Complete check of the [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") for all errors that can be detected statically. Programs with errors in the extended program check can be executed, but usually raise an [exception](javascript:call_link\('abenexception_glosry.htm'\) "Glossary Entry"). Called in [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry") by choosing Program → Check → Extended Program Check or using the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SLIN or using [ABAP Test Cockpit](javascript:call_link\('abenabap_test_cockpit_glosry.htm'\) "Glossary Entry"). Settings for the extended program check can be made in the administration transaction SLIN\_ADMIN.
