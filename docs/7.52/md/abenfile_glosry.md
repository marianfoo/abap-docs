file

Repository of data with a name in a persistent storage medium. The statements of the [ABAP file interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfile_interface_glosry.htm "Glossary Entry") can be used to edit files on the [application server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry"). For files on the [presentation servers](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpresentation_server_glosry.htm "Glossary Entry"), [function modules](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfunction_module_glosry.htm "Glossary Entry") or [methods](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmethod_glosry.htm "Glossary Entry") of a global [class](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") are used. The files of the [database server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_server_glosry.htm "Glossary Entry") are part of the [database](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_glosry.htm "Glossary Entry") and are managed by the [database system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_system_glosry.htm "Glossary Entry").
