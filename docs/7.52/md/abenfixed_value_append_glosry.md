fixed value append

A fixed value append defines [fixed values](javascript:call_link\('abenfixed_value_glosry.htm'\) "Glossary Entry") and/or intervals appended to an existing [domain](javascript:call_link\('abendomain_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). Domains delivered by SAP can be enhanced in customer systems using fixed value appends without making modifications.
