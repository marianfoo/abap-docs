floating point arithmetic

Calculation using [floating point numbers](javascript:call_link\('abenfloating_point_number_2_glosry.htm'\) "Glossary Entry"), which is executed directly on the platform of the current [application server](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry"). Either [binary](javascript:call_link\('abenbinfloat_arith_glosry.htm'\) "Glossary Entry") or [decimal floating point arithmetic](javascript:call_link\('abendecfloat_arith_glosry.htm'\) "Glossary Entry").
