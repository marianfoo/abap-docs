floating point arithmetic

Calculation using [floating point numbers](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfloating_point_number_2_glosry.htm "Glossary Entry"), which is executed directly on the platform of the current [application server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry"). Either [binary](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbinfloat_arith_glosry.htm "Glossary Entry") or [decimal floating point arithmetic](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendecfloat_arith_glosry.htm "Glossary Entry").
