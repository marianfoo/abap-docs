Floating point number

Either a [decimal floating point number](javascript:call_link\('abendecfloat_glosry.htm'\) "Glossary Entry") or a [binary floating point number](javascript:call_link\('abenbinfloat_glosry.htm'\) "Glossary Entry"). In floating point numbers, the size of the [fractional portion](javascript:call_link\('abenfractional_portion_glosry.htm'\) "Glossary Entry") is part of the value and not part of the data type, as is the case in [packed numbers](javascript:call_link\('abenpacked_number_glosry.htm'\) "Glossary Entry"). [Floating point arithmetic](javascript:call_link\('abenfloating_point_arith_glosry.htm'\) "Glossary Entry") is used for calculations with floating point numbers.
