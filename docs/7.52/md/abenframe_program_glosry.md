master program

Term for a program organized into [include programs](javascript:call_link\('abeninclude_program_glosry.htm'\) "Glossary Entry"). An include program should only be assigned to a single master program. The [ABAP Unit](javascript:call_link\('abenabap_unit_glosry.htm'\) "Glossary Entry") tests for the units in the include programs are usually created in the [test include](javascript:call_link\('abentest_include_glosry.htm'\) "Glossary Entry") of the master program.
