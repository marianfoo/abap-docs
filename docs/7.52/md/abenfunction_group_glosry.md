function group

The only [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that can contain [function modules](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry"); usually loaded using a function module. A function group can currently contain a maximum of 99 function modules. It supports its own [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry").
