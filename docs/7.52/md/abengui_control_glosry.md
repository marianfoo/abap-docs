GUI control

Standalone software component of the [presentation layer](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpresentation_layer_glosry.htm "Glossary Entry"). Examples of GUI controls are the Picture Control, the Text Edit Control, and the Tree Control. GUI controls are installed with [SAP GUI](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_gui_glosry.htm "Glossary Entry") and are accessed in ABAP using [classes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") in [Control Framework](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencontrol_framework_glosry.htm "Glossary Entry").
