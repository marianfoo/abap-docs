GUI status

Groups the [menu bar](javascript:call_link\('abenmenu_bar_glosry.htm'\) "Glossary Entry"), [standard toolbar](javascript:call_link\('abenstandard_toolbar_glosry.htm'\) "Glossary Entry"), and [application toolbar](javascript:call_link\('abenapplication_toolbar_glosry.htm'\) "Glossary Entry") of a [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry") together with the [function key settings](javascript:call_link\('abenfunction_key_setting_glosry.htm'\) "Glossary Entry"). A GUI status is set using the statement [SET PF-STATUS](javascript:call_link\('abapset_pf-status_dynpro.htm'\)), and is created using the [Menu Painter](javascript:call_link\('abenmenu_painter_glosry.htm'\) "Glossary Entry") tool.
