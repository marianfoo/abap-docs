SAP HANA

[SAP HANA](http://help.sap.com/hana) platform for SAP In-Memory Computing based on the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). In ABAP programs, SAP HANA can be accessed using [secondary connections](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry"). Alternatively, an SAP HANA database is used as a [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"), making direct reads possible here.
