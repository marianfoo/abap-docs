SAP HANA Studio

Tool based on Eclipse for [XSV](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenxsc_glosry.htm "Glossary Entry") objects saved in the [SAP HANA repository](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_hana_repository_glosry.htm "Glossary Entry"). In [XSA](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenxsa_glosry.htm "Glossary Entry"), SAP HANA Studio is replaced by [SAP Web IDE for SAP HANA](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_web_ide_for_sap_han_glosry.htm "Glossary Entry") and [SAP HANA Cockpit](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_hana_cockpit_glosry.htm "Glossary Entry").
