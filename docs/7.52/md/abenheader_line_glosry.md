header line

Obsolete [work area](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenwork_area_glosry.htm "Glossary Entry") of an [internal table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninternal_table_glosry.htm "Glossary Entry") that has the type of the [row type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrow_type_glosry.htm "Glossary Entry") and the same name as the internal table. If an internal table with a header line is used in an [operand position](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoperand_position_glosry.htm "Glossary Entry"), the header line is usually addressed. To enforce access to the [table body](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentable_body_glosry.htm "Glossary Entry"), square brackets \[\] can be entered after the table name.
