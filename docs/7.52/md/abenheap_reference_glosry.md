heap reference

[Reference](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") or part of an object in the [heap](javascript:call_link\('abenheap_glosry.htm'\) "Glossary Entry"). A heap reference keeps an object alive. See also [stack reference](javascript:call_link\('abenstack_reference_glosry.htm'\) "Glossary Entry").
