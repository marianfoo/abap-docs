ICF server

Instance of a system that provides services called using [ICF](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenicf_glosry.htm "Glossary Entry"). An ICF server can be a HTTP/HTTPS server or an SMTP server.
