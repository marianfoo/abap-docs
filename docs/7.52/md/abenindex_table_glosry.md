index table

General term for [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") that are maintained using a [primary table index](javascript:call_link\('abenprimary_table_index_glosry.htm'\) "Glossary Entry") ([standard tables](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry") and [sorted tables](javascript:call_link\('abensorted_table_glosry.htm'\) "Glossary Entry")).

You can assign a [secondary table key](javascript:call_link\('abensecondary_table_key_glosry.htm'\) "Glossary Entry") to enable access using a [hash algorithm](javascript:call_link\('abenhash_algorithm_glosry.htm'\) "Glossary Entry") The corresponding [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [index table](javascript:call_link\('abenbuilt_in_types_generic.htm'\)).
