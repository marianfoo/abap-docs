instance component

Superordinate term encompassing [instance attributes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstance_attribute_glosry.htm "Glossary Entry"), [instance events](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstance_event_glosry.htm "Glossary Entry"), and [instance methods](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstance_method_glosry.htm "Glossary Entry"). The instance components of a [class](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") can only be addressed in [instances](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstance_glosry.htm "Glossary Entry") of the class.
