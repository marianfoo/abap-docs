Instance

Concrete occurence of a [type](javascript:call_link\('abentype_glosry.htm'\) "Glossary Entry"). The instance of a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") is a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). The instance of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") is often simply described as an object. For more information, see [instantiation](javascript:call_link\('abeninstantiation_glosry.htm'\) "Glossary Entry").
