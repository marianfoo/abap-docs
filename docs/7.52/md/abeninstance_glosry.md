Instance

Concrete occurence of a [type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentype_glosry.htm "Glossary Entry"). The instance of a [data type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_type_glosry.htm "Glossary Entry") is a [data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry"). The instance of a [class](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") is often simply described as an object. For more information, see [instantiation](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstantiation_glosry.htm "Glossary Entry").
