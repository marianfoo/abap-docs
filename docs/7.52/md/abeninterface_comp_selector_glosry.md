interface component selector

The ~ character. A [component](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencomponent_glosry.htm "Glossary Entry") [comp](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_ifac_components.htm) of an [interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninterface_oo_glosry.htm "Glossary Entry") intf can be addressed in an implementing [class](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") or a [compound interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencompound_interface_glosry.htm "Glossary Entry") using intf~comp.
