interface pool

[ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that contains the definition of exactly one [global interface](javascript:call_link\('abenglobal_interface_glosry.htm'\) "Glossary Entry") and is loaded using the interface. Does not support its own [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry").
