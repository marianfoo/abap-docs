iteration expression

Expression that performs an [iteration](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeniteration_glosry.htm "Glossary Entry"). An iteration expression is subexpression of a [constructor expression](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenconstructor_expression_glosry.htm "Glossary Entry") introduced using an [iteration operator](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeniteration_operator_glosry.htm "Glossary Entry") [FOR](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfor.htm) and performs [conditional iterations](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfor_conditional.htm) or [table iterations](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentable_iteration_glosry.htm "Glossary Entry").
