join

Link between multiple [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") in which each pair of tables involved are joined using conditions between one or more columns. Joins can be implemented declaratively as [views](javascript:call_link\('abenview_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"), or programmatically as [join expressions](javascript:call_link\('abenjoin_expression_glosry.htm'\) "Glossary Entry") in the [SELECT](javascript:call_link\('abapselect_join.htm'\)) statement or in the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry").
