JSON

JavaScript Object Notation ([http://www.json.org/](http://www.json.org/)). Data format in text form for data exchange between applications. In ABAP, [sXML Library](javascript:call_link\('abensxml_library_glosry.htm'\) "Glossary Entry") and [transformations](javascript:call_link\('abenabap_json_trafos.htm'\)) can be used to access JSON. This is enabled by a special [JSON-XML](javascript:call_link\('abenjson_xml_glosry.htm'\) "Glossary Entry") supported by sXML Library.
