Key table

Superordinate term for [sorted tables](javascript:call_link\('abensorted_table_glosry.htm'\) "Glossary Entry") and [hash tables](javascript:call_link\('abenhashed_table_glosry.htm'\) "Glossary Entry") for which the [primary table key](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry"), unlike for [standard tables](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry"), has its own stable administration that permits optimized access.
