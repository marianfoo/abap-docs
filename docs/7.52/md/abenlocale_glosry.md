locale

Part of the [text environment](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentext_environment_glosry.htm "Glossary Entry"). Defines the language-specific and country-specific properties of characters, such as the sort order and the handling of uppercase and lowercase letters etc. Local properties are defined in a library independent of an operating system.
