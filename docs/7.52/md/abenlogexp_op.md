rel\_exp - Relational Operators

Depending on the operands that are compared with each other, four groups of relational operators for relational expressions can be distinguished:

-   Relational operators for all data types
    [\=, EQ, <>, NE, \>, GT, <, LT, \>=, GE, <=, LE](javascript:call_link\('abenlogexp_any_operand.htm'\))
    [BETWEEN](javascript:call_link\('abenlogexp_between.htm'\))
    [IN](javascript:call_link\('abenlogexp_select_option.htm'\))
    

-   Relational operators for character-like data types
    [CO, CN, CA, NA, CS, NS, CP, NP](javascript:call_link\('abenlogexp_strings.htm'\))

-   Relational operators for byte-like data types
    [BYTE-CO, BYTE-CN, BYTE-CA, BYTE-NA, BYTE-CS, BYTE-NS](javascript:call_link\('abenlogexp_bytes.htm'\))

-   Relational operators for bit patterns
    [O, Z, M](javascript:call_link\('abenlogexp_bitmasks.htm'\))
