rel\_exp - Relational Operators

Depending on the operands that are compared with each other, four groups of relational operators for relational expressions can be distinguished:

-   Relational operators for all data types
    [\=, EQ, <>, NE, \>, GT, <, LT, \>=, GE, <=, LE](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_any_operand.htm)
    [BETWEEN](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_between.htm)
    [IN](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_select_option.htm)
    

-   Relational operators for character-like data types
    [CO, CN, CA, NA, CS, NS, CP, NP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_strings.htm)

-   Relational operators for byte-like data types
    [BYTE-CO, BYTE-CN, BYTE-CA, BYTE-NA, BYTE-CS, BYTE-NS](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_bytes.htm)

-   Relational operators for bit patterns
    [O, Z, M](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogexp_bitmasks.htm)
