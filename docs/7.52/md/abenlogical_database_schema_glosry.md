logical database schema

[Logical schema](javascript:call_link\('abenlogical_schema_glosry.htm'\) "Glossary Entry") in which the mapping of a physical [ABAP database schema](javascript:call_link\('abenabap_db_schema_glosry.htm'\) "Glossary Entry") in the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") is represented in a mapping table. A logical database schema is represented by a transportable [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that defines the properties of the schema.
