logical file name

Cross-platform name that can be associated with [physical file names](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenphysical_filename_glosry.htm "Glossary Entry") using the transactions FILE and SF01. Logical file names are used for platform-independent programming to [validate file names](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendataset_auth_self.htm).
