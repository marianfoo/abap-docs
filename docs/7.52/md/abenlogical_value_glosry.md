truth value

A truth value is either true or false. In ABAP, truth values only occur as the result of [logical expressions](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogical_expression_glosry.htm "Glossary Entry"). [Data types](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_type_glosry.htm "Glossary Entry") for [Boolean data objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenboolean_data_object_glosry.htm "Glossary Entry") that can save truth values are not supported. Instead, a truth value can be evaluated as the result of a [Boolean function](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenboole_function_glosry.htm "Glossary Entry").
