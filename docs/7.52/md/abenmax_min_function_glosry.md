Extreme Value Function

[Function](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfunction.htm) which returns the maximum or minimum value from a set of numeric arguments ([nmax, nmin](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abennmax_nmin_functions.htm)) or character-type arguments ([cmax, cmin](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencmax_cmin_functions.htm)).
