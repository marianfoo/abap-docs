Extreme Value Function

[Function](javascript:call_link\('abenfunction.htm'\)) which returns the maximum or minimum value from a set of numeric arguments ([nmax, nmin](javascript:call_link\('abennmax_nmin_functions.htm'\))) or character-type arguments ([cmax, cmin](javascript:call_link\('abencmax_cmin_functions.htm'\))).
