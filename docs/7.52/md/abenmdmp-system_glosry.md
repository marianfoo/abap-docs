MDMP system

Abbreviation of multi-display, multi-processing system. An MDMP system contains multiple non- [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") [system code pages](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry"). The [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry") of an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") is determined by its current [text environment](javascript:call_link\('abentext_environment_glosry.htm'\) "Glossary Entry"). MDMP systems are no longer supported from [SAP NetWeaver](javascript:call_link\('abensap_netweaver_glosry.htm'\) "Glossary Entry"), Release 7.0.
