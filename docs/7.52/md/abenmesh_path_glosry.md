mesh path

Chaining of multiple [mesh nodes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmesh_node_glosry.htm "Glossary Entry") using their [mesh associations](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmesh_association_glosry.htm "Glossary Entry"). A mesh path describes a set of rows in its final path node. These rows are accessed when a mesh path is used.
