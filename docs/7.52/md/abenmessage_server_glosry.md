message server

Component of the [application layer](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_layer_glosry.htm "Glossary Entry") that realizes communication between the [application servers](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry"), and which determines the application server used when a [user](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenuser_glosry.htm "Glossary Entry") logs on to an [AS ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_nw_abap_glosry.htm "Glossary Entry").
