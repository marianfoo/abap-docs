message server

Component of the [application layer](javascript:call_link\('abenapplication_layer_glosry.htm'\) "Glossary Entry") that realizes communication between the [application servers](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry"), and which determines the application server used when a [user](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry") logs on to an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry").
