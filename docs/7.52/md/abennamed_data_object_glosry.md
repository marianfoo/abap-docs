named data object

[Data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry") that can be addressed using a name. Named data objects are [constants](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenconstant_glosry.htm "Glossary Entry"), [variables](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenvariable_glosry.htm "Glossary Entry"), and [text symbols](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentext_symbol_glosry.htm "Glossary Entry"). Named data objects can be stored in the [stack](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenstack_glosry.htm "Glossary Entry") or as [instance attributes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninstance_attribute_glosry.htm "Glossary Entry") in the [heap](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenheap_glosry.htm "Glossary Entry"). Also see [unnamed data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenunnamed_data_object_glosry.htm "Glossary Entry").
