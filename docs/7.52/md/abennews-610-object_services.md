Object Services in Release 6.10

Object services are low-level services used for object-oriented programming with ABAP Objects, but which are not implemented as language elements. In Release 6.10, persistence services and transaction services are available. For more details, see [ABAP - Object Services](javascript:call_link\('abenabap_object_services.htm'\)).
