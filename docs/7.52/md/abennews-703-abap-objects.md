Modification 1

Chained Method Call

From Release 7.31, the [chained method call](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenchained_method_call_glosry.htm "Glossary Entry") can not only be specified in functions but also executed in operations.
