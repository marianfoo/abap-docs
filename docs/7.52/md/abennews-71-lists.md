[1\. Support for the new 12 hour format](#!ABAP_MODIFICATION_1@1@)

Modification 1

Support for the New 12 Hour Format

The new addition [ENVIRONMENT TIME FORMAT](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapwrite_to_options.htm) to the WRITE TO and WRITE statements supports the new [12 hour format](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abennews-710-others.htm).

The [predefined time formats](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenwrite_formats.htm) of the WRITE TO and WRITE statements cannot be influenced automatically due to downward compatibility.

Note

SP14 includes this change in Release 7.0 as well.
