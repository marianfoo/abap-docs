Modification 1

Search in Subtrees in the SAP GUI Version of the ABAP Keyword Documentation

In the documentation display of the SAP GUI version of the ABAP keyword documentation, the search for a search term can be restricted to the current subtree. This option can be selected from the context menu of a node or in the dialog window of the documentation.

Modification 2

Standard ABAP with Restricted Object Use

In Release 7.52, a new [ABAP version](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_version_glosry.htm "Glossary Entry") [Standard ABAP with Restricted Object Use](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenstandard_abap_restr_obj_glosry.htm "Glossary Entry") was introduced. The internal version ID in column UCCHECK of system table TRDIR is "4".
