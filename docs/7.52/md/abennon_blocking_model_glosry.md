non-blocking mode

Any statements that block the program flow are forbidden in [ABAP programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry") that run in non-blocking mode. Non-blocking mode is available in different instances for [APC](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapc_non_blocking_model.htm) and [ADF](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenadf_non_blocking_model.htm).
