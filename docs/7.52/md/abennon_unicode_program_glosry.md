obsolete ABAP (non-Unicode)

Obsolete ABAP is a [language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") in which the [Unicode checks](javascript:call_link\('abenunicode_check_glosry.htm'\) "Glossary Entry") are ignored. A program with this language version cannot be executed in [Unicode systems](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry"). In the current [SAP NetWeaver](javascript:call_link\('abensap_netweaver_glosry.htm'\) "Glossary Entry") release, only Unicode systems are supported. Therefore, obsolete ABAP can no longer be used.
