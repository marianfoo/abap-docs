numerical expression

Represents a numerical value. A numerical expression is an [arithmetical expression](javascript:call_link\('abenarithmetic_expression_glosry.htm'\) "Glossary Entry"), or a [function](javascript:call_link\('abenfunction_glosry.htm'\) "Glossary Entry"), whose return value has a [numerical data type](javascript:call_link\('abennumeric_data_type_glosry.htm'\) "Glossary Entry").
