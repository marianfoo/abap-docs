object reference

[Reference](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") that points to an [instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry"). An non-initial object reference is always a [Heapreference](javascript:call_link\('abenheap_reference_glosry.htm'\) "Glossary Entry"). For more information, see [weak reference](javascript:call_link\('abenweak_reference_glosry.htm'\) "Glossary Entry").
