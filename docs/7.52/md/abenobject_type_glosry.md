object type

Description of an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") in [ABAP Objects](javascript:call_link\('abenabap_objects_glosry.htm'\) "Glossary Entry"). Either a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") for the entire object or [interface](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry") for part of an object. The associated [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [object](javascript:call_link\('abenbuilt_in_types_generic.htm'\)). Object types are defined either locally in an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") or globally as part of the [class library](javascript:call_link\('abenclass_library_glosry.htm'\) "Glossary Entry").
