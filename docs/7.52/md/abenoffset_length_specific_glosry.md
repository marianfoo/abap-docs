offset/length specification

Provides access to subareas of a [data object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_object_glosry.htm "Glossary Entry") by appending *\[*+off*\]**\[*(len)*\]* to the name of the data object.
