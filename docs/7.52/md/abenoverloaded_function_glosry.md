general numerical function

[Numerical function](javascript:call_link\('abenmathematical_funktion_glosry.htm'\) "Glossary Entry") that expects a [numeric data object](javascript:call_link\('abennumeric_data_object_glosry.htm'\) "Glossary Entry") as argument. The [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of the argument determines the data type of the [return value](javascript:call_link\('abenreturn_value_glosry.htm'\) "Glossary Entry") of the function.
