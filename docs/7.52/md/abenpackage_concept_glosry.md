package concept

Rule set that is based on [packages](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpackage_glosry.htm "Glossary Entry") and specifies the mutual usability of [repository objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrepository_object_glosry.htm "Glossary Entry"). Currently, only some of the rules are checked and only by the extended program check.
