parameter interface

[Interface](javascript:call_link\('abeninterface_glosry.htm'\) "Glossary Entry") of a [procedure](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry"). The parameter interface consists of [formal parameters](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") and specifies the possible [exceptions](javascript:call_link\('abenexception_glosry.htm'\) "Glossary Entry") of the procedure.
