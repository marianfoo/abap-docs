parameter interface

[Interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninterface_glosry.htm "Glossary Entry") of a [procedure](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenprocedure_glosry.htm "Glossary Entry"). The parameter interface consists of [formal parameters](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenformal_parameter_glosry.htm "Glossary Entry") and specifies the possible [exceptions](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenexception_glosry.htm "Glossary Entry") of the procedure.
