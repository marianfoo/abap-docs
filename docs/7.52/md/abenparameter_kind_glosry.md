parameter category

Category of [formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry"). [Input parameters](javascript:call_link\('abeninput_parameter_glosry.htm'\) "Glossary Entry"), [output parameters](javascript:call_link\('abenoutput_parameter_glosry.htm'\) "Glossary Entry"), [input/output parameters](javascript:call_link\('abeninput_output_parameter_glosry.htm'\) "Glossary Entry"), and [return values](javascript:call_link\('abenreturn_value_glosry.htm'\) "Glossary Entry") are all possible categories. The parameter category is specified in the definition of a formal parameter.
