physical file name

Platform-dependent name of a [file](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfile_glosry.htm "Glossary Entry") on the [application server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry"). The statements of the [ABAP file interface](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfile_interface_glosry.htm "Glossary Entry") work directly with the physical file names. Physical file names can be associated with [logical file names](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogical_filename_glosry.htm "Glossary Entry") to enable platform-independent and secure programming.
