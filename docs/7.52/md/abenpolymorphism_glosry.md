Polymorphism

Polymorphism means - in object-oriented programming - to be able to address differently implemented [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") that belong to different [objects](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of different [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") with the same descriptor via the same interface.
