pooled table

[Table category](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenddic_database_tables_tab_type.htm) of a [database table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_table_glosry.htm "Glossary Entry") in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A pooled table does not exist as a variant on the [database](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_glosry.htm "Glossary Entry"). Instead, multiple pooled tables are saved in a [table pool](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentable_pool_glosry.htm "Glossary Entry") in the database. Only [Open SQL](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenopen_sql_glosry.htm "Glossary Entry") can be used to access pooled tables. Certain other restrictions also apply.
