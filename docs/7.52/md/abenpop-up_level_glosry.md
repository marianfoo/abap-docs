popup level

Hierarchy level of a [GUI window](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abengui_window_glosry.htm "Glossary Entry"). The first GUI window that appears in a [logon](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlogon_glosry.htm "Glossary Entry") to [AS ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") has the popup level 0. Higher popup levels are created by stacking modal [dialog boxes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendialog_box_glosry.htm "Glossary Entry") using the statement [CALL SCREEN](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapcall_screen.htm) or [CALL SELECTION-SCREEN](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapcall_selection_screen.htm). The maximum popup level is 9.
