popup level

Hierarchy level of a [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry"). The first GUI window that appears in a [logon](javascript:call_link\('abenlogon_glosry.htm'\) "Glossary Entry") to [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") has the popup level 0. Higher popup levels are created by stacking modal [dialog boxes](javascript:call_link\('abendialog_box_glosry.htm'\) "Glossary Entry") using the statement [CALL SCREEN](javascript:call_link\('abapcall_screen.htm'\)) or [CALL SELECTION-SCREEN](javascript:call_link\('abapcall_selection_screen.htm'\)). The maximum popup level is 9.
