Presentation Server

Desktop PCs belonging to users of an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"). The [presentation layer](javascript:call_link\('abenpresentation_layer_glosry.htm'\) "Glossary Entry") on these PCs is provided either by installing a [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry") or by a Web browser.
