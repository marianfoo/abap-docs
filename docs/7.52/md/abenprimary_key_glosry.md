primary key

[table key](javascript:call_link\('abentable_key_glosry.htm'\) "Glossary Entry") that is determined when you define a [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") and that distinctively defines a table row. For [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") the [primary table key](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry").
