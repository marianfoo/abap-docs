program directive

A program directive is a special construct in [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") that modifies the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") rather than affecting the program flow directly.
