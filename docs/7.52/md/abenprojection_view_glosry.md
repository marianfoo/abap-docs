projection view

Special [classic view](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclassical_view_glosry.htm "Glossary Entry") in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") for a [database table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_table_glosry.htm "Glossary Entry"). A projection view is not defined as an SQL view on the [database](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_glosry.htm "Glossary Entry").
