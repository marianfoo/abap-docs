Reference type

A reference type describes [reference variables](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"), that is [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") that contain references. See also [static type](javascript:call_link\('abenstatic_type_glosry.htm'\) "Glossary Entry").
