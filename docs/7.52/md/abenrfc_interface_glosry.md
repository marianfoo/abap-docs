RFC interface

Interface for [remote function calls](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenremote_function_call_glosry.htm "Glossary Entry") (RFC). The RFC interface consists of both an interface for [ABAP programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry") and interfaces for non-ABAP programs.
