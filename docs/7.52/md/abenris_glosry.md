Repository Information System

Tool in [ABAP Workbench](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_workbench_glosry.htm "Glossary Entry") used when searching for [repository objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrepository_object_glosry.htm "Glossary Entry"). Called from [Object Navigator](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenobject_navigator_glosry.htm "Glossary Entry") or [transaction code](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE15.
