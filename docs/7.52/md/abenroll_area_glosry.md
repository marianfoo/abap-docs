program memory

Memory for the modifiable parts of a program in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry"). The program memory can be further divided into the areas [heap](javascript:call_link\('abenheap_glosry.htm'\) "Glossary Entry") and [stack](javascript:call_link\('abenstack_glosry.htm'\) "Glossary Entry"). The non-modifiable parts of a program are saved in [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry"), which is referenced in the program memory.
