root class

A [superclass](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensuperclass_glosry.htm "Glossary Entry") shared by all [subclasses](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensubclass_glosry.htm "Glossary Entry") in an [inheritance tree](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninheritance_tree_glosry.htm "Glossary Entry") in [single inheritance](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensingle_inheritance_glosry.htm "Glossary Entry"). In [ABAP Objects](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_objects_glosry.htm "Glossary Entry"), all [classes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") are subclasses of the predefined [abstract](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabstract_glosry.htm "Glossary Entry") root class [object](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbuilt_in_types_generic.htm).
