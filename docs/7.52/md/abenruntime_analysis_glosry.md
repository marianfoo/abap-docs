runtime analysis

Tool for analyzing the execution of program parts or individual statements and for measuring their runtime in SAP GUI. Called using [transaction code](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentransaction_code_glosry.htm "Glossary Entry") SAT. Replaced by the ABAP Profiler in the [ABAP Development Tools (ADT)](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenadt_glosry.htm "Glossary Entry").
