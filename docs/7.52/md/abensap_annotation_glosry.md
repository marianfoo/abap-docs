SAP annotation

[CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") delivered by SAP as a [CDS annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry"). There are [ABAP annotations](javascript:call_link\('abencore_annotation_glosry.htm'\) "Glossary Entry"), which are evaluated by the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") for every [CDS object, and](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") [framework-specific annotations](javascript:call_link\('abencomponent_annotation_glosry.htm'\) "Glossary Entry"), which are evaluated by frameworks in other software components.
