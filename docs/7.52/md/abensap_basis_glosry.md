SAP Basis

Before Release 6.10, SAP Basis was the central platform for all SAP applications written in ABAP. From Release 6.10, the functions of SAP Basis are fully integrated into [Application Server ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_personality_glosry.htm "Glossary Entry").
