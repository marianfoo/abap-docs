SAP Code Page Number

SAP internal identification for a [code page](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencodepage_glosry.htm "Glossary Entry") that is stored in the TCP00 database table. If the [byte order](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbyte_order_glosry.htm "Glossary Entry") of a code page is relevant, an individual SAP code page number is assigned to it there for each [endian](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenendian_glosry.htm "Glossary Entry"). For example, the code page numbers for [UTF-16](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenutf16_glosry.htm "Glossary Entry") are 4102 ([big endian](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbig_endian_glosry.htm "Glossary Entry")) and 4103 ([little endian](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenlittle_endian_glosry.htm "Glossary Entry")).
