SAP GUI

Software component which is installed on the [presentation server](javascript:call_link\('abenpresentation_server_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"). Implements the SAP-specific [GUI](javascript:call_link\('abengui_glosry.htm'\) "Glossary Entry") of the [application layer](javascript:call_link\('abenapplication_layer_glosry.htm'\) "Glossary Entry") applications and supports additional functions on the [presentation layer](javascript:call_link\('abenpresentation_layer_glosry.htm'\) "Glossary Entry"), such as [GUI controls](javascript:call_link\('abengui_control_glosry.htm'\) "Glossary Entry") and [OLE automation](javascript:call_link\('abenole_automation_glosry.htm'\) "Glossary Entry").
