SAP system

Software system based on the [SAPApplication Server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_nw_app_server_glosry.htm "Glossary Entry") in [SAP NetWeaver](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_netweaver_glosry.htm "Glossary Entry"). An SAP system that is based on the [Application Server ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_personality_glosry.htm "Glossary Entry") is an [ABAP-based SAP system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_sap_system_glosry.htm "Glossary Entry").
