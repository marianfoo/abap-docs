SAP system

Software system based on the [SAPApplication Server](javascript:call_link\('abensap_nw_app_server_glosry.htm'\) "Glossary Entry") in [SAP NetWeaver](javascript:call_link\('abensap_netweaver_glosry.htm'\) "Glossary Entry"). An SAP system that is based on the [Application Server ABAP](javascript:call_link\('abenabap_personality_glosry.htm'\) "Glossary Entry") is an [ABAP-based SAP system](javascript:call_link\('abenabap_sap_system_glosry.htm'\) "Glossary Entry").
