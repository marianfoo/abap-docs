seam

Section of source code that can be inserted ("injected") or replaced dynamically. Seams appear in ABAP as [test seams](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentest_seam_glosry.htm "Glossary Entry").
