secondary database

Target database of a [secondary connection](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") defined in a database table DBCON. The database system must be a system supported by SAP and the software required for use with ABAP must be installed as a database shared library (DBSL) on the current application server.
