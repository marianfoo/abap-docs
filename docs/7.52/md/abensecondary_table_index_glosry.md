secondary table index

[Table index](javascript:call_link\('abentable_index_glosry.htm'\) "Glossary Entry") for the administration of [sorted](javascript:call_link\('abensorted_key_glosry.htm'\) "Glossary Entry") [secondary table keys](javascript:call_link\('abensecondary_table_key_glosry.htm'\) "Glossary Entry"). A secondary table index of a non-unique key is not updated instantly after a line is added to, or deleted from, an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry"), it is only updated when the internal table is accessed via the associated secondary key.
