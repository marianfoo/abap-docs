selection table

Global [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") with a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry"), the columns SIGN, OPTION, LOW, and HIGH and the name of a [selection criterion](javascript:call_link\('abenselection_criterion_glosry.htm'\) "Glossary Entry"). Used as an internal repository for selection conditions. Declaration with the statement [SELECT-OPTIONS](javascript:call_link\('abapselect-options.htm'\)); can be evaluated in a [comparison expression](javascript:call_link\('abenrelational_expression_glosry.htm'\) "Glossary Entry") using the [relational operator](javascript:call_link\('abenrelational_operator_glosry.htm'\) "Glossary Entry") [IN](javascript:call_link\('abenlogexp_select_option.htm'\)), where each row of the table represents a separate comparison.
