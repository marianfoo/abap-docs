short dump

Error log displayed and saved after a [runtime error](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenruntime_error_glosry.htm "Glossary Entry") or an [exit message](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenexit_message_glosry.htm "Glossary Entry"). Saved short dumps can be displayed using the [ABAP dump analysis](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_dump_analsyis_glosry.htm "Glossary Entry") tool.
