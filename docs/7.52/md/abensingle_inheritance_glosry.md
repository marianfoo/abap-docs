single inheritance

[Inheritance](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninheritance_glosry.htm "Glossary Entry") whose class can have several [subclasses](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensubclass_glosry.htm "Glossary Entry") but only one direct [superclass](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensuperclass_glosry.htm "Glossary Entry") in contrast to a multiple inheritance. See also [inheritance hierarchy](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninheritance_hierarchy_glosry.htm "Glossary Entry").
