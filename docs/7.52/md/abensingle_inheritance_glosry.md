single inheritance

[Inheritance](javascript:call_link\('abeninheritance_glosry.htm'\) "Glossary Entry") whose class can have several [subclasses](javascript:call_link\('abensubclass_glosry.htm'\) "Glossary Entry") but only one direct [superclass](javascript:call_link\('abensuperclass_glosry.htm'\) "Glossary Entry") in contrast to a multiple inheritance. See also [inheritance hierarchy](javascript:call_link\('abeninheritance_hierarchy_glosry.htm'\) "Glossary Entry").
