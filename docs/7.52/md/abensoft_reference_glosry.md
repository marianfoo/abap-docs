soft reference

Reference to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") where the object is not passed to the reference if the available memory is running low. The class CL\_ABAP\_SOFT\_REFERENCE is planned for soft references but not yet implemented. See also [weak reference](javascript:call_link\('abenweak_reference_glosry.htm'\) "Glossary Entry").
