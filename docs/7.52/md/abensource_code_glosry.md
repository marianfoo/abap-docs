source code

Text representation of programs or other [development objects](javascript:call_link\('abendevelopment_object_glosry.htm'\) "Glossary Entry"). These can include [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") for [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"), [CDS source code](javascript:call_link\('abencds_source_code_glosry.htm'\) "Glossary Entry") for [CDS objects](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry"), or [transformation source code](javascript:call_link\('abentrafo_source_code_glosry.htm'\) "Glossary Entry") for [XSLT programs](javascript:call_link\('abenxsl_transformation_2_glosry.htm'\) "Glossary Entry") and [simple transformations](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry").
