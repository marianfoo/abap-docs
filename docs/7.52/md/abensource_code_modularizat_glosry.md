source code modularization

Splitting of the [source code](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensource_code_glosry.htm "Glossary Entry") of a program into its individual parts. The source code modularization methods applied in [ABAP programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry") are [include programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninclude_program_glosry.htm "Glossary Entry") and [macros](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmacro_glosry.htm "Glossary Entry").
