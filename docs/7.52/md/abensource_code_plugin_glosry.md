source code plug-in

Source code plug-ins are used to make modification-free [enhancements](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") to [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry"). Source code plug-ins are wrapped by the statements [ENHANCEMENT](javascript:call_link\('abapenhancement.htm'\)) and [ENDENHANCEMENT](javascript:call_link\('abapendenhancement.htm'\)) and created using the [Enhancement Builder](javascript:call_link\('abenenhancement_builder_glosry.htm'\) "Glossary Entry") tool.
