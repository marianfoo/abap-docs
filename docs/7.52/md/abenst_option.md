ST - option, Mapping Rules

Syntax

... option="...,..."

Effect

The attribute option can be used to override the associated mapping rule from and to [asXML](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_xslt_asxml.htm) for certain data types (the rule is defined by the data type). The possible options can be specified as the content of the attribute, in a list separated by commas. Any options that are mutually exclusive produced a syntax check warning.

-   [Mapping Rules for XML Fragments](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenst_option_xsdany.htm)
