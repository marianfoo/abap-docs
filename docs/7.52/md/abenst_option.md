ST - option, Mapping Rules

Syntax

... option="...,..."

Effect

The attribute option can be used to override the associated mapping rule from and to [asXML](javascript:call_link\('abenabap_xslt_asxml.htm'\)) for certain data types (the rule is defined by the data type). The possible options can be specified as the content of the attribute, in a list separated by commas. Any options that are mutually exclusive produced a syntax check warning.
