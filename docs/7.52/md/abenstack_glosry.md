stack

Part of the [program memory](javascript:call_link\('abenroll_area_glosry.htm'\) "Glossary Entry") in which [named data objects](javascript:call_link\('abennamed_data_object_glosry.htm'\) "Glossary Entry") of programs and [procedures](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry") are stacked. The memory used by a procedure is released again once the procedure finishes. See also [heap](javascript:call_link\('abenheap_glosry.htm'\) "Glossary Entry").
