standard toolbar

Component of a [GUI window](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abengui_window_glosry.htm "Glossary Entry"). Contains predefined pushbuttons and [icon](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenicon_glosry.htm "Glossary Entry") [push buttons](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenpushbutton_glosry.htm "Glossary Entry") for using the program, as well as a special [command field](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencommand_field_glosry.htm "Glossary Entry") for the direct input of [function codes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfunction_code_glosry.htm "Glossary Entry"). Part of the [GUI status](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abengui_status_glosry.htm "Glossary Entry") set using [SET PF STATUS](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapset_pf-status_dynpro.htm). Created using the [Menu Painter](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmenu_painter_glosry.htm "Glossary Entry") tool. Known as the [symbol toolbar](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensymbol_toolbar_glosry.htm "Glossary Entry") in [ABAP Workbench](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_workbench_glosry.htm "Glossary Entry").
