static constructor

[Constructor](javascript:call_link\('abenconstructor_glosry.htm'\) "Glossary Entry") declared as a [static method](javascript:call_link\('abenstatic_method_glosry.htm'\) "Glossary Entry") under the name [class\_constructor](javascript:call_link\('abenconstructor.htm'\)) in the [public](javascript:call_link\('abenpublic_glosry.htm'\) "Glossary Entry") [visibility section](javascript:call_link\('abenvisibility_section_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry"). It is called automatically in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") before the class is used for the first time.
