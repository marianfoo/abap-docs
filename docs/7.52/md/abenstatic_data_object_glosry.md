Static data object

A [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") for which all attributes, including memory consumption, are specified as static by the [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry"). Apart from [reference variables](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"), all static data objects are [flat](javascript:call_link\('abenflat_glosry.htm'\) "Glossary Entry"). Also see [dynamic data object](javascript:call_link\('abendynamic_data_object_glosry.htm'\) "Glossary Entry").
