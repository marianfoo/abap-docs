static method

[Method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that can be used independently of a class [instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry"). Declared with [CLASS METHODS](javascript:call_link\('abapclass-methods.htm'\)). Static methods can only access [static attributes](javascript:call_link\('abenstatic_attribute_glosry.htm'\) "Glossary Entry") and [static events](javascript:call_link\('abenstatic_event_glosry.htm'\) "Glossary Entry") of their own class.
