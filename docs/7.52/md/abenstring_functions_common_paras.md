Common Parameters of Character String Functions

-   [case - Case sensitive](javascript:call_link\('abenstring_functions_case.htm'\))

-   [occ - Occurrence](javascript:call_link\('abenstring_functions_occ.htm'\))

-   [off, len - Offset and length](javascript:call_link\('abenstring_functions_off_len.htm'\))

-   [regex - Regular expression](javascript:call_link\('abenstring_functions_regex.htm'\))

-   [sub - Substring](javascript:call_link\('abenstring_functions_sub.htm'\))

-   [val - Input value](javascript:call_link\('abenstring_functions_val.htm'\))
