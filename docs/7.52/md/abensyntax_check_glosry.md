syntax check

Static check performed on [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") to check whether the syntax is formally correct. The syntax check is called in [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry") by choosing Program → Check → Syntax or using the statement [SYNTAX-CHECK](javascript:call_link\('abapsyntax-check_for_itab.htm'\)). An automatic syntax check can be configured in the [ABAP Development Tools (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry"), which checks the source code while it is being entered.
