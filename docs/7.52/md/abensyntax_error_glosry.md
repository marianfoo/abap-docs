syntax error

Violation of the ABAP syntax rules. Syntax errors are displayed after the [syntax check](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensyntax_check_glosry.htm "Glossary Entry") by the [ABAP Editor](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_editor_glosry.htm "Glossary Entry"). A program that has syntax errors cannot be executed.
