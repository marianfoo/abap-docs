syntax error

Violation of the ABAP syntax rules. Syntax errors are displayed after the [syntax check](javascript:call_link\('abensyntax_check_glosry.htm'\) "Glossary Entry") by the [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry"). A program that has syntax errors cannot be executed.
