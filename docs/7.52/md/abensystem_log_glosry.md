system log

Central repository for the recording of system errors on an [application server](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry"). The system log is called using the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SM21.
