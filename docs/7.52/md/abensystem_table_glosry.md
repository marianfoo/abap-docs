System Table

[Database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") without a [client column](javascript:call_link\('abenclient_column_glosry.htm'\) "Glossary Entry"). A system table contains cross-client data that is generally only edited by [system programs](javascript:call_link\('abensystem_program_glosry.htm'\) "Glossary Entry").
