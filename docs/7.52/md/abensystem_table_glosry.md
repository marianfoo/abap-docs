System Table

[Database table](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendatabase_table_glosry.htm "Glossary Entry") without a [client column](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclient_column_glosry.htm "Glossary Entry"). A system table contains cross-client data that is generally only edited by [system programs](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensystem_program_glosry.htm "Glossary Entry").
