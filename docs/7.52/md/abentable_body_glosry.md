table body

Name of the actual internal table for [internal tables](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninternal_table_glosry.htm "Glossary Entry") with a [header line](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenheader_line_glosry.htm "Glossary Entry"). The table body can be addressed by appending \[\] to the table name. Internal tables and table bodies are identical for internal tables without a header line.
