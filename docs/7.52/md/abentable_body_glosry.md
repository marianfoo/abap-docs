table body

Name of the actual internal table for [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") with a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry"). The table body can be addressed by appending \[\] to the table name. Internal tables and table bodies are identical for internal tables without a header line.
