table type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry"). Defines the [row type](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry"), [table category](javascript:call_link\('abentable_category_glosry.htm'\) "Glossary Entry"), and [table key](javascript:call_link\('abentable_key_glosry.htm'\) "Glossary Entry") in an internal table. The table type can be [generic](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") with respect to the key.
