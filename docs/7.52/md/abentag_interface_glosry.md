tag interface

Special predefined global interface. The integration of a tag interface gives [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") or other [interfaces](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry") a special status within the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry"). A tag interface generally does not contain its own [interface components](javascript:call_link\('abeninterface_component_glosry.htm'\) "Glossary Entry") and instead assigns a particular task to the integrating classes or interfaces and changes the way they are handled by [ABAP Compiler](javascript:call_link\('abenabap_compiler_glosry.htm'\) "Glossary Entry").
