tag interface

Special predefined global interface. The integration of a tag interface gives [classes](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenclass_glosry.htm "Glossary Entry") or other [interfaces](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninterface_oo_glosry.htm "Glossary Entry") a special status within the [ABAP runtime environment](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry"). A tag interface generally does not contain its own [interface components](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abeninterface_component_glosry.htm "Glossary Entry") and instead assigns a particular task to the integrating classes or interfaces and changes the way they are handled by [ABAP Compiler](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_compiler_glosry.htm "Glossary Entry").
