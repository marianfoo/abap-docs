termination message

[Message](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmessage_glosry.htm "Glossary Entry") with [message type](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenmessage_type_glosry.htm "Glossary Entry") A. Termination messages are displayed in a [dialog box](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendialog_box_glosry.htm "Glossary Entry") and terminate the program that is running.
