text file

[File](javascript:call_link\('abenfile_glosry.htm'\) "Glossary Entry") on an [application server](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") that is opened using the addition [TEXT](javascript:call_link\('abapopen_dataset_mode.htm'\)) of the statement [OPEN DATASET](javascript:call_link\('abapopen_dataset_mode.htm'\)). In a text file, the contents are handled as text in the [character representation](javascript:call_link\('abenchar_representation_glosry.htm'\) "Glossary Entry") chosen when the file was opened.
