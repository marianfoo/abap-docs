text file

[File](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfile_glosry.htm "Glossary Entry") on an [application server](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenapplication_server_glosry.htm "Glossary Entry") that is opened using the addition [TEXT](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapopen_dataset_mode.htm) of the statement [OPEN DATASET](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abapopen_dataset_mode.htm). In a text file, the contents are handled as text in the [character representation](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenchar_representation_glosry.htm "Glossary Entry") chosen when the file was opened.
