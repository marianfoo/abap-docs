text language

Identification of a component of type LANG of a [structure](javascript:call_link\('abenstructure_glosry.htm'\) "Glossary Entry") defined in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") or a [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry"). Only one component of this type can be identified as the text language. The text language is used when the structure is transferred between [MDMP systems](javascript:call_link\('abenmdmp-system_glosry.htm'\) "Glossary Entry") and [Unicode systems](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry") for the conversion of the character-like components by using the [IMPORT](javascript:call_link\('abenimport_text_language.htm'\)) statement or corespondingly structured TABLES parameter using the binary [RFC protocol](javascript:call_link\('abenrfc_protocol.htm'\)).
