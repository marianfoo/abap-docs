title bar

Component of the [GUI](javascript:call_link\('abengui_glosry.htm'\) "Glossary Entry") that contains the [GUI title](javascript:call_link\('abengui_title_glosry.htm'\) "Glossary Entry") of a [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry"). The title bar is set using the [SET TITLEBAR](javascript:call_link\('abapset_titlebar_dynpro.htm'\)) statement.
