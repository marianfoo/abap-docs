transaction code

Twenty-character name associated with a [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry"), with another transaction code, or with a [method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and which is used to execute programs. Transaction codes associated with dynpros are possible for [executable programs](javascript:call_link\('abenexecutable_program_glosry.htm'\) "Glossary Entry"), [module pools](javascript:call_link\('abenmodul_pool_glosry.htm'\) "Glossary Entry"), and [function groups](javascript:call_link\('abenfunction_group_glosry.htm'\) "Glossary Entry"). [Parameter transactions](javascript:call_link\('abenparameter_transaction_glosry.htm'\) "Glossary Entry") and [variant transactions](javascript:call_link\('abenvariant_transaction_glosry.htm'\) "Glossary Entry") are associated with other transaction codes. Transaction codes associated with methods are possible for all [program types](javascript:call_link\('abenprogram_type_glosry.htm'\) "Glossary Entry") that contain methods. Transaction codes are edited and managed using transaction code SE93.
