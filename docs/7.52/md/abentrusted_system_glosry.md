trusted system

[AS ABAP](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") classified as a trusted caller of [remote function calls](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenremote_function_call_glosry.htm "Glossary Entry") in a [trusting system](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentrusting_system_glosry.htm "Glossary Entry"). The trusted system provides the [RFC client](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenrfc_client_glosry.htm "Glossary Entry"). The system is classified in [transaction](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abentransaction_glosry.htm "Glossary Entry") SMT1. Trusted systems log on to the trusting system without sending a password. User-specific logon data is checked in the trusting system.
