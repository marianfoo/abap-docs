type

Abstract description of an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry"). Either a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") for a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") or [object type](javascript:call_link\('abenobject_type_glosry.htm'\) "Glossary Entry") for an object in [ABAP objects](javascript:call_link\('abenabap_objects_glosry.htm'\) "Glossary Entry"). The corresponding [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [any](javascript:call_link\('abenbuilt_in_types_generic.htm'\)). All types are in a collective [name space](javascript:call_link\('abennaming_conventions.htm'\)).
