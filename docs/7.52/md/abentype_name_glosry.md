type name

Name of a [type](javascript:call_link\('abentype_glosry.htm'\) "Glossary Entry"). In a program, only the [relative type name](javascript:call_link\('abenrelative_type_name_glosry.htm'\) "Glossary Entry") is known and statically usable. For dynamic type specifications, you can also use the [absolute type name](javascript:call_link\('abenabsolute_typename_glosry.htm'\) "Glossary Entry").
