user interface

Full set of [control elements](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abencontrol_element_glosry.htm "Glossary Entry") in an [ABAP program](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_program_glosry.htm "Glossary Entry") in [SAP GUI](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abensap_gui_glosry.htm "Glossary Entry"). Comprises the graphical interface (GUI) in a [GUI window](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abengui_window_glosry.htm "Glossary Entry") and the [function keys](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfunction_key_glosry.htm "Glossary Entry") and other keys of an input device (keyboard or hardware components that simulate keyboards, such as bar code readers).
