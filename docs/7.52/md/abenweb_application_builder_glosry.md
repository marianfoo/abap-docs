Web Application Builder

Tool in [ABAP Workbench](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_workbench_glosry.htm "Glossary Entry") used to create [Business Server Pages](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenbusiness_server_pages_glosry.htm "Glossary Entry"). Called by choosing Create → BSP Library → BSP Application in [Object Navigator](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenobject_navigator_glosry.htm "Glossary Entry").
