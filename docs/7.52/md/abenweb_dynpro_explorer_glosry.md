Web Dynpro Explorer

[ABAP Workbench](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenabap_workbench_glosry.htm "Glossary Entry") tool used to create [Web Dynpros](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenweb_dynpro_glosry.htm "Glossary Entry"). Called by choosing Create → Web Dynpro → ... in [Object Navigator](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenobject_navigator_glosry.htm "Glossary Entry").
