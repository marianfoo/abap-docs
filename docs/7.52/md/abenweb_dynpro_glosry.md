Web Dynpro

Technology used to create platform-independent Web-based interfaces, based on the Model View Controller (MVC) approach. The technology is available on both [Application Server Java](javascript:call_link\('abenjava_personality_glosry.htm'\) "Glossary Entry") and [Application Server ABAP](javascript:call_link\('abenabap_personality_glosry.htm'\) "Glossary Entry"). Before [SAPUI5](javascript:call_link\('abensapui5_glosry.htm'\) "Glossary Entry") was introduced, Web Dynpro for ABAP was SAP's standard UI technology for developing Web applications in the ABAP environment. Web Dynpros for ABAP are created in [Web Dynpro Explorer](javascript:call_link\('abenweb_dynpro_explorer_glosry.htm'\) "Glossary Entry").
