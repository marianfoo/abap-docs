writer stream

Also called [output stream](javascript:call_link\('abenoutput_stream_glosry.htm'\) "Glossary Entry"). Either a [data stream](javascript:call_link\('abendata_stream_glosry.htm'\) "Glossary Entry") connected to a [data sink](javascript:call_link\('abendata_sink_glosry.htm'\) "Glossary Entry") or a [filter stream](javascript:call_link\('abenfilter_stream_glosry.htm'\) "Glossary Entry") which writes data to another stream. The direction of a writer stream is always towards the data sink. Only data can be passed in a writer stream.
