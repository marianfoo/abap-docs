writer stream

Also called [output stream](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoutput_stream_glosry.htm "Glossary Entry"). Either a [data stream](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_stream_glosry.htm "Glossary Entry") connected to a [data sink](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abendata_sink_glosry.htm "Glossary Entry") or a [filter stream](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenfilter_stream_glosry.htm "Glossary Entry") which writes data to another stream. The direction of a writer stream is always towards the data sink. Only data can be passed in a writer stream.
