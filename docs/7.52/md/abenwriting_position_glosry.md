writer position

[Operand position](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoperand_position_glosry.htm "Glossary Entry") in which the [operand](https://help.sap.com/doc/abapdocu_752_index_htm/7.52/en-US/abenoperand_glosry.htm "Glossary Entry") is written to. As well as writer-only positions, positions that read an operand are also available.
