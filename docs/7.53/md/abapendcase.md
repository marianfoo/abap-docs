ENDCASE

[Quick Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapcase_shortref.htm)

Syntax

ENDCASE.

Effect

The statement ENDCASE closes a [case distinction](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencase_distinction_glosry.htm "Glossary Entry") introduced using [CASE](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapcase.htm) or [CASE TYPE OF](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapcase_type.htm).
