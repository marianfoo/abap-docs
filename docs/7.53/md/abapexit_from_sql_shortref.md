EXIT FROM SQL - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapexit_sql.htm)

Syntax

EXIT FROM SQL.

Effect

Obsolete: Cancels implicit cursor processing in EXEC SQL. Processing is resumed after ENDEXEC.
