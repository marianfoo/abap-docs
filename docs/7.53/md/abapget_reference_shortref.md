GET REFERENCE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapget_reference.htm)

Syntax

GET REFERENCE OF dobj INTO dref.

Effect

Gets a reference to the data object dobj in the reference variable dref.
