GET TIME STAMP - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapget_time-stamp.htm)

Syntax

GET TIME STAMP FIELD time\_stamp.

Effect

Returns the current [time stamp](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentime_stamp_glosry.htm "Glossary Entry") in time\_stamp of the type TIMESTAMP or TIMESTAMPL.
