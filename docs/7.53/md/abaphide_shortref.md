HIDE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abaphide.htm)

Syntax

HIDE dobj.

Effect

Saves the content of the variable dobj with the current list line in the current list level.
