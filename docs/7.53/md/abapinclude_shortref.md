INCLUDE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapinclude_prog.htm)

Syntax

INCLUDE incl *\[*IF FOUND*\]*.

Effect

Includes the [include program](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninclude_program_glosry.htm "Glossary Entry") incl in the current position in the source code.

Addition

-   IF FOUND
    Suppresses the syntax check error message if the include program is not found.
