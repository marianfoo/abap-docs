INSERT TEXTPOOL - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapinsert_textpool.htm)

Syntax

INSERT TEXTPOOL prog FROM itab LANGUAGE lang.

Effect

Inserts the text elements in the internal table itab of the type TEXTPOOL into the [repository](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_glosry.htm "Glossary Entry"). They are inserted as a text pool of the language specified in lang for the program specified in prog.
