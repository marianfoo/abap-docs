METHOD - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapmethod.htm)

Syntax

METHOD meth.
  ...
ENDMETHOD.

Effect

Implements a method meth in the implementation part of a class. A method is declared in the declaration part of the class and is implemented between METHOD and ENDMETHOD.
