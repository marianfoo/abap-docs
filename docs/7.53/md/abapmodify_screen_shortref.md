MODIFY SCREEN - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapmodify_screen.htm)

Syntax

MODIFY SCREEN FROM wa.

Effect

In LOOP AT SCREEN, overwrites the properties of the current screen element with the content of the predefined work area wa of the type [SCREEN](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenscreen.htm).
