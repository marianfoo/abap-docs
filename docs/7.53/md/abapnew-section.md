NEW-SECTION

[Quick Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapnew-section_shortref.htm)

Obsolete Syntax

NEW-SECTION.

Effect

This statement (which cannot be used in classes) is used to create a new spool request, has the same function as the following statement, and is replaced by this statement:

[NEW-PAGE](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapnew-page.htm) [PRINT ON NEW-SECTION](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapnew-page_print.htm).

Note

The obsolete statement does not allow the mandatory [spool parameters](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprint_parameters.htm) to be specified.
