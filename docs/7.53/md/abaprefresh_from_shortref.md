REFRESH - FROM - Quick reference

[Reference](javascript:call_link\('abaprefresh_obsolete.htm'\))

Syntax

REFRESH itab FROM TABLE dbtab.

Effect

Obsolete: Deletes all rows of an internal table itab and fills the table with rows from the database table or view dbtab. The content of the [table work area](javascript:call_link\('abentable_work_area_glosry.htm'\) "Glossary Entry") dbtab is used as the search key.
