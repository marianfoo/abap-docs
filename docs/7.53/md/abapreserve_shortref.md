RESERVE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapreserve.htm)

Syntax

RESERVE n LINES.

Effect

Creates a page break on a list if there are no more n lines left on the current page of the list.
