SELECTION-SCREEN DYNAMIC SELECTIONS - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapselection-screen_ldb_dynamic.htm)

Syntax

SELECTION-SCREEN DYNAMIC SELECTIONS
                 FOR *{*NODE*|*TABLE*}* node *\[*ID id*\]*.

Effect

Obsolete: Flags a node node of the structure of a [logical database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlogical_data_base_glosry.htm "Glossary Entry") for dynamic selections.

Addition

-   ID id
    Defines an ID id for the node, with a maximum of three characters.
