SET RUN TIME CLOCK - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapset_run_time_clock_resolution.htm)

Syntax

SET RUN TIME CLOCK RESOLUTION *{*HIGH*|*LOW*}*.

Effect

Sets the measurement accuracy for the GET RUN TIME statement.
