SUBTRACT - Quick reference

[Reference](javascript:call_link\('abapsubtract.htm'\))

Syntax

SUBTRACT dobj1 FROM dobj2.

Effect

Subtracts the content of a numeric data object dobj1 from the content of a numeric data object dobj2 and assigns the event dobj2.
