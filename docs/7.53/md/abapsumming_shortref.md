SUMMING - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapsumming.htm)

Syntax

SUMMING dobj.

Effect

Obsolete: After the statement SUMMING, adds the value of dobj to the content of sum\_dobj after each WRITE statement.
