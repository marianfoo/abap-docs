SUPPLY - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapsupply.htm)

Syntax

SUPPLY key1 = f1 key2 = f2 ... TO CONTEXT context\_ref.

Effect

Obsolete: Fills the key fields key1, key2, ... of a [context](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencontext_glosry.htm "Glossary Entry") instance referenced by context\_ref with the values of data objects f1, f2, ...
