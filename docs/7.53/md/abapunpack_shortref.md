UNPACK - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapunpack.htm)

Syntax

UNPACK source TO destination.

Effect

Assigns the digits of a packed number source to the character-like variable destination.
