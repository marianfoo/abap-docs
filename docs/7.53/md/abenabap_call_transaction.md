Calling Transactions

A transaction is the execution of a program using a [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry"). The process flow of the program depends on which [initial dynpro](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninitial_dynpro_glosry.htm "Glossary Entry") or [selection screen](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenselection_screen_glosry.htm "Glossary Entry"), or which method, is associated with the transaction code.

There are two statements for calling transactions:

-   [CALL TRANSACTION](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapcall_transaction.htm)

-   [LEAVE TO TRANSACTION](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapleave_to_transaction.htm)
