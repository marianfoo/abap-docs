ABAP Compiler

ABAP Compiler creates a [byte code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbytecode_glosry.htm "Glossary Entry") as interim code when generating a program from the [ABAP source code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_source_code_glosry.htm "Glossary Entry"). This interim code is stored in the database as a [load](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenload_glosry.htm "Glossary Entry") program and is loaded to [PXA](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpxa_glosry.htm "Glossary Entry") when required.
