ABAP Daemon class

Global subclass of CL\_ABAP\_DAEMON\_EXT\_BASE. The instances of an ABAP Daemon class are [ABAP Daemons](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") created by the [ABAP Daemon framework](javascript:call_link\('abenabap_daemon_framework_glosry.htm'\) "Glossary Entry"), each of which are executed in a separate [ABAP Daemon session](javascript:call_link\('abenabap_daemon_session_glosry.htm'\) "Glossary Entry"). An ABAP Daemon class implements interface methods inherited from the superclass to respond to [ABAP Daemon events](javascript:call_link\('abenabap_daemon_event_glosry.htm'\) "Glossary Entry").
