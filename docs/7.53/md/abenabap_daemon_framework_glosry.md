ABAP Daemon Framework

ADF for short. A framework for using [ABAP Daemons](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_daemon_glosry.htm "Glossary Entry"), based on [classes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") and [interfaces](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninterface_oo_glosry.htm "Glossary Entry"). ADF includes APIs both for communication with ABAP Daemons in [ABAP programs](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_program_glosry.htm "Glossary Entry") and for the handling of events by ABAP Daemons in [ABAP Daemon sessions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_daemon_session_glosry.htm "Glossary Entry").
