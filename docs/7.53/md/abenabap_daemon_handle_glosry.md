ABAP Daemon handle

[ABAP Daemon Framework](javascript:call_link\('abenabap_daemon_framework_glosry.htm'\) "Glossary Entry") object that implements the [interface](javascript:call_link\('abeninterface_glosry.htm'\) "Glossary Entry") IF\_ABAP\_DAEMON\_HANDLE. An ABAP Daemon handle enables consumers to use the method SEND to send [PCP](javascript:call_link\('abenpcp_glosry.htm'\) "Glossary Entry") messages to the daemon.
