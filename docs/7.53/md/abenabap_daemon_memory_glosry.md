ABAP Daemon memory

A part of the [user memory](javascript:call_link\('abenuser_memory_glosry.htm'\) "Glossary Entry") in which an [ABAP Daemon](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") can save data in [PCP](javascript:call_link\('abenpcp_glosry.htm'\) "Glossary Entry") format. An ABAP daemon uses approximately 1 megabyte of ABAP Daemon memory.
