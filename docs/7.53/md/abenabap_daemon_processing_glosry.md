ABAP Daemon processing

Processing of an [ABAP Daemon](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_daemon_glosry.htm "Glossary Entry"). An ABAP Daemon is processed in a dedicated [ABAP Daemon session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_daemon_session_glosry.htm "Glossary Entry") and in [non-blocking mode](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennon_blocking_model_glosry.htm "Glossary Entry") (in which no blocking statements are allowed).
