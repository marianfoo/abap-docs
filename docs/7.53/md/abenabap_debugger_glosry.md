ABAP Debugger

Tool used to execute [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") line by line or section by section. In this way it is possible to edit the contents of [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") and to check the program logic. See also [breakpoint](javascript:call_link\('abenbreakpoint_glosry.htm'\) "Glossary Entry").
