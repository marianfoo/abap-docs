ABAP Doc Comment

Special [comment](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencomment_glosry.htm "Glossary Entry") consisting of one or more [comment lines](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencomment_line_glosry.htm "Glossary Entry"), each of which is prefixed by "!. ABAP Doc comments can be placed in front of [declarations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendeclaration_glosry.htm "Glossary Entry") and are used by [ABAP Doc](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_doc_glosry.htm "Glossary Entry") for documentation purposes.
