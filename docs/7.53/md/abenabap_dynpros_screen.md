Screen and Screen Elements

The screen of a [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") is created in [Layout Editor](javascript:call_link\('abenlayout_editor_glosry.htm'\) "Glossary Entry") in [Screen Painter](javascript:call_link\('abenscreen_painter_glosry.htm'\) "Glossary Entry"). The screen of the program DEMO\_DYNPRO contains all possible screen elements. Above the screen, the menu bar, standard toolbar, title bar, and the application toolbar can be seen, with the status bar at the bottom.
