Processing External Data

External data generally covers all data saved outside of the [internal session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_session_glosry.htm "Glossary Entry"). The following sections describe the processing of external data.

-   [ABAP Database Access](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_sql.htm)

-   [Data Clusters](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_cluster.htm)

-   [ABAP File Interface](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_language_files.htm)

Note

As well as the statements described here, there is also a description of the [obsolete processing of external data](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_storage_obsolete.htm).
