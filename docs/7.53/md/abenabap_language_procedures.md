Procedures

Procedures are processing blocks with [parameter interface](javascript:call_link\('abenformal_parameters_oview.htm'\)) and a local [data area](javascript:call_link\('abendata_area_glosry.htm'\) "Glossary Entry"). They can be called from within ABAP programs. The possible types of procedure are:

-   [Methods](javascript:call_link\('abenabap_methods.htm'\))

-   [Function modules](javascript:call_link\('abenabap_functions.htm'\))

The following is an obsolete type of procedure:

-   [Subroutines](javascript:call_link\('abenabap_subroutines.htm'\))
