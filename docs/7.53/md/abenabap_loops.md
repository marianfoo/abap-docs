Loops

This section describes the loops defined using the following:

-   [DO - ENDDO](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapdo.htm)

-   [WHILE - ENDWHILE](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapwhile.htm)

Further keywords for defining loops are:

-   [LOOP - ENDLOOP](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abaploop_at_itab.htm)

-   [PROVIDE - ENDPROVIDE](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapprovide.htm) for loops across internal tables

-   [SELECT - ENDSELECT](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapselect.htm) for loops across the results set of a database access.
