ABAP program

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that contains [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") and has a [program type](javascript:call_link\('abenprogram_type_glosry.htm'\) "Glossary Entry") . An ABAP program is associated with additional [components](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry") such as [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") or the [GUI status](javascript:call_link\('abengui_status_glosry.htm'\) "Glossary Entry").
