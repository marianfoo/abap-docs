ABAP Timer handler

[Class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") that implements the [interface](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninterface_glosry.htm "Glossary Entry") IF\_ABAP\_TIMER\_HANDLER. The interface method ON\_TIMEOUT of the timer handler specified when an [ABAP Timer](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_timer_glosry.htm "Glossary Entry") is started in [ABAP Timer Manager](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_timer_manager_glosry.htm "Glossary Entry") handles the timeout event of the timer.
