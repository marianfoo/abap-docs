ABAP Development Tools

Abbreviated as ADT. Eclipse-based tools used to edit [repository objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_object_glosry.htm "Glossary Entry") in [ABAP Repository](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_repository_glosry.htm "Glossary Entry") and their components. The ADT currently cover a subset of the [ABAP Workbench](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_workbench_glosry.htm "Glossary Entry") functions. In the case of some newer repository objects, only the ADT can be used.
