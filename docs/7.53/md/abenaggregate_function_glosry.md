aggregate function

Function that determines a value from the values of multiple rows of a column in the results set of an [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry"). The value is calculated in the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry"). When specified as the argument of an aggregate function, a column of a database table forms an [aggregate expression](javascript:call_link\('abenaggregate_expression_glosry.htm'\) "Glossary Entry").
