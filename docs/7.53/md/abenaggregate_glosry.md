aggregate

Grouping of data from multiple rows of a [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry"). Can be determined using [aggregate functions](javascript:call_link\('abenaggregate_function_glosry.htm'\) "Glossary Entry") in the [SELECT](javascript:call_link\('abapselect.htm'\)) statement.
