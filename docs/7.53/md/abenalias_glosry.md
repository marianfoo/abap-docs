alias name

Either a name for an [interface component](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninterface_component_glosry.htm "Glossary Entry") in a [class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") or [composite interface](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencompound_interface_glosry.htm "Glossary Entry") declared using the statement [ALIASES](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapaliases.htm), or a name for the [primary table key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprimary_key_glosry.htm "Glossary Entry") of an [internal table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry") defined using the addition [ALIAS](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abaptypes_primary_key.htm).
