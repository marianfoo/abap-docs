AMDP class

[Global class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenglobal_class_glosry.htm "Glossary Entry") with one or more [tag interfaces](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentag_interface_glosry.htm "Glossary Entry") IF\_AMDP\_MARKER\_ ... for [ABAP Managed Database Procedures](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_managed_db_proc_glosry.htm "Glossary Entry"). An AMDP class can contain [AMDP methods](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenamdp_method_glosry.htm "Glossary Entry") for various [database systems](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_system_glosry.htm "Glossary Entry").
