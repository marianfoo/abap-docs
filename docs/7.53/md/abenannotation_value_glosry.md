annotation value

A value specified for a [](javascript:call_link\('abencds_annotation_syntax_glosry.htm'\) "Glossary Entry")[CDS](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") annotation in the [CDS annotation syntax](javascript:call_link\('abencds_annotation_syntax_glosry.htm'\) "Glossary Entry"). The possible values can be specified in the [CDS annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry") using a type specification, enumeration symbols, and a default value.
