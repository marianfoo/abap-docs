anonymous data object

[Unnamed data object](javascript:call_link\('abenunnamed_data_object_glosry.htm'\) "Glossary Entry") created by the statement [CREATE DATA](javascript:call_link\('abapcreate_data.htm'\)) or the [instance operator](javascript:call_link\('abeninstance_operator_glosry.htm'\) "Glossary Entry") [NEW](javascript:call_link\('abenconstructor_expression_new.htm'\)) and addressed using [reference variables](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry") or [field symbols](javascript:call_link\('abenfield_symbol_glosry.htm'\) "Glossary Entry"). Anonymous data objects are stored in the [heap](javascript:call_link\('abenheap_glosry.htm'\) "Glossary Entry"). An anonymous data object is kept alive by [heap references](javascript:call_link\('abenheap_reference_glosry.htm'\) "Glossary Entry") or by field symbols pointing to it. See also [named data object](javascript:call_link\('abennamed_data_object_glosry.htm'\) "Glossary Entry") and [literal](javascript:call_link\('abenliteral_glosry.htm'\) "Glossary Entry").
