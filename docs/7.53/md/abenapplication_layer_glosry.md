application layer

Software layer of a [client-server system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclient_server_system_glosry.htm "Glossary Entry") that contains the application logic. [AS ABAP](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") represents the application layer of an [ABAP system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_system_glosry.htm "Glossary Entry") in which [ABAP programs](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_program_glosry.htm "Glossary Entry") are executed.
