application table

[Database table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry") with [client column](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclient_column_glosry.htm "Glossary Entry"). An application table contains client-specific data that is edited by [application programs](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenapplication_program_glosry.htm "Glossary Entry").
