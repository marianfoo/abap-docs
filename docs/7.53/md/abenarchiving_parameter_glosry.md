archiving parameter

Parameter that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenspool_request_glosry.htm "Glossary Entry") if its data is to be archived using [ArchiveLink](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_archivelink_glosry.htm "Glossary Entry").
