area menu

Tree structure whose nodes can be associated with [transaction codes](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry"), [executable programs](javascript:call_link\('abenexecutable_program_glosry.htm'\) "Glossary Entry"), and other area menus. Area menus are edited using transaction code SE43 and displayed from [SAP Easy Access](javascript:call_link\('abensap_easy_access_glosry.htm'\) "Glossary Entry").
