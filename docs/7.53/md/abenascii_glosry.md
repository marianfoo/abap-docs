ASCII

The abbreviation for American Standard Code for Information Interchange. 7-bit [character set](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencharacter_set_glosry.htm "Glossary Entry") in accordance with the ISO-646 standard. Expanded to 8-bit character sets in [ISO-8859](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeniso-8859_glosry.htm "Glossary Entry").
