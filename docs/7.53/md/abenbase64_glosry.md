Base64

Method of encoding binary data into printable characters. Every byte is converted into a short character string whose characters are obtained from a set of 64 [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry")\-independent [ASCII](javascript:call_link\('abenascii_glosry.htm'\) "Glossary Entry") characters.
