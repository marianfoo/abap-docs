batch input session

Groups a series of transaction calls together with input data and [user actions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenuser_action_glosry.htm "Glossary Entry"). A batch input session can be used to execute [dialog transactions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendialog_transaction_glosry.htm "Glossary Entry") in [batch input](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbatch_input_glosry.htm "Glossary Entry"), where some or all of the [screens](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenscreen_glosry.htm "Glossary Entry") are processed by the session. Batch input sessions are saved persistently and are managed by the batch input manager in transaction SM35. See also [batch input tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbatch_input_table_glosry.htm "Glossary Entry").
