business function set

Abbreviated as BFS. [Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that groups multiple [business functions](javascript:call_link\('abenbusiness_function_glosry.htm'\) "Glossary Entry") to form a unit that covers the demands of an industry solution within the scope of [Switch Framework](javascript:call_link\('abenswitch_framework_glosry.htm'\) "Glossary Entry"). For every [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"), one business function set can be switched on. Within a business function set, it is guaranteed that all business functions are compatible with each other. Business function sets are created in [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SFW3 and switched in transaction SFW5.
