byte-like expression

Represents a [byte string](javascript:call_link\('abenbyte_string_glosry.htm'\) "Glossary Entry"). A byte-like expression is a [bit expression](javascript:call_link\('abenbit_expression_glosry.htm'\) "Glossary Entry") or a [function](javascript:call_link\('abenfunction_glosry.htm'\) "Glossary Entry"), whose return value has a [numeric data type](javascript:call_link\('abenbyte_like_data_typ_glosry.htm'\) "Glossary Entry").
