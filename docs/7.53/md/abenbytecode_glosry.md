byte code

Result of generating an ABAP program using [ABAP Compiler](javascript:call_link\('abenabap_compiler_glosry.htm'\) "Glossary Entry"). The statements in the byte code are associated with C functions. When the [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") is executed, the byte code is loaded into the [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry") as the program [Load](javascript:call_link\('abenload_glosry.htm'\) "Glossary Entry") and interpreted by the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") (virtual machine), which accesses the associated C functions.
