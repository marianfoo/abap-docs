character set

Defined set of characters. Examples are [ASCII](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenascii_glosry.htm "Glossary Entry"), [EBCDIC](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenebcdic_glosry.htm "Glossary Entry") or [Unicode](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenunicode_glosry.htm "Glossary Entry"). Character sets are mapped to [character representations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenchar_representation_glosry.htm "Glossary Entry") using [code pages](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencodepage_glosry.htm "Glossary Entry").
