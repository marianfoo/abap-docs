ABAP SQL - System Classes

The following system classes are available for extending the scope of the ABAP SQL language:

-   [CL\_OSQL\_EXTENDED\_RESULT](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencl_osql_extended_result.htm) handles extended results

-   s[CL\_OSQL\_REPLACE](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencl_osql_replace.htm) provides a replacement service for executing [module tests](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenunit_test_glosry.htm "Glossary Entry")
