class component selector

The \=> characters. Each visible [static component](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_component_glosry.htm "Glossary Entry") comp of a [class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") class can be addressed using class=>comp.
