client ID

Three-digit ID of a [client](javascript:call_link\('abenclient_glosry.htm'\) "Glossary Entry"), specified when [logging on](javascript:call_link\('abenlogon_glosry.htm'\) "Glossary Entry") to [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"). In [application tables](javascript:call_link\('abenapplication_table_glosry.htm'\) "Glossary Entry"), client-specific data is indicated by the client ID entered in the [client column](javascript:call_link\('abenclient_column_glosry.htm'\) "Glossary Entry").
