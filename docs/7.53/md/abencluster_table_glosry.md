cluster table

Before Release 7.53, [table category](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenddic_database_tables_tab_type.htm) of a [database table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry") in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A cluster table did not exist as a table instance on the [database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_glosry.htm "Glossary Entry"). Instead, multiple cluster tables were saved in a [table cluster](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_cluster_glosry.htm "Glossary Entry") in the database. Only [ABAP SQL](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenopen_sql_glosry.htm "Glossary Entry") could be used to access cluster tables. Certain other restrictions also applied. Cluster tables are no longer supported from Release 7.53. Any existing cluster tables were transformed to [transparent tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransparent_table_glosry.htm "Glossary Entry").
