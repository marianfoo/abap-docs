Code Inspector

Tool for checking [repository objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_object_glosry.htm "Glossary Entry") in terms of performance, security, syntax, and adherence to naming conventions. Called using [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SCI.
