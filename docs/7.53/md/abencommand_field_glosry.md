command field

Input field in the [standard toolbar](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstandard_toolbar_glosry.htm "Glossary Entry"). This input field accepts [function codes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfunction_code_glosry.htm "Glossary Entry"), [transaction codes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry"), and other shortcuts such as "/h" for starting [ABAP Debugger](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_debugger_glosry.htm "Glossary Entry").
