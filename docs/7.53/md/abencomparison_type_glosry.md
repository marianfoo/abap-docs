comparison type

Property of [comparison expressions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencomparison_expression_glosry.htm "Glossary Entry") whose [operands](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperand_glosry.htm "Glossary Entry") have [elementary data types](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenelementary_data_type_glosry.htm "Glossary Entry"), including [calculation expressions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencalculation_expression_glosry.htm "Glossary Entry"). The comparison type is determined from the operands and defines how the comparison is performed. The comparison type can have any [built-in ABAP type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpredefined_abap_type_glosry.htm "Glossary Entry"). Operands with other types are converted to the comparison type before the comparison.
