component interface

[Interface](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninterface_oo_glosry.htm "Glossary Entry") included in a [compound interface](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencompound_interface_glosry.htm "Glossary Entry") using the statement [INTERFACES](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapinterfaces.htm). A component interface can itself be a compound interface.
