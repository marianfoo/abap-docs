Core Data Services

Infrastructure for defining and consuming semantically rich data models. The current variants are the database-independent [ABAP CDS](javascript:call_link\('abenabap_core_data_services_glosry.htm'\) "Glossary Entry") and [HANA CDS](https://help.sap.com/viewer/09b6623836854766b682356393c6c416/2.0.00/en-us).
