currency ID

Content of a [currency key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencurrency_key_glosry.htm "Glossary Entry"). Valid currency IDs are defined in the database table TCURC. The database table TCURX contains the currency code for currencies whose number of decimal places is not two.
