Data Browser

Tool in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") used to access the content of [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry").
