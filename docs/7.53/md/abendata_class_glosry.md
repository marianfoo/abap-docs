shared memory-enabled class

[Class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") whose [instances](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") can be stored as [shared objects](javascript:call_link\('abenshared_objects_glosry.htm'\) "Glossary Entry") in [area instance versions](javascript:call_link\('abenarea_instance_version_glosry.htm'\) "Glossary Entry") of the [shared memory](javascript:call_link\('abenshared_memory_glosry.htm'\) "Glossary Entry"). Shared memory-enabled classes are defined by the addition [SHARED MEMORY ENABLED](javascript:call_link\('abapclass_options.htm'\)) of the statement CLASS. In particular, all [area root classes](javascript:call_link\('abenroot_data_class_glosry.htm'\) "Glossary Entry") must be shared memory-enabled.
