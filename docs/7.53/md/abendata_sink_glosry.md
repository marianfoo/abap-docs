Data Sink

Repository in which a write [data stream](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_stream_glosry.htm "Glossary Entry") stores its data. A data sink can be an external resource such as an [LOB](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlob_glosry.htm "Glossary Entry") column of a [database table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry") or a [file](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfile_glosry.htm "Glossary Entry"), or an internal resource such as a [string](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_glosry.htm "Glossary Entry") or an [internal table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry"). [Streaming](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstreaming_glosry.htm "Glossary Entry") is currently implemented for [database tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry"), [strings](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_glosry.htm "Glossary Entry"), and [internal tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry").
