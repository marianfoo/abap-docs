database commit

End of a [database LUW](javascript:call_link\('abendatabase_luw_glosry.htm'\) "Glossary Entry"), in which changed data records are committed to the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry"). All changes are temporary until they have been written using a database commit. Changes are always directly visible to the executing program. The [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") that is used determines whether other users can see the database changes immediately or only after the database commit. In [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"), database commits are triggered implicitly, as well as by being requested explicitly.
