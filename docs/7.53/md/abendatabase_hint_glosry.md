database note

Extension of the SQL standard, which allows it to inform the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") how to execute access. Database notes are usually specific to the database system and release. In [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry"), database hints can be specified using the addition [%\_HINTS](javascript:call_link\('abenosql_db_hints.htm'\)).
