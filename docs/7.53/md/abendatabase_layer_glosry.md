database layer

Software layer of a [client-server system](javascript:call_link\('abenclient_server_system_glosry.htm'\) "Glossary Entry") in which persistent data is saved and managed. The [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") of an [ABAP system](javascript:call_link\('abenabap_system_glosry.htm'\) "Glossary Entry") is always connected with a database layer in the form of a [database server](javascript:call_link\('abendatabase_server_glosry.htm'\) "Glossary Entry"), which itself contains a [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") with the [standard AS ABAP database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry").
