database layer

Software layer of a [client-server system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclient_server_system_glosry.htm "Glossary Entry") in which persistent data is saved and managed. The [AS ABAP](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") of an [ABAP system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_system_glosry.htm "Glossary Entry") is always connected with a database layer in the form of a [database server](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_server_glosry.htm "Glossary Entry"), which itself contains a [database system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_system_glosry.htm "Glossary Entry") with the [standard AS ABAP database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstandard_db_glosry.htm "Glossary Entry").
