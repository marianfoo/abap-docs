database lock

Physical [locks](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlock_glosry.htm "Glossary Entry") on rows in [database tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry"), set by the [database system](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_system_glosry.htm "Glossary Entry"). A database lock persists until the end of the current [database LUW](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_luw_glosry.htm "Glossary Entry").
