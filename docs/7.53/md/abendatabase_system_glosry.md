database system

System for storing large amounts of data in a [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry"). Besides the database, a database system consists of a database management system that administers the database and offers access to the data using a programming interface ([SQL](javascript:call_link\('abensql_glosry.htm'\) "Glossary Entry") for [relational databases](javascript:call_link\('abenrelational_database_glosry.htm'\) "Glossary Entry")).
