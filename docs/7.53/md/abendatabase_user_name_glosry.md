database user name

Identity used by a [database user](javascript:call_link\('abendatabase_user_glosry.htm'\) "Glossary Entry") to log on to a [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry"). A database user name is assigned a [database schema](javascript:call_link\('abendatabase_schema_glosry.htm'\) "Glossary Entry"), whose objects can be accessed by the database user (implicitly or without qualifications). The user name used in an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") read can be determined using the function module DB\_DBUSER. The [ABAP database schema](javascript:call_link\('abenabap_db_schema_glosry.htm'\) "Glossary Entry") is assigned to this user.
