double-byte code

A character in double-byte representation. Used before the introduction of [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") to display Asian characters.
