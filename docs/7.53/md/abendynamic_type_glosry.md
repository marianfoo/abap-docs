dynamic type

In a [reference variable](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreference_variable_glosry.htm "Glossary Entry"), the [data type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_type_glosry.htm "Glossary Entry") or the [class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") of the [object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenobject_glosry.htm "Glossary Entry"), to which the reference variable points. The dynamic type is always more specific than or the same as the [static type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_type_glosry.htm "Glossary Entry").
