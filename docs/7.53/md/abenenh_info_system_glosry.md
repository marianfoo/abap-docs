enhancement information system

Tool in [ABAP Workbench](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_workbench_glosry.htm "Glossary Entry") used to search for [enhancements](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_glosry.htm "Glossary Entry") in the [enhancement concept](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_concept_glosry.htm "Glossary Entry"). Called in [Object Navigator](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenobject_navigator_glosry.htm "Glossary Entry") or by using the [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SPAU\_ENH.
