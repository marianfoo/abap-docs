enhancement information system

Tool in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") used to search for [enhancements](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") in the [enhancement concept](javascript:call_link\('abenenhancement_concept_glosry.htm'\) "Glossary Entry"). Called in [Object Navigator](javascript:call_link\('abenobject_navigator_glosry.htm'\) "Glossary Entry") or by using the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SPAU\_ENH.
