enhancement point

Position in a [repository object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_object_glosry.htm "Glossary Entry") that can be enhanced in accordance with the [enhancement concept](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_concept_glosry.htm "Glossary Entry"). Enhancement points can be defined explicitly or be available implicitly. Explicitly defined enhancement points are realized using [enhancement spot element definitions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_spot_def_glosry.htm "Glossary Entry") and [enhancement spot invocations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_spot_invoc_glosry.htm "Glossary Entry"), and are administered using [enhancement spots](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenhancement_spot_glosry.htm "Glossary Entry").
