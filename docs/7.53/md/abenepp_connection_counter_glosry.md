EPP connection counter

Component of an [Extended Passport (EPP)](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenepp_glosry.htm "Glossary Entry"). Counts connections between clients and servers with the same [EPP connection ID](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenepp_connection_id_glosry.htm "Glossary Entry").
