EPP precomponent ID

Component of an [Extended Passport (EPP)](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenepp_glosry.htm "Glossary Entry"). Contains the [system ID](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensystem_id_glosry.htm "Glossary Entry") and the name of the [application server instance](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenapplication_server_glosry.htm "Glossary Entry") of the source [ABAP session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmain_session_glosry.htm "Glossary Entry") of the current connection.
