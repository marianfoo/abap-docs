EVA

Possible process flow in program execution. The EVA principle dictates that a program is executed in the order input, processing, and then output of the results. See also [reporting](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreporting_glosry.htm "Glossary Entry").
