extended program check

Complete check of the [ABAP source code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_source_code_glosry.htm "Glossary Entry") for all errors that can be detected statically. Programs with errors in the extended program check can be executed, but usually raise an [exception](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenexception_glosry.htm "Glossary Entry"). Called in [ABAP Editor](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_editor_glosry.htm "Glossary Entry") by choosing Program → Check → Extended Program Check or using the [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SLIN or using [ABAP Test Cockpit](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_test_cockpit_glosry.htm "Glossary Entry"). Settings for the extended program check can be made in the administration transaction SLIN\_ADMIN.
