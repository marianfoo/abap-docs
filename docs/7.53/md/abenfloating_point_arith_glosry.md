floating point arithmetic

Calculation using [floating point numbers](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfloating_point_number_2_glosry.htm "Glossary Entry"). Is performed directly on the [host computer](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhost_computer_glosry.htm "Glossary Entry") of the current [AS Instance](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenapplication_server_glosry.htm "Glossary Entry") Can be either [binary floating point arithmetic](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbinfloat_arith_glosry.htm "Glossary Entry") or [decimal floating point arithmetic](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendecfloat_arith_glosry.htm "Glossary Entry").
