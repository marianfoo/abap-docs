[Numeric function](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmathematical_funktion_glosry.htm "Glossary Entry") that expects a [floating point number](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfloating_point_number_2_glosry.htm "Glossary Entry") as an argument and produces a floating point number as a [return value](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreturn_value_glosry.htm "Glossary Entry").
