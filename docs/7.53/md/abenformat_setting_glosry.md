format setting

Part of the [language environment](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlanguage_environment_glosry.htm "Glossary Entry") where the formats for numbers, dates, and times are defined. Defined using either fixed values in the [user master record](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenuser_master_record_glosry.htm "Glossary Entry") or by [country-specific formats](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencountry_formats.htm).
