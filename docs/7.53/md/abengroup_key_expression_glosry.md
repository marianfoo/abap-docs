group key expression

Expression for determining the [group key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abengroup_key_glosry.htm "Glossary Entry") of rows in the grouping of [internal tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry") using [LOOP AT itab ... GROUP BY](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abaploop_at_itab_group_by.htm) or [FOR GROUPS ... OF](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfor_groups_of.htm).
