grouping set

Grouping sets are used in the [GROUP BY](javascript:call_link\('abapgroupby_clause.htm'\)) clause of a [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") to aggregate the results set of the query. Each grouping set aggregates the rows with identical content in the columns and/or the rows with the identical results in SQL expressions. The GROUP BY addition GROUPING SETS makes it possible to group multiple aggregations under one [SELECT](javascript:call_link\('abapselect.htm'\)) statement.
