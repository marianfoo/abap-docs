SAP HANA Studio

Eclipse-based tool for database objects in the [SAP HANA database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhana_database_glosry.htm "Glossary Entry"), more specifically for [XSC](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsc_glosry.htm "Glossary Entry") objects saved in [SAP HANA Repository](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_hana_repository_glosry.htm "Glossary Entry"). In [XSA](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsa_glosry.htm "Glossary Entry"), SAP HANA Studio is replaced by [SAP Web IDE for SAP HANA](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_web_ide_for_sap_han_glosry.htm "Glossary Entry") and [SAP HANA Cockpit](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_hana_cockpit_glosry.htm "Glossary Entry").
