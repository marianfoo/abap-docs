SAP HANA view

An SAP HANA view is used to select data, analyze data, or perform calculations with data from the [SAP HANA database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhana_database_glosry.htm "Glossary Entry"). There are various different types of view: attribute views, analytic views and calculation views. SAP HANA views can be defined as objects in [SAP HANA XS](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxs_glosry.htm "Glossary Entry"). [ABAP programs](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_program_glosry.htm "Glossary Entry") provide special [methods for accessing](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_hana_access.htm) these views.
