hashed table

[Table category](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_category_glosry.htm "Glossary Entry") of an [internal table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry") whose rows are stored internally in accordance with a [hash algorithm](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhash_algorithm_glosry.htm "Glossary Entry") and can be accessed using a unique [hash key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhash_key_glosry.htm "Glossary Entry"). A hashed table does not have a [primary table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprimary_table_index_glosry.htm "Glossary Entry"), but it can be assigned a [secondary table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensecondary_table_index_glosry.htm "Glossary Entry") using a [secondary table key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensecondary_table_key_glosry.htm "Glossary Entry"). The associated [generic data type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [hashed table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbuilt_in_types_generic.htm).
