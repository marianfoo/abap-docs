header line

Obsolete [work area](javascript:call_link\('abenwork_area_glosry.htm'\) "Glossary Entry") of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") that has the type of the [row type](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry") and the same name as the internal table. If an internal table with a header line is used in an [operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry"), the header line is usually addressed. To enforce access to the [table body](javascript:call_link\('abentable_body_glosry.htm'\) "Glossary Entry"), square brackets \[\] can be entered after the table name.
