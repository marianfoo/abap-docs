hierarchy level

Level of a [hierarchy](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhierarchy_glosry.htm "Glossary Entry"). Each object in a hierarchy is part of a hierarchy level. In a hierarchy defined using [parent-child relationships](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpcr_glosry.htm "Glossary Entry") between the objects, hierarchy levels are arranged linearly and are counted from the first parent level.
