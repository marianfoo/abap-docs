hierarchy level

Level of a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry"). Each object in a hierarchy is part of a hierarchy level. In a hierarchy defined using [parent-child relationships](javascript:call_link\('abenpcr_glosry.htm'\) "Glossary Entry") between the objects, hierarchy levels are arranged linearly and are counted from the first parent level.
