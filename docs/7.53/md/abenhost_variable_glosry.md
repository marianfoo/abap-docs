host variable

ABAP [data object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_object_glosry.htm "Glossary Entry") (usually a [variable](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenvariable_glosry.htm "Glossary Entry")) specified in an operand position of an [ABAP SQL](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenopen_sql_host_variables.htm) statement or of a statically embedded [Native SQL](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapexec_host.htm) statement . In ABAP SQL, a host variable must be prefixed with the [escape character](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenescape_character_glosry.htm "Glossary Entry") @ ; in Native SQL, with the escape character : . In [ADBC](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenadbc_glosry.htm "Glossary Entry"), the placeholder ? (to which ABAP data objects are bound) is used instead of host variables.
