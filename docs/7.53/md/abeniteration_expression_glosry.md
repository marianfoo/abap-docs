iteration expression

Expression that performs an [iteration](javascript:call_link\('abeniteration_glosry.htm'\) "Glossary Entry"). An iteration expression is subexpression of a [constructor expression](javascript:call_link\('abenconstructor_expression_glosry.htm'\) "Glossary Entry") introduced using an [iteration operator](javascript:call_link\('abeniteration_operator_glosry.htm'\) "Glossary Entry") [FOR](javascript:call_link\('abenfor.htm'\)) and performs [conditional iterations](javascript:call_link\('abenfor_conditional.htm'\)) or [table iterations](javascript:call_link\('abentable_iteration_glosry.htm'\) "Glossary Entry").
