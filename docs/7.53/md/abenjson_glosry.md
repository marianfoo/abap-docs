JSON

JavaScript Object Notation ([http://www.json.org/](http://www.json.org/)). Data format in text form for data exchange between applications. In ABAP, the [sXML Library](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensxml_library_glosry.htm "Glossary Entry") and [transformations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_json_trafos.htm) can be used to access JSON. This is enabled by a special JSON-XML supported by [sXML](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenjson_xml_glosry.htm "Glossary Entry") Library.
