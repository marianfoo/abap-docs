JSON reader

[XML](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxml_reader_glosry.htm "Glossary Entry") reader that reads [JSON](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenjson_glosry.htm "Glossary Entry") data. A JSON reader handles the JSON data internally like [JSON-XML](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenjson_xml_glosry.htm "Glossary Entry").
