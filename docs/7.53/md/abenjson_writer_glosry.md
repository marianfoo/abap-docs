JSON writer

[XML writer](javascript:call_link\('abenxml_writer_glosry.htm'\) "Glossary Entry") that writes [JSON](javascript:call_link\('abenjson_glosry.htm'\) "Glossary Entry") data. A JSON writer expects input for [JSON-XML](javascript:call_link\('abenjson_xml_glosry.htm'\) "Glossary Entry") and creates JSON data from this input.
