L procedure

A [database procedure](javascript:call_link\('abendatabase_procedure_glosry.htm'\) "Glossary Entry") in the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") programmed in the language [L](javascript:call_link\('abenllang_glosry.htm'\) "Glossary Entry"), used internally at SAP. SQLScript procedures can be created interactively in [SAP Web IDE for SAP HANA](javascript:call_link\('abensap_web_ide_for_sap_han_glosry.htm'\) "Glossary Entry"). In addition, [ABAP Managed Database Procedures (AMDP)](javascript:call_link\('abenabap_managed_db_proc_glosry.htm'\) "Glossary Entry") enable L procedures to be implemented in [AMDP procedure implementations](javascript:call_link\('abenamdp_procedure_method_glosry.htm'\) "Glossary Entry") of [AMDP classes](javascript:call_link\('abenamdp_class_glosry.htm'\) "Glossary Entry").
