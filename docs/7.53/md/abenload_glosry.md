load

In the ABAP environment, a load describes a binary representation of a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") which is optimized for fast access, in the memory or on the database. Examples of loads are those from the [byte code](javascript:call_link\('abenbytecode_glosry.htm'\) "Glossary Entry") of the existing program load of a program or the package load.
