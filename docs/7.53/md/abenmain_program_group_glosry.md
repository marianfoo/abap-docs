main program group

First [program group](javascript:call_link\('abenprogram_group_glosry.htm'\) "Glossary Entry") in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry"). It is created when the internal session is created by a program call. The first program in an internal session is the [main program](javascript:call_link\('abenmain_program_glosry.htm'\) "Glossary Entry") of the main program group. The runtime of the main program in the main program group determines its lifespan and the lifespan of the entire internal session.
