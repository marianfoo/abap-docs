ABAP session

Instance for a [user session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenuser_session_glosry.htm "Glossary Entry") in [AS Instance](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenapplication_server_glosry.htm "Glossary Entry") with [ABAP Memory](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_memory_glosry.htm "Glossary Entry") as a separate memory area. Each [GUI window](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abengui_window_glosry.htm "Glossary Entry") of a user session is associated with its own ABAP session. A maximum of 16 ABAP sessions can be managed for a single user session. An [ABAP program](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_program_glosry.htm "Glossary Entry") is executed in an [internal session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_session_glosry.htm "Glossary Entry") of an ABAP session.
