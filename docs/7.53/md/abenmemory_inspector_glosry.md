Memory Inspector

Tool for displaying and analyzing [memory snapshots](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmemory_snapshot_glosry.htm "Glossary Entry"). Called using [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") S\_MEMORY\_INSPECTOR.
