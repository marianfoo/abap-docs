mesh node

Component of a [mesh](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_glosry.htm "Glossary Entry"). Mesh nodes are always [internal tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry") with a structured [row type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrow_type_glosry.htm "Glossary Entry"). Mesh nodes can have [mesh associations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_association_glosry.htm "Glossary Entry") used to address them in [mesh paths](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_path_glosry.htm "Glossary Entry").
