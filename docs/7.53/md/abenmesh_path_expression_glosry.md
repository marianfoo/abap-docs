mesh path expression

Standalone specification of a [mesh path](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_path_glosry.htm "Glossary Entry") in an [operand position](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperand_position_glosry.htm "Glossary Entry"), where the same rules apply as to a [table expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_expression_glosry.htm "Glossary Entry").
