mesh type

[Data type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_type_glosry.htm "Glossary Entry") of a [mesh](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_glosry.htm "Glossary Entry"). Special [structured type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstructured_type_glosry.htm "Glossary Entry") containing only [table types](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_type_glosry.htm "Glossary Entry") with structured [row types](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrow_type_glosry.htm "Glossary Entry") as [components](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencomponent_glosry.htm "Glossary Entry"), associated with each other using [mesh associations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmesh_association_glosry.htm "Glossary Entry").
