namespace prefix

Name of a [prefix namespace](javascript:call_link\('abenprefix_name_space_glosry.htm'\) "Glossary Entry") with five to ten characters and a forward slash character (/.../) in the first and final places. The names of the [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") of a prefix namespace are introduced using the associated namespace prefix.
