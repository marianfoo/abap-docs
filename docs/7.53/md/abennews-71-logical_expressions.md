Logical Expressions in Release 7.0, EhP2

[1\. Boolean functions](#!ABAP_MODIFICATION_1@1@)

[2\. Predicate functions](#!ABAP_MODIFICATION_2@2@)

[3\. New Boolean operator EQUIV](#!ABAP_MODIFICATION_3@3@)

Note

The program DEMO\_EXPRESSIONS shows examples of the logical expressions in Release 7.0, EhP2.

Modification 1

Boolean Functions

The new built-in [Boolean functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenboole_functions.htm) boolc and boolx accept logical expressions as arguments and return their truth values in the form of character strings or byte strings.

Modification 2

Predicate Functions

The new [predicate functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpredicate_functions.htm) return truth values and can be used like relational expressions. Predicate functions for character-like arguments were introduced in Release 7.0, EhP2.

Modification 3

New Boolean Operator EQUIV

The new Boolean operator [EQUIV](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlogexp_equiv.htm) enables an equivalence relationship between two [logical expressions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlogical_expression_glosry.htm "Glossary Entry").
