null reference

[Initial value](javascript:call_link\('abeninitial_value_glosry.htm'\) "Glossary Entry") of a [reference variable](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"). The null reference does not point to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") and can be assigned to any reference variable.
