numeric expression position

[Reader position](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreading_position_glosry.htm "Glossary Entry") in which not only a [numeric data object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennumeric_data_object_glosry.htm "Glossary Entry") convertible to the operand type can be specified, but also a [numerical expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennumerical_expression_glosry.htm "Glossary Entry").
