numeric operand position

[Operand position](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperand_position_glosry.htm "Glossary Entry") with numeric [operand type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperand_type_glosry.htm "Glossary Entry"), in which a [numeric data object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennumeric_data_object_glosry.htm "Glossary Entry") convertible to the operand type can be specified. See also [numeric expression position](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennumerical_expr_position_glosry.htm "Glossary Entry").
