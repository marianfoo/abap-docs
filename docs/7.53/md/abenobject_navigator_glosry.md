Object Navigator

Development environment for the central editing of [repository objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_object_glosry.htm "Glossary Entry"). Object Navigator displays repository objects as nodes of different tree representations (browser), such as [Repository Browser](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_browser_glosry.htm "Glossary Entry"), [Repository Information System](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenris_glosry.htm "Glossary Entry"), or [Enhancement Information System](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenenh_info_system_glosry.htm "Glossary Entry"), in the navigation area. From here, they can be edited using [forward navigation](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenforward_navigation_glosry.htm "Glossary Entry") in the tool area. Called using [transaction](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE80.
