object reference variable

[Reference variable](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreference_variable_glosry.htm "Glossary Entry") for [object references](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenobject_reference_glosry.htm "Glossary Entry").
