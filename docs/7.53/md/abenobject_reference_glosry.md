object reference

[Reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreference_glosry.htm "Glossary Entry") that points to an [instance](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninstance_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry"). An non-initial object reference is always a [heap reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenheap_reference_glosry.htm "Glossary Entry"). See also [weak reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenweak_reference_glosry.htm "Glossary Entry").
