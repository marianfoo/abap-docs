ABAP SQL - Conditions sql\_cond

Syntax

...   rel\_exp
    *|* *\[*NOT*\]* sql\_cond *\[*AND*|*OR sql\_cond*\]* ...

Effect

Logical expression sql\_cond for formulating a condition in ABAP SQL. A logical expression consists of one or more relational expressions rel\_exp that are linked with [AND](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenwhere_logexp_andornot.htm) or [OR](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenwhere_logexp_andornot.htm) and can be negated with [NOT](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenwhere_logexp_andornot.htm). There are different types of relational expressions:

-   [Relational expressions for statements](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenwhere_logexp.htm)

-   [Relational expressions for expressions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenosql_expr_logexp.htm)
