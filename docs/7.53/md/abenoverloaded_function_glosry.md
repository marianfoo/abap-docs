general numeric function

[Numeric function](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmathematical_funktion_glosry.htm "Glossary Entry") that expects a [numeric data object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abennumeric_data_object_glosry.htm "Glossary Entry") as an argument. The [data type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_type_glosry.htm "Glossary Entry") of the argument determines the data type of the [return value](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreturn_value_glosry.htm "Glossary Entry") of the function.
