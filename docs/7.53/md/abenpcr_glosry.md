parent-child relationship

PCR for short. A parent-child relationship between two objects defines one object as the parent (a predecessor object) and one as the child (a successor object). A set of objects in parent-child relationships comprise a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry") and each object is assigned to a [hierarchy level](javascript:call_link\('abenhierarchy_level_glosry.htm'\) "Glossary Entry").
