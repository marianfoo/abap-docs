parent-child relationship

PCR for short. A parent-child relationship between two objects defines one object as the parent (a predecessor object) and one as the child (a successor object). A set of objects in parent-child relationships comprise a [hierarchy](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhierarchy_glosry.htm "Glossary Entry") and each object is assigned to a [hierarchy level](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhierarchy_level_glosry.htm "Glossary Entry").
