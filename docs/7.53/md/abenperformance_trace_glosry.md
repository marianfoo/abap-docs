performance trace

Test tool combining multiple trace functions. These functions enable system behavior to be analyzed in [database reads](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_sql.htm) ([SQL trace](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensql_trace_glosry.htm "Glossary Entry")), when the [table buffer](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_buffer_glosry.htm "Glossary Entry") is read, in [lock management](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_lock.htm) reads, and external reads using [RFC](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrfc_glosry.htm "Glossary Entry") and [ICF](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenicf_glosry.htm "Glossary Entry"). Called using [transaction](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") ST05.
