PCFG condition

Part of an [access condition](javascript:call_link\('abenaccess_condition_glosry.htm'\) "Glossary Entry") in a [CDS role](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry") that uses an [authorization object](javascript:call_link\('abenauthorization_object_glosry.htm'\) "Glossary Entry") to associate an element of a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") with [authorizations](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry") of the current [user](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry").
