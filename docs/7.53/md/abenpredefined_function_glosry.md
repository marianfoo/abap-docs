built-in function

A predefined function in ABAP that can be used in [certain operand positions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperands_expressions.htm). Available as [numeric functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmathematical_funktion_glosry.htm "Glossary Entry"), [string functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_function_glosry.htm "Glossary Entry"), [table functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_function_glosry.htm "Glossary Entry"), and [byte string functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbinary_function_glosry.htm "Glossary Entry"). The return values have the corresponding data type. [Description functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendescription_function_glosry.htm "Glossary Entry"), which provide information about data objects, are also available. The names of the built-in functions are a subset of the [reserved names](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_words.htm). Built-in functions are also available in [ABAP SQL](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenopen_sql_functions.htm) and in the [CDS DDL](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencds_f1_builtin_functions.htm) in ABAP [CDS](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_cds_glosry.htm "Glossary Entry").
