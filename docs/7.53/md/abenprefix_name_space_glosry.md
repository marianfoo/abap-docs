prefix namespace

Namespace for [repository objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrepository_object_glosry.htm "Glossary Entry") belonging to SAP departments, SAP partners, and SAP customers, and which can be requested from SAP. The names of the repository objects of a prefix namespace are introduced using a [namespace prefix](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenname_space_prefix_glosry.htm "Glossary Entry").
