primary index

Unique [index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenindex_glosry.htm "Glossary Entry") of a [database table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry") defined in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). The index is constructed from the [key fields](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenkey_field_glosry.htm "Glossary Entry") of the [primary key](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprimary_key_glosry.htm "Glossary Entry"). When a table is activated in ABAP Dictionary, the primary index is created automatically using the primary key fields of the table. Certain [internal tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry") have a [primary table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprimary_table_index_glosry.htm "Glossary Entry").
