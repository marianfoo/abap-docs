primary table index

[Table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_index_glosry.htm "Glossary Entry") of [index tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenindex_table_glosry.htm "Glossary Entry") ([standard tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstandard_table_glosry.htm "Glossary Entry") and [sorted tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensorted_table_glosry.htm "Glossary Entry")). Every index table has a primary table index, managed by the [ABAP runtime environment](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry"), which contains a unique row number for each table row. The primary table index of an index table is updated each time a row is added or deleted from an internal table.
