pseudo reference

Label super that can be specified in the implementation of a redefined [instance method](javascript:call_link\('abeninstance_method_glosry.htm'\) "Glossary Entry") or an [instance constructor](javascript:call_link\('abeninstance_constructor_glosry.htm'\) "Glossary Entry") instead of a [reference variable](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"). It enables the implementation of the [method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") to be called in the direct [superclass](javascript:call_link\('abensuperclass_glosry.htm'\) "Glossary Entry").
