pseudo reference

Label super that can be specified in the implementation of a redefined [instance method](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninstance_method_glosry.htm "Glossary Entry") or an [instance constructor](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninstance_constructor_glosry.htm "Glossary Entry") instead of a [reference variable](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreference_variable_glosry.htm "Glossary Entry"). It enables the implementation of the [method](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenmethod_glosry.htm "Glossary Entry") to be called in the direct [superclass](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensuperclass_glosry.htm "Glossary Entry").
