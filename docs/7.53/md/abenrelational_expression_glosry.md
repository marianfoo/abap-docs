relational expression

[Expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenexpression_glosry.htm "Glossary Entry") whose result is a [truth value](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlogical_value_glosry.htm "Glossary Entry"). Relational expressions include both [comparison expressions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencomparison_expression_glosry.htm "Glossary Entry") and [predicates](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpredicate_glosry.htm "Glossary Entry"). A single relational expression or multiple relational expressions joined using [Boolean operators](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenboolean_operator_glosry.htm "Glossary Entry") create a [logical expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlogical_expression_glosry.htm "Glossary Entry").
