remote-enabled function module

Abbreviated as [RFM](javascript:call_link\('abenrfm_glosry.htm'\) "Glossary Entry"). [Function module](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") that is flagged as remote-enabled in its properties, meaning it can be called using [remote function call](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry").
