return value

Numeric value for many [ABAP statements](javascript:call_link\('abenabap_statement_glosry.htm'\) "Glossary Entry") that is entered in the [system field](javascript:call_link\('abensystem_field_glosry.htm'\) "Glossary Entry") [sy-subrc](javascript:call_link\('abensystem_fields.htm'\)) when the statement is executed. The value 0 usually means processing has been completed successfully. In [functional methods](javascript:call_link\('abenfunctional_method_glosry.htm'\) "Glossary Entry") and [built-in functions](javascript:call_link\('abenpredefined_function_glosry.htm'\) "Glossary Entry") the return value is the result. The result for functional methods is the [formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") defined by the addition [RETURNING](javascript:call_link\('abapmethods_functional.htm'\)) of the statement [*\[*CLASS-*\]*](javascript:call_link\('abapclass-methods.htm'\))[METHODS](javascript:call_link\('abapmethods.htm'\)).
