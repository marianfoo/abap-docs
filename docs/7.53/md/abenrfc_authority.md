RFC Authorizations

If the [profile parameter](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprofile_parameter_glosry.htm "Glossary Entry") auth/rfc\_authority\_check is set to 1, an automatic authorization check is carried out when a function module is called by remote call.

More Information

For more details about RFC authorizations, see the RFC documentation on [SAP Help Portal](http://help.sap.com).
