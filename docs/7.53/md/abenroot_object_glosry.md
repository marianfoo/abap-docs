root object

Special [shared object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenshared_object_glosry.htm "Glossary Entry"). [Instance](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninstance_glosry.htm "Glossary Entry") of the [area root class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenroot_data_class_glosry.htm "Glossary Entry"). A non-empty [area instance version](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenarea_instance_version_glosry.htm "Glossary Entry") must contain a root object. [Attribute](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenattribute_glosry.htm "Glossary Entry") ROOT of an associated [area handle](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenarea_handle_glosry.htm "Glossary Entry") points to the root object. This can contain references to the other shared objects of the area instance version.
