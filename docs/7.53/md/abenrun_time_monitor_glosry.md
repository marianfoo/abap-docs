Runtime Monitor

Tool for recording events raised by [system classes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensystem_class_glosry.htm "Glossary Entry") when a program is executed. Called using [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SRTM.
