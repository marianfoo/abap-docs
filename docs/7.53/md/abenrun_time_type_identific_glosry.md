runtime type identification

Abbreviation RTTI. The identification of [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") at program runtime. Implemented using description methods in [type description classes](javascript:call_link\('abentype_class_glosry.htm'\) "Glossary Entry"). Together with [RTTC](javascript:call_link\('abenrun_time_type_creation_glosry.htm'\) "Glossary Entry"), constitutes [RTTS](javascript:call_link\('abenrun_time_type_services_glosry.htm'\) "Glossary Entry").
