runtime type services

Abbreviated as RTTS. Combines RTTC and [RTTI](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenrun_time_type_identific_glosry.htm "Glossary Entry") in a class hierarchy. The RTTS methods get information about [data types](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendata_type_glosry.htm "Glossary Entry") or [classes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") and [interfaces](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninterface_glosry.htm "Glossary Entry") of [objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenobject_glosry.htm "Glossary Entry"), or create new data types as [type description objects](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentype_object_glosry.htm "Glossary Entry"). RTTS are implemented as a hierarchy of [type description classes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentype_class_glosry.htm "Glossary Entry") from which type description objects can be created.
