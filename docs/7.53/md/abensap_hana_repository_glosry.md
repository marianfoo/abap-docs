SAP HANA repository

Repository of objects in the [SAP HANA database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhana_database_glosry.htm "Glossary Entry"). More specifically, [XSC](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsc_glosry.htm "Glossary Entry") objects are saved in SAP HANA Repository. [XSA](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsa_glosry.htm "Glossary Entry") objects, on the other hand, are not saved in SAP HANA Repository and are usually deployed using [HDI](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenhdi_glosry.htm "Glossary Entry").
