SAP spool system

[AS Instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") process for managing sequential data streams that are saved as [spool lists](javascript:call_link\('abenprint_list_glosry.htm'\) "Glossary Entry"), printed on a printer, or stored using [ArchiveLink](javascript:call_link\('abensap_archivelink_glosry.htm'\) "Glossary Entry"). [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") can create [spool requests](javascript:call_link\('abenspool_request_glosry.htm'\) "Glossary Entry") when lists are created by including pages of lists in [spool list](javascript:call_link\('abenprint_list_glosry.htm'\) "Glossary Entry") output.
