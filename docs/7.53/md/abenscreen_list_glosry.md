screen list

[List](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlist_glosry.htm "Glossary Entry") displayed on the screen in a [list dynpro](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlist_dynpro_glosry.htm "Glossary Entry"). When it is being compiled, a screen list is stored in the [list buffer](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenlist_buffer_glosry.htm "Glossary Entry"). See also [spool list](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprint_list_glosry.htm "Glossary Entry").
