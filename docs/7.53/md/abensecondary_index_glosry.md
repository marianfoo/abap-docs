secondary index

[Index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenindex_glosry.htm "Glossary Entry") of [database tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_table_glosry.htm "Glossary Entry") defined in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). Created alongside the [primary index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprimary_index_glosry.htm "Glossary Entry"). Creating secondary indexes can improve the performance of database reads performed on the indexes of the [database](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendatabase_glosry.htm "Glossary Entry"). A secondary index can be unique, but does not have to be. A [secondary table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensecondary_table_index_glosry.htm "Glossary Entry") can be assigned to [internal tables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry").
