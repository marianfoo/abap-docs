secondary table index

[Table index](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentable_index_glosry.htm "Glossary Entry") for the administration of [sorted](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensorted_key_glosry.htm "Glossary Entry") [secondary table keys](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensecondary_table_key_glosry.htm "Glossary Entry"). A secondary table index of a non-unique key is not updated instantly after a row is added to, or deleted from, an [internal table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninternal_table_glosry.htm "Glossary Entry"); it is only updated when the internal table is accessed using the associated secondary key.
