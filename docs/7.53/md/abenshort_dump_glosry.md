short dump

Error log created and saved after a [runtime error](javascript:call_link\('abenruntime_error_glosry.htm'\) "Glossary Entry") or by a [runtime warning](javascript:call_link\('abenruntime_warning_glosry.htm'\) "Glossary Entry"). Saved short dumps can be displayed using the [ABAP dump analysis](javascript:call_link\('abenabap_dump_analsyis_glosry.htm'\) "Glossary Entry") tool.
