soft reference

Reference to an [object](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenobject_glosry.htm "Glossary Entry") where the object is not passed to the reference if the available memory is running low. The class CL\_ABAP\_SOFT\_REFERENCE is planned for soft references but not yet implemented. See also [weak reference](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenweak_reference_glosry.htm "Glossary Entry").
