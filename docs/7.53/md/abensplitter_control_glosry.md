splitter control

Splitter controls make it possible to display two [subscreens](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensubscreen_glosry.htm "Glossary Entry") beside each other or one above the other. A splitter control is a [screen element](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenscreen_element_glosry.htm "Glossary Entry") that divides the area that it covers into two [subscreen areas](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensubscreen_arrea_glosry.htm "Glossary Entry"). The boundary between the areas can be moved using a [sash](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensash_glosry.htm "Glossary Entry").
