standalone selection screen

[Selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry") defined between the statements [SELECTION-SCREEN *{*BEGIN*|*END*}* OF SCREEN](javascript:call_link\('abapselection-screen_definition.htm'\)). Standalone selection screens can be defined in all programs that can contain [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry"). See also [standard selection screen](javascript:call_link\('abenstandard_selscreen_glosry.htm'\) "Glossary Entry").
