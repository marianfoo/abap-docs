static component

Umbrella term for [static attribute](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_attribute_glosry.htm "Glossary Entry"), [static event](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_event_glosry.htm "Glossary Entry"), and [static method](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_method_glosry.htm "Glossary Entry"). The static components of a [class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenclass_glosry.htm "Glossary Entry") can be addressed using the name of the class. A class that only contains static components is referred to as a [static class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_class_glosry.htm "Glossary Entry").
