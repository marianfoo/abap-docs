String Functions

String functions are part of the [built-in functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenbuilt_in_functions.htm). They are divided into:

-   [Description functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendescriptive_functions.htm)

-   [Processing functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenprocess_functions.htm)

Description functions have a numeric result and processing functions have a character-like result.

Parameters that are used by more than one function in the same way are described under [Shared Parameters of String Functions](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_functions_common_paras.htm).

Note

The program DEMO\_EXPRESSIONS shows examples of how to use string expressions.
