string operator

Joins two character-like [operands](javascript:call_link\('abenoperand_glosry.htm'\) "Glossary Entry") in a [string expression](javascript:call_link\('abenstring_expression_glosry.htm'\) "Glossary Entry"). The only string operator currently possible is the [concatenation operator](javascript:call_link\('abenconcatenation_operator_glosry.htm'\) "Glossary Entry") [&&](javascript:call_link\('abenstring_operators.htm'\)), which joins the two operands in a single string.
