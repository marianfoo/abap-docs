string operator

Joins two character-like [operands](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenoperand_glosry.htm "Glossary Entry") in a [string expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_expression_glosry.htm "Glossary Entry"). The only string operator currently possible is the [concatenation operator](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenconcatenation_operator_glosry.htm "Glossary Entry") [&&](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_operators.htm), which joins the two operands in a single string.
