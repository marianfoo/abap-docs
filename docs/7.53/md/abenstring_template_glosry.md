string template

A string template creates a [string](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abencharacter_string_1_glosry.htm "Glossary Entry") from literal text, embedded expressions, and control characters in a [string expression](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstring_expression_glosry.htm "Glossary Entry").
