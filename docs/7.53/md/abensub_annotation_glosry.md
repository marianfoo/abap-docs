subannotation

[CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") specified in curly brackets [{ }](javascript:call_link\('abencds_f1_define_annotation.htm'\)) in an [annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry") behind another annotation. When a subannotation is specified in [annotation syntax](javascript:call_link\('abencds_annotation_syntax_glosry.htm'\) "Glossary Entry"), curly brackets [{ }](javascript:call_link\('abencds_annotations_syntax.htm'\)) are also used (or [structured names](javascript:call_link\('abencds_annotations_syntax_name.htm'\)) separated using periods).
