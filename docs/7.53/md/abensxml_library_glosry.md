sXML Library

Global [interfaces](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenglobal_interface_glosry.htm "Glossary Entry") and [classes](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenglobal_class_glosry.htm "Glossary Entry") for handling [XML](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxml_glosry.htm "Glossary Entry") data in various [formats](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_sxml_lib_formats.htm). The "serial" XML Library provides XML readers and XML writers for various sources and targets. The classes and interfaces in sXML Library start with the prefixes CL\_SXML and IF\_SXML respectively.
