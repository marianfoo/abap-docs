system class

[Class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") of the [class library](javascript:call_link\('abenclass_library_glosry.htm'\) "Glossary Entry") in which language-relevant functions are implemented. Its name usually starts with CL\_ABAP\_, CL\_SYSTEM\_, or CX\_SY\_ (in the case of [exception classes](javascript:call_link\('abenexception_class_glosry.htm'\) "Glossary Entry")). An overview of all ABAP system classes can be displayed using a [Class Builder](javascript:call_link\('abenclass_builder_glosry.htm'\) "Glossary Entry") [input help](javascript:call_link\('abeninput_help_glosry.htm'\) "Glossary Entry").
