system ID

SID for short. ID or name of an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") and hence the ID or name of its [ABAP system](javascript:call_link\('abenabap_system_glosry.htm'\) "Glossary Entry") too. A system ID has three characters and is the content of the [system field](javascript:call_link\('abensystem_field_glosry.htm'\) "Glossary Entry") [sy-sysid](javascript:call_link\('abensystem_fields.htm'\)).
