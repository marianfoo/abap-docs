table expression

A table expression reads a row from an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") and returns it as a result. [Mesh path expressions](javascript:call_link\('abenmesh_path_expression_glosry.htm'\) "Glossary Entry") are a special form of table expression that can be used in exactly the same way as table expressions.
