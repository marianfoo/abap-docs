table parameter

Obsolete [formal parameter](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenformal_parameter_glosry.htm "Glossary Entry") of [function modules](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfunction_module_glosry.htm "Glossary Entry") or [subroutines](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensubroutine_glosry.htm "Glossary Entry") typed as an internal [standard table](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstandard_table_glosry.htm "Glossary Entry") with a [header line](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenheader_line_glosry.htm "Glossary Entry"). A table parameter is defined using the addition TABLES of the statements [FUNCTION](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenfunction.htm) and [FORM](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapform.htm) .
