test method

Special [instance method](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abeninstance_method_glosry.htm "Glossary Entry") of a [test class](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentest_class_glosry.htm "Glossary Entry") in which a test is implemented. Test methods are defined using the addition [FOR TESTING](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapmethods_testing.htm) of the statement [METHODS](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abapmethods.htm), and called during a [test run](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentest_run_glosry.htm "Glossary Entry") of the [ABAP Unit](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_unit_glosry.htm "Glossary Entry") tool.
