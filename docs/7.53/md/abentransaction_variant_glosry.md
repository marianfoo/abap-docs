transaction variant

Transaction variants can be used to assign default values to the input fields on multiple [dynpros](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendynpro_glosry.htm "Glossary Entry") of a [transaction](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_glosry.htm "Glossary Entry"), to change the attributes of [screen elements](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenscreen_element_glosry.htm "Glossary Entry"), and to hide entire [screens](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenscreen_glosry.htm "Glossary Entry"). Transaction variants are edited and managed using the [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") SHD0.
