Transformation Editor

Tool in [ABAP Workbench](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_workbench_glosry.htm "Glossary Entry") used for creating [XSLT programs](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsl_transformation_2_glosry.htm "Glossary Entry") or [simple transformations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensimple_transformation_glosry.htm "Glossary Entry") in [transformation source code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsl_transformation_2_glosry.htm "Glossary Entry"). Called using the [transaction code](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abentransaction_code_glosry.htm "Glossary Entry") STRANS.
