up cast

Also known as a widening cast. [Assignment](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenassignment_glosry.htm "Glossary Entry") between [reference variables](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenreference_variable_glosry.htm "Glossary Entry"), where the [static type](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenstatic_type_glosry.htm "Glossary Entry") of the target variables is similar or identical to the static type of the source variables. An up cast does not have to be performed explicitly. See [down cast](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abendown_cast_glosry.htm "Glossary Entry").
