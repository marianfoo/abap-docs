user interface

Full set of [control elements](javascript:call_link\('abencontrol_element_glosry.htm'\) "Glossary Entry") in an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"). The user interface can be implemented using various different technologies, such as the browser-based [SAPUI5](javascript:call_link\('abensapui5_glosry.htm'\) "Glossary Entry"), [Web Dynpro](javascript:call_link\('abenweb_dynpro_glosry.htm'\) "Glossary Entry"), or the user-installed [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry").
