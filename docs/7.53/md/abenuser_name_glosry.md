user name

Name of a [user](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry") required for logging on to a [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"). Together with the [password](javascript:call_link\('abenpassword_glosry.htm'\) "Glossary Entry"), the user name allows authorized use of AS ABAP.
