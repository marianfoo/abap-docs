user name

Name of a [user](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenuser_glosry.htm "Glossary Entry") required for logging on to a [user session](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenuser_session_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abensap_nw_abap_glosry.htm "Glossary Entry"). Together with the [password](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenpassword_glosry.htm "Glossary Entry"), the user name allows authorized use of AS ABAP.
