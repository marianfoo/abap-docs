Web Application Builder

Tool in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") used to create [Business Server Pages](javascript:call_link\('abenbusiness_server_pages_glosry.htm'\) "Glossary Entry"). Called by choosing Create → BSP Library → BSP Application in [Object Navigator](javascript:call_link\('abenobject_navigator_glosry.htm'\) "Glossary Entry").
