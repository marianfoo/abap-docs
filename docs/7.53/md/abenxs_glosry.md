SAP HANA XS

SAP HANA Extended Application Services. XS is a framework for developing and managing native applications in [SAP HANA](javascript:call_link\('abenhana_glosry.htm'\) "Glossary Entry"). The current version, [XSA](javascript:call_link\('abenxsa_glosry.htm'\) "Glossary Entry"), replaces the previous version, [XSC](javascript:call_link\('abenxsc_glosry.htm'\) "Glossary Entry"). XS objects can be accessed from ABAP.
