XSLT

XSLT (Extensible Stylesheet Language Transformation, see [www.w3.org/TR/xslt](http://www.w3.org/tr/xslt)) enables the conversion of [XML](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxml_glosry.htm "Glossary Entry") formats into any other XML formats. The [ABAP runtime environment](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") contains an [SAP XSLT processor](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxslt_processor_glosry.htm "Glossary Entry") for performing [XSL transformations](https://help.sap.com/doc/abapdocu_753_index_htm/7.53/en-US/abenxsl_transformation_1_glosry.htm "Glossary Entry").
