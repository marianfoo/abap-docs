AT LINE-SELECTION - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapat_line-selection.htm)

Syntax

AT LINE-SELECTION.

Effect

Introduces an event block whose result is triggered by the selection of a list line using the predefined function code "PICK".
