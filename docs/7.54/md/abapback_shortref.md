BACK - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapback.htm)

Syntax

BACK.

Effect

Positions the [list cursor](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlist_cursor_glosry.htm "Glossary Entry") at the first position of a line block defined with RESERVE, a [page header](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpage_header_glosry.htm "Glossary Entry") defined after TOP-OF-PAGE, or the first line underneath the page header.
