CLASS-EVENTS - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapclass-events.htm)

Syntax

CLASS-EVENTS evt options.

Effect

Declares a [static event](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatic_event_glosry.htm "Glossary Entry") evt of a class or an interface.

Additions

-   options
    As in [EVENTS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapevents_shortref.htm).
