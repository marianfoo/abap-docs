DELETE DATASET - Quick reference

[Reference](javascript:call_link\('abapdelete_dataset.htm'\))

Syntax

DELETE DATASET dset.

Effect

Deletes the file specified in dset on the [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") of the current [AS Instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry").
