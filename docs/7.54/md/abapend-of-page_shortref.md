END-OF-PAGE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapend-of-page.htm)

Syntax

END-OF-PAGE.

Effect

Initiates an event block whose event is raised when the [page footer](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpage_footer_glosry.htm "Glossary Entry") is reached when a basic list is created.
