END-TEST-INJECTION

[Quick Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abaptest-injection_shortref.htm)

Syntax

END-TEST-INJECTION.

Effect

The statement END-TEST-INJECTION closes a test injection that was initiated using [TEST-INJECTION](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abaptest-injection.htm).
