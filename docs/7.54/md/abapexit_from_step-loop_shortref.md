EXIT FROM STEP-LOOP - Quick reference

[Reference](javascript:call_link\('abapexit_step-loop.htm'\))

Syntax

EXIT FROM STEP-LOOP.

Effect

Exits the loop processing of a [table control](javascript:call_link\('abentable_control_glosry.htm'\) "Glossary Entry") or [step loop](javascript:call_link\('abenstep_loop_glosry.htm'\) "Glossary Entry") in the current [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry"). Processing continues in the [dynpro flow logic](javascript:call_link\('abendynpro_flow_logic_glosry.htm'\) "Glossary Entry") after the loop.
