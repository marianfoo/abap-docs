FIELDS

[Quick Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapfields_shortref.htm)

Obsolete Syntax

FIELDS dobj.

Effect

This statement (forbidden in classes) addresses a data object dobj of the program. A warning from the extended program check can be avoided if the data object dobj is addressed in the program dynamically and not statically.

Note

This statement (forbidden in classes) is replaced by the [pragma](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpragma_glosry.htm "Glossary Entry") ##NEEDED.
