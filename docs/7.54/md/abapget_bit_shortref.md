GET BIT - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapget_bit.htm)

Syntax

GET BIT bitpos OF byte\_string INTO val.

Effect

Returns the bit at the bit position bitpos of the data object byte\_string in val.
