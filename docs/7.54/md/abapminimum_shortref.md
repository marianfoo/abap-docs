MINIMUM - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapminimum.htm)

Syntax

MINIMUM dobj.

Effect

Obsolete: After the statement MINIMUM, determines the minimum value of dobj and assigns it to the data object min\_dobj for every WRITE statement for dobj.
