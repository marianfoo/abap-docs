RESUME - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapresume.htm)

Syntax

RESUME.

Effect

Resumes processing when handling a [resumable exception](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenresumable_exception_glosry.htm "Glossary Entry").
