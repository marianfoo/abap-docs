SELECTION-SCREEN - VERSION - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapselection-screen_ldb_version.htm)

Syntax

SELECTION-SCREEN BEGIN OF VERSION vers text.
...
SELECTION-SCREEN END OF VERSION vers.

Effect

Obsolete: Defines a version vers of the standard selection screen in a [logical database](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlogical_data_base_glosry.htm "Glossary Entry").

Addition

-   text
    Specifies a help text for the version.
