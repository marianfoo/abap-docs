SET LOCALE LANGUAGE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapset_locale.htm)

Syntax

SET LOCALE LANGUAGE lang *\[*COUNTRY cntry*\]* *\[*MODIFIER mod*\]*.

Effect

Sets the [text environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentext_environment_glosry.htm "Glossary Entry").

Additions

-   LANGUAGE lang
    Specifies the language in lang.
    
-   [COUNTRY cntry](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapset_locale_obsolete.htm)
    Obsolete: Specifies the country key in cntry.
    
-   [MODIFIER mod](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapset_locale_obsolete.htm)
    Obsolete: Specifies the ID for a specific locale in mod.
