SUBTRACT - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapsubtract_multiply_divide.htm)

Syntax

SUBTRACT dobj1 FROM dobj2.

Effect

Obsolete: Subtracts the content of a numeric data object dobj1 from the content of a numeric data object dobj2 and assigns the event dobj2.
