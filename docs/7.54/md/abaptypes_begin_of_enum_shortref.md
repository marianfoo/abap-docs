TYPES BEGIN OF ENUM - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abaptypes_enum.htm)

Syntax

TYPES BEGIN OF ENUM enum\_type *\[*...*\]*.
  val1 *\[*...*\]*, val2 *\[*...*\]*, ...
TYPES END OF ENUM enum\_type *\[*...*\]*.

Effect

Defines an enumerated type enum\_type.
