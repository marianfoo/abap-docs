Calling Transactions

A transaction is the execution of a program using a [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry"). The program process flow depends on which [initial screen](javascript:call_link\('abeninitial_dynpro_glosry.htm'\) "Glossary Entry") or [selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry"), or which method, the transaction code is linked to.

There are two statements for calling transactions:

-   [CALL TRANSACTION](javascript:call_link\('abapcall_transaction.htm'\))

-   [LEAVE TO TRANSACTION](javascript:call_link\('abapleave_to_transaction.htm'\))
