ABAP command injection

Injection of malicious ABAP statements by an attacker. ABAP command injections are possible whenever programs use external statements (or parts of statements) that are then are passed to the [runtime environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") without being checked.
