ABAP Daemon memory

A part of the [user memory](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_memory_glosry.htm "Glossary Entry") in which an [ABAP Daemon](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_daemon_glosry.htm "Glossary Entry") can save data in [PCP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpcp_glosry.htm "Glossary Entry") format. An ABAP daemon uses approximately 1 megabyte of ABAP Daemon memory.
