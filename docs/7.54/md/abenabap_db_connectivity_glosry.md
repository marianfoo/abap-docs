ABAP database connectivity

Abbreviated as ADBC. Class-based API for the [Native SQL interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abennative_sql_interface_glosry.htm "Glossary Entry"). ADBC enables object-oriented execution of dynamically specified [Native SQL](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abennative_sql_glosry.htm "Glossary Entry").
