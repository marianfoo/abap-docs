ABAP Doc Comment

Special [comment](javascript:call_link\('abencomment_glosry.htm'\) "Glossary Entry") consisting of one or more [comment lines](javascript:call_link\('abencomment_line_glosry.htm'\) "Glossary Entry"), each of which is prefixed by "!. ABAP Doc comments can be placed in front of [declarations](javascript:call_link\('abendeclaration_glosry.htm'\) "Glossary Entry") and are used by [ABAP Doc](javascript:call_link\('abenabap_doc_glosry.htm'\) "Glossary Entry") for documentation purposes.
