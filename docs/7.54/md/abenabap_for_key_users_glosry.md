ABAP for Key Users

[ABAP language version](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_version_glosry.htm "Glossary Entry") with restricted language scope for [enhancements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_glosry.htm "Glossary Entry") at dedicated enhancement points made by key users. ABAP for key users is available in the special extensibility tools for key users.
