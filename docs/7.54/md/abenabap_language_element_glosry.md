ABAP language element

[ABAP keyword](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_keyword_glosry.htm "Glossary Entry") or combination of an ABAP keyword and one or more succeeding [ABAP words](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_word_glosry.htm "Glossary Entry") or [operands](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenoperand_glosry.htm "Glossary Entry"). An ABAP language element uniquely describes the semantics of an [ABAP statement](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_statement_glosry.htm "Glossary Entry"). Examples of different ABAP language elements are GET BIT, GET TIME, DELETE itab, or DELETE dbtab. ABAP language elements can be enhanced using [ABAP language element additions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_language_elmnt_add_glosry.htm "Glossary Entry").
