Processing External Data

External data generally covers all data saved outside of the [internal session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninternal_session_glosry.htm "Glossary Entry"). The following sections describe the processing of external data.

-   [Data Clusters](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_cluster.htm)

-   [ABAP File Interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_language_files.htm)
