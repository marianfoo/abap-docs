Procedures

Procedures are processing blocks with [parameter interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenformal_parameters_oview.htm) and a local [data area](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_area_glosry.htm "Glossary Entry"). They can be called from within ABAP programs. The possible types of procedure are:

-   [Methods](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_methods.htm)

-   [Function modules](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_functions.htm)

The following is an obsolete type of procedure:

-   [Subroutines](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_subroutines.htm)
