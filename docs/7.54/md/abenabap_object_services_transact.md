transaction service

The transaction service implicitly or explicitly handles [updates](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenupdate_glosry.htm "Glossary Entry") of persistent objects created in the program or whose persistent attributes were changed.
