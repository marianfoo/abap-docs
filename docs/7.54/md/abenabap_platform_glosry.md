ABAP Platform

ABAP Platform is used to develop and run ABAP-based [application programs](javascript:call_link\('abenapplication_program_glosry.htm'\) "Glossary Entry"). It is implemented mainly using [ABAP systems](javascript:call_link\('abenabap_system_glosry.htm'\) "Glossary Entry"). ABAP Platform also includes a range of development environments, including [ABAP Development Tools (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") and [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") .
