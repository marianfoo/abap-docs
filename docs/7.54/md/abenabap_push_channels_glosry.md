ABAP Push Channels

Abbreviated as APC. Framework for bidirectional communication between an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") and the Internet using the WebSocket protocol or TCP socket protocol. ABAP Push Channels can be associated with [ABAP Messaging Channels](javascript:call_link\('abenabap_messaging_channels_glosry.htm'\) "Glossary Entry").
