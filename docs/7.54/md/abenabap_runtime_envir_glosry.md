ABAP runtime environment

Hardware, operating system, and database-independent platform (virtual machine) of an [ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry"). Processes of the ABAP runtime environment control the execution of an ABAP program by calling the [processing blocks](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprocessing_block_glosry.htm "Glossary Entry") of the program. The ABAP runtime environment is provided by the Application Server ABAP. [ABAP programs](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") can be executed only in AS ABAP.
