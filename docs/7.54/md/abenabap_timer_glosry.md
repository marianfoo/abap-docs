ABAP Timer

A timer for wait times in [non-blocking mode](javascript:call_link\('abennon_blocking_model_glosry.htm'\) "Glossary Entry"), integrated into the [ABAP runtime environment](javascript:call_link\('abenabap_timer_manager_glosry.htm'\) "Glossary Entry"). An ABAP Timer is started for a defined runtime using a [timer manager](javascript:call_link\('abenabap_timer_manager_glosry.htm'\) "Glossary Entry"). At the end of this runtime, a timeout event is raised that is responded to using a [timer handler](javascript:call_link\('abenabap_timer_handler_glosry.htm'\) "Glossary Entry").
