ABAP Unit

Test tool integrated in the [ABAP runtime environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") that is used for checking the functions of code sections in a program ([unit tests](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunit_test_glosry.htm "Glossary Entry")). The individual tests are implemented as [test methods](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentest_method_glosry.htm "Glossary Entry") of local [test classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentest_class_glosry.htm "Glossary Entry") in ABAP programs. The tests in a test class all use the same [fixture](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfixture_glosry.htm "Glossary Entry"). Global test classes can only be used in local test classes. The attributes of a test can be specified in the definition of a local test class. Tests for several ABAP programs can be grouped together into [test tasks](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentest_task_glosry.htm "Glossary Entry").
