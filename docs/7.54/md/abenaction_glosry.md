Action

A change operation that differs from the standard and is assigned to a [business object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbusiness_object_glosry.htm "Glossary Entry").

Actions can either reference an instance (the default case) or they can be static (specified using the addition static).
