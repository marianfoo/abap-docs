actual parameter

[Data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") whose content is passed as an argument to a [formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") or is taken from this formal parameter when a [procedure](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry") is called. The required data type and the type of passing (by [reference](javascript:call_link\('abenpass_by_reference_glosry.htm'\) "Glossary Entry") or [value](javascript:call_link\('abenpass_by_value_glosry.htm'\) "Glossary Entry")) are specified by the definition of the formal parameter. You can also pass [functional methods](javascript:call_link\('abenfunctional_method_glosry.htm'\) "Glossary Entry") and [calculation expressions](javascript:call_link\('abencalculation_expression_glosry.htm'\) "Glossary Entry") as actual parameters to the [input parameters](javascript:call_link\('abeninput_parameter_glosry.htm'\) "Glossary Entry") of [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry").
