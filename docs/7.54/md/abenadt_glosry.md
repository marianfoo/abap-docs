ABAP Development Tools

Abbreviated as ADT. Eclipse-based tools used to edit [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in [ABAP Repository](javascript:call_link\('abenabap_repository_glosry.htm'\) "Glossary Entry") and their components. The ADT currently cover a subset of the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") functions. In the case of some newer repository objects, only the ADT can be used.
