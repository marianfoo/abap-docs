aggregate

Grouping of data from multiple rows of a [database source](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_source_glosry.htm "Glossary Entry"). Can be determined using [aggregate functions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenaggregate_function_glosry.htm "Glossary Entry") in the [SELECT](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapselect.htm) statement.
