AMDP function

[Database function](javascript:call_link\('abendatabase_function_glosry.htm'\) "Glossary Entry") in the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") implemented in an [AMDP function implementation](javascript:call_link\('abenamdp_function_method_glosry.htm'\) "Glossary Entry") of an [](javascript:call_link\('abenamdp_table_function_glosry.htm'\) "Glossary Entry")[AMDP](javascript:call_link\('abenamdp_class_glosry.htm'\) "Glossary Entry") class and replicated from here to the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry"). There are [AMDP table functions](javascript:call_link\('abenamdp_table_function_glosry.htm'\) "Glossary Entry") or [AMDP scalar functions](javascript:call_link\('abenamdp_scalar_function_glosry.htm'\) "Glossary Entry"). See also [AMDP procedure](javascript:call_link\('abenamdp_procedure_glosry.htm'\) "Glossary Entry").
