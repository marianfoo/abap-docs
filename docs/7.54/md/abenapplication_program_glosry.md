application program

[ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") that is part of the business applications running on [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry"). Application programs generally only operate with the data from the current [application client](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenapplication_client_glosry.htm "Glossary Entry") stored in [application tables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenapplication_table_glosry.htm "Glossary Entry").
