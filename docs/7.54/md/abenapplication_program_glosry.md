application program

[ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that is part of the business applications running on [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"). Application programs generally only operate with the data from the current [application client](javascript:call_link\('abenapplication_client_glosry.htm'\) "Glossary Entry") stored in [application tables](javascript:call_link\('abenapplication_table_glosry.htm'\) "Glossary Entry").
