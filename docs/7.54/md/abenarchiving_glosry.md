archiving

Archiving of [spool lists](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_list_glosry.htm "Glossary Entry") using the [ArchiveLink](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_archivelink_glosry.htm "Glossary Entry") tool. Any spool lists stored in the [SAP spool system](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_spool_system_glosry.htm "Glossary Entry") can be archived. When being archived, spool lists from the spool system are saved in exactly the same format used there.
