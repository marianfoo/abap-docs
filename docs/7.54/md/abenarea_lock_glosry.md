area lock

Exclusive, shared, or update lock on an [area instance version](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenarea_instance_version_glosry.htm "Glossary Entry") in the [shared objects memory](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenshared_objects_memory_glosry.htm "Glossary Entry"). Area locks are set by attaching [area handles](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenarea_handle_glosry.htm "Glossary Entry") to area instance versions.
