authorization profile

Combination of multiple [authorizations](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenauthorization_glosry.htm "Glossary Entry"). Multiple authorization profiles can be assigned to a single authorization. Entering authorization profiles in the [user master record](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_master_record_glosry.htm "Glossary Entry") assigns [user names](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_name_glosry.htm "Glossary Entry") to authorizations. Authorization profiles are edited directly using [transaction code](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentransaction_code_glosry.htm "Glossary Entry") SU02 or are generated using the profile generator from the role management tool (transaction PFCG).
