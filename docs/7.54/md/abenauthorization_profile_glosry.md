authorization profile

Combination of multiple [authorizations](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry"). Multiple authorization profiles can be assigned to a single authorization. Entering authorization profiles in the [user master record](javascript:call_link\('abenuser_master_record_glosry.htm'\) "Glossary Entry") assigns [user names](javascript:call_link\('abenuser_name_glosry.htm'\) "Glossary Entry") to authorizations. Authorization profiles are edited directly using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SU02 or are generated using the profile generator from the role management tool (transaction PFCG).
