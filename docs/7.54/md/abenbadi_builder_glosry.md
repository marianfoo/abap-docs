BAdI Builder

[ABAP Workbench](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_workbench_glosry.htm "Glossary Entry") tool that can be used to create and edit [BAdIs](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbadi_glosry.htm "Glossary Entry"). BAdI Builder is the BAdI-specific variant of [Enhancement Builder](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_builder_glosry.htm "Glossary Entry"). Previously, BAdI Builder was used only to edit classic BAdIs. Called using [transactions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE18 (definition) and SE19 (implementation).
