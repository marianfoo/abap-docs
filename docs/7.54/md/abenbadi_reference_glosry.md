BAdI Reference

Reference to a [BAdI object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbadi_object_glosry.htm "Glossary Entry"). BAdI references occur only as contents of [BAdI reference variables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbadi_reference_variable_glosry.htm "Glossary Entry"). Technically, BAdI references are [object references](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_reference_glosry.htm "Glossary Entry") to [instances](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninstance_glosry.htm "Glossary Entry") of [BAdI classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbadi_class_glosry.htm "Glossary Entry").
