Base64

Method of encoding binary data into printable characters. Every byte is converted into a short character string whose characters are obtained from a set of 64 [code page](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencodepage_glosry.htm "Glossary Entry")\-independent [ASCII](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenascii_glosry.htm "Glossary Entry") characters.
