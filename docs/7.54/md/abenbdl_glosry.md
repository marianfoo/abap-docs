Behavior Definition Language

Short: BDL

Declarative language for behavior modelling of a [business object](javascript:call_link\('abenbusiness_object_glosry.htm'\) "Glossary Entry") in the context of the [ABAP RESTful Programming Model](javascript:call_link\('abenr_a_p_glosry.htm'\) "Glossary Entry"). The language is based on data modeling using [CDS views](javascript:call_link\('abencds_f1_define_view.htm'\)). The corresponding source text artifact in the ABAP repository is the [behavior definition](javascript:call_link\('abenbehavior_definition_glosry.htm'\) "Glossary Entry").
