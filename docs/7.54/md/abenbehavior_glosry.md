Business Object Behavior

Short: Behavior

An artifact in the [ABAP RESTful Programming Model](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrestful_abap_programming.htm) that combines ABAP Core Data Services ([ABAP CDS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds.htm)), Behavior Definition Language ([BDL](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_bdl.htm)) and ABAP. It contains the [behavior definition](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_behavior_definitions.htm) and the [behavior implementation](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_behavior_implementations.htm).
