Business Object Behavior

Short: Behavior

An artifact in the [ABAP RESTful Programming Model](javascript:call_link\('abenrestful_abap_programming.htm'\)) that combines ABAP Core Data Services ([ABAP CDS](javascript:call_link\('abencds.htm'\))), Behavior Definition Language ([BDL](javascript:call_link\('abenabap_bdl.htm'\))) and ABAP. It contains the [behavior definition](javascript:call_link\('abenabap_behavior_definitions.htm'\)) and the [behavior implementation](javascript:call_link\('abenabap_behavior_implementations.htm'\)).
