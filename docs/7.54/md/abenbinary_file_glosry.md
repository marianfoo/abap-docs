binary file

[File](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfile_glosry.htm "Glossary Entry") on the [host computer](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhost_computer_glosry.htm "Glossary Entry") of an [AS Instance](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenapplication_server_glosry.htm "Glossary Entry") opened using the addition [BINARY](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapopen_dataset_mode.htm) of the statement [OPEN DATASET](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapopen_dataset_mode.htm). The content of a binary file is read or written without special handling.
