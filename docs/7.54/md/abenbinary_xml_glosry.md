Binary XML

SAP-specific format for storing [XML](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenxml_glosry.htm "Glossary Entry") data in binary form. In Binary XML, redundant information is suppressed and [UTF-8](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenutf8_glosry.htm "Glossary Entry") coding is used, which produces performance benefits when storing and transporting data. In [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry"), Binary XML is one of the supported [formats](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_sxml_lib_formats.htm) in the [sXML Library](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_sxml_lib.htm).
