Binary XML

SAP-specific format for storing [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data in binary form. In Binary XML, redundant information is suppressed and [UTF-8](javascript:call_link\('abenutf8_glosry.htm'\) "Glossary Entry") coding is used, which produces performance benefits when storing and transporting data. In [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"), Binary XML is one of the supported [formats](javascript:call_link\('abenabap_sxml_lib_formats.htm'\)) in the [sXML Library](javascript:call_link\('abenabap_sxml_lib.htm'\)).
