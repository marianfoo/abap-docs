binary floating point arithmetic

Calculation with [binary floating point numbers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbinfloat_glosry.htm "Glossary Entry"). Can produce rounding errors since floating point numbers are not represented internally using the decimal system.
