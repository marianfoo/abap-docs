Byte Chain

Sequence of bytes. Also byte sequence. Content of every data object. Direct access to the bytes of a byte chain is possible only in [byte-like data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbyte_like_data_object_glosry.htm "Glossary Entry"), however, or by using appropriately typed [field symbols](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfield_symbol_glosry.htm "Glossary Entry").
