Byte-Like Data Object

[Data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") with a [byte-like data type](javascript:call_link\('abenbyte_like_data_typ_glosry.htm'\) "Glossary Entry") whose content is interpreted as non-encoded.
