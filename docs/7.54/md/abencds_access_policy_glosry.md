CDS access policy

A container for other definitions defined using [DEFINE ACCESSPOLICY](javascript:call_link\('abencds_f1_define_accesspolicy.htm'\)) in the [ABAP Core Data Services](javascript:call_link\('abenabap_core_data_services_glosry.htm'\) "Glossary Entry") [DCL](javascript:call_link\('abendcl_glosry.htm'\) "Glossary Entry"). Currently, a single CDS access policy is used as a container for the definition of precisely one [PFCG mapping](javascript:call_link\('abencds_pfcg_mapping_glosry.htm'\) "Glossary Entry").
