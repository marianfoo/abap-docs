CDS access policy

A container for other definitions defined using [DEFINE ACCESSPOLICY](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds_f1_define_accesspolicy.htm) in the [ABAP Core Data Services](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_core_data_services_glosry.htm "Glossary Entry") [DCL](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendcl_glosry.htm "Glossary Entry"). Currently, a single CDS access policy is used as a container for the definition of precisely one [PFCG mapping](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds_pfcg_mapping_glosry.htm "Glossary Entry").
