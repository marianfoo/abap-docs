CDS annotation syntax

Syntax for specifying a [CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") in [CDS source code](javascript:call_link\('abencds_source_code_glosry.htm'\) "Glossary Entry"). The CDS annotation syntax is fixed and rules for its use are added to it using the definition of a CDS annotation in a [CDS annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry").
