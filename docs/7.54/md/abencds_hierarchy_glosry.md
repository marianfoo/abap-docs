CDS hierarchy

[CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") defined using [DEFINE HIERARCHY](javascript:call_link\('abencds_f1_define_hierarchy.htm'\)) in a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). In [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") read statements, a CDS hierarchy can be accessed as a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry"). Here, additional [hierarchy columns](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") containing the [hierarchy attributes](javascript:call_link\('abenhierarchy_attribute_glosry.htm'\) "Glossary Entry") can be read.
