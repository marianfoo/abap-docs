CDS view extension

Transportable extension of a [CDS view](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds_view_glosry.htm "Glossary Entry") defined in [DDL source code](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenddl_source_code_glosry.htm "Glossary Entry") in [CDS DDL](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds_ddl_glosry.htm "Glossary Entry") in [ABAP Core Data Services](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_core_data_services_glosry.htm "Glossary Entry"). It can be used to add fields from the entities used by the view (and other clauses) to the view without making modifications.
