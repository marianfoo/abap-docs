Child Entity

A node in the structure of a [business object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbusiness_object_glosry.htm "Glossary Entry") that is connected directly with another node when traversing away from the [root entity](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenroot_entity_glosry.htm "Glossary Entry").

In [ABAP CDS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds.htm), entities are connected using [compositions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencomposition_glosry.htm "Glossary Entry"). A child entity is a CDS entity, which is the target of a composition.
