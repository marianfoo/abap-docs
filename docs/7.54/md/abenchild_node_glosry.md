child node

[Hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") that is connected to a [parent node](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry") using a [parent-child relationship](javascript:call_link\('abenpcr_glosry.htm'\) "Glossary Entry"). A child node can have multiple parent nodes. A child node is a [descendant node](javascript:call_link\('abendescendant_node_glosry.htm'\) "Glossary Entry") of its parent nodes (and of their parent nodes and so on). A hierarchy in which every child node has only one parent node is a tree-like hierarchy. Here, every [root node](javascript:call_link\('abenroot_node_glosry.htm'\) "Glossary Entry") can be the origin of a tree.
