System Class for the File Interface

The static methods of the class CL\_ABAP\_FILE\_UTILITIES do the following:

-   Provide information about files on the [host computer](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhost_computer_glosry.htm "Glossary Entry") of the current AS Instance

-   Create a file with a UTF 8 byte order mark
