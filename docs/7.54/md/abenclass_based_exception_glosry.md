class-based exception

[Handleable exception](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentreatable_exception_glosry.htm "Glossary Entry") represented by an [exception object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenexception_object_glosry.htm "Glossary Entry") of an [exception class](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenexception_class_glosry.htm "Glossary Entry"). Class-based exceptions replace the obsolete [catchable runtime errors](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencatchable_runtime_error_glosry.htm "Glossary Entry") and non-class-based exceptions. A class-based exception can either terminate the current context or be a [resumable exception](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenresumable_exception_glosry.htm "Glossary Entry").
