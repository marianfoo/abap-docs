class pool

[ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that contains the definition of exactly one [global class](javascript:call_link\('abenglobal_class_glosry.htm'\) "Glossary Entry") and is loaded when the class is used. Does not support its own [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") and the only possible [processing blocks](javascript:call_link\('abenprocessing_block_glosry.htm'\) "Glossary Entry") are [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry"). In addition to the global class, a class pool can contain further [local classes](javascript:call_link\('abenlocal_class_glosry.htm'\) "Glossary Entry").
