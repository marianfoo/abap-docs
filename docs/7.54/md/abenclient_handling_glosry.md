client handling

The handling of the application data in a [client](javascript:call_link\('abenclient_glosry.htm'\) "Glossary Entry"). When accessing [client-specific](javascript:call_link\('abenclient_dependence_glosry.htm'\) "Glossary Entry") [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") or [views](javascript:call_link\('abenview_glosry.htm'\) "Glossary Entry") with implicit client handling, [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") statements use implicit client handling. This accesses the current client by default and additions can be specified to switch it to other clients.
