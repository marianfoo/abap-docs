client-server system

Software system consisting of multiple software layers. These are usually the [presentation layer](javascript:call_link\('abenpresentation_layer_glosry.htm'\) "Glossary Entry"), [application layer](javascript:call_link\('abenapplication_layer_glosry.htm'\) "Glossary Entry"), and [database layer](javascript:call_link\('abendatabase_layer_glosry.htm'\) "Glossary Entry"). [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") represents the application layer of a client-server in which [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") are executed.
