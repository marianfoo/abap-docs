cluster

Grouping of different data in a shared repository Currently only indicates [data clusters](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_cluster_glosry.htm "Glossary Entry") in various memory media for [data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry"). Before Release 7.53 also in use as [table clusters](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentable_cluster_glosry.htm "Glossary Entry") for [database tables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_table_glosry.htm "Glossary Entry") ([cluster tables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencluster_table_glosry.htm "Glossary Entry")).
