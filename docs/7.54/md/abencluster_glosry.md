cluster

Grouping of different data in a shared repository Currently only indicates [data clusters](javascript:call_link\('abendata_cluster_glosry.htm'\) "Glossary Entry") in various memory media for [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). Before Release 7.53 also in use as [table clusters](javascript:call_link\('abentable_cluster_glosry.htm'\) "Glossary Entry") for [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") ([cluster tables](javascript:call_link\('abencluster_table_glosry.htm'\) "Glossary Entry")).
