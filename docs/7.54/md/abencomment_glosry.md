comment

Explanation inserted into [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") that is ignored by [ABAP Compiler](javascript:call_link\('abenabap_compiler_glosry.htm'\) "Glossary Entry") when the program is generated. Either a [comment line](javascript:call_link\('abencomment_line_glosry.htm'\) "Glossary Entry") or an [line end comment](javascript:call_link\('abenendline_comment_glosry.htm'\) "Glossary Entry"). See also [pseudo comment](javascript:call_link\('abenpseudo_comment_glosry.htm'\) "Glossary Entry").
