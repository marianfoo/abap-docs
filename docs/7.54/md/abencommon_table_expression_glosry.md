common table expression

Common table expressions (CTE for short) are defined in the [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") statement [WITH](javascript:call_link\('abapwith.htm'\)) using [subqueries](javascript:call_link\('abensubquery_glosry.htm'\) "Glossary Entry"). The [main query](javascript:call_link\('abenmainquery_glosry.htm'\) "Glossary Entry") of the WITH statement accesses the common table expressions.
