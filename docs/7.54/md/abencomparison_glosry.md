comparison

[Relational expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrelational_expression_glosry.htm "Glossary Entry") that compares one or more [operands](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenoperand_glosry.htm "Glossary Entry"). Implemented in ABAP using [comparison expressions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencomparison_expression_glosry.htm "Glossary Entry").
