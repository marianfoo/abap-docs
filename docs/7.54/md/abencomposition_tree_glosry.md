Composition Tree

Hierarchy of nodes in the structure of a [business object](javascript:call_link\('abenbusiness_object_glosry.htm'\) "Glossary Entry") where the nodes are linked by the [composition relationship](javascript:call_link\('abencomposition_glosry.htm'\) "Glossary Entry").

Each node in a composition hierarchy has elements that are modeled in the [ABAP RESTful Programming Model](javascript:call_link\('abenr_a_p_glosry.htm'\) "Glossary Entry") using CDS entities, with the [root entity](javascript:call_link\('abenroot_entity_glosry.htm'\) "Glossary Entry") as the top node in the business object’s structure.
