compound interface

Interface that includes at least one [component interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencomponent_interface_glosry.htm "Glossary Entry") when declared using [INTERFACES](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapinterfaces_ifac.htm). Also known as a nested interface.
