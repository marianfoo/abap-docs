compound interface

Interface that includes at least one [component interface](javascript:call_link\('abencomponent_interface_glosry.htm'\) "Glossary Entry") when declared using [INTERFACES](javascript:call_link\('abapinterfaces_ifac.htm'\)). Also known as a nested interface.
