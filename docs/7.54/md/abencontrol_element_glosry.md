control element

Element in the [user interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_interface_glosry.htm "Glossary Entry") for which a [user action](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_action_glosry.htm "Glossary Entry") can be executed. Most [SAP GUI](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_gui_glosry.htm "Glossary Entry") UI elements can be associated with a [function code](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfunction_code_glosry.htm "Glossary Entry").
