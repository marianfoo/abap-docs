control structure

Method of structuring a [processing block](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprocessing_block_glosry.htm "Glossary Entry") in [statement blocks](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatement_block_glosry.htm "Glossary Entry"). Possible control structures are [sequences](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensequence_glosry.htm "Glossary Entry"), [branches](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbranch_1_glosry.htm "Glossary Entry"), and [loops](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenloop_glosry.htm "Glossary Entry") as well as special control structures for exception handling. Control structures can be nested, so that closed statement blocks can contain other control structures.
