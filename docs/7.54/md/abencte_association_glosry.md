CTE association

[Association](javascript:call_link\('abenassociation_glosry.htm'\) "Glossary Entry") defined for a [common table expression](javascript:call_link\('abencommon_table_expression_glosry.htm'\) "Glossary Entry") ([CTE](javascript:call_link\('abencte_glosry.htm'\) "Glossary Entry")) and used in the [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") of its [WITH](javascript:call_link\('abapwith.htm'\)) statement in [SQL path expressions](javascript:call_link\('abencds_path_expression_glosry.htm'\) "Glossary Entry"). In the definition of the common table expression, a CTE association is created using the addition [WITH ASSOCIATIONS JOIN](javascript:call_link\('abapwith_associations_defining.htm'\)).
