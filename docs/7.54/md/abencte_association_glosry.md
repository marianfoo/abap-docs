CTE association

[Association](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenassociation_glosry.htm "Glossary Entry") defined for a [common table expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencommon_table_expression_glosry.htm "Glossary Entry") ([CTE](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencte_glosry.htm "Glossary Entry")) and used in the [queries](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenquery_glosry.htm "Glossary Entry") of its [WITH](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapwith.htm) statement in [SQL path expressions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencds_path_expression_glosry.htm "Glossary Entry"). In the definition of the common table expression, a CTE association is created using the addition [WITH ASSOCIATIONS JOIN](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapwith_associations_defining.htm).
