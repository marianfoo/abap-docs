CTE hierarchy

[Common table expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencommon_table_expression_glosry.htm "Glossary Entry") ([CTE](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencte_glosry.htm "Glossary Entry")) that can be used as a [hierarchy](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhierarchy_glosry.htm "Glossary Entry") in the subsequent [queries](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenquery_glosry.htm "Glossary Entry") of its [WITH](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapwith.htm) statement. In the definition of the common table expression, a CTE hierarchy is created using the addition [WITH HIERARCHY](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapwith_hierarchy.htm).
