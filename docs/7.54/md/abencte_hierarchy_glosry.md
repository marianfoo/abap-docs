CTE hierarchy

[Common table expression](javascript:call_link\('abencommon_table_expression_glosry.htm'\) "Glossary Entry") ([CTE](javascript:call_link\('abencte_glosry.htm'\) "Glossary Entry")) that can be used as a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry") in the subsequent [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") of its [WITH](javascript:call_link\('abapwith.htm'\)) statement. In the definition of the common table expression, a CTE hierarchy is created using the addition [WITH HIERARCHY](javascript:call_link\('abapwith_hierarchy.htm'\)).
