Change and Transport System

Abbreviated to CTS. The Change and Transport System encompasses all [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") tools used to manage and transport [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") and other objects between [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") systems. The interface between a repository object and CTS is defined using the [package](javascript:call_link\('abenpackage_glosry.htm'\) "Glossary Entry") of the object.
