cursor

A cursor determines a position. The term "cursor" means the position of a pointer in a display window. A [list cursor](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlist_cursor_glosry.htm "Glossary Entry") determines the output position in a [list](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlist_glosry.htm "Glossary Entry") and a [database cursor](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_cursor_glosry.htm "Glossary Entry") reads data from [database tables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_table_glosry.htm "Glossary Entry"). In this sense, the [file pointer](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfile_pointer_glosry.htm "Glossary Entry") can be viewed as a cursor as well.
