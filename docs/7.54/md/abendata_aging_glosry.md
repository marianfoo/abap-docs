data aging

[SAP HANA database](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhana_database_glosry.htm "Glossary Entry") concept in which current data is distinguished from old data using entries in a [temperature column](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentemperature_column_glosry.htm "Glossary Entry"). Only current data exists in the memory.
