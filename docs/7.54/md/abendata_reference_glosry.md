data reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_glosry.htm "Glossary Entry") that points to a [data object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry") or a part of a data object. A non-initial data reference can be a [heap reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenheap_reference_glosry.htm "Glossary Entry") or a [stack reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstack_reference_glosry.htm "Glossary Entry").
