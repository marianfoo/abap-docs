database procedure

General synonym for [stored procedure](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstored_procedure_glosry.htm "Glossary Entry"). In [SAP HANA databases](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhana_database_glosry.htm "Glossary Entry"), name for an [SQL Script procedure](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensql_script_procedure_glosry.htm "Glossary Entry") programmed in [SQL Script](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensql_script_glosry.htm "Glossary Entry") or an [L procedure](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenl_procedure_glosry.htm "Glossary Entry") programmed in [L](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenllang_glosry.htm "Glossary Entry") . Database procedures can be managed as [AMDP procedures](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenamdp_procedure_glosry.htm "Glossary Entry") in the [ABAP Managed Database Procedures](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_managed_db_proc_glosry.htm "Glossary Entry") (AMDP). See also [database function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_function_glosry.htm "Glossary Entry").
