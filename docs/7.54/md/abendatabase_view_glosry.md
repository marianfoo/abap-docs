database view

General [classic view](javascript:call_link\('abenclassical_view_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") for multiple [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry"). When created as an [ABAP-managed database object](javascript:call_link\('abenabap_managed_db_object_glosry.htm'\) "Glossary Entry"), a database view is an [SQL view](javascript:call_link\('abensql_view_glosry.htm'\) "Glossary Entry") in the [ABAP database schema](javascript:call_link\('abenabap_db_schema_glosry.htm'\) "Glossary Entry") of the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry").
