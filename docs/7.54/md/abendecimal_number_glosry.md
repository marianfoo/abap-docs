decimal number

Number in the decimal numbering system. It consists of [places](javascript:call_link\('abenplace_glosry.htm'\) "Glossary Entry"), which can contain the decimal digits 0-9, and can also include a [decimal separator](javascript:call_link\('abendecimal_point_separat_glosry.htm'\) "Glossary Entry") and a sign.
