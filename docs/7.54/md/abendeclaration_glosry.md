declaration

Defines the properties of units local to the program. It is possible to declare [data types](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_type_glosry.htm "Glossary Entry"), [data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry"), [classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_glosry.htm "Glossary Entry"), [interfaces](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_glosry.htm "Glossary Entry"), and [field symbols](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfield_symbol_glosry.htm "Glossary Entry"). Declarations are made using either [declaration statements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendeclaration_statement_glosry.htm "Glossary Entry") or as [inline declarations](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninline_declaration_glosry.htm "Glossary Entry") using [declaration expressions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendeclaration_expression_glosry.htm "Glossary Entry"). The declared unit is visible statically as soon as it is declared and is valid across the entire [context](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencontext_2_glosry.htm "Glossary Entry").
