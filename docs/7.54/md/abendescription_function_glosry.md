description function

[Built-in function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredefined_function_glosry.htm "Glossary Entry") that gets properties of [data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry"). There are [description functions for character-like](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendescriptive_functions.htm), [table-like](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendescriptive_functions_table.htm), and [byte-like](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendescriptive_functions_binary.htm) arguments.
