dialog module object

Obsolete [repository object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrepository_object_glosry.htm "Glossary Entry"). Predecessor of [function modules](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfunction_module_glosry.htm "Glossary Entry"). Like a [transaction code](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentransaction_code_glosry.htm "Glossary Entry"), is associated with an [initial dynpro](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninitial_dynpro_glosry.htm "Glossary Entry") of a [dynpro sequence](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_sequence_glosry.htm "Glossary Entry") of any [ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry"). Can contain interface parameters that have to be declared in the ABAP program as global data. Different dialog modules can be assigned to one ABAP program. Called using [CALL DIALOG](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapcall_dialog.htm). Dialog modules are edited using the transaction code SE35.
