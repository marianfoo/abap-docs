double-byte code

A character in double-byte representation. Used before the introduction of [Unicode](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunicode_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") to display Asian characters.
