elementary data type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of fixed or variable length that is not [structured](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry"), not a [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry"), and not a [reference type](javascript:call_link\('abenreference_type_glosry.htm'\) "Glossary Entry"). The [built-in ABAP types](javascript:call_link\('abenpredefined_abap_type_glosry.htm'\) "Glossary Entry") are a particular kind of elementary data type.
