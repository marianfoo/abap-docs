enhancement point

Position in a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that can be enhanced in accordance with the [enhancement concept](javascript:call_link\('abenenhancement_concept_glosry.htm'\) "Glossary Entry"). Enhancement points can be defined explicitly or be available implicitly. Explicitly defined enhancement points are realized using [enhancement spot element definitions](javascript:call_link\('abenenhancement_spot_def_glosry.htm'\) "Glossary Entry") and [enhancement spot invocations](javascript:call_link\('abenenhancement_spot_invoc_glosry.htm'\) "Glossary Entry"), and are administered using [enhancement spots](javascript:call_link\('abenenhancement_spot_glosry.htm'\) "Glossary Entry").
