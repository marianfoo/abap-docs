enhancement spot element definition

Part of the definition of an explicit [enhancement point](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_point_glosry.htm "Glossary Entry"). An enhancement spot element definition must be assigned to at least one [enhancement spot](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_spot_glosry.htm "Glossary Entry").
