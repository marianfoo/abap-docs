enhancement spot element invocation

Part of the definition of an explicit [enhancement point](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_point_glosry.htm "Glossary Entry"). Position where an [enhancement spot element definition](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_spot_def_glosry.htm "Glossary Entry") is invoked. An enhancement spot element definition can be called in multiple places, as well as in [source code enhancements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensource_code_enhancement_glosry.htm "Glossary Entry").
