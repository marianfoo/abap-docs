environment code page

The active [system code page](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry") of an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry"). The current release only supports [unicode systems](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry") and the environment codepage is always [UTF-16](javascript:call_link\('abenutf16_glosry.htm'\) "Glossary Entry").
