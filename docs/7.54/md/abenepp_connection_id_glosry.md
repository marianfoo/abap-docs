EPP connection ID

Component of an [Extended Passport (EPP)](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenepp_glosry.htm "Glossary Entry"). ID of a connection produced by calls from [ABAP sessions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenmain_session_glosry.htm "Glossary Entry") or external connections in ABAP sessions.
