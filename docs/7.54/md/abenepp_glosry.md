Extended Passport

Abbreviated as EPP. Internal data structure that exists for every [ABAP session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenmain_session_glosry.htm "Glossary Entry"). It contains information about the session and its external connections.
