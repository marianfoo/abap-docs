EVA

Possible process flow in program execution. The EVA principle dictates that a program is executed in the order input, processing, and then output of the results. See also [reporting](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreporting_glosry.htm "Glossary Entry").
