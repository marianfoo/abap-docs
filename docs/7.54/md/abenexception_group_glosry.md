exception group

Grouping of multiple [catchable runtime errors](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencatchable_runtime_error_glosry.htm "Glossary Entry") in an organizational unit. The catchable runtime errors of an exception group, together with the obsolete statement [CATCH - ENDCATCH](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapcatch_sys.htm), can be handled by specifying the group name.
