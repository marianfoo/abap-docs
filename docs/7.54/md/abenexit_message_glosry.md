exit message

[Message](javascript:call_link\('abenmessage_glosry.htm'\) "Glossary Entry") with the [message type](javascript:call_link\('abenmessage_type_glosry.htm'\) "Glossary Entry") X. Exit messages create a [database rollback](javascript:call_link\('abendatabase_rollback_glosry.htm'\) "Glossary Entry") and cancel the running program with the [runtime error](javascript:call_link\('abenruntime_error_glosry.htm'\) "Glossary Entry") MESSAGE\_TYPE\_X. The short text of the message is in the [short dump](javascript:call_link\('abenshort_dump_glosry.htm'\) "Glossary Entry").
