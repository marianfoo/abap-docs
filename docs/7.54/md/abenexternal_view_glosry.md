external view

A special [view](javascript:call_link\('abenview_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that defines an [SAP HANA view](javascript:call_link\('abenhana_view_glosry.htm'\) "Glossary Entry") from [SAP HANA Repository](javascript:call_link\('abensap_hana_repository_glosry.htm'\) "Glossary Entry") in ABAP programs. An external view can be used like a [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry") and specified in reading [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") statements as a source.
