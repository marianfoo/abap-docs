field symbol

Symbolic name for a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") to which actual memory areas can be assigned at program runtime. A field symbol can be used as a placeholder for a data object at the operand position. A field symbol is [typed](javascript:call_link\('abentyping_glosry.htm'\) "Glossary Entry") either generically or fully. Field symbols are declared using the statement [FIELD-SYMBOLS](javascript:call_link\('abapfield-symbols.htm'\)) or the [declaration operator](javascript:call_link\('abendeclaration_operator_glosry.htm'\) "Glossary Entry") [FIELD-SYMBOL](javascript:call_link\('abenfield-symbol_inline.htm'\)); memory areas are assigned to them using [ASSIGN](javascript:call_link\('abapassign.htm'\)).
