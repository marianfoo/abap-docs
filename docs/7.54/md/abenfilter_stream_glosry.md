filter stream

[Stream](javascript:call_link\('abenstream_glosry.htm'\) "Glossary Entry") that is not connected directly to a [data source](javascript:call_link\('abendata_source_glosry.htm'\) "Glossary Entry") or a [data sink](javascript:call_link\('abendata_sink_glosry.htm'\) "Glossary Entry") but to another stream instead. A filter stream is either a [reader stream](javascript:call_link\('abenreader_stream_glosry.htm'\) "Glossary Entry"), which reads and filters the data of another reader stream, or it is a [writer stream](javascript:call_link\('abenwriter_stream_glosry.htm'\) "Glossary Entry") which writes filtered data to another writer stream.
