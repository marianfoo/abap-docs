floating point arithmetic

Calculation using [floating point numbers](javascript:call_link\('abenfloating_point_number_2_glosry.htm'\) "Glossary Entry"). Is performed directly on the [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") of the current [AS Instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") Can be either [binary floating point arithmetic](javascript:call_link\('abenbinfloat_arith_glosry.htm'\) "Glossary Entry") or [decimal floating point arithmetic](javascript:call_link\('abendecfloat_arith_glosry.htm'\) "Glossary Entry").
