SAP HANA

[SAP HANA](http://help.sap.com/hana) platform for SAP In-Memory Computing based on the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). If a SAP HANA database is used as the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry"), ABAP programs can use [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") to access SAP HANA directly. As before, [secondary connections](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") can also be used to access SAP HANA from ABAP.
