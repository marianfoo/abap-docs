SAP HANA view

An SAP HANA view is used to select data, analyze data, or perform calculations with data from the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). There are various different types of view: attribute views, analytic views and calculation views. SAP HANA views can be defined as objects in [SAP HANA XS](javascript:call_link\('abenxs_glosry.htm'\) "Glossary Entry"). [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") provide special [methods for accessing](javascript:call_link\('abensap_hana_access.htm'\)) these views.
