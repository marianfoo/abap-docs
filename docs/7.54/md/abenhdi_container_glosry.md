HDI container

An HDI container is a [database schema](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_schema_glosry.htm "Glossary Entry") of the [SAP HANA database](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhana_database_glosry.htm "Glossary Entry") for [HDI objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhdi_object_glosry.htm "Glossary Entry"). Special [ABAP Managed HDI Containers (AMHC)](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenamhc_glosry.htm "Glossary Entry") can be created for HDI objects accessed in ABAP.
