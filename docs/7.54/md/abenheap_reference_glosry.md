heap reference

[Reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_glosry.htm "Glossary Entry") to an [object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_glosry.htm "Glossary Entry") or part of an object in the [heap](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenheap_glosry.htm "Glossary Entry"). A heap reference keeps an object alive. See also [stack reference](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstack_reference_glosry.htm "Glossary Entry").
