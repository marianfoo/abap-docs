hide area

Area in the [list buffer](javascript:call_link\('abenlist_buffer_glosry.htm'\) "Glossary Entry"), in which [global](javascript:call_link\('abenglobal_data_glosry.htm'\) "Glossary Entry") [variables](javascript:call_link\('abenvariable_glosry.htm'\) "Glossary Entry") can be stored for every row of a [screen list](javascript:call_link\('abenscreen_list_glosry.htm'\) "Glossary Entry") using the statement [HIDE](javascript:call_link\('abaphide.htm'\)).
