hierarchy function

Special [table function](javascript:call_link\('abentable_function_glosry.htm'\) "Glossary Entry") in [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") used to edit [hierarchies](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry"). A [hierarchy generator](javascript:call_link\('abenhierarchy_generator_glosry.htm'\) "Glossary Entry") is used to create hierarchies and [hierarchy navigators](javascript:call_link\('abenhierarchy_navigator_glosry.htm'\) "Glossary Entry") are used to access them.
