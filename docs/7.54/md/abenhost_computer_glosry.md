host computer

Computer on which the current [AS Instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") is instantiated. The name of the current host computer can be read from the [system field](javascript:call_link\('abensystem_field_glosry.htm'\) "Glossary Entry") sy-host.
