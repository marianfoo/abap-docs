ICF processing

Processing of a HTTP/HTTPS query or SMTP query using [ICF](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenicf_glosry.htm "Glossary Entry") in an appropriate handler class and in a separate [ICF session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenicf_session_glosry.htm "Glossary Entry").
