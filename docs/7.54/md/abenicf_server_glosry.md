ICF server

Instance of a system that provides services called using [ICF](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenicf_glosry.htm "Glossary Entry"). An ICF server can be an HTTP/HTTPS server or an SMTP server.
