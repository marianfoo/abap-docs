include program

[ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") that is not generated as a standalone program, but which is integrated into another ABAP program using the statement [INCLUDE](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapinclude_prog.htm).
