initial dynpro

First [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") in a [dynpro sequence](javascript:call_link\('abendynpro_sequence_glosry.htm'\) "Glossary Entry"). In a [dialog transaction](javascript:call_link\('abendialog_transaction_glosry.htm'\) "Glossary Entry"), the dynpro that is associated with the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry"). For dynpro sequences called using CALL SCREEN, the initial dynpro is the dynpro specified in this command.
