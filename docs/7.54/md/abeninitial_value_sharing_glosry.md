initial value sharing

The state of a [boxed component](javascript:call_link\('abenboxed_component_glosry.htm'\) "Glossary Entry") as long as only reads are performed on it. The [initial value](javascript:call_link\('abeninitial_value_glosry.htm'\) "Glossary Entry") of all type-friendly boxed components is only saved once on each [AS Instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") in [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry").
