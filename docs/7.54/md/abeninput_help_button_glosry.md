input help button

Single-character [pushbutton](javascript:call_link\('abenpushbutton_glosry.htm'\) "Glossary Entry") with an [icon](javascript:call_link\('abenicon_glosry.htm'\) "Glossary Entry") that is displayed on the left of an [input field](javascript:call_link\('abeninput_output_field_glosry.htm'\) "Glossary Entry") on a [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") if an [input help](javascript:call_link\('abeninput_help_glosry.htm'\) "Glossary Entry") is defined for that field. Selecting the pushbutton has the same effect as pressing F4 for the field.
