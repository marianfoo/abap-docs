instantiation

Creation of an [instance](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninstance_glosry.htm "Glossary Entry"). [Data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry") as instances of [data types](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_type_glosry.htm "Glossary Entry") are created implicitly when a [context](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencontext_2_glosry.htm "Glossary Entry") is loaded, or explicitly using [CREATE DATA](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapcreate_data.htm) or [NEW](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenconstructor_expression_new.htm). [Objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_glosry.htm "Glossary Entry") as instances of [classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_glosry.htm "Glossary Entry") are only created explicitly using [CREATE OBJECT](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapcreate_object.htm) or [NEW](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenconstructor_expression_new.htm).
