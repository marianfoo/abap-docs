interface component selector

The ~ character. A [component](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry") [comp](javascript:call_link\('abenclass_ifac_components.htm'\)) of an [interface](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry") intf can be addressed in an implementing [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") or a [compound interface](javascript:call_link\('abencompound_interface_glosry.htm'\) "Glossary Entry") using intf~comp.
