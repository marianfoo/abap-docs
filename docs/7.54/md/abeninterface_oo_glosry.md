interface (OO)

Template for the [public](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpublic_glosry.htm "Glossary Entry") [interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_glosry.htm "Glossary Entry") of [classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_glosry.htm "Glossary Entry"). Defined with [INTERFACE - ENDINTERFACE](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapinterface.htm) as either a [global interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenglobal_interface_glosry.htm "Glossary Entry") in an [interface pool](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_pool_glosry.htm "Glossary Entry") or a [local interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlocal_interface_glosry.htm "Glossary Entry") in a different [ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry"). An interface contains the declaration of [interface components](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_component_glosry.htm "Glossary Entry") but does not contain method implementations. Can be implemented in classes using [INTERFACES](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapinterfaces.htm), which extends the external interface of the class to include interface components.
