interface pool

[ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") that contains the definition of exactly one [global interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenglobal_interface_glosry.htm "Glossary Entry") and is loaded using the interface. Does not support its own [dynpros](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_glosry.htm "Glossary Entry").
