interface reference variable

[Object reference variable](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_refer_variable_glosry.htm "Glossary Entry") of the [static type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatic_type_glosry.htm "Glossary Entry") of an [interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_oo_glosry.htm "Glossary Entry"). An interface reference variable can be used to provide precise access to the visible [interface components](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_component_glosry.htm "Glossary Entry") of an object that exist in the associated interface. This applies to both dynamic and static access.
