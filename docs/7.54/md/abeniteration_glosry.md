iteration

Repeated processing of a [statement block](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatement_block_glosry.htm "Glossary Entry") in a [loop](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenloop_glosry.htm "Glossary Entry") or an [expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenexpression_glosry.htm "Glossary Entry") using an [iteration expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeniteration_expression_glosry.htm "Glossary Entry").
