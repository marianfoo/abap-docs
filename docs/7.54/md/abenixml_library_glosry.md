iXML Library

Global [classes](javascript:call_link\('abenglobal_class_glosry.htm'\) "Glossary Entry") used to handle [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") documents in [DOM](javascript:call_link\('abendom_glosry.htm'\) "Glossary Entry") representation, accessed using [interfaces](javascript:call_link\('abeninterface_oo_glosry.htm'\) "Glossary Entry"). The "integrated" iXML Library provides services such as an XML parser, an XML renderer, and writes and reads on the DOM. The interfaces in iXML Library all begin with the prefix IF\_IXML.
