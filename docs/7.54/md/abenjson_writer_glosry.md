JSON writer

[XML writer](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenxml_writer_glosry.htm "Glossary Entry") that writes [JSON](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenjson_glosry.htm "Glossary Entry") data. A JSON writer expects input for [JSON-XML](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenjson_xml_glosry.htm "Glossary Entry") and creates JSON data from this input.
