JSON-XML

SAP-specific representation of [JSON](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenjson_glosry.htm "Glossary Entry") data in [XML](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenxml_glosry.htm "Glossary Entry") format. [JSON readers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenjson_reader_glosry.htm "Glossary Entry") and [JSON writers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenjson_writer_glosry.htm "Glossary Entry") both use JSON-XML to [parse](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_json_sxml.htm) and [render](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_json_sxml.htm) JSON data.
