list buffer

Memory area for saving [screen lists](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenscreen_list_glosry.htm "Glossary Entry"). Subdivided into [list levels](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlist_level_glosry.htm "Glossary Entry"). For each [dynpro sequence](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_sequence_glosry.htm "Glossary Entry"), the list buffer contains a hierarchy consisting of a [basic list](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbasic_list_glosry.htm "Glossary Entry") and up to 20 [details lists](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendetails_list_glosry.htm "Glossary Entry").
