Obsolete Statements in List Processing

[Classic lists](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_dynpro_list.htm) should no longer be used in live application programs. Use SAP List Viewer (ALV) instead of classic lists. If classic lists are still used for any reason, then the following constructs must be avoided:
