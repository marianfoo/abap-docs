Literals

Alongside the [named data objects](javascript:call_link\('abennamed_data_object_glosry.htm'\) "Glossary Entry") that can be addressed by their names in ABAP programs (this includes the text symbols) and the [anonymous data objects](javascript:call_link\('abenanonymous_data_object_glosry.htm'\) "Glossary Entry") created by the statement [CREATE DATA](javascript:call_link\('abapcreate_data.htm'\)), there are also literals defined in the source code of programs by specifying a string that represents a value. Possible literals are:

-   [Numeric literals](javascript:call_link\('abenuntyped_number_literals.htm'\))

-   [Character literals](javascript:call_link\('abenuntyped_character_literals.htm'\))

The [literal operator](javascript:call_link\('abenliteral_operator.htm'\)) & can be used to associated multiple literals of the same type with each other.
