local data queue

Persistence layer into which data can be placed and read only in the order in which it was logged. The local data queue triggers the no-send scenario in [qRFC](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenqrfc_glosry.htm "Glossary Entry").
