lock function module

Special [function module](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") for setting and removing [SAP locks](javascript:call_link\('abensap_lock_glosry.htm'\) "Glossary Entry"). To create a [lock object](javascript:call_link\('abenlock_object_glosry.htm'\) "Glossary Entry"), a function module with the prefix ENQUEUE\_ is created to set the lock and a function module with the prefix DEQUEUE\_ is created to remove the lock.
