logical function

[Built-in function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredefined_function_glosry.htm "Glossary Entry"). Either a [Boolean function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenboole_function_glosry.htm "Glossary Entry") that evaluates a [logical expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlogical_expression_glosry.htm "Glossary Entry") or a [predicate function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredicate_function_glosry.htm "Glossary Entry") that returns a [logical value](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlogical_value_glosry.htm "Glossary Entry").
