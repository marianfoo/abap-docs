logical schema

A logical schema is a symbolic name for a [database schema](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_schema_glosry.htm "Glossary Entry") in the [SAP HANA database](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhana_database_glosry.htm "Glossary Entry") that can be used in an [AMDP macro](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenamdp_macro_glosry.htm "Glossary Entry"). [Logical database schemas](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlogical_database_schema_glosry.htm "Glossary Entry") define the mapping of the database schema in a mapping table. In [logical HDI containers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlogical_hdi_container_glosry.htm "Glossary Entry"), on the other hand, the mapping is part of the definition of an [ABAP-managed HDI container (AMHC)](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenamhc_glosry.htm "Glossary Entry").
