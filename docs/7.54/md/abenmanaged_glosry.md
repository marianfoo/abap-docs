Managed

An implementation type of the [business object behavior](javascript:call_link\('abenbehavior_glosry.htm'\) "Glossary Entry") in the context of the [ABAP RESTful Programming Model](javascript:call_link\('abenr_a_p_glosry.htm'\) "Glossary Entry").

The implementation type managed is best used when implementing completely new business logic. In this case, a [behavior definition](javascript:call_link\('abenbehavior_definition_glosry.htm'\) "Glossary Entry") is sufficient to obtain a ready-to-run [business object](javascript:call_link\('abenbehavior_implement_glosry.htm'\) "Glossary Entry").
