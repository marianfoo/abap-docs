MDMP system

Abbreviation of multi-display, multi-processing system. An MDMP system contains multiple non- [Unicode](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunicode_glosry.htm "Glossary Entry") [system code pages](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_codepage_glosry.htm "Glossary Entry"). The [code page](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencodepage_glosry.htm "Glossary Entry") of an [internal session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninternal_session_glosry.htm "Glossary Entry") is determined by its current [text environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentext_environment_glosry.htm "Glossary Entry"). MDMP systems are no longer supported from [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry"), Release 7.0.
