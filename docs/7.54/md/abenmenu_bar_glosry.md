menu bar

Component of a [GUI window](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abengui_window_glosry.htm "Glossary Entry"). Contains [menus](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenmenu_glosry.htm "Glossary Entry") for using the program. Part of the [GUI status](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abengui_status_glosry.htm "Glossary Entry") set using [SET PF STATUS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapset_pf-status_dynpro.htm). Created using the [Menu Painter](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenmenu_painter_glosry.htm "Glossary Entry") tool.
