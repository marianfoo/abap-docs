mesh node

Component of a [mesh](javascript:call_link\('abenmesh_glosry.htm'\) "Glossary Entry"). Mesh nodes are always [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") with a structured [row type](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry"). Mesh nodes can have [mesh associations](javascript:call_link\('abenmesh_association_glosry.htm'\) "Glossary Entry") used to address them in [mesh paths](javascript:call_link\('abenmesh_path_glosry.htm'\) "Glossary Entry").
