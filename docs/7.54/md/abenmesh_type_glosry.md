mesh type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of a [mesh](javascript:call_link\('abenmesh_glosry.htm'\) "Glossary Entry"). Special [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry") containing only [table types](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry") with structured [row types](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry") as [components](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry"), associated with each other using [mesh associations](javascript:call_link\('abenmesh_association_glosry.htm'\) "Glossary Entry").
