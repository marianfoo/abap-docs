message type

When the statement [MESSAGE](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapmessage.htm) is used to send a [message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenmessage_glosry.htm "Glossary Entry"), a message type must be specified. This message type determines the representation of the message in [SAP GUI](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_gui_glosry.htm "Glossary Entry") and the remaining program flow. The possible message types are [information message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninformation_message_glosry.htm "Glossary Entry"), [status message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatus_message_glosry.htm "Glossary Entry"), [error message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenerror_message_glosry.htm "Glossary Entry"), [warning](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenwarning_glosry.htm "Glossary Entry"), [termination message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentermination_message_glosry.htm "Glossary Entry"), and [exit message](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenexit_message_glosry.htm "Glossary Entry").
