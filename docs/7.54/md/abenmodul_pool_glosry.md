module pool

[ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") that normally contains [dynpros](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_glosry.htm "Glossary Entry") and [dialog modules](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendialog_module_glosry.htm "Glossary Entry"), and is started using [transaction codes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentransaction_code_glosry.htm "Glossary Entry").
