namespace prefix

Name of a [prefix namespace](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprefix_name_space_glosry.htm "Glossary Entry") with five to ten characters and a forward slash character (/.../) in the first and final places. The names of the [repository objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrepository_object_glosry.htm "Glossary Entry") of a prefix namespace are introduced using the associated namespace prefix.
