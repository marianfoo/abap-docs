name table

nametab for short. Repository for the [runtime objects](javascript:call_link\('abenruntime_object_glosry.htm'\) "Glossary Entry") of [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). The name table is split across two native [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry"): DDNTT for the [nametab header](javascript:call_link\('abenname_tab_header_glosry.htm'\) "Glossary Entry") and DDNTF for the [nametab field descriptions](javascript:call_link\('abenname_tab_fields_glosry.htm'\) "Glossary Entry"). These database tables are not defined in the ABAP Dictionary. The name table is accessed internally in the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") or by using APIs.
