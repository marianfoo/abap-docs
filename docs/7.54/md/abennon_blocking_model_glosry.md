non-blocking mode

Any statements that block the program flow are forbidden in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that run in non-blocking mode. Non-blocking mode is available in different instances for [APC](javascript:call_link\('abenapc_non_blocking_model.htm'\)) and [ADF](javascript:call_link\('abenadf_non_blocking_model.htm'\)).
