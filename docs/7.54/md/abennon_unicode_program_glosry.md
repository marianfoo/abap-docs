Non-Unicode ABAP

Obsolete [language version](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_version_glosry.htm "Glossary Entry") of an [ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") in which the [Unicode checks](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunicode_check_glosry.htm "Glossary Entry") are ignored. A program with this language version cannot be executed in [Unicode systems](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunicode_system_glosry.htm "Glossary Entry"). In the current [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") release, only Unicode systems are supported. Therefore, non-Unicode ABAP can no longer be used.
