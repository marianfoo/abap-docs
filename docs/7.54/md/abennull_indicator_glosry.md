Null Indicator

[Substructure](javascript:call_link\('abensubstructure_glosry.htm'\) "Glossary Entry") of a structured target area of an [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry") [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry"). The components of the substructure indicate which columns of the results set contain the [null value](javascript:call_link\('abennull_value_glosry.htm'\) "Glossary Entry") and which do not.
