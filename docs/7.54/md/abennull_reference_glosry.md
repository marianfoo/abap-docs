null reference

[Initial value](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninitial_value_glosry.htm "Glossary Entry") of a [reference variable](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_variable_glosry.htm "Glossary Entry"). The null reference does not point to an [object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_glosry.htm "Glossary Entry") and can be assigned to any reference variable. [Data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry") accessed using the null reference generally raise a [non-handleable exception](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuntreatable_exception_glosry.htm "Glossary Entry"). Method calls using the null reference produce a [handleable exception](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentreatable_exception_glosry.htm "Glossary Entry").
