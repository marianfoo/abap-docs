null reference

[Initial value](javascript:call_link\('abeninitial_value_glosry.htm'\) "Glossary Entry") of a [reference variable](javascript:call_link\('abenreference_variable_glosry.htm'\) "Glossary Entry"). The null reference does not point to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") and can be assigned to any reference variable. [Data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") accessed using the null reference generally raise a [non-handleable exception](javascript:call_link\('abenuntreatable_exception_glosry.htm'\) "Glossary Entry"). Method calls using the null reference produce a [handleable exception](javascript:call_link\('abentreatable_exception_glosry.htm'\) "Glossary Entry").
