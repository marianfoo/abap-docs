object reference variable

[Reference variable](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_variable_glosry.htm "Glossary Entry") for [object references](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_reference_glosry.htm "Glossary Entry").
