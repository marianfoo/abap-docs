offset

Specified position for a character or byte in a [character-like](javascript:call_link\('abencharlike_data_object_glosry.htm'\) "Glossary Entry") or [byte-like data object](javascript:call_link\('abenbyte_like_data_object_glosry.htm'\) "Glossary Entry") whose counter starts with 0. The value of an offset can also be interpreted as a label for the start of the line, for the gaps between the characters or bytes, and for the end of the line.
