ABAP SQL - Built-In Functions sql\_func

The following [built-in database functions](javascript:call_link\('abenddic_builtin_functions.htm'\)) managed by ABAP Dictionary can either be specified in ABAP SQL as [SQL expressions](javascript:call_link\('abapsql_expr.htm'\)), in all places where these are possible:

-   [SQL functions](javascript:call_link\('abenopen_sql_functions.htm'\))

-   [Special functions](javascript:call_link\('abenopen_sql_special_functions.htm'\))

Note

[Certain SQL functions](javascript:call_link\('abenbuffer_expressions.htm'\)) can be executed in the [table buffer](javascript:call_link\('abentable_buffer_glosry.htm'\) "Glossary Entry"). If any other built-in functions are used, [table buffering](javascript:call_link\('abensap_buffering_glosry.htm'\) "Glossary Entry") is bypassed.
