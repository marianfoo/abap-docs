orphan node

[Hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") that does not have any [parent nodes](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry") and that is not part of the [root node set](javascript:call_link\('abenroot_node_set_glosry.htm'\) "Glossary Entry").
