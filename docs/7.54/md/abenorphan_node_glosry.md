orphan node

[Hierarchy node](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") that does not have any [parent nodes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenparent_node_glosry.htm "Glossary Entry") and that is not part of the [root node set](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenroot_node_set_glosry.htm "Glossary Entry").
