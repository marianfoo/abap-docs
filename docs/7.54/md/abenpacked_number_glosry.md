packed number

[Numeric data type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abennumeric_data_type_glosry.htm "Glossary Entry") for saving [fixed point numbers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfixed_point_number_glosry.htm "Glossary Entry") with a fixed number of [decimal places](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfractional_portion_glosry.htm "Glossary Entry") or a number saved in a similar way. The associated [built-in ABAP type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredefined_abap_type_glosry.htm "Glossary Entry") is [p](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbuiltin_types_numeric.htm). The number of decimal places is a property of the data type and not (as in [floating point numbers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfloating_point_number_2_glosry.htm "Glossary Entry")) a property of the data itself. Calculations with packed numbers are performed using [fixed point arithmetic](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfixed_point_arithmetic_glosry.htm "Glossary Entry"). The internal representation uses the [BCD](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenbcd_glosry.htm "Glossary Entry") format.
