parameter

A component of a [selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry") (defined using [PARAMETERS](javascript:call_link\('abapparameters.htm'\))) displayed as an [input field](javascript:call_link\('abeninput_output_field_glosry.htm'\) "Glossary Entry") and represented internally as an [elementary data object](javascript:call_link\('abenelementary_data_object_glosry.htm'\) "Glossary Entry"). A parameter is a also a component of the [parameter interface](javascript:call_link\('abenparameter_interface_glosry.htm'\) "Glossary Entry") of a [procedure](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry").
