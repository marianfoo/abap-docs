parent node

[Hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") connected to a [child node](javascript:call_link\('abenchild_node_glosry.htm'\) "Glossary Entry") in a [parent-child relationship](javascript:call_link\('abenpcr_glosry.htm'\) "Glossary Entry"). A parent node can have multiple child nodes. A parent node is an [ancestor node](javascript:call_link\('abenancestor_node_glosry.htm'\) "Glossary Entry") of its child nodes (and of their child nodes, and so on).
