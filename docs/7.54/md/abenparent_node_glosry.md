parent node

[Hierarchy node](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") connected to a [child node](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenchild_node_glosry.htm "Glossary Entry") in a [parent-child relationship](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpcr_glosry.htm "Glossary Entry"). A parent node can have multiple child nodes. A parent node is an [ancestor node](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenancestor_node_glosry.htm "Glossary Entry") of its child nodes (and of their child nodes, and so on).
