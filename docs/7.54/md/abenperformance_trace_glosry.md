performance trace

Test tool combining multiple trace functions. These functions enable system behavior to be analyzed in [database reads](javascript:call_link\('abenabap_sql.htm'\)) ([SQL trace](javascript:call_link\('abensql_trace_glosry.htm'\) "Glossary Entry")), when the [table buffer](javascript:call_link\('abentable_buffer_glosry.htm'\) "Glossary Entry") is read, in [lock management](javascript:call_link\('abensap_lock.htm'\)) reads, and external reads using [RFC](javascript:call_link\('abenrfc_glosry.htm'\) "Glossary Entry") and [ICF](javascript:call_link\('abenicf_glosry.htm'\) "Glossary Entry"). Called using [transaction](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") ST05.
