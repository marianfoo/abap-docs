presentation server

Front-end computer belonging to an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") user. The [presentation layer](javascript:call_link\('abenpresentation_layer_glosry.htm'\) "Glossary Entry") on these computers is provided either by installing [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry") or by a web browser.
