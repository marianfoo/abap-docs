spool parameter

Parameters that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenspool_request_glosry.htm "Glossary Entry"). Spool parameters can be augmented by [archiving parameters](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenarchiving_parameter_glosry.htm "Glossary Entry").
