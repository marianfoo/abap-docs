program constructor event

[ABAP runtime environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") [event](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenevent_glosry.htm "Glossary Entry") that occurs once only when an [ABAP program](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") (not a [class pool](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_pool_glosry.htm "Glossary Entry")) is loaded to the [internal session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninternal_session_glosry.htm "Glossary Entry") to call the [program constructor](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprogram_constructor_glosry.htm "Glossary Entry"). The associated [event keyword](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenevent_keyword_glosry.htm "Glossary Entry") is [LOAD-OF-PROGRAM](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapload-of-program.htm).
