program constructor

[Event block](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenevent_block_glosry.htm "Glossary Entry") initiated by the [LOAD-OF-PROGRAM](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapload-of-program.htm) [event keyword](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenevent_keyword_glosry.htm "Glossary Entry") that, with the exception of [type groups](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_group_1_glosry.htm "Glossary Entry"), [class pools](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_pool_glosry.htm "Glossary Entry"), and [interface pools](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_pool_glosry.htm "Glossary Entry"), is executed when a program is loaded into an [internal session](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninternal_session_glosry.htm "Glossary Entry").
