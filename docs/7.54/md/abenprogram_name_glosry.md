program name

Unique identification of an ABAP program in the [repository](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrepository_glosry.htm "Glossary Entry"). A program name must consist of at least one character. It can have a maximum of 30 alphanumeric characters, including underscores.
