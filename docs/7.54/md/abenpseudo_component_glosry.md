pseudo component

table\_line expression that can be specified in reads on [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") instead of a [component](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry"). The entire row of the internal table is then interpreted as a single component. The name table\_line is a reserved name. In ABAP and in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary.htm'\)), no structure components called table\_line can be defined or created.
