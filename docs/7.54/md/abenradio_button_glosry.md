radio button

[Screen element](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenscreen_element_glosry.htm "Glossary Entry") associated with a [dynpro field](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_field_glosry.htm "Glossary Entry") and in which the user can enter the value "X" or " ". Radio buttons are grouped together in [radio button groups](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenradio_button_group_glosry.htm "Glossary Entry").
