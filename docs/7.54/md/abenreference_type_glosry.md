reference type

A reference type describes [reference variables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_variable_glosry.htm "Glossary Entry"), in other words the [data objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_object_glosry.htm "Glossary Entry") that contain [references](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_glosry.htm "Glossary Entry"). See also [static type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatic_type_glosry.htm "Glossary Entry").
