relational database

[Database](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_glosry.htm "Glossary Entry") whose data is managed in accordance with the [relational data model](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrelational_data_model_glosry.htm "Glossary Entry"), namely in [database tables](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_table_glosry.htm "Glossary Entry"). Further objects, such as [database views](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_view_glosry.htm "Glossary Entry") or [stored procedures](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstored_procedure_glosry.htm "Glossary Entry") can be created to access the data. These objects are organized in [database schemas](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_schema_glosry.htm "Glossary Entry"). The objects are generally defined and accessed using [SQL](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensql_glosry.htm "Glossary Entry").
