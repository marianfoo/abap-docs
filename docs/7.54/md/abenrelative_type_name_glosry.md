relative type name

[Type name](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_name_glosry.htm "Glossary Entry") given when defining a [data type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_type_glosry.htm "Glossary Entry"), a [class](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclass_glosry.htm "Glossary Entry"), or an [interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninterface_oo_glosry.htm "Glossary Entry"). The actual [type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_glosry.htm "Glossary Entry") described by a relative type name depends on the [context](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencontext_2_glosry.htm "Glossary Entry") because local definitions obscure global definitions. See also [absolute type name](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabsolute_typename_glosry.htm "Glossary Entry").
