remote-enabled function module

Abbreviated as [RFM](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrfm_glosry.htm "Glossary Entry"). [Function module](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfunction_module_glosry.htm "Glossary Entry") that is flagged as remote-enabled in its properties, meaning it can be called using [remote function call](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenremote_function_call_glosry.htm "Glossary Entry").
