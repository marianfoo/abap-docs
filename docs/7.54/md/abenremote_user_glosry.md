remote user

[User](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry") who is logged on to [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") using [remote function call](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry"). The logon data and the [authorization](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry") can be declared in the [RFC destination](javascript:call_link\('abendestination_glosry.htm'\) "Glossary Entry").
