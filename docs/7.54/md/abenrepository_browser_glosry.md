Repository Browser

Standard display of [Object Navigator](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenobject_navigator_glosry.htm "Glossary Entry") when representing [repository objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrepository_object_glosry.htm "Glossary Entry") sorted according to type.
