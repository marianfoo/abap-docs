RFC server group

Group of multiple [AS Instances](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry")s in a single [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") which are available for the parallel processing of asynchronous [remote function calls](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry") (RFC). RFC server groups are created using [transaction code](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") RZ12.
