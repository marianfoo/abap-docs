root node

[Hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") that does not have any [parent nodes](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry") but is part of the [root node set](javascript:call_link\('abenroot_node_set_glosry.htm'\) "Glossary Entry"). The [hierarchy column](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") [HIERARCHY\_PARENT\_RANK](javascript:call_link\('abenddddl_hierarchy.htm'\)) of a root node contains the value 0 and the hierarchy column [HIERARCHY\_LEVEL](javascript:call_link\('abenddddl_hierarchy.htm'\)) contains the value 1.
