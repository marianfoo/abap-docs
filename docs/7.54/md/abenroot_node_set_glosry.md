root node set

The set of [hierarchy nodes](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") in a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry") that is determined by their start condition. The elements of the root node set are [root nodes](javascript:call_link\('abenroot_node_glosry.htm'\) "Glossary Entry").
