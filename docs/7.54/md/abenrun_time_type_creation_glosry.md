runtime type creation

Abbreviated as RTTC. The creation of [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") at program runtime. Implemented using methods of the [type description classes](javascript:call_link\('abentype_class_glosry.htm'\) "Glossary Entry") of [RTTS](javascript:call_link\('abenrun_time_type_services_glosry.htm'\) "Glossary Entry").
