runtime type creation

Abbreviated as RTTC. The creation of [data types](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_type_glosry.htm "Glossary Entry") at program runtime. Implemented using methods of the [type description classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_class_glosry.htm "Glossary Entry") of [RTTS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrun_time_type_services_glosry.htm "Glossary Entry").
