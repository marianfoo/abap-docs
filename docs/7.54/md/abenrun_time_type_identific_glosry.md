runtime type identification

Abbreviation RTTI. The identification of [data types](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_type_glosry.htm "Glossary Entry") at program runtime. Implemented using description methods in [type description classes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_class_glosry.htm "Glossary Entry"). Together with [RTTC](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrun_time_type_creation_glosry.htm "Glossary Entry"), constitutes [RTTS](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrun_time_type_services_glosry.htm "Glossary Entry").
