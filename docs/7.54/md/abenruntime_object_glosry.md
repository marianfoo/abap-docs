runtime object

Summary of all relevant attributes of a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") defined in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). A runtime object is generated when a Dictionary object is activated and is saved as an entry in the [name table](javascript:call_link\('abenname_table_glosry.htm'\) "Glossary Entry") ([nametab](javascript:call_link\('abenname_tab_glosry.htm'\) "Glossary Entry")). The [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") accesses the runtime object when types from the ABAP Dictionary are used.
