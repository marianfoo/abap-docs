runtime warning

A runtime warning indicates any critical situations, such as high memory use, that arise when executing a program and that could produce a [runtime error](javascript:call_link\('abenruntime_error_glosry.htm'\) "Glossary Entry"). A runtime warning creates a [short dump](javascript:call_link\('abenshort_dump_glosry.htm'\) "Glossary Entry"), but does not cancel the current program.
