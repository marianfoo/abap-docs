SAP spool system

[AS Instance](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenapplication_server_glosry.htm "Glossary Entry") process for managing sequential data streams that are saved as [spool lists](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_list_glosry.htm "Glossary Entry"), printed on a printer, or stored using [ArchiveLink](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_archivelink_glosry.htm "Glossary Entry"). [ABAP programs](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_program_glosry.htm "Glossary Entry") can create [spool requests](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenspool_request_glosry.htm "Glossary Entry") when lists are created by including pages of lists in [spool list](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_list_glosry.htm "Glossary Entry") output.
