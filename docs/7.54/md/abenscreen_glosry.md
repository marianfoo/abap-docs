screen

Component of a [GUI window](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abengui_window_glosry.htm "Glossary Entry"). Contains [SAP GUI](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_gui_glosry.htm "Glossary Entry"). Contains [screen elements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenscreen_element_glosry.htm "Glossary Entry"). For a user, the visible part of a [dynpro](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_glosry.htm "Glossary Entry"). In the case of general dynpros, the screen is created using [Layout Editor](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlayout_editor_glosry.htm "Glossary Entry") in [Screen Painter](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenscreen_painter_glosry.htm "Glossary Entry"). In the case of special dynpros ([selection screens](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenselection_screen_glosry.htm "Glossary Entry") and [list dynpros](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenlist_dynpro_glosry.htm "Glossary Entry")), the screen is created implicitly.
