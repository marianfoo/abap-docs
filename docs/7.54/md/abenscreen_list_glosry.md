screen list

[List](javascript:call_link\('abenlist_glosry.htm'\) "Glossary Entry") displayed on the screen in a [list dynpro](javascript:call_link\('abenlist_dynpro_glosry.htm'\) "Glossary Entry"). When it is being compiled, a screen list is stored in the [list buffer](javascript:call_link\('abenlist_buffer_glosry.htm'\) "Glossary Entry"). See also [spool list](javascript:call_link\('abenprint_list_glosry.htm'\) "Glossary Entry").
