self reference

[Object reference variable](javascript:call_link\('abenobject_refer_variable_glosry.htm'\) "Glossary Entry") called [me](javascript:call_link\('abenme.htm'\)) that is predefined as a local [constant](javascript:call_link\('abenconstant_glosry.htm'\) "Glossary Entry") and is available in every [instance method](javascript:call_link\('abeninstance_method_glosry.htm'\) "Glossary Entry"). It always points to the [instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") in which the method is currently being executed. The static type of me is the [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") in which the instance method is implemented.
