short dump

Error log created and saved after a [runtime error](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenruntime_error_glosry.htm "Glossary Entry") or by a [runtime warning](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenruntime_warning_glosry.htm "Glossary Entry"). Saved short dumps can be displayed using the [ABAP dump analysis](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_dump_analsyis_glosry.htm "Glossary Entry") tool.
