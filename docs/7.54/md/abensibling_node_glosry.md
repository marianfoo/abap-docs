sibling node

All [child nodes](javascript:call_link\('abenchild_node_glosry.htm'\) "Glossary Entry") in a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry") that have the same [parent node](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry") are sibling nodes. They have the same value in the [hierarchy column](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") [HIERARCHY\_PARENT\_RANK](javascript:call_link\('abenddddl_hierarchy.htm'\)).
