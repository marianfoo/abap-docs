simple enhancement implementation

Semantically related enhancement of a [repository object](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrepository_object_glosry.htm "Glossary Entry"). A simple enhancement implementation manages one or more [enhancement implementation elements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenenhancement_impl_elem_glosry.htm "Glossary Entry"). A simple enhancement implementation can exist independently or as a component of a [composite enhancement implementation](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencomp_enhancement_impl_glosry.htm "Glossary Entry").
