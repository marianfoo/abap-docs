spool request

Output request sent to the [SAP spool system](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_spool_system_glosry.htm "Glossary Entry"). In a spool request, the pages of a [spool list](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_list_glosry.htm "Glossary Entry") are passed to the SAP spool system. [Spool parameters](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_parameter_glosry.htm "Glossary Entry") are defined when a spool request is opened and these parameters cannot be changed afterwards. A spool request is always assigned to the current [spool list level](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenprint_list_level_glosry.htm "Glossary Entry"). The spool requests of stacked spool list levels are also stacked.
