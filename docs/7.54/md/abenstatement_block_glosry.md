statement block

Structural unit within [control structures](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencontrol_structure_glosry.htm "Glossary Entry") that consists of one or more sequential statements. The statements of a statement block are processed sequentially. Statement blocks themselves can contain control structures.
