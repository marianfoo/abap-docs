Common Parameters of Character String Functions

The following parameters are used to achieve the same effect with multiple [character string functions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions.htm) and [predicate functions for character-type arguments](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredicate_functions_strgs.htm):

-   [case - Case sensitive](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_case.htm)

-   [occ - Occurrence](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_occ.htm)

-   [off, len - Offset and length](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_off_len.htm)

-   [regex - Regular expression](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_regex.htm)

-   [sub - Substring](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_sub.htm)

-   [val - Input value](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstring_functions_val.htm)
