system ID

SID for short. ID or name of an [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") and hence the ID or name of its [ABAP system](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_system_glosry.htm "Glossary Entry") too. A system ID has three characters and is the content of the [system field](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_field_glosry.htm "Glossary Entry") [sy-sysid](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_fields.htm).
