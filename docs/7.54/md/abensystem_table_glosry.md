system table

[Database table](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendatabase_table_glosry.htm "Glossary Entry") without a [client column](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenclient_column_glosry.htm "Glossary Entry"). A system table contains cross-client data that is generally only processed by [system programs](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_program_glosry.htm "Glossary Entry").
