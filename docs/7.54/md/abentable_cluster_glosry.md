table cluster

Before Release 7.53, [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") in the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry") that contains the data of multiple [cluster tables](javascript:call_link\('abencluster_table_glosry.htm'\) "Glossary Entry"). Cluster tables are no longer supported from Release 7.53 and all table clusters were removed.
