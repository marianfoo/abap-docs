table key

Columns of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") or [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") whose content identifies table rows. For internal tables, a distinction is made between [primary table keys](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry") and [secondary table keys](javascript:call_link\('abensecondary_table_key_glosry.htm'\) "Glossary Entry").
