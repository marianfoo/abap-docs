temperature column

Column of a [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") with the name \_DATAAGING. On a [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"), the values in this column specify whether [data aging](javascript:call_link\('abendata_aging_glosry.htm'\) "Glossary Entry") data is current or old.
