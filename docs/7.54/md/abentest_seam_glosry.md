test seam

[Seam](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenseam_glosry.htm "Glossary Entry") enclosed by the statements [TEST-SEAM](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abaptest-seam.htm) and [END-TEST-SEAM](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapend-test-seam.htm) in production code and replaced by an [injection](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abeninjection_glosry.htm "Glossary Entry") when [unit tests](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenunit_test_glosry.htm "Glossary Entry") are executed.
