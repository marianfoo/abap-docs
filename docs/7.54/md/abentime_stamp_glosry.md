time stamp

Combined [date](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendate_glosry.htm "Glossary Entry")/ [time](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentime_glosry.htm "Glossary Entry") for identifying a point in time. Time stamps in the [ABAP runtime environment](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") are [POSIX time stamps](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenposix_timestamp_glosry.htm "Glossary Entry") independent of time zone. Time stamps are represented by a [time stamp field](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentimestamp_field_glosry.htm "Glossary Entry") with a [time stamp type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentimestamp_field_glosry.htm "Glossary Entry") or as [time stamps in packed numbers](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentime_stamps_packed.htm) whose data type is defined using the [data elements](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendata_element_glosry.htm "Glossary Entry") TIMESTAMP or TIMESTAMPL from [ABAP Dictionary](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_dictionary_glosry.htm "Glossary Entry").
