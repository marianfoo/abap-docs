Time Stamp Functions

The following [built-in functions](javascript:call_link\('abenpredefined_function_glosry.htm'\) "Glossary Entry") for editing time stamps in time stamp fields are available:

-   [utclong\_current](javascript:call_link\('abenutclong_current.htm'\)) creates a current time stamp

-   [utclong\_add](javascript:call_link\('abenutclong_current.htm'\)) adds values to a time stamp

-   [utclong\_diff](javascript:call_link\('abenutclong_current.htm'\)) determines the difference between two time stamps
