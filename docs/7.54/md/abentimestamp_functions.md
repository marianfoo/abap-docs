Time Stamp Functions

The following [built-in functions](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpredefined_function_glosry.htm "Glossary Entry") for editing time stamps in time stamp fields are available:

-   [utclong\_current](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenutclong_current.htm) creates a current time stamp

-   [utclong\_add](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenutclong_current.htm) adds values to a time stamp

-   [utclong\_diff](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenutclong_current.htm) determines the difference between two time stamps
