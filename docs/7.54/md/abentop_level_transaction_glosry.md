top level transaction

The first program in a [call sequence](javascript:call_link\('abencall_sequence_glosry.htm'\) "Glossary Entry"). The [ABAP memory](javascript:call_link\('abenabap_memory_glosry.htm'\) "Glossary Entry") assigned to a call sequence is persisted for as long as the [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") of the top level transaction remains loaded. The [SAP Easy Access](javascript:call_link\('abensap_easy_access_glosry.htm'\) "Glossary Entry") program does not usually function as a top level transaction, except in [area instance versions](javascript:call_link\('abenarea_instance_version_glosry.htm'\) "Glossary Entry").
