Transformation Editor

Tool in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") used for creating [XSLT programs](javascript:call_link\('abenxsl_transformation_2_glosry.htm'\) "Glossary Entry") or [simple transformations](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry") in [transformation source code](javascript:call_link\('abenxsl_transformation_2_glosry.htm'\) "Glossary Entry"). Called using the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") STRANS.
