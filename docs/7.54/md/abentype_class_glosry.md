type description class

[System class](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_class_glosry.htm "Glossary Entry") of the [runtime type services](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenrun_time_type_services_glosry.htm "Glossary Entry"). The [static methods](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenstatic_method_glosry.htm "Glossary Entry") of the type description classes create [type description objects](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_object_glosry.htm "Glossary Entry") from existing types or construct new types.
