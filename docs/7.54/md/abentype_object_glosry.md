type description object

Instance of a [type description class](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_class_glosry.htm "Glossary Entry"). Type description objects describe a [type](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentype_glosry.htm "Glossary Entry"). Type description objects are created using the static methods of the type description classes. The type description class and any attributes of type description objects determine the [technical attributes](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abentechnical_type_attr_glosry.htm "Glossary Entry") of the type. [References](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenreference_glosry.htm "Glossary Entry") to type description objects can be used, for example, after the addition HANDLE of the statements [CREATE DATA](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapcreate_data_handle.htm) and [ASSIGN](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abapassign_casting.htm).
