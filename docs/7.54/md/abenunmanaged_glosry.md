unmanaged

An implementation type of the [business object behavior](javascript:call_link\('abenbehavior_glosry.htm'\) "Glossary Entry") in the context of the [ABAP RESTful programming model](javascript:call_link\('abenr_a_p_glosry.htm'\) "Glossary Entry").

For this implementation type, all important components of the REST contract must be implemented first. All required operations (create, update, delete, plus any application-specific actions) must be defined in the associated [behavior definition](javascript:call_link\('abenbehavior_definition_glosry.htm'\) "Glossary Entry") before being implemented in ABAP.
