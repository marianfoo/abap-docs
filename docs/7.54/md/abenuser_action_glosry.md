user action

Action performed by a user on a [control element](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abencontrol_element_glosry.htm "Glossary Entry") on the [user interface](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenuser_interface_glosry.htm "Glossary Entry"). Raises the [dynpro event](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abendynpro_event_glosry.htm "Glossary Entry") [PAI](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenpai_glosry.htm "Glossary Entry") if the selected control element is associated with a [function code](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenfunction_code_glosry.htm "Glossary Entry").
