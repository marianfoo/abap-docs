UUID

Universal Unique Identifier. Universally unique identifier, generated in accordance with [OSF standards](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/http://www.opengroup.org/onlinepubs/9629399/apdxa.htm) from the system time stamp and the network address. In ABAP, a UUID can be created using the [system class](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensystem_class_glosry.htm "Glossary Entry") CL\_SYSTEM\_UUID and the [SQL function](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensql_function_glosry.htm "Glossary Entry") [UUID](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensql_uuid.htm).
