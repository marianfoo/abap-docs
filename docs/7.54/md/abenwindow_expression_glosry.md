window expression

In [ABAP SQL](javascript:call_link\('abenopen_sql_glosry.htm'\) "Glossary Entry"), a window expression uses [OVER](javascript:call_link\('abapselect_over.htm'\)) to define a subset of the results set of a [query](javascript:call_link\('abenwindow_function_glosry.htm'\) "Glossary Entry") and applies a [window function](javascript:call_link\('abenwindow_function_glosry.htm'\) "Glossary Entry") to this query.
