WebSocket protocol

Abbreviated as WSP. Log based on TCP for bidirectional communication between a client and a server. Using [ABAP Push Channels](javascript:call_link\('abenabap_push_channels_glosry.htm'\) "Glossary Entry"), an [AS ABAP](javascript:call_link\('abensap_nw_abap_glosry.htm'\) "Glossary Entry") can be used as a server and as a client for the WebSocket protocol.
