WebSocket protocol

Abbreviated as WSP. Log based on TCP for bidirectional communication between a client and a server. Using [ABAP Push Channels](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_push_channels_glosry.htm "Glossary Entry"), an [AS ABAP](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abensap_nw_abap_glosry.htm "Glossary Entry") can be used as a server and as a client for the WebSocket protocol.
