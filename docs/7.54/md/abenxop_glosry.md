XOP

XML-binary Optimized Packaging (see [http://www.w3.org/TR/xop10/](http://www.w3.org/TR/xop10/)). Representation of [XML](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenxml_glosry.htm "Glossary Entry") data in which any binary raw data is saved separately from the data represented by characters. In ABAP, XOP is supported by a [format](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_sxml_lib_formats.htm) from the [sXML Library](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenabap_sxml_lib.htm).
