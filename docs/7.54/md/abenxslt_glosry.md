XSLT

XSLT (Extensible Stylesheet Language Transformation, see [www.w3.org/TR/xslt](http://www.w3.org/TR/xslt)) enables the conversion of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") formats into any other XML formats. The [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") contains an [SAP XSLT processor](javascript:call_link\('abenxslt_processor_glosry.htm'\) "Glossary Entry") for performing [XSL transformations](javascript:call_link\('abenxsl_transformation_1_glosry.htm'\) "Glossary Entry").
