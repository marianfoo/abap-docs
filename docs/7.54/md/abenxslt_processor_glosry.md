SAP XSLT processor

System program integrated into the [ABAP kernel](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenkernel_glosry.htm "Glossary Entry") and used to perform [XSL transformations](https://help.sap.com/doc/abapdocu_754_index_htm/7.54/en-US/abenxsl_transformation_1_glosry.htm "Glossary Entry").
