AT USER-COMMAND - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapat_user-command.htm)

Syntax

AT USER-COMMAND.

Effect

Introduces an event block whose result is raised by a function with a self-defined function code on a list.
