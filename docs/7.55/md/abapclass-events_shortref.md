CLASS-EVENTS - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapclass-events.htm)

Syntax

CLASS-EVENTS evt options.

Effect

Declares a [static event](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstatic_event_glosry.htm "Glossary Entry") evt of a class or an interface.

Additions

-   options
    As in [EVENTS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapevents_shortref.htm).
