CLASS, DEFERRED, LOAD - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapclass_deferred.htm)

Syntax

CLASS class DEFINITION *{*DEFERRED *\[*PUBLIC*\]**}* *|* LOAD.

Effect

Makes a class public.

Additions

-   DEFERRED *\[*PUBLIC*\]*
    Makes a local or global class public before it is actually declared in the program.
    

-   [LOAD](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapclass_interface_load.htm)
    Obsolete: Previously loaded a global class from the Class Library.
