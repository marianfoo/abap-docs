CLOSE DATASET - Quick reference

[Reference](javascript:call_link\('abapclose_dataset.htm'\))

Syntax

CLOSE DATASET dset.

Effect

Closes a file on the current [AS instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") that was specified in dset and opened previously using [OPEN DATASET](javascript:call_link\('abapopen_dataset.htm'\)).
