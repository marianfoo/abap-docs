CONTINUE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcontinue.htm)

Syntax

CONTINUE.

Effect

Exits a loop. The current loop pass is exited and the loop is continued with the next loop pass.
