END-OF-DEFINITION

[Short Reference](javascript:call_link\('abapdefine_shortref.htm'\))

Syntax

END-OF-DEFINITION.

Effect

The statement END-OF-DEFINITION closes a macro definition that was initiated by [DEFINE](javascript:call_link\('abapdefine.htm'\)).
