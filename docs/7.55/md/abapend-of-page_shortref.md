END-OF-PAGE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapend-of-page.htm)

Syntax

END-OF-PAGE.

Effect

Initiates an event block whose event is raised when the [page footer](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpage_footer_glosry.htm "Glossary Entry") is reached when a basic list is created.
