END-TEST-INJECTION

[Short Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abaptest-injection_shortref.htm)

Syntax

END-TEST-INJECTION.

Effect

The statement END-TEST-INJECTION closes a test injection that was initiated using [TEST-INJECTION](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abaptest-injection.htm).
