ENDCASE

[Short Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcase_shortref.htm)

Syntax

ENDCASE.

Effect

The statement ENDCASE closes a [case distinction](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencase_distinction_glosry.htm "Glossary Entry") introduced using [CASE](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcase.htm) or [CASE TYPE OF](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcase_type.htm).
