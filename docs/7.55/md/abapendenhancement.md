ENDENHANCEMENT

[Short Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapenhancement_shortref.htm)

Syntax

ENDENHANCEMENT.

Effect

The statement ENDENHANCEMENT closes a [source code plug-in](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensource_code_plugin_glosry.htm "Glossary Entry") introduced using [ENHANCEMENT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapenhancement.htm).
