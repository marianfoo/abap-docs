ENDMETHOD

[Short Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmethod_shortref.htm)

Syntax

ENDMETHOD.

Effect

The statement ENDMETHOD closes a method implementation introduced using [METHOD](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmethod.htm).
