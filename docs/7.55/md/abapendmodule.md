ENDMODULE

[Short Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmodule_shortref.htm)

Syntax

ENDMODULE.

Effect

The statement ENDMODULE closes a module definition introduced using [MODULE](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmodule.htm).
