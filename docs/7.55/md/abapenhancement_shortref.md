ENHANCEMENT - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapenhancement.htm)

Syntax

ENHANCEMENT id.
  ...
ENDENHANCEMENT.

Effect

Implements a [source code enhancement](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensource_code_enhancement_glosry.htm "Glossary Entry") as a [source code plug-in](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensource_code_plugin_glosry.htm "Glossary Entry") with the ID id.
