[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapexit_sql.htm)

Syntax

EXIT FROM SQL.

Effect

Obsolete: Cancels implicit cursor processing in EXEC SQL. Processing is resumed after ENDEXEC.
