GET TIME STAMP - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapget_time-stamp.htm)

Syntax

GET TIME STAMP FIELD time\_stamp.

Effect

Returns the current [time stamp](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentime_stamp_glosry.htm "Glossary Entry") in time\_stamp of the type TIMESTAMP or TIMESTAMPL.
