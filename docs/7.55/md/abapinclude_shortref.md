INCLUDE - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapinclude_prog.htm)

Syntax

INCLUDE incl *\[*IF FOUND*\]*.

Effect

Includes the [include program](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninclude_program_glosry.htm "Glossary Entry") incl in the current position in the source code.

Addition

-   IF FOUND
    Suppresses the syntax check error message if the include program is not found.
