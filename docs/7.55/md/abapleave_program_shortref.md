LEAVE PROGRAM - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapleave_program.htm)

Syntax

LEAVE PROGRAM.

Effect

Exits the current program and continues processing as specified by the program call after the call position of the program or the [call sequence](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencall_sequence_glosry.htm "Glossary Entry").
