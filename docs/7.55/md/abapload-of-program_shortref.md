LOAD-OF-PROGRAM - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapload-of-program.htm)

Syntax

LOAD-OF-PROGRAM.

Effect

Initializes an event block whose event is raised in an [internal session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninternal_session_glosry.htm "Glossary Entry") when the program is loaded.
