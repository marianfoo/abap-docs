LOOP AT SCREEN - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abaploop_at_screen.htm)

Syntax

LOOP AT SCREEN INTO wa.
  ...
ENDLOOP.

Effect

Reads the properties of the screen elements of the current [dynpro](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendynpro_glosry.htm "Glossary Entry") in a loop and assigns the results to a work area wa of the structure [SCREEN](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenscreen.htm).
