MODIFY SCREEN - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmodify_screen.htm)

Syntax

MODIFY SCREEN FROM wa.

Effect

In LOOP AT SCREEN, overwrites the properties of the current screen element with the content of the predefined work area wa of the type [SCREEN](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenscreen.htm).
