SELECTION-SCREEN, LINE - Quick reference

[Reference](javascript:call_link\('abapselection-screen_line.htm'\))

Syntax

SELECTION-SCREEN BEGIN OF LINE.
...
SELECTION-SCREEN END OF LINE.

Effect

Defines a line on the current [selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry") in which screen elements can be positioned using [SELECTION-SCREEN POSITION](javascript:call_link\('abapselection-screen_shortref.htm'\)).
