SELECTION-SCREEN FIELD SELECTION - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselection-screen_ldb_field.htm)

Syntax

SELECTION-SCREEN FIELD SELECTION
                 FOR *{*NODE*|*TABLE*}* node *\[*ID id*\]*.

Effect

Obsolete: Flags a node node of the structure of a [logical database](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlogical_data_base_glosry.htm "Glossary Entry") for [field selection](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfield_selection_glosry.htm "Glossary Entry").

Addition

-   ID id
    Defines an ID id for the node, with a maximum of three characters.
