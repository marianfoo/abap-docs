SET RUN TIME CLOCK - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapset_run_time_clock_resolution.htm)

Syntax

SET RUN TIME CLOCK RESOLUTION *{*HIGH*|*LOW*}*.

Effect

Sets the measurement accuracy for the statement GET RUN TIME.
