SET USER-COMMAND - Quick reference

[Reference](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapset_user-command.htm)

Syntax

SET USER-COMMAND fcode.

Effect

Raises a list event with a [function code](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunction_code_glosry.htm "Glossary Entry") specified in fcode.
