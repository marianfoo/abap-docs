ABAP Compiler

ABAP Compiler creates a [byte code](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbytecode_glosry.htm "Glossary Entry") as an intermediate code when generating a program from an [ABAP source code](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_source_code_glosry.htm "Glossary Entry") that is stored in the database as a program [load](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenload_glosry.htm "Glossary Entry") and is loaded into the [PXA](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpxa_glosry.htm "Glossary Entry") if required.
