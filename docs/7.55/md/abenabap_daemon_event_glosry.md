ABAP Daemon event

Event in the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") to which an [ABAP Daemon](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") reacts using the [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of the [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") IF\_ABAP\_DAEMON\_EXTENSION. ABAP Daemon events can be raised in ABAP programs or by the ABAP runtime environment.
