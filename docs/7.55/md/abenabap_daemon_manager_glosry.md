ABAP Daemon manager

The class CL\_ABAP\_DAEMON\_CLIENT\_MANAGER. The static methods of the ABAP Daemon Manager are used to create and use [ABAP Daemons](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_daemon_glosry.htm "Glossary Entry") in [ABAP programs](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry").
