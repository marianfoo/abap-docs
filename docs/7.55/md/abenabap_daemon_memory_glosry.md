ABAP Daemon memory

Part of the [user memory](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_memory_glosry.htm "Glossary Entry") in which an [ABAP Daemon](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_daemon_glosry.htm "Glossary Entry") can store data in [PCP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpcp_glosry.htm "Glossary Entry") format. An ABAP daemon can use approximately 1 megabyte of the ABAP Daemon memory.
