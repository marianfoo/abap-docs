ABAP language element addition

[ABAP word](javascript:call_link\('abenabap_word_glosry.htm'\) "Glossary Entry") or combination of multiple ABAP words. A language element addition can be used in [ABAP statements](javascript:call_link\('abenabap_statement_glosry.htm'\) "Glossary Entry") as an addition to a suitable [ABAP language element](javascript:call_link\('abenabap_language_element_glosry.htm'\) "Glossary Entry"). Examples of different ABAP language element additions are WITH KEY and WITH TABLE KEY.
