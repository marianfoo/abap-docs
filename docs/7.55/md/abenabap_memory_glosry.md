ABAP memory

Memory area within each [ABAP session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_session_glosry.htm "Glossary Entry") that can be accessed jointly by the programs of a [call sequence](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencall_sequence_glosry.htm "Glossary Entry") using the statements [EXPORT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapexport_data_cluster.htm) and [IMPORT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapimport_data_cluster.htm), and whose data remains as long as the [top-level transaction](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentop_level_transaction_glosry.htm "Glossary Entry") of a call sequence.
