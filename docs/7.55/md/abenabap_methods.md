Methods

Methods are the [procedures](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprocedure_glosry.htm "Glossary Entry") of a class whose functionality is implemented between the following statements:

[METHOD](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmethod.htm)
  ...
[ENDMETHOD](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapendmethod.htm)

The statement METHOD does not usually have any additions. Exceptions to this rule are:

-   Methods used to implement [ABAP Managed Database Procedures](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenamdp.htm) (AMDP)

-   [Kernel methods](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmethod_kernel_module_internal.htm) for internal use
