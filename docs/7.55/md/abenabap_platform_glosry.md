ABAP platform

The ABAP Platform is used to develop and run ABAP-based [application programs](javascript:call_link\('abenapplication_program_glosry.htm'\) "Glossary Entry"). It is implemented mainly using [ABAP systems](javascript:call_link\('abenabap_system_glosry.htm'\) "Glossary Entry"). Furthermore, the ABAP Platform includes various development environments such as the [ABAP Development Tools (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") and the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") . The [SAP Cloud Platform, ABAP Environment](javascript:call_link\('abensap_cp_abap_env_glosry.htm'\) "Glossary Entry") is a special variant of the ABAP Platform for the [SAP Cloud Platform](javascript:call_link\('abensap_cloud_platform_glosry.htm'\) "Glossary Entry") and other cloud developments.
