ABAP programming model for SAP Fiori

Obsolete programming model for SAP HANA-optimized Fiori apps based on the [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry"). Its successor is the [ABAP RESTful Application Programming Model (RAP)](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenarap_glosry.htm "Glossary Entry").
