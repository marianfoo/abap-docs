Runtime Measurements

The statements in this section make it possible to measure the runtime of a program and make corresponding settings.

-   [GET RUN TIME](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapget_run_time.htm)

-   [SET RUN TIME CLOCK RESOLUTION](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapset_run_time_clock_resolution.htm)

-   [SET RUN TIME ANALYZER](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapset_run_time_analyzer.htm)

Alternatively, a [class for runtime measurements](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencl_abap_runtime.htm) (CL\_ABAP\_RUNTIME) can be used.
