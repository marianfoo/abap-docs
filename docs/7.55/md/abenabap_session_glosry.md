ABAP session

Instance for a [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry") of an [AS instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry") with the [ABAP Memory](javascript:call_link\('abenabap_memory_glosry.htm'\) "Glossary Entry") as a separate memory area. Each [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry") of a user session is associated with its own ABAP session. A maximum of 16 ABAP sessions can be managed by a single user session. An [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") is executed in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") of an ABAP session.
