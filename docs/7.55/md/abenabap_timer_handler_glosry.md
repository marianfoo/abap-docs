ABAP timer handler

[Class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that implements the [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") IF\_ABAP\_TIMER\_HANDLER. The interface method ON\_TIMEOUT of the timer handler, which is specified with a [ABAP Timer Manager](javascript:call_link\('abenabap_timer_manager_glosry.htm'\) "Glossary Entry") when an [ABAP Timer](javascript:call_link\('abenabap_timer_glosry.htm'\) "Glossary Entry") is started. It handles the timeout event of the timer.
