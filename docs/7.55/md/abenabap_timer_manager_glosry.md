ABAP timer manager

Object of the [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") CL\_ABAP\_TIMER\_MANAGER that is accessed using the [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") IF\_ABAP\_TIMER\_MANAGER. A timer manager allows an [ABAP Timer](javascript:call_link\('abenabap_timer_glosry.htm'\) "Glossary Entry") to be started and stopped in [non-blocking mode](javascript:call_link\('abennon_blocking_model_glosry.htm'\) "Glossary Entry"). When the timer is started, a runtime and an [ABAP Timer handler](javascript:call_link\('abenabap_timer_handler_glosry.htm'\) "Glossary Entry") are defined.
