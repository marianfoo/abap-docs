ABAP Unit

Test tool integrated into the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") that is used for checking the functions of code units in a program ([unit tests](javascript:call_link\('abenunit_test_glosry.htm'\) "Glossary Entry")). Individual tests are implemented as [test methods](javascript:call_link\('abentest_method_glosry.htm'\) "Glossary Entry") of local [test classes](javascript:call_link\('abentest_class_glosry.htm'\) "Glossary Entry") in ABAP programs. The tests of one test class use the same [fixture](javascript:call_link\('abenfixture_glosry.htm'\) "Glossary Entry"). Global test classes can only be used in local test classes. The properties of a test can be specified in the definition of a local test class. Tests of multiple ABAP programs can be combined into [test tasks](javascript:call_link\('abentest_task_glosry.htm'\) "Glossary Entry").
