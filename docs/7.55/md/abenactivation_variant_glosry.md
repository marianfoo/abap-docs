activation variant

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that contains activation settings for [activatable checkpoints](javascript:call_link\('abenactivatable_checkpoint_glosry.htm'\) "Glossary Entry"). An activation variant can contain settings for [checkpoint groups](javascript:call_link\('abencheckpoint_group_glosry.htm'\) "Glossary Entry") and [compilation units](javascript:call_link\('abencompilation_unit_glosry.htm'\) "Glossary Entry"). Activation variants are created and the activation settings are maintained using [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SAAB.
