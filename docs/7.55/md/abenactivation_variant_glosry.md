activation variant

[Repository object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrepository_object_glosry.htm "Glossary Entry") that contains activation settings for [activatable checkpoints](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenactivatable_checkpoint_glosry.htm "Glossary Entry"). An activation variant can contain settings for [checkpoint groups](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencheckpoint_group_glosry.htm "Glossary Entry") and [compilation units](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencompilation_unit_glosry.htm "Glossary Entry"). Activation variants are created and the activation settings are maintained using [transaction](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentransaction_glosry.htm "Glossary Entry") SAAB.
