aggregate function

Function that determines a value from the values of multiple rows of a column in the result set of an [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry"). The calculation is done in the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry"). An aggregate function can be used in an [aggregate expression](javascript:call_link\('abenaggregate_expression_glosry.htm'\) "Glossary Entry"), as a [window function](javascript:call_link\('abenwindow_function_glosry.htm'\) "Glossary Entry") in a [window expression](javascript:call_link\('abenwindow_expression_glosry.htm'\) "Glossary Entry"), and in a [hierarchy aggregate navigator](javascript:call_link\('abenhierarchy_agg_navi_glosry.htm'\) "Glossary Entry").
