aggregate

A summary of the data of multiple rows of an [SQL source](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensql_data_source_glosry.htm "Glossary Entry"). Can be determined using [aggregate functions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenaggregate_function_glosry.htm "Glossary Entry") in the [SELECT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselect.htm) statement.
