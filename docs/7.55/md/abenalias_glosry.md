alias name

A name that can or must be used instead of the actual name of something. The following alias names can be defined in ABAP:

-   The statement [ALIASES](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapaliases.htm) can be used to declare an alias name for an [interface component](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninterface_component_glosry.htm "Glossary Entry") in a [class](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_glosry.htm "Glossary Entry") or in a [compound interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencompound_interface_glosry.htm "Glossary Entry").

-   The addition [ALIAS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abaptypes_primary_key.htm) can be used to specify an alias for the [primary table key](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprimary_key_glosry.htm "Glossary Entry") when declaring an [internal table](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninternal_table_glosry.htm "Glossary Entry").

-   In [ABAP SQL](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_glosry.htm "Glossary Entry"), the addition AS can be used to define aliases for columns of the result set in a [SELECT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselect_list.htm) list or for data sources in the [FROM](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapfrom_clause.htm) clause.
