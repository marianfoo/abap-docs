AMDP macro

Syntax specified in the implementation of an [AMDP method](javascript:call_link\('abenamdp_method_glosry.htm'\) "Glossary Entry") in [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry"), which is replaced by the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry") when the [database procedure](javascript:call_link\('abendatabase_procedure_glosry.htm'\) "Glossary Entry") is generated on the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry") in compliance with the assigned rules.
