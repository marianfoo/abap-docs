annotation array

[CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") defined in its [annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry") using the addition [array of](javascript:call_link\('abencds_f1_define_annotation.htm'\)). When an annotation array is specified in [annotation syntax](javascript:call_link\('abencds_annotation_syntax_glosry.htm'\) "Glossary Entry"), any number of similar elements are listed in square brackets [\[ \]](javascript:call_link\('abencds_annotations_syntax.htm'\)).
