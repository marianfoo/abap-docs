application program

[ABAP program](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry") that is part of the business applications executed on [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry"). Application programs generally only work with the data of the current [application client](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenapplication_client_glosry.htm "Glossary Entry") that is stored in [application tables](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenapplication_table_glosry.htm "Glossary Entry").
