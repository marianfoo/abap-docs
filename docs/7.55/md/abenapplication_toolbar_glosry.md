application toolbar

Component of a [GUI window](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abengui_window_glosry.htm "Glossary Entry"). Contains [pushbuttons](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpushbutton_glosry.htm "Glossary Entry") defined in the [Menu Painter](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmenu_painter_glosry.htm "Glossary Entry") tool. Part of the [GUI status](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abengui_status_glosry.htm "Glossary Entry") set using [SET PF-STATUS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapset_pf-status_dynpro.htm). Also referred to as the [pushbutton toolbar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpush_button_toolbar_glosry.htm "Glossary Entry") in [ABAP Workbench](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_workbench_glosry.htm "Glossary Entry")
