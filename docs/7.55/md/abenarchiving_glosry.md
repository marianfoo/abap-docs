archiving

Storing [spool lists](javascript:call_link\('abenspool_list_glosry.htm'\) "Glossary Entry") using the [ArchiveLink](javascript:call_link\('abenarchivelink_glosry.htm'\) "Glossary Entry") tool. Spool lists stored in the [SAP spool system](javascript:call_link\('abensap_spool_system_glosry.htm'\) "Glossary Entry") can be archived. During archiving, spool lists are transferred from the spool system are saved in the same format that was used there.
