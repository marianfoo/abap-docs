archiving parameter

Parameter that must be transferred to a [spool request](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenspool_request_glosry.htm "Glossary Entry") if its data is to be archived using [ArchiveLink](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenarchivelink_glosry.htm "Glossary Entry").
