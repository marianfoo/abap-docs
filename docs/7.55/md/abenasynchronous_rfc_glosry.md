asynchronous Remote Function Call

aRFC for short. [Remote function call](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenremote_function_call_glosry.htm "Glossary Entry") that does not wait for the end of processing of the remotely called function.
