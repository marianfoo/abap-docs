Base64

Method for coding binary data into printable characters. Each byte is converted into a short character string whose characters are obtained from a record of 64 [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry")\-independent [ASCII](javascript:call_link\('abenascii_glosry.htm'\) "Glossary Entry") characters.
