batch input

Data transfer technique that allows data sets to be transferred automatically to the [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") of [transactions](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry"), and thus to [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Batch input is controlled by [batch input sessions](javascript:call_link\('abenbatch_input_session_glosry.htm'\) "Glossary Entry"). The processing of batch input sessions in batch input is partly implemented directly in the [ABAP kernel](javascript:call_link\('abenkernel_glosry.htm'\) "Glossary Entry"). See also [batch input tables](javascript:call_link\('abenbatch_input_table_glosry.htm'\) "Glossary Entry").
