BCD

Abbreviation of Binary Coded Decimals. Code for representing decimal numbers that stores each decimal digit in 4 bits. See also [packed number](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpacked_number_glosry.htm "Glossary Entry").
