BLOB

Abbreviation for Binary Large Object. Term for a [database field](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_field_glosry.htm "Glossary Entry") of the [RAWSTRING](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddic_builtin_types.htm) data type or the result of an [SQL expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapsql_expr.htm) that returns the data type [RAWSTRING](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddic_builtin_types.htm).
