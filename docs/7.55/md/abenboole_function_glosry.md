Boolean function

[Logical function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlogic_function_glosry.htm "Glossary Entry") that analyzes a [logical expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlogical_expression_glosry.htm "Glossary Entry") and returns the [truth value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentruth_value_glosry.htm "Glossary Entry") in character-like or byte-like form.
