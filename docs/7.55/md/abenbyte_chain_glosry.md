byte chain

Sequence of bytes, also called byte sequence. Content of each data object. However direct access to the bytes of a byte chain is possible only in [byte-like data objects](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbyte_like_data_object_glosry.htm "Glossary Entry"), however, or by using appropriately typed [field symbols](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfield_symbol_glosry.htm "Glossary Entry").
