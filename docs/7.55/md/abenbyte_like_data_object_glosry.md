byte-like data object

[Data object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_object_glosry.htm "Glossary Entry") with a [byte-like data type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbyte_like_data_typ_glosry.htm "Glossary Entry") whose content is interpreted as non-encoded.
