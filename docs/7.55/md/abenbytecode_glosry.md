byte code

Result of the generation of an ABAP program using the [ABAP Compiler](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_compiler_glosry.htm "Glossary Entry"). The statements in the byte code are associated with C functions. When the [ABAP program](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry") is executed, the byte code is loaded into the [PXA](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpxa_glosry.htm "Glossary Entry") as program [Load](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenload_glosry.htm "Glossary Entry") and is interpreted by the [ABAP runtime environment](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") (virtual machine), whereby the associated C functions are called.
