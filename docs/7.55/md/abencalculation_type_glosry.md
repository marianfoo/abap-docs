calculation Type

Property of an [arithmetic expression](javascript:call_link\('abenarithmetic_expression_glosry.htm'\) "Glossary Entry") depending on the [numeric data types](javascript:call_link\('abennumeric_data_type_glosry.htm'\) "Glossary Entry") involved. The calculation type determines the calculation rule used to evaluate an arithmetic expression. There are different rules for the calculation types i, int8, p, decfloat34, and f.
