Call Event Blocks

Event blocks are processing blocks in an ABAP program that are executed when the associated event occurs in the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry"). In two cases, such events can be raised programmatically and the event blocks, if implemented, can be called:

-   Using the statement [SET USER-COMMAND](javascript:call_link\('abapset_user-command.htm'\)) in classic list processing

-   Using the statement [PUT](javascript:call_link\('abapput.htm'\)) in logical databases

Both statements are effectively obsolete. For more information, see the relevant sections.
