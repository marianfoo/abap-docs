CDS abstract entity

[CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") defined using [DEFINE ABSTRACT ENTITY](javascript:call_link\('abencds_f1_define_abstract_entity.htm'\)) in a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). A CDS abstract entity defines the type properties of a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") without defining a database object.
