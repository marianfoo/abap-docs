CDS behavior definition

A CDS behavior definition defines a [RAP Business Object behavior](javascript:call_link\('abenrap_bo_behavior_glosry.htm'\) "Glossary Entry"). It is a [CDS RAP object](javascript:call_link\('abencds_rap_object_glosry.htm'\) "Glossary Entry") that defines the behavior of a [RAP business object](javascript:call_link\('abenrap_bo_glosry.htm'\) "Glossary Entry") in [BDL source code](javascript:call_link\('abenbdl_source_code_glosry.htm'\) "Glossary Entry") using language elements in [CDS BDL](javascript:call_link\('abencds_bdl_glosry.htm'\) "Glossary Entry") of [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry").
