Syntax

Effect

CDS DDL expressions are expressions that can be specified in a CDS DDL operand position of a [CDS view entity](javascript:call_link\('abencds_v2_view_glosry.htm'\) "Glossary Entry").

The following CDS DDL expressions exist:

-   [Arithmetic expressions](javascript:call_link\('abencds_arithmetic_expression_v2.htm'\))

-   [Aggregate expressions](javascript:call_link\('abencds_aggregate_functions_v2.htm'\))

-   [Case distinctions](javascript:call_link\('abencds_case_expression_v2.htm'\))

-   [Cast expressions](javascript:call_link\('abencds_cast_expression_v2.htm'\))

-   [Calls of built-in CDS functions](javascript:call_link\('abencds_builtin_functions_v2.htm'\))
