ABAP CDS - DCL Statements

The data control language CDS DCL in ABAP CDS follows the [general CDS syntax rules](javascript:call_link\('abencds_general_syntax_rules.htm'\)) and comprises the following statements for the definition of [CDS roles](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry") and [CDS access policies](javascript:call_link\('abencds_access_policy_glosry.htm'\) "Glossary Entry") in [DCL source code](javascript:call_link\('abendcl_source_code_glosry.htm'\) "Glossary Entry"):

-   [DEFINE ROLE](javascript:call_link\('abencds_f1_define_role.htm'\))

-   [DEFINE ACCESSPOLICY](javascript:call_link\('abencds_f1_define_accesspolicy.htm'\))

Hints

-   The [DCL source code](javascript:call_link\('abendcl_source_code_glosry.htm'\) "Glossary Entry") for CDS access control has its own editor in [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry"), which is documented there.

-   The names of the items of DCL source code for [CDS roles](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry") and [CDS access policies](javascript:call_link\('abencds_access_policy_glosry.htm'\) "Glossary Entry") are in the same namespace.

-   DCL source code can also be displayed in [Repository Browser](javascript:call_link\('abenrepository_browser_glosry.htm'\) "Glossary Entry") in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry").
