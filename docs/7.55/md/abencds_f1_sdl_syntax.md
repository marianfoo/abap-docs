ABAP CDS - SDL for Service Definitions

The [CDS service definitions](javascript:call_link\('abencds_service_definition_glosry.htm'\) "Glossary Entry") in ABAP CDS are created using the data definition language [CDS SDL](javascript:call_link\('abencds_sdl_glosry.htm'\) "Glossary Entry") in [SDL source code](javascript:call_link\('abensdl_source_code_glosry.htm'\) "Glossary Entry"). CDS SDL complies with the [general CDS syntax rules](javascript:call_link\('abencds_general_syntax_rules.htm'\)) and contains the following language element for service definitions:

-   [DEFINE SERVICE](javascript:call_link\('abensrvd_define_service.htm'\))

Hint

The [SDL source code](javascript:call_link\('abensdl_source_code_glosry.htm'\) "Glossary Entry") of service definitions has its own editor in [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry"), which is documented there. SDL source code can also be displayed in [Repository Browser](javascript:call_link\('abenrepository_browser_glosry.htm'\) "Glossary Entry") in [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry").
