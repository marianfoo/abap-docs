CDS DDL - CDS View Entity, Elementary CDS Operands

Syntax

... [literal](javascript:call_link\('abencds_literal_v2.htm'\))
  *|* [field](javascript:call_link\('abencds_field_v2.htm'\))
  *|* [$parameters.pname](javascript:call_link\('abencds_parameter_v2.htm'\))
  *|* [$session.vname](javascript:call_link\('abencds_session_variable_v2.htm'\)) ...

Effect

Specifies a literal [literal](javascript:call_link\('abencds_literal_v2.htm'\)), a field [field](javascript:call_link\('abencds_field_v2.htm'\)), a parameter [$parameters.pname](javascript:call_link\('abencds_parameter_v2.htm'\)), or a session variable [$session.vname](javascript:call_link\('abencds_session_variable_v2.htm'\)) as an elementary [CDS operand](javascript:call_link\('abencds_operand_glosry.htm'\) "Glossary Entry") in a CDS DDL operand position.
