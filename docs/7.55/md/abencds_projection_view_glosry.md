CDS projection view

Special [CDS view](javascript:call_link\('abencds_view_glosry.htm'\) "Glossary Entry") which is based on another [CDS view](javascript:call_link\('abencds_view_glosry.htm'\) "Glossary Entry") (called 'projected entity') and exposes only a subset of elements of the projected entity. A CDS projection view is defined with the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") statement [DEFINE VIEW ENTITY AS PROJECTION](javascript:call_link\('abencds_define_view_as_projection.htm'\)).
