class-based exception

[Catchable exception](javascript:call_link\('abencatchable_exception_glosry.htm'\) "Glossary Entry"), which is represented by an [exception object](javascript:call_link\('abenexception_object_glosry.htm'\) "Glossary Entry") of an [exception class](javascript:call_link\('abenexception_class_glosry.htm'\) "Glossary Entry"). Class-based exceptions replace the obsolete [catchable runtime errors](javascript:call_link\('abencatchable_runtime_error_glosry.htm'\) "Glossary Entry") and non-class-based exceptions. A class-based exception can either terminate the current context or be a [resumable exception](javascript:call_link\('abenresumable_exception_glosry.htm'\) "Glossary Entry").
