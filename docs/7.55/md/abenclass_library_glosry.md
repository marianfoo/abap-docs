class library

The class library consists of all [global](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenglobal_type_glosry.htm "Glossary Entry") [object types](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_type_glosry.htm "Glossary Entry"), that means of all [global classes](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenglobal_class_glosry.htm "Glossary Entry") and [interfaces](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenglobal_interface_glosry.htm "Glossary Entry").
