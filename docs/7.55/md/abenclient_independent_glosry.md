client-independent

A [SQL data source](javascript:call_link\('abensql_data_source_glosry.htm'\) "Glossary Entry") that is [client-independent](javascript:call_link\('abenclient_dependent_glosry.htm'\) "Glossary Entry"). Client-independent SQL data sources contain data that is not specific for a [client](javascript:call_link\('abenclient_glosry.htm'\) "Glossary Entry") and accessed generally by [system programs](javascript:call_link\('abensystem_program_glosry.htm'\) "Glossary Entry"). See also [system table](javascript:call_link\('abensystem_table_glosry.htm'\) "Glossary Entry").
