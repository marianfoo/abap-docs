client-server system

Software system that consists of multiple software layers. These are usually the [presentation layer](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpresentation_layer_glosry.htm "Glossary Entry"), the [application layer](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenapplication_layer_glosry.htm "Glossary Entry"), and the [database layer](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_layer_glosry.htm "Glossary Entry"). [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry") represents the application layer of a client-server in which [ABAP programs](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry") are executed.
