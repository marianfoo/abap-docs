CLOB

Short for Character Large Object. Name for a [database field](javascript:call_link\('abendatabase_field_glosry.htm'\) "Glossary Entry") with the data type [STRING](javascript:call_link\('abenddic_builtin_types.htm'\)) or the result of an [SQL expression](javascript:call_link\('abapsql_expr.htm'\)) that returns the data type [STRING](javascript:call_link\('abenddic_builtin_types.htm'\)).
