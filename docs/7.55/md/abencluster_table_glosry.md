cluster table

Before Release 7.53, [table category](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddic_database_tables_tab_type.htm) of a [DDIC database table](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddic_db_table_glosry.htm "Glossary Entry") in [ABAP Dictionary](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A cluster table was not defined as such on the [database](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_glosry.htm "Glossary Entry"). Instead, multiple cluster tables were stored in a [table cluster](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentable_cluster_glosry.htm "Glossary Entry") in the database. Only [ABAP SQL](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_glosry.htm "Glossary Entry") could be used to access cluster tables, whereby certain restrictions applied. Cluster tables are no longer supported as of Release 7.53. Any existing cluster tables were converted to [transparent tables](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentransparent_table_glosry.htm "Glossary Entry").
