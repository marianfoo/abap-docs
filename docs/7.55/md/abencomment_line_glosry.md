comment line

Line in a program that contains nothing but a [comment](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencomment_glosry.htm "Glossary Entry"). A comment line is either completely identified as a comment by an asterisk \* in the first position or is a line that contains nothing except a comment introduced by " in any position.
