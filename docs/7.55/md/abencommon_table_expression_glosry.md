Common Table Expression

CTE for short. Common table expressions are defined in the [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") statement [WITH](javascript:call_link\('abapwith.htm'\)) via [subqueries](javascript:call_link\('abensubquery_glosry.htm'\) "Glossary Entry"). The [main query](javascript:call_link\('abenmainquery_glosry.htm'\) "Glossary Entry") of the WITH statement accesses its common table expressions.
