comparison type

Property of [comparison expressions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencomparison_expression_glosry.htm "Glossary Entry") whose [operands](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenoperand_glosry.htm "Glossary Entry") have [elementary data types](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenelementary_data_type_glosry.htm "Glossary Entry"), including [calculation expressions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencalculation_expression_glosry.htm "Glossary Entry"). The comparison type is determined from the operands and defines how the comparison is executed. The comparison type can be of any [built-in ABAP type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbuiltin_abap_type_glosry.htm "Glossary Entry"). Operands of another type are converted to the comparison type before the comparison.
