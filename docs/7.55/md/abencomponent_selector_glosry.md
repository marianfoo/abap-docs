component selector

Defined characters that can be used to address [components](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencomponent_glosry.htm "Glossary Entry") of higher units. There is the [structure component selector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstructure_component_sel_glosry.htm "Glossary Entry") (\-), the [class component selector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_component_select_glosry.htm "Glossary Entry") (\=>), the [interface component selector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninterface_comp_selector_glosry.htm "Glossary Entry") (~), and the [object component selector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_component_select_glosry.htm "Glossary Entry") (\->). [ABAP SQL](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_glosry.htm "Glossary Entry") provides the additional [column selector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentable_comp_selector_glosry.htm "Glossary Entry") ~.
