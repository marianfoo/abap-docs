control element

Element in the [user interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_interface_glosry.htm "Glossary Entry") for which a [user action](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_action_glosry.htm "Glossary Entry") can be executed. Most [SAP GUI](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensap_gui_glosry.htm "Glossary Entry") UI elements can be linked to a [function code](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunction_code_glosry.htm "Glossary Entry").
