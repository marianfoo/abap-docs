control structure

Means of structuring a [processing block](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprocessing_block_glosry.htm "Glossary Entry") in [statement blocks](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstatement_block_glosry.htm "Glossary Entry"). Possible control structures are [sequence](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensequence_glosry.htm "Glossary Entry"), [branch](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbranch_glosry.htm "Glossary Entry"), and [loop](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenloop_glosry.htm "Glossary Entry") as well as special control structures for exception handling. Control structures can be nested, so that statement blocks can contain other control structures.
