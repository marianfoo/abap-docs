Assigning Structure Components

The following special language constructs are used to assign components from structures or columns of internal tables:

-   [MOVE-CORRESPONDING](javascript:call_link\('abapmove-corresponding.htm'\)) for the assignment of identically named components.

-   Component operator [CORRESPONDING](javascript:call_link\('abenconstructor_expr_corresponding.htm'\)) for the assignment of components using identical names or a static mapping rule.

-   System class [CL\_ABAP\_CORRESPONDING](javascript:call_link\('abencl_abap_corresponding.htm'\)) for dynamic mapping rules.
