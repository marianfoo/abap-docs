CTE association

[Association](javascript:call_link\('abenassociation_glosry.htm'\) "Glossary Entry") that is defined for a [common table expression](javascript:call_link\('abencommon_table_expression_glosry.htm'\) "Glossary Entry") ([CTE](javascript:call_link\('abencte_glosry.htm'\) "Glossary Entry")) and can be used in the [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") of its [WITH](javascript:call_link\('abapwith.htm'\)) statement in [SQL path expressions](javascript:call_link\('abensql_path_expression_glosry.htm'\) "Glossary Entry"). A CTE association is defined in the definition of the common table expression using the addition [WITH ASSOCIATIONS JOIN](javascript:call_link\('abapwith_associations_defining.htm'\)).
