CTE association path

Sequence of [CTE associations](javascript:call_link\('abencte_association_glosry.htm'\) "Glossary Entry") that link [common table expressions](javascript:call_link\('abencommon_table_expression_glosry.htm'\) "Glossary Entry") with [association targets](javascript:call_link\('abenassociation_target_glosry.htm'\) "Glossary Entry"). CTE association paths can be specified in the form of [SQL path expressions](javascript:call_link\('abensql_path_expression_glosry.htm'\) "Glossary Entry") in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry").
