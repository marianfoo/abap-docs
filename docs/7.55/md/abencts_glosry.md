Change and Transport System

Abbreviated to CTS. The Change and Transport System encompasses all [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") tools used to manage and transport [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") and other objects between different [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") systems. A repository object is connected to CTS using its [package](javascript:call_link\('abenpackage_glosry.htm'\) "Glossary Entry").
