data type

Property of a [data object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_object_glosry.htm "Glossary Entry"). ABAP interprets the content of a data object in accordance with its data type. Data types exist either [bound](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbound_data_type_glosry.htm "Glossary Entry") as a property of data objects or [standalone](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstand-alone_data_type_glosry.htm "Glossary Entry"). Standalone data types can be defined globally in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") or locally using [TYPES](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abaptypes.htm) in an [ABAP program](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry"). The [generic data type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") of all data types is [data](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbuilt_in_types_generic.htm).
