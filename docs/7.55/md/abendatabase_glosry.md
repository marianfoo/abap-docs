database

System for storing and managing data between which logical dependencies exist. The data in the database of an [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry") is stored on the [database server](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_server_glosry.htm "Glossary Entry") and is managed by the [database system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_system_glosry.htm "Glossary Entry"). See also [relational database](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrelational_database_glosry.htm "Glossary Entry").
