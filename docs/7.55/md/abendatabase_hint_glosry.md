database hints

Extension of the SQL standard that allows the informing of the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") on how access should be executed. Database hints are usually dependent on the database system and its release. In [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), database hints can be specified using the addition [%\_HINTS](javascript:call_link\('abenosql_db_hints.htm'\)).
