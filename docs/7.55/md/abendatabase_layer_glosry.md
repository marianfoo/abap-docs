database layer

Software layer of a [client-server system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclient_server_system_glosry.htm "Glossary Entry") in which persistent data is stored and managed. The [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry") of an [ABAP system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_system_glosry.htm "Glossary Entry") is always connected to a database layer in the form of a [database server](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_server_glosry.htm "Glossary Entry") that contains a [database system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendatabase_system_glosry.htm "Glossary Entry") with the [standard AS ABAP database](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstandard_db_glosry.htm "Glossary Entry").
