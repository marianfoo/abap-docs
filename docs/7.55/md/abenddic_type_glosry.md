DDIC type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") defined as a [global type](javascript:call_link\('abenglobal_type_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). DDIC types are always data types, while global [object types](javascript:call_link\('abenobject_type_glosry.htm'\) "Glossary Entry") are defined in the [class library](javascript:call_link\('abenclass_library_glosry.htm'\) "Glossary Entry"). In particular [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") and [DDIC views](javascript:call_link\('abendictionary_view_glosry.htm'\) "Glossary Entry") can be used as DDIC types in ABAP programs.
