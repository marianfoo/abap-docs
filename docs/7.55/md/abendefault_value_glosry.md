default value

A value that appears in an [input field](javascript:call_link\('abeninput_output_field_glosry.htm'\) "Glossary Entry") when a [screen](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry") is called and can be overwritten by the user. A default value is usually set in the corresponding [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") before the screen is called.
