dependency rule

[Database object](javascript:call_link\('abendb_object_glosry.htm'\) "Glossary Entry") of the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). A dependency rule can be defined for [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") and derives additional conditions from a selection condition, with which the optimizer of the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") extends the read access to these database tables. On an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"), dependency rules can be defined in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry").
