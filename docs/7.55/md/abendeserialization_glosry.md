deserialization

Transformation of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data or [JSON](javascript:call_link\('abenjson_glosry.htm'\) "Glossary Entry") data to ABAP data objects. Performed either by calling an [XSLT program](javascript:call_link\('abenxsl_transformation_glosry.htm'\) "Glossary Entry") or a [simple transformation](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry"). If use [XSLT](javascript:call_link\('abenxslt_glosry.htm'\) "Glossary Entry") is used, the result of the transformation (in XML) must be a [canonical XML representation](javascript:call_link\('abencanonical_xml_represent_glosry.htm'\) "Glossary Entry") (([asXML](javascript:call_link\('abenasxml_glosry.htm'\) "Glossary Entry")).
