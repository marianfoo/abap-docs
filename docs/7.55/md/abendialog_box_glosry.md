dialog box

Also referred to as a modal dialog box. Display of the [GUI window](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abengui_window_glosry.htm "Glossary Entry") of a [dynpro sequence](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendynpro_sequence_glosry.htm "Glossary Entry") called with [CALL SCREEN](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcall_screen.htm), where the previous GUI window remains visible, although it is inactive. Up to nine dialog boxes can be stacked over a GUI window (see [popup level](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpop-up_level_glosry.htm "Glossary Entry")). Modeless dialog boxes, where the previous GUI window remains active, can be implemented in [SAP GUI](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensap_gui_glosry.htm "Glossary Entry") only as [GUI controls](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abengui_control_glosry.htm "Glossary Entry").
