double-byte code

Representation of a character in two bytes. Was used before the introduction of [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") for representing Asian characters.
