enhancement option

Position in a [Repository object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrepository_object_glosry.htm "Glossary Entry") that can be enhanced in accordance with the [enhancement concept](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_concept_glosry.htm "Glossary Entry"). Enhancement options can be explicitly defined or implicitly available. Explicitly defined enhancement options are implemented using [enhancement spot element definitions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_spot_def_glosry.htm "Glossary Entry") and [enhancement spot invocations](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_spot_invoc_glosry.htm "Glossary Entry") and are administrated using [enhancement spots](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_spot_glosry.htm "Glossary Entry").
