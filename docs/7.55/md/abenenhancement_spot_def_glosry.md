enhancement spot element definition

Part of the definition of an explicit [enhancement option](javascript:call_link\('abenenhancement_point_glosry.htm'\) "Glossary Entry"). An enhancement spot element definition must be assigned to at least one [enhancement spot](javascript:call_link\('abenenhancement_spot_glosry.htm'\) "Glossary Entry").
