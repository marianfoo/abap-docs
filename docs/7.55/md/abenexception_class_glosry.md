exception class

Special [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that is the basis for [catchable](javascript:call_link\('abencatchable_exception_glosry.htm'\) "Glossary Entry") exceptions. If an [exception](javascript:call_link\('abenexception_glosry.htm'\) "Glossary Entry") is raised, an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of an exception class is created. There are predefined exception classes for exceptions of the runtime environment, and self-defined exception classes for custom applications. The superclasses for all exception classes are CX\_STATIC\_CHECK, CX\_DYNAMIC\_CHECK, or CX\_NO\_CHECK, through which exceptions are divided into [exception categories](javascript:call_link\('abenexception_category_glosry.htm'\) "Glossary Entry").
