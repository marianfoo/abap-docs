exception group

Grouping of multiple [catchable runtime errors](javascript:call_link\('abencatchable_runtime_error_glosry.htm'\) "Glossary Entry") in an organizational unit. The catchable runtime errors of an exception group, together with the obsolete statement [CATCH - ENDCATCH](javascript:call_link\('abapcatch_sys.htm'\)), can be treated by specifying the group name.
