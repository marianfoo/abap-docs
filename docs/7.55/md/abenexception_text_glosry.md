exception text

Text that describes an exception and is specified in an [exception class](javascript:call_link\('abenexception_class_glosry.htm'\) "Glossary Entry"). An exception text can be given parameters using the attributes of the exception class. Exception texts are usually defined with reference to [messages](javascript:call_link\('abenmessage_glosry.htm'\) "Glossary Entry") and in [system classes](javascript:call_link\('abensystem_class_glosry.htm'\) "Glossary Entry") with reference to the Online Text Repository (OTR).
