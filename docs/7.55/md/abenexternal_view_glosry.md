DDIC external view

A special [dictionary view](javascript:call_link\('abendictionary_view_glosry.htm'\) "Glossary Entry") that defines an [SAP HANA view](javascript:call_link\('abenhana_view_glosry.htm'\) "Glossary Entry") of the [SAP HANA Repository](javascript:call_link\('abensap_hana_repository_glosry.htm'\) "Glossary Entry") in ABAP programs. A DDIC external view can be used like a [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry") and specified as a source in reading [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") statements.
