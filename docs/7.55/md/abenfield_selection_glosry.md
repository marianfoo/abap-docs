field selection

Property of a node of a [logical database](javascript:call_link\('abenlogical_data_base_glosry.htm'\) "Glossary Entry"). If a node is defined for field selection, it is possible to control which [fields](javascript:call_link\('abenfield_glosry.htm'\) "Glossary Entry") of the node are supposed to be read from outside the logical database. Field selections are declared using the addition [FIELD SELECTION](javascript:call_link\('abapselection-screen_ldb_field.htm'\)) of the statement [SELECTION-SCREEN](javascript:call_link\('abapselection-screen.htm'\)) in the logical database.
