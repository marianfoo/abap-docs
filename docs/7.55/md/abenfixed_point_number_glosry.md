fixed point number

Number with a fixed number of [decimal places](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendecimal_place_glosry.htm "Glossary Entry"). In ABAP, fixed point numbers are implemented using [packed numbers](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpacked_number_glosry.htm "Glossary Entry") for which a string of digits in a position defined by the [data type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_type_glosry.htm "Glossary Entry") contains a [decimal separator](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendecimal_point_separat_glosry.htm "Glossary Entry") .
