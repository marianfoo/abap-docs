floating point function

[Numeric function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmathematical_funktion_glosry.htm "Glossary Entry") that expects a [floating point number](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfloating_point_number_glosry.htm "Glossary Entry") as an argument and returns a floating point number as a [return value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenreturn_value_glosry.htm "Glossary Entry").
