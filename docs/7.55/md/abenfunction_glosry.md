function

A function calculates a [return value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenreturn_value_glosry.htm "Glossary Entry") from one or more arguments. In ABAP, functions are either predefined as [built-in functions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbuiltin_function_glosry.htm "Glossary Entry") or can be defined as [functional methods](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunctional_method_glosry.htm "Glossary Entry").
