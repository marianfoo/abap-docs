functional method

[Method](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmethod_glosry.htm "Glossary Entry") with only one [return value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenreturn_value_glosry.htm "Glossary Entry") declared using [RETURNING](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapmethods_functional.htm) and any number of [formal parameters](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenformal_parameter_glosry.htm "Glossary Entry"). Can be used in [functional method calls](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunctional_method_call_glosry.htm "Glossary Entry").
