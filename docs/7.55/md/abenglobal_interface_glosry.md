global interface

[Interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") that is defined as a [global type](javascript:call_link\('abenglobal_type_glosry.htm'\) "Glossary Entry"). Global interfaces are declared in the [interface pools](javascript:call_link\('abeninterface_pool_glosry.htm'\) "Glossary Entry") and are marked as such there using the addition PUBLIC of the statement [INTERFACE](javascript:call_link\('abapinterface.htm'\)). An interface pool cannot contain anything except for a global interface.
