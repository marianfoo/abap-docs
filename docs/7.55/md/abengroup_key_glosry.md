group key

Value of any [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that is constructed for each read line in a [group key expression](javascript:call_link\('abengroup_key_expression_glosry.htm'\) "Glossary Entry") when grouping [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") using [LOOP AT itab ... GROUP BY](javascript:call_link\('abaploop_at_itab_group_by.htm'\)) or [FOR GROUPS ... OF](javascript:call_link\('abenfor_groups_of.htm'\)). All table lines for which the same value is produced are assigned to each group key. In [group level processing](javascript:call_link\('abengroup_level_processing_glosry.htm'\) "Glossary Entry") for internal tables, the group key is the content of all columns to the left of a column specified after [AT](javascript:call_link\('abapat_itab.htm'\)).
