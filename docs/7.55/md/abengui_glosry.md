GUI

Short for Graphical User Interface. Graphical part of the [user interface](javascript:call_link\('abenuser_interface_glosry.htm'\) "Glossary Entry"). The GUI is displayed in a [window](javascript:call_link\('abenwindow_glosry.htm'\) "Glossary Entry").
