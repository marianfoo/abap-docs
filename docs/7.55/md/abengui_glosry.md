GUI

Short for Graphical User Interface. Graphical part of the [user interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_interface_glosry.htm "Glossary Entry"). The GUI is displayed in a [window](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenwindow_glosry.htm "Glossary Entry").
