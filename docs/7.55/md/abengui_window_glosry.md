GUI window

[Window](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenwindow_glosry.htm "Glossary Entry") of the [SAP GUI](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensap_gui_glosry.htm "Glossary Entry"). Consists of a [title bar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentitle_bar_glosry.htm "Glossary Entry"), [menu bar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmenu_bar_glosry.htm "Glossary Entry"), [standard toolbar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstandard_toolbar_glosry.htm "Glossary Entry"), [application toolbar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenapplication_toolbar_glosry.htm "Glossary Entry"), [screen](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenscreen_glosry.htm "Glossary Entry"), [status bar](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstatus_bar_glosry.htm "Glossary Entry"), and any [scroll bars](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenscroll_bar_glosry.htm "Glossary Entry"). Each GUI window is associated with an [ABAP session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_session_glosry.htm "Glossary Entry"). A maximum of 16 ABAP sessions and therefore GUI windows can be managed by a single [user session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_session_glosry.htm "Glossary Entry").
