ICF server

Instance of a system that provides services that can be called using [ICF](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenicf_glosry.htm "Glossary Entry"). An ICF server can be an HTTP/HTTPS server or an SMTP server.
