include program

[ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that cannot be compiled independently but must be included into another ABAP program using the statement [INCLUDE](javascript:call_link\('abapinclude_prog.htm'\)). For compilation, an include program must be part of a [compilation unit](javascript:call_link\('abencompilation_unit_glosry.htm'\) "Glossary Entry"). An include program should only be assigned to a single compilation unit.
