indicator structure

[Substructure](javascript:call_link\('abensubstructure_glosry.htm'\) "Glossary Entry") of a [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry"). The indicator structure is the last component of the structured type and mirrors the preceding components as indicators of type x of length 1. Indicator structures can be defined with the addition [WITH INDICATORS](javascript:call_link\('abaptypes_indicators.htm'\)) of the statement [TYPES](javascript:call_link\('abaptypes_indicators.htm'\)) and can be used as [ABAP SQL indicators](javascript:call_link\('abenabap_sql_indicator_glosry.htm'\) "Glossary Entry").
