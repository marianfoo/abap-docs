inner join

[Join type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenjoin_type_glosry.htm "Glossary Entry") for which each row of the result set of the [join](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenjoin_glosry.htm "Glossary Entry") fulfills the [join conditions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenjoin_condition_glosry.htm "Glossary Entry").
