instance attribute

[Attribute](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenattribute_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_glosry.htm "Glossary Entry") that is declared using [DATA](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapdata.htm). Instance attributes are only valid in the [context](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobj_context_glosry.htm "Glossary Entry") of an [instance](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninstance_glosry.htm "Glossary Entry") of the class. The content of the instance attributes determines the status of the [objects](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_glosry.htm "Glossary Entry") of a class.
