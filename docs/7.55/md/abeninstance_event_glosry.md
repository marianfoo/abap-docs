instance event

[Event](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenevent_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_glosry.htm "Glossary Entry") declared using [EVENTS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapevents.htm). Instance events usually indicate status changes of [objects](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_glosry.htm "Glossary Entry").
