interface component

[Component](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry") of an [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry"). Declared in the [declaration part](javascript:call_link\('abendeclaration_part_glosry.htm'\) "Glossary Entry") of an interface. Possible interface components are: [Attributes](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry"), [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry"), and [events](javascript:call_link\('abenevent_glosry.htm'\) "Glossary Entry").
