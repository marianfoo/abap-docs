interface reference variable

[Object reference variable](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_refer_variable_glosry.htm "Glossary Entry") of the [static type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstatic_type_glosry.htm "Glossary Entry") of an [interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenoo_intf_glosry.htm "Glossary Entry"). An interface reference variable can be used to access the visible [interface components](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninterface_component_glosry.htm "Glossary Entry") of an object that exist in the corresponding interface. This applies to both dynamic and static access.
