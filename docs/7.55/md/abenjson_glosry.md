JSON

JavaScript Object Notation ([http://www.json.org/](http://www.json.org/)). Data format in text form for data exchange between applications. In ABAP, the [sXML Library](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensxml_library_glosry.htm "Glossary Entry") and [transformations](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_json_trafos.htm) can be used to access JSON, for which a special JSON-XML supported by [sXML](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenjson_xml_glosry.htm "Glossary Entry") Library is used.
