JSON reader

[XML](javascript:call_link\('abenxml_reader_glosry.htm'\) "Glossary Entry") reader that accesses [JSON](javascript:call_link\('abenjson_glosry.htm'\) "Glossary Entry") data. A JSON reader handles the JSON data internally like [JSON-XML](javascript:call_link\('abenjson_xml_glosry.htm'\) "Glossary Entry").
