LDB - Statements for Logical Databases

The following additions and statements are intended for use in executable programs associated with logical databases only:

-   Addition [NO DATABASE SELECTION](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselect-options_no_db_sel.htm) of the statement SELECT-OPTIONS

-   Event block [GET node](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapget-.htm)

-   Event block [END-OF-SELECTION](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapend-of-selection.htm)

-   Statement [CHECK SELECT-OPTIONS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapcheck_select-options.htm)

-   Statement [REJECT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapreject.htm)

The following addition is only intended for calling an executable program that is associated with a logical database:

-   Statement [SUBMIT WITH FREE SELECTIONS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapsubmit_selscreen_parameters.htm)

If logical databases are no longer used, these language elements are obsolete.
