lock object

[Repository object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrepository_object_glosry.htm "Glossary Entry") defined in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") that is used as the basis for [SAP locks](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensap_lock_glosry.htm "Glossary Entry"). In a lock object [DDIC database tables](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddic_db_table_glosry.htm "Glossary Entry") are specified to which a common [lock](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlock_glosry.htm "Glossary Entry") is to be applied and which are linked by [foreign key dependencies](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenforeign_key_dependency_glosry.htm "Glossary Entry"). When a lock object is created, two [lock function modules](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlock_function_module_glosry.htm "Glossary Entry") are automatically generated.
