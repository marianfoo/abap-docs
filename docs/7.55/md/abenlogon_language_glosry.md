logon language

Language that a [user](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_glosry.htm "Glossary Entry") specifies when logging on to an [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry"). The logon language is persistent for the entire [user session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_session_glosry.htm "Glossary Entry"). The logon language affects [messages](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmessage_glosry.htm "Glossary Entry"), [text elements](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentext_element_glosry.htm "Glossary Entry"), and the [text environment](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentext_environment_glosry.htm "Glossary Entry"). It can be determined using the method GET\_LOGON\_LANGUAGE of the class CL\_ABAP\_SYST.
