managed RAP Business Object

Managed RAP BO for short. An implementation type of the [business object behavior](javascript:call_link\('abenrap_bo_behavior_glosry.htm'\) "Glossary Entry") in the context of [ABAP RAP](javascript:call_link\('abenabap_rap_glosry.htm'\) "Glossary Entry"). The implementation type managed is suitable if a completely new business logic is to be implemented. In this case, a [behavior definition](javascript:call_link\('abencds_behavior_definition_glosry.htm'\) "Glossary Entry") is sufficient to get a finished and executable [business object](javascript:call_link\('abenbehavior_implement_glosry.htm'\) "Glossary Entry").
