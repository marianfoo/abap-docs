[Processing function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprocess_function_glosry.htm "Glossary Entry") that returns a substring of a character string that matches a [regular expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenregular_expression_glosry.htm "Glossary Entry").
