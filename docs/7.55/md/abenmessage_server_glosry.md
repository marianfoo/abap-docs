message server

Component of the [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") that implements the communication between the [AS instances](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry"), and which determines the AS instance used when a [user](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry") logs on to an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry").
