Logical Expressions in Release 7.0, EhP2

[1\. Boolean functions](#!ABAP_MODIFICATION_1@1@)
[2\. Predicate functions](#!ABAP_MODIFICATION_2@2@)
[3\. New Boolean operator EQUIV](#!ABAP_MODIFICATION_3@3@)

Hint

The program DEMO\_EXPRESSIONS shows examples of the logical expressions in Release 7.0, EhP2.

Modification 1

Boolean Functions

The new built-in [Boolean functions](javascript:call_link\('abenboole_functions.htm'\)) boolc and boolx accept logical expressions as arguments and return their truth values in the form of character strings or byte strings.

Modification 2

Predicate Functions

The new [predicate functions](javascript:call_link\('abenpredicate_functions.htm'\)) return truth values and can be used like relational expressions. Predicate functions for character-like arguments were introduced in Release 7.0, EhP2.

Modification 3

New Boolean Operator EQUIV

The new Boolean operator [EQUIV](javascript:call_link\('abenlogexp_equiv.htm'\)) enables an equivalence relationship between two [logical expressions](javascript:call_link\('abenlogical_expression_glosry.htm'\) "Glossary Entry").
