non-blocking mode

Any statements that block the program flow are not allowed in an [ABAP programs](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry") that is processed in non-blocking mode. There are values of the non-blocking mode for [APC](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenapc_non_blocking_model.htm) and [ADF](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenadf_non_blocking_model.htm).
