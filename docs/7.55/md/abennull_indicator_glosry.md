null indicator

[ABAP SQL indicator](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_indicator_glosry.htm "Glossary Entry") in a structured target area of an [ABAP SQL](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_glosry.htm "Glossary Entry") [query](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenquery_glosry.htm "Glossary Entry"). The components of the null indicator indicate which columns of the result set contain the [null value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abennull_value_glosry.htm "Glossary Entry") and which do not.
