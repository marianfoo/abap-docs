Numeric operand position

[Operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry") with numeric [operand type](javascript:call_link\('abenoperand_type_glosry.htm'\) "Glossary Entry"), in which a [numeric data object](javascript:call_link\('abennumeric_data_object_glosry.htm'\) "Glossary Entry") that can be converted into the operand type can be specified. See also [numeric expression position](javascript:call_link\('abennumerical_expr_position_glosry.htm'\) "Glossary Entry").
