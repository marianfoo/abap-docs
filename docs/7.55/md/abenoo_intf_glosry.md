OO Interface

Interface for short. Template for the [public](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpublic_glosry.htm "Glossary Entry") [interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenintf_gen_glosry.htm "Glossary Entry") of [classes](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_glosry.htm "Glossary Entry") in [ABAP Objects](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_objects_glosry.htm "Glossary Entry"). Defined with [INTERFACE - ENDINTERFACE](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapinterface.htm), either as a [global interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenglobal_interface_glosry.htm "Glossary Entry") in an [interface pool](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninterface_pool_glosry.htm "Glossary Entry") or as a [local interface](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlocal_interface_glosry.htm "Glossary Entry") in another [ABAP program](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry"). An interface contains the declaration of [interface components](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninterface_component_glosry.htm "Glossary Entry") but does not contain method implementations. Can be implemented in classes using [INTERFACES](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapinterfaces.htm), which enhances the external interface of the class by adding interface components. OO interfaces are generally referred to as interfaces, unless the distinction to [general interfaces](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenintf_gen_glosry.htm "Glossary Entry") needs to be made.
