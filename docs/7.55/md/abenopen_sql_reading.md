ABAP SQL - Read Access

-   [SELECT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselect.htm)

-   [WITH](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapwith.htm)

-   [CLOSE CURSOR](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapclose_cursor.htm)
