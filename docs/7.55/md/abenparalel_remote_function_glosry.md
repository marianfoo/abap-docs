parallel Remote Function Call

pRFC for short. Extension of the [asynchronous remote function call](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenasynchronous_rfc_glosry.htm "Glossary Entry") to enable the controlling of parallelization.
