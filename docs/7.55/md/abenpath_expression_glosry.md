path expression

A sequence of [associations](javascript:call_link\('abenassociation_glosry.htm'\) "Glossary Entry") specified at an operand position. Either an [SQL path expression](javascript:call_link\('abensql_path_expression_glosry.htm'\) "Glossary Entry") or a [mesh path expression](javascript:call_link\('abenmesh_path_expression_glosry.htm'\) "Glossary Entry").
