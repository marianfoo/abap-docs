path

Sequence of relationships that link objects. In ABAP, sequences of [associations](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenassociation_glosry.htm "Glossary Entry") have a important consequences. Paths like these can be specified using [path expressions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpath_expression_glosry.htm "Glossary Entry"). See [CDS association path](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencds_association_path_glosry.htm "Glossary Entry"), [CDS composition path](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencds_composition_path_glosry.htm "Glossary Entry"), and [mesh path](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmesh_path_glosry.htm "Glossary Entry").
