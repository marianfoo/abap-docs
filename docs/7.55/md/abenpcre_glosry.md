Perl Compatible Regular Expression

PCRE for short. [Regular expressions](javascript:call_link\('abenregular_expression_glosry.htm'\) "Glossary Entry") using the same syntax and semantics as [Perl 5](https://www.perl.org/). In general, ABAP supports PCRE by implementing the PCRE2 library in the [ABAP kernel](javascript:call_link\('abenkernel_glosry.htm'\) "Glossary Entry"). [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") supports PCRE by using the PCRE1 library implemented in the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry").
