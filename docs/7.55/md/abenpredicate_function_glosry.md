predicate function

[Logical function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlogic_function_glosry.htm "Glossary Entry") that returns a [truth value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentruth_value_glosry.htm "Glossary Entry") and can be used in the same way as a [relational expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrelational_expression_glosry.htm "Glossary Entry").
