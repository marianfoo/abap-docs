primary key

[Table key](javascript:call_link\('abentable_key_glosry.htm'\) "Glossary Entry") that is determined when a [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") is defined and which uniquely identifies a table row. In [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry"), there is a [primary table key](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry").
