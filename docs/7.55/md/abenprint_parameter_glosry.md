spool parameter

Parameters that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenspool_request_glosry.htm "Glossary Entry"). Spool parameters can be extended with [archiving parameters](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenarchiving_parameter_glosry.htm "Glossary Entry").
