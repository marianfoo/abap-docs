pseudo comment

[Program directive](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprogram_directive_glosry.htm "Glossary Entry") in the form of a special [line end comment](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenendline_comment_glosry.htm "Glossary Entry"). A pseudo comment begins with the expression #EC or #AU and suppresses either a message of the [extended program check](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenextended_program_check_glosry.htm "Glossary Entry") or of the [Code Inspector](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencode_inspector_glosry.htm "Glossary Entry"), or determines the test properties for a [test class](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentest_class_glosry.htm "Glossary Entry"). Pseudo comments are only still required for suppressing Code Inspector messages and are otherwise obsolete.
