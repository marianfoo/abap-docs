Push Channel Protocol

PCP for short. Describes an SAP communications format used to exchange messages between programs and internal or external interfaces. Currently, PCP is used by [ABAP Channels](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_channel_glosry.htm "Glossary Entry") and [ABAP Daemons](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_daemon_glosry.htm "Glossary Entry").
