queued Remote Function Call

qRFC for short. Obsolete extension of the [transactional remote function call](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentrfc_1_glosry.htm "Glossary Entry") to be able to determine a call sequence.
