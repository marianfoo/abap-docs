relational database

[Database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry") whose data is managed according to the [relational data model](javascript:call_link\('abenrelational_data_model_glosry.htm'\) "Glossary Entry"), that is, in [database tables](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry"). Further objects such as [views](javascript:call_link\('abenview_glosry.htm'\) "Glossary Entry") or [stored procedures](javascript:call_link\('abenstored_procedure_glosry.htm'\) "Glossary Entry") can be created to access the data. These objects are organized in [database schemas](javascript:call_link\('abendatabase_schema_glosry.htm'\) "Glossary Entry"). The objects are generally defined and accessed using [SQL](javascript:call_link\('abensql_glosry.htm'\) "Glossary Entry").
