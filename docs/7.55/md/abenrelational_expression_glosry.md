relational expression

[Expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenexpression_glosry.htm "Glossary Entry") whose result is a [truth value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentruth_value_glosry.htm "Glossary Entry"). Relational expressions include [comparison expressions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencomparison_expression_glosry.htm "Glossary Entry") and [predicates](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpredicate_glosry.htm "Glossary Entry"). A single relational expression or multiple relational expressions joined using [Boolean operators](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenboolean_operator_glosry.htm "Glossary Entry") form an ABAP [logical expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenlogical_expression_glosry.htm "Glossary Entry").
