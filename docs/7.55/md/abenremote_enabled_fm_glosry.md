Remote-enabled Function Module

[RFM](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrfm_glosry.htm "Glossary Entry") for short. [Function module](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunction_module_glosry.htm "Glossary Entry") that is marked as remote-enabled in its properties and can be called using [remote function call](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenremote_function_call_glosry.htm "Glossary Entry").
