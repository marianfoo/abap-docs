remote user

[User](javascript:call_link\('abenuser_glosry.htm'\) "Glossary Entry") that is logged on to an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") via the [remote function call](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry"). The logon data and the [authorization](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry") can be declared in the [RFC destination](javascript:call_link\('abenrfc_dest_glosry.htm'\) "Glossary Entry").
