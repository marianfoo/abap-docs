RFC - Authorizations

If the [profile parameter](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprofile_parameter_glosry.htm "Glossary Entry") auth/rfc\_authority\_check is set to 1, an automatic authorization check is carried out when a function module is called by remote call.

More Information

Detailed information about RFC authorizations can be found in RFC documentation in SAP Help Portal.
