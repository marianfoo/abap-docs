RFC destination

An RFC destination contains the connection parameters for [remote function calls](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry"). It contains the connection type, the target system, and the partner program. Connections that only affect another [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") can be classified as [trusted systems](javascript:call_link\('abentrusted_system_glosry.htm'\) "Glossary Entry"). RFC destinations are managed statically using the [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SM59 or dynamically using the class CL\_DYNAMIC\_DESTINATION.
