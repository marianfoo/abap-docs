Run Time Type Identification

RTTI for short. Determination of [data types](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_type_glosry.htm "Glossary Entry") during program runtime. Implemented using description methods in [type description classes](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentype_class_glosry.htm "Glossary Entry"). Constitutes [RTTS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrun_time_type_services_glosry.htm "Glossary Entry") together with [RTTC](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrun_time_type_creation_glosry.htm "Glossary Entry").
