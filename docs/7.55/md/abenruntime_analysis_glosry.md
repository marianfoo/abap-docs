runtime analysis

Tool for analyzing the execution of program parts or individual statements and for measuring their runtime in SAP GUI. Called using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SAT. Replaced by the ABAP Profiler in the [ABAP Development Tools (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry").
