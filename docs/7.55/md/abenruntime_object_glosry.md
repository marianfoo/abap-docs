runtime object

Summary of all relevant properties of a [data type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_type_glosry.htm "Glossary Entry") defined in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A runtime object is created when a Dictionary object is activated and is stored as an entry in the [name table](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenname_table_glosry.htm "Glossary Entry") ([nametab](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenname_tab_glosry.htm "Glossary Entry")). The [ABAP runtime environment](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_runtime_envir_glosry.htm "Glossary Entry") accesses the runtime object when types from the ABAP Dictionary are used.
