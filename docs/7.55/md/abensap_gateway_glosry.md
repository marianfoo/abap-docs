SAP Gateway

Framework that makes it possible to access [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") using standardized open protocols such as [OData](javascript:call_link\('abenodata_glosry.htm'\) "Glossary Entry"). [Business services](javascript:call_link\('abenbusiness_service_glosry.htm'\) "Glossary Entry") and [RAP business objects](javascript:call_link\('abenrap_bo_glosry.htm'\) "Glossary Entry") based on the data and behavior modeled in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") use SAP Gateway internally as part of the [RAP runtime framework](javascript:call_link\('abenrap_runtime_framework_glosry.htm'\) "Glossary Entry").
