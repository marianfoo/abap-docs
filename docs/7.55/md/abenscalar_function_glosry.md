scalar function

In the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"), a [database function](javascript:call_link\('abendatabase_function_glosry.htm'\) "Glossary Entry") with a scalar or elementary return value. In [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry"), scalar functions can be used as operands in the same places as elementary variables or columns of database tables or views. A scalar function can be managed as an [AMDP function](javascript:call_link\('abenamdp_function_glosry.htm'\) "Glossary Entry") in an [AMDP class](javascript:call_link\('abenamdp_class_glosry.htm'\) "Glossary Entry").
