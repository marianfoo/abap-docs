secondary language

Language specified in the [profile parameter](javascript:call_link\('abenprofile_parameter_glosry.htm'\) "Glossary Entry") zcsa/second\_language. The secondary language is used for [messages](javascript:call_link\('abenmessage_glosry.htm'\) "Glossary Entry"), [text pools](javascript:call_link\('abentext_pool_glosry.htm'\) "Glossary Entry"), and translatable [CDS annotations](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") if the desired language is not found.
