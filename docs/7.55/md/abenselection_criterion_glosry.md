selection criterion

Component of a [selection screen](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenselection_screen_glosry.htm "Glossary Entry") defined using [SELECT-OPTIONS](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapselect-options.htm). Displayed on the selection screen via two [input fields](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninput_output_field_glosry.htm "Glossary Entry") for interval boundaries and a pushbutton for [multiple selection](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmultiple_selection_glosry.htm "Glossary Entry"). It is represented internally by a [selection table](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenselection_table_glosry.htm "Glossary Entry").
