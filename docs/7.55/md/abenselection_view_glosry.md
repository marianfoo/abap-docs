selection view

If [dynamic selections](javascript:call_link\('abendynamic_selection_glosry.htm'\) "Glossary Entry") are required for a node of a [logical database](javascript:call_link\('abenlogical_data_base_glosry.htm'\) "Glossary Entry"), a selection view can be used to determine the node [fields](javascript:call_link\('abenfield_glosry.htm'\) "Glossary Entry") for which the user can define dynamic selections.
