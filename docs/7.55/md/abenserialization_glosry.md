serialization

Transformation of ABAP data into [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data or [JSON](javascript:call_link\('abenjson_glosry.htm'\) "Glossary Entry") data. Performed either by calling an [XSLT program](javascript:call_link\('abenxsl_transformation_glosry.htm'\) "Glossary Entry") or a [simple transformation](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry"). If [XSLT](javascript:call_link\('abenxslt_glosry.htm'\) "Glossary Entry") is used, the system first creates a [canonical XML representation](javascript:call_link\('abencanonical_xml_represent_glosry.htm'\) "Glossary Entry") ([asXML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry")).
