session

Organizational unit of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") for executing ABAP programs. An ABAP program is executed in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") in an [ABAP session](javascript:call_link\('abenabap_session_glosry.htm'\) "Glossary Entry"), which itself is in a [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry"). Within the program, there is access to memory areas assigned to the sessions and to the [shared memory](javascript:call_link\('abenshared_memory_glosry.htm'\) "Glossary Entry") of the current [AS instance](javascript:call_link\('abenapplication_server_glosry.htm'\) "Glossary Entry").
