sibling node

Sibling nodes are all [child nodes](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenchild_node_glosry.htm "Glossary Entry") of a [hierarchy](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenhierarchy_glosry.htm "Glossary Entry") that have the same [parent node](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenparent_node_glosry.htm "Glossary Entry"). In an [SQL hierarchy](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensql_hierarchy_glosry.htm "Glossary Entry"), sibling nodes have the same values in the [hierarchy column](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenhierarchy_column_glosry.htm "Glossary Entry") [HIERARCHY\_PARENT\_RANK](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenddddl_hierarchy.htm).
