soft reference

Reference to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") that does not keep an object alive when the available memory becomes insufficient. The class CL\_ABAP\_SOFT\_REFERENCE is intended for soft references but has not yet been implemented. See also [weak reference](javascript:call_link\('abenweak_reference_glosry.htm'\) "Glossary Entry").
