source code plug-in

Source code plug-ins are used to make modification-free [enhancements](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_glosry.htm "Glossary Entry") to [ABAP source code](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_source_code_glosry.htm "Glossary Entry"). Source code plug-ins are included in code by means of the statements [ENHANCEMENT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapenhancement.htm) and [ENDENHANCEMENT](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapendenhancement.htm) and are created using the [Enhancement Builder](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenenhancement_builder_glosry.htm "Glossary Entry") tool.
