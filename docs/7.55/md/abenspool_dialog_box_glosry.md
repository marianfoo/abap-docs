spool dialog box

Also referred to as print dialog box. Standard [dialog box](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendialog_box_glosry.htm "Glossary Entry") for entering [spool parameters](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenprint_parameter_glosry.htm "Glossary Entry"). The spool dialog box is either called automatically or using the [function module](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenfunction_module_glosry.htm "Glossary Entry") GET\_PRINT\_PARAMETERS.
