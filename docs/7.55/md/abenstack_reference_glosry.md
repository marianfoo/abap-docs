stack reference

[Reference](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") to an [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") or part of an object in the [stack](javascript:call_link\('abenstack_glosry.htm'\) "Glossary Entry"). A stack reference does not keep an object alive. See also [heap reference](javascript:call_link\('abenheap_reference_glosry.htm'\) "Glossary Entry").
