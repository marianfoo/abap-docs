Standard ABAP

Standard ABAP is a [language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") in which the [Unicode checks](javascript:call_link\('abenunicode_check_glosry.htm'\) "Glossary Entry") are effective. Only [Unicode systems](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry") are supported in the current release, therefore only programs with the ABAP language version Standard ABAP and higher are supported.
