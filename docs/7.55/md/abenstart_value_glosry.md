start value

Value specified when a [data object](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abendata_object_glosry.htm "Glossary Entry") is declared. The data object is filled with this value when it is created. The start value is set using the [VALUE](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapdata_options.htm) addition of the [DATA](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapdata.htm) statement. If the VALUE addition is not used, the system uses the type-specific [initial value](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninitial_value_glosry.htm "Glossary Entry").
