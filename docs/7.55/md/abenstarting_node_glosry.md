starting node

[Hierarchy node](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") from which a [hierarchy node navigator](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenhierarchy_node_navi_glosry.htm "Glossary Entry") determines further hierarchy nodes and adds them to its result set.
