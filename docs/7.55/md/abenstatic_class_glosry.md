static class

[Class](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenclass_glosry.htm "Glossary Entry") that only contains [static components](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstatic_component_glosry.htm "Glossary Entry") and no [instance components](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninstance_component_glosry.htm "Glossary Entry").
