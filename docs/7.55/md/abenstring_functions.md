String Functions (string\_func)

String functions are part of the [built-in functions](javascript:call_link\('abenbuilt_in_functions.htm'\)). They are divided into:

-   [Description functions](javascript:call_link\('abendescriptive_functions.htm'\))

-   [Processing functions](javascript:call_link\('abenprocess_functions.htm'\))

Description functions have a numeric result and processing functions have a character-like result.

Parameters that are used by more than one function in the same way are described under [Shared Parameters of String Functions](javascript:call_link\('abenstring_functions_common_paras.htm'\)).

Hint

The program DEMO\_EXPRESSIONS shows examples of how to use string expressions.
