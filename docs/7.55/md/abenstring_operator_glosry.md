string operator

Links two character-like [operands](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_operand_glosry.htm "Glossary Entry") in a [string expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstring_expression_glosry.htm "Glossary Entry"). The only string operator currently possible is the [concatenation operator](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenconcatenation_operator_glosry.htm "Glossary Entry") [&&](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstring_operators.htm), which concatenates the two operands into one string.
