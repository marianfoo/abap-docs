character string template

String template for short. A character string template creates a [character string](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencharacter_string_glosry.htm "Glossary Entry") from literal text, embedded expressions, and control characters in a [string expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstring_expression_glosry.htm "Glossary Entry").
