structured type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of a [structure](javascript:call_link\('abenstructure_glosry.htm'\) "Glossary Entry"). It contains other data types as [components](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry"). A named structured type is either defined as a [DDIC structure](javascript:call_link\('abenddic_structure_glosry.htm'\) "Glossary Entry"), a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") or inside an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") using the statement [TYPES BEGIN OF](javascript:call_link\('abaptypes_struc.htm'\)).
