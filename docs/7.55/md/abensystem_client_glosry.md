system client

Special [client](javascript:call_link\('abenclient_glosry.htm'\) "Glossary Entry") in [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") with the [client ID](javascript:call_link\('abenclient_identifier_glosry.htm'\) "Glossary Entry") "000". The system client is used to execute system-specific [system programs](javascript:call_link\('abensystem_program_glosry.htm'\) "Glossary Entry").
