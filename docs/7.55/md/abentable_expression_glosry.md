table expression

A table expression reads a line from an [internal table](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abeninternal_table_glosry.htm "Glossary Entry") and returns it as a result. [Mesh path expressions](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenmesh_path_expression_glosry.htm "Glossary Entry") are a special form of table expressions that can be used in the same way as table expressions.
