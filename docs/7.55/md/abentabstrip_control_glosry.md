tabstrip control

Tabstrip controls allow tab pages to be displayed on dynpros. A tabstrip control is a [screen element](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenscreen_element_glosry.htm "Glossary Entry") built from several [tab pages](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentab_page_glosry.htm "Glossary Entry") or tabs.
