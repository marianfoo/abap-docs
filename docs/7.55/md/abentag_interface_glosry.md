tag interface

Special predefined global interface. By implementing or including a tag interface, [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") or other [interfaces](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") are particularly tagged for the [ABAP runtime environment](javascript:call_link\('abenabap_runtime_envir_glosry.htm'\) "Glossary Entry"). A tag interface generally does not contain its own [interface components](javascript:call_link\('abeninterface_component_glosry.htm'\) "Glossary Entry") but instead assigns a particular task to the integrating classes or interfaces and changes the way they are handled by the [ABAP Compiler](javascript:call_link\('abenabap_compiler_glosry.htm'\) "Glossary Entry").
