test task

Combination of tests of one or several ABAP programs in [ABAP Unit](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_unit_glosry.htm "Glossary Entry") for which a [test run](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentest_run_glosry.htm "Glossary Entry") can be executed.
