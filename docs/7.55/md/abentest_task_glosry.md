test task

Combination of tests of one or several ABAP programs in [ABAP Unit](javascript:call_link\('abenabap_unit_glosry.htm'\) "Glossary Entry") for which a [test run](javascript:call_link\('abentest_run_glosry.htm'\) "Glossary Entry") can be executed.
