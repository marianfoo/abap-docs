text environment

The text environment is part of the [language settings](javascript:call_link\('abenlanguage_setting_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and consists of a [text environment language](javascript:call_link\('abentext_env_langu_glosry.htm'\) "Glossary Entry"), the [system code page](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry"), and a [locale](javascript:call_link\('abenlocale_glosry.htm'\) "Glossary Entry"). All programs of an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") have a common text environment.
