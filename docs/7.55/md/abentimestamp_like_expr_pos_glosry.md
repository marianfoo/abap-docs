time-stamp-like expression position

[Read position](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenreading_position_glosry.htm "Glossary Entry") in which a [time stamp field](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentimestamp_field_glosry.htm "Glossary Entry"), as well as an [expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencharlike_expression_glosry.htm "Glossary Entry") can be specified whose result has a [time stamp type](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentimestamp_type_glosry.htm "Glossary Entry").
