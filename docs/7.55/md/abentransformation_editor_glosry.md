Transformation Editor

Tool in the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") for creating [XSLT programs](javascript:call_link\('abenxslt_program_glosry.htm'\) "Glossary Entry") or [simple transformations](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry") in [transformation source code](javascript:call_link\('abenxslt_program_glosry.htm'\) "Glossary Entry"). Called using the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") STRANS.
