trusted system

[AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry") that is classified as a trusted caller of [remote function calls](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenremote_function_call_glosry.htm "Glossary Entry") in a [trusting system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentrusting_system_glosry.htm "Glossary Entry"). The trusted system provides the [RFC client](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrfc_client_glosry.htm "Glossary Entry"). The system is classified in [transaction](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentransaction_glosry.htm "Glossary Entry") SMT1. Trusted systems log on to the trusting system without passing a password. User-specific logon data is checked in the trusting system.
