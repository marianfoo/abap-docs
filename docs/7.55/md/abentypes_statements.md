Declaring Data Types

In ABAP programs, data types local to the program can be defined, which can be used for [typing](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abentyping_glosry.htm "Glossary Entry") or declaring further data types or data objects, depending on their validity and visibility.

-   [INCLUDE TYPE](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abapinclude_type.htm)
