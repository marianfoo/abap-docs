Unicode check

[Syntax check](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abensyntax_check_glosry.htm "Glossary Entry") for [ABAP programs](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_program_glosry.htm "Glossary Entry") that are to be executed in a [Unicode system](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenunicode_system_glosry.htm "Glossary Entry"). The Unicode check is activated by the [ABAP language version](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_version_glosry.htm "Glossary Entry") [Standard ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenstandard_abap_glosry.htm "Glossary Entry") or higher. Only Unicode systems are supported in the current [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry") release and therefore no programs of the obsolete ABAP language version [Obsolete ABAP (Non-Unicode)](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abennon_unicode_abap_glosry.htm "Glossary Entry") are supported.
