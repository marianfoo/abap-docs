Unicode

Language-independent [character set](javascript:call_link\('abencharacter_set_glosry.htm'\) "Glossary Entry") for international data processing ([www.unicode.org](http://www.unicode.org)). Subset of the [UCS](javascript:call_link\('abenucs_glosry.htm'\) "Glossary Entry") character set according to ISO-10646. Unicode can represent 1114111 different characters numbered hexadecimally from U+0000 to U+10FFFF. The Unicode character set is normally accessed using a [UTF](javascript:call_link\('abenutf_glosry.htm'\) "Glossary Entry") or [UCS](javascript:call_link\('abenucs2_glosry.htm'\) "Glossary Entry") [character representation](javascript:call_link\('abenchar_representation_glosry.htm'\) "Glossary Entry").
