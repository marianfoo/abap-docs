unmanaged RAP Business Object

Unmanaged RAP BO for short. An implementation type of the [business object behavior](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenrap_bo_behavior_glosry.htm "Glossary Entry") in the context of [ABAP RAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_rap_glosry.htm "Glossary Entry"). For this implementation type, all major components of the REST contract must be implemented first. All required operations (create, update, delete, plus any application-specific actions) must be defined in the corresponding [behavior definition](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abencds_behavior_definition_glosry.htm "Glossary Entry") before being implemented in ABAP.
