unmanaged RAP Business Object

Unmanaged RAP BO for short. An implementation type of the [business object behavior](javascript:call_link\('abenrap_bo_behavior_glosry.htm'\) "Glossary Entry") in the context of [ABAP RAP](javascript:call_link\('abenabap_rap_glosry.htm'\) "Glossary Entry"). For this implementation type, all major components of the REST contract must be implemented first. All required operations (create, update, delete, plus any application-specific actions) must be defined in the corresponding [behavior definition](javascript:call_link\('abencds_behavior_definition_glosry.htm'\) "Glossary Entry") before being implemented in ABAP.
