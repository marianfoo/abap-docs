user name

Name of a [user](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_glosry.htm "Glossary Entry") that is required for logging on to a [user session](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenuser_session_glosry.htm "Glossary Entry") in the [AS ABAP](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenas_abap_glosry.htm "Glossary Entry"). Together with the [password](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenpassword_glosry.htm "Glossary Entry"), the user name allows authorized use of an AS ABAP.
