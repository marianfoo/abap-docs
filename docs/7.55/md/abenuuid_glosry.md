UUID

Short for Universal Unique Identifier. Universally unique identifier that is generated in accordance with [OSF standards](javascript:call_link\('http://www.opengroup.org/onlinepubs/9629399/apdxa.htm'\)) from the system time stamp and the network address. In ABAP, a UUID can be created using the [system class](javascript:call_link\('abensystem_class_glosry.htm'\) "Glossary Entry") CL\_SYSTEM\_UUID and the [SQL function](javascript:call_link\('abensql_function_glosry.htm'\) "Glossary Entry") [UUID](javascript:call_link\('abensql_uuid.htm'\)).
