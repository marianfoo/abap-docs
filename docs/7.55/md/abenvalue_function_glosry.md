value function

[Window function](javascript:call_link\('abenwindow_function_glosry.htm'\) "Glossary Entry") that determines a value from a different row of the window for a row of a [window](javascript:call_link\('abenwindow_glosry.htm'\) "Glossary Entry") in the result set of a [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"). The calculation is executed in the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry").
