Web Application Builder

Tool in [ABAP Workbench](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for creating [Business Server Pages](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenbusiness_server_pages_glosry.htm "Glossary Entry"). It can be called via Create → BSP Library → BSP Application in [Object Navigator](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenobject_navigator_glosry.htm "Glossary Entry").
