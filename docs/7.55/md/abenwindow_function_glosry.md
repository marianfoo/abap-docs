window function

Function that is specified in front of the definition of a [window](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenwindow_glosry.htm "Glossary Entry") in an [ABAP SQL](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenabap_sql_glosry.htm "Glossary Entry") [window expression](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenwindow_expression_glosry.htm "Glossary Entry") and that evaluates the rows of the window. A window function is either an [aggregate function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenaggregate_function_glosry.htm "Glossary Entry"), a [ranking function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenranking_function_glosry.htm "Glossary Entry"), or a [value function](https://help.sap.com/doc/abapdocu_755_index_htm/7.55/en-US/abenvalue_function_glosry.htm "Glossary Entry").
