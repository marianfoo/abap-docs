XML writer

[Instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") of the [sXML Library](javascript:call_link\('abensxml_library_glosry.htm'\) "Glossary Entry") that implements the [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") IF\_SXML\_WRITER. An XML writer enables the rendering of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") in different [formats](javascript:call_link\('abenabap_sxml_lib_formats.htm'\)).
