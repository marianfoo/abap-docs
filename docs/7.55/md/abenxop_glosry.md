XOP

Short for XML-binary Optimized Packaging (see [http://www.w3.org/TR/xop10/](http://www.w3.org/TR/xop10/)). Representation of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data in which any binary raw data is stored separately from the data represented by characters. In ABAP, XOP is supported by a [format](javascript:call_link\('abenabap_sxml_lib_formats.htm'\)) of the [sXML Library](javascript:call_link\('abenabap_sxml_lib.htm'\)).
