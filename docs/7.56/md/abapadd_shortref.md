ADD - Short Reference

[Reference](javascript:call_link\('abapadd.htm'\))

Syntax

ADD dobj1 TO dobj2.

Effect

Obsolete: Adds the content of a numeric data object dobj1 to the content of a numeric data object dobj2 and assigns the result to dobj2.
