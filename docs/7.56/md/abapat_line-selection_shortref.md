AT LINE-SELECTION - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapat_line-selection.htm)

Syntax

AT LINE-SELECTION.

Effect

Introduces an event block whose event is raised by the selection of a list line using the predefined function code "PICK".
