Syntax

BACK.

Effect

Positions the [list cursor](javascript:call_link\('abenlist_cursor_glosry.htm'\) "Glossary Entry") at the first position of a line block defined with RESERVE, a [page header](javascript:call_link\('abenpage_header_glosry.htm'\) "Glossary Entry") defined after TOP-OF-PAGE, or the first line underneath the page header.
