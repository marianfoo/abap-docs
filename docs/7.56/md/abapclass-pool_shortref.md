CLASS-POOL - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapclass-pool.htm)

Syntax

CLASS-POOL *\[*MESSAGE-ID mid*\]*.

Effect

Introduces a [class pool](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_pool_glosry.htm "Glossary Entry")

Addition

-   MESSAGE-ID mid
    Specifies a message class mid for short forms of MESSAGE.
