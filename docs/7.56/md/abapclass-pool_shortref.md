CLASS-POOL - Short Reference

[Reference](javascript:call_link\('abapclass-pool.htm'\))

Syntax

CLASS-POOL *\[*MESSAGE-ID mid*\]*.

Effect

Introduces a [class pool](javascript:call_link\('abenclass_pool_glosry.htm'\) "Glossary Entry")

Addition

-   MESSAGE-ID mid
    Specifies a message class mid for short forms of MESSAGE.
