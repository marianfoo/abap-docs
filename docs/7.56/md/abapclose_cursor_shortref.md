CLOSE CURSOR - Short Reference

[Reference](javascript:call_link\('abapclose_cursor.htm'\))

Syntax

CLOSE CURSOR @dbcur.

Effect

[ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") statement. Closes a [database cursor](javascript:call_link\('abendatabase_cursor_glosry.htm'\) "Glossary Entry") linked with dbcur and that was opened using OPEN CURSOR.
