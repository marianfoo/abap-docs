CLOSE CURSOR - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapclose_cursor.htm)

Syntax

CLOSE CURSOR @dbcur.

Effect

[ABAP SQL](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sql_glosry.htm "Glossary Entry") statement. Closes a [database cursor](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_cursor_glosry.htm "Glossary Entry") linked with dbcur and that was opened using OPEN CURSOR.
