CONDENSE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapcondense.htm)

Syntax

CONDENSE text *\[*NO-GAPS*\]*.

Effect

Condenses the character string in text by removing leading and trailing blanks and replaces multiple consecutive blanks with a single blank.

Addition

-   NO-GAPS
    Removes all blanks from the character string.
