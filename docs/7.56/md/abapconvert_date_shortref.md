CONVERT DATE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapconvert_date.htm)

Syntax

CONVERT *{*DATE dat1 INTO INVERTED-DATE dat2*}*
      *|* *{*INVERTED-DATE dat1 INTO DATE dat2*}*.

Effect

Obsolete: Both variants convert the digits of a character-like data object dat1 into their complement on nine and assign the result to the data object dat2.
