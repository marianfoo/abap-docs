EXPORT NAMETAB

This statement is no longer supported.

Syntax

EXPORT NAMETAB ...

Effect

This statement was used internally to export a DDIC structure description in an internal format to the [name table](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenname_table_glosry.htm "Glossary Entry") of the ABAP Dictionary.

The statement is no longer supported and produces a runtime error.

[Exceptions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_language_exceptions.htm)

Uncatchable Exceptions

-   Cause: Statement is no longer supported.
    Runtime error: EXPORT\_NAMETAB\_OUTDATED
