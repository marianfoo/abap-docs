GET PARAMETER - Short Reference

[Reference](javascript:call_link\('abapget_parameter.htm'\))

Syntax

GET PARAMETER ID pid FIELD dobj.

Effect

Returns the value of the [SPA/GPA parameters](javascript:call_link\('abenspa_gpa_parameter_glosry.htm'\) "Glossary Entry") specified in pid in dobj.
