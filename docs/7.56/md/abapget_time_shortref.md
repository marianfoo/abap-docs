GET TIME - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapget_time.htm)

Syntax

GET TIME *\[*FIELD tim*\]*.

Effect

Updates the system fields sy-datlo, sy-datum, sy-timlo, and sy-uzeit.

Addition

-   FIELD tim
    Returns the current time in tim, without updating the system fields.
