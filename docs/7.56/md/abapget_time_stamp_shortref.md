GET TIME STAMP - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapget_time-stamp.htm)

Syntax

GET TIME STAMP FIELD time\_stamp.

Effect

Returns the current [time stamp](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentime_stamp_glosry.htm "Glossary Entry") in time\_stamp of the type TIMESTAMP or TIMESTAMPL.
