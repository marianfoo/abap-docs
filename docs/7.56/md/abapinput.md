Obsolete Syntax

INPUT.

Effect

This statement is forbidden in classes, has the same function as the following statement and is replaced by it:

[FORMAT](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapformat.htm) INPUT ON.
