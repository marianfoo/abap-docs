INTERFACE, DEFERRED, LOAD - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapinterface_deferred.htm)

Syntax

INTERFACE intf *{*DEFERRED *\[*PUBLIC*\]**}* *|* LOAD.

Effect

Makes an interface public.

Additions

-   DEFERRED *\[*PUBLIC*\]*
    Makes a local or global interface public before its actual declaration in the program.
-   [LOAD](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapclass_interface_load.htm)
    Obsolete: Previously loaded a global interface from the Class Library.
