LOOP AT SCREEN - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abaploop_at_screen.htm)

Syntax

LOOP AT SCREEN INTO wa.
  ...
ENDLOOP.

Effect

Reads the properties of the screen elements of the current [dynpro](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_glosry.htm "Glossary Entry") in a loop and assigns the results to a work area wa of the structure [SCREEN](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscreen.htm).
