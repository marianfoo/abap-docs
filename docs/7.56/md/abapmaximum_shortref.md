MAXIMUM - Short Reference

[Reference](javascript:call_link\('abapmaximum.htm'\))

Syntax

MAXIMUM dobj.

Effect

Obsolete: After the statement MAXIMUM, determines the maximum value of dobj and assigns it to the data object max\_dobj for every WRITE statement for dobj.
