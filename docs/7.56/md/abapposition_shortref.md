POSITION - Short Reference

[Reference](javascript:call_link\('abapposition.htm'\))

Syntax

POSITION pos.

Effect

Positions the [list cursor](javascript:call_link\('abenlist_cursor_glosry.htm'\) "Glossary Entry") on the position specified in pos for the current list line.
