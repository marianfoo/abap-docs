RESUME - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapresume.htm)

Syntax

RESUME.

Effect

Resumes processing when handling a [resumable exception](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenresumable_exception_glosry.htm "Glossary Entry").
