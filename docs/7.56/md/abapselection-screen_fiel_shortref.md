SELECTION-SCREEN FIELD SELECTION - Short Reference

[Reference](javascript:call_link\('abapselection-screen_ldb_field.htm'\))

Syntax

SELECTION-SCREEN FIELD SELECTION
                 FOR *{*NODE*|*TABLE*}* node *\[*ID id*\]*.

Effect

Obsolete: Flags a node node of the structure of a [logical database](javascript:call_link\('abenlogical_data_base_glosry.htm'\) "Glossary Entry") for [field selection](javascript:call_link\('abenfield_selection_glosry.htm'\) "Glossary Entry").

Addition

-   ID id
    Defines an ID id for the node, with a maximum of three characters.
