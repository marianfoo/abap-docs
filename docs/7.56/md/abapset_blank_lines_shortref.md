SET BLANK LINES - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapset_blank_lines.htm)

Syntax

SET BLANK LINES *{*ON*|*OFF*}*.

Effect
