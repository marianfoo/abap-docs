SET SCREEN - Short Reference

[Reference](javascript:call_link\('abapset_screen.htm'\))

Syntax

SET SCREEN dynnr.

Effect

Defines the [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") with the [dynpro number](javascript:call_link\('abendynpro_number_glosry.htm'\) "Glossary Entry") dynnr as the [next dynpro](javascript:call_link\('abennext_dynpro_glosry.htm'\) "Glossary Entry") to follow the current dynpro.
