START-OF-SELECTION - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapstart-of-selection.htm)

Syntax

START-OF-SELECTION.

Effect

Initiates an event block whose event is raised in [executable programs](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenexecutable_program_glosry.htm "Glossary Entry") after the [standard selection screen](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstandard_selscreen_glosry.htm "Glossary Entry") is processed.
