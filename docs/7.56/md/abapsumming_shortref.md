SUMMING - Short Reference

[Reference](javascript:call_link\('abapsumming.htm'\))

Syntax

SUMMING dobj.

Effect

Obsolete: After the statement SUMMING, adds the value of dobj to the content of sum\_dobj after each WRITE statement.
