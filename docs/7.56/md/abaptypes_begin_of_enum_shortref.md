[Reference](javascript:call_link\('abaptypes_enum.htm'\))

Syntax

TYPES BEGIN OF ENUM enum\_type *\[*...*\]*.

TYPES END OF ENUM enum\_type *\[*...*\]*.

Effect

Defines an enumerated type enum\_type.
