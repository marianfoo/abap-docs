TYPES BEGIN OF ENUM - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abaptypes_enum.htm)

Syntax

TYPES BEGIN OF ENUM enum\_type *\[*...*\]*.
  val1 *\[*...*\]*, val2 *\[*...*\]*, ...
TYPES END OF ENUM enum\_type *\[*...*\]*.

Effect

Defines an enumerated type enum\_type.
