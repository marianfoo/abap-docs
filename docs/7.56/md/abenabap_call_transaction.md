Calling Transactions

A transaction is the execution of a program using a [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry"). The process flow of the program depends on which [initial dynpro](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninitial_dynpro_glosry.htm "Glossary Entry"), [selection screen](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenselection_screen_glosry.htm "Glossary Entry"), or method the transaction code is linked to.

There are two statements for calling transactions:

-   [CALL TRANSACTION](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapcall_transaction.htm)
-   [LEAVE TO TRANSACTION](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapleave_to_transaction.htm)
