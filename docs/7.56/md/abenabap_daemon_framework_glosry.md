ABAP daemon framework

ADF for short. A framework based on [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") and [interfaces](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") for using [ABAP daemons](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry"). ADF includes APIs both for communication with ABAP daemons in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and for handling events using ABAP daemons in [ABAP daemon sessions](javascript:call_link\('abenabap_daemon_session_glosry.htm'\) "Glossary Entry").
