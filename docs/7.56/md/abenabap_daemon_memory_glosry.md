ABAP daemon memory

Part of the [user memory](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenuser_memory_glosry.htm "Glossary Entry") in which an [ABAP daemon](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_daemon_glosry.htm "Glossary Entry") can store data in [PCP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpcp_glosry.htm "Glossary Entry") format. An ABAP daemon can use approximately 1 megabyte of the ABAP daemon memory.
