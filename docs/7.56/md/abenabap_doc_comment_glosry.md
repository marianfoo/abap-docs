ABAP Doc comment

Special [comment](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomment_glosry.htm "Glossary Entry") consisting of one or more [comment lines](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomment_line_glosry.htm "Glossary Entry"), each of which is initiated by "!. ABAP Doc comments can precede [declarations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendeclaration_glosry.htm "Glossary Entry") and are used by [ABAP Doc](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_doc_glosry.htm "Glossary Entry") for their documentation.
