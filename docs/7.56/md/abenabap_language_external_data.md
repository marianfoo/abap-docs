Processing External Data

External data generally covers all data that is stored outside of the [internal session](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninternal_session_glosry.htm "Glossary Entry"). The following sections describe the processing of external data.

-   [ABAP Database Access](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendb_access.htm)
-   [Data Clusters](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_cluster.htm)
-   [ABAP File Interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_language_files.htm)
