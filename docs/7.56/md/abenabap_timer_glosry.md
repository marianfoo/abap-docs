ABAP timer

A timer for wait times in [non-blocking mode](javascript:call_link\('abennon_blocking_model_glosry.htm'\) "Glossary Entry") integrated into the [ABAP runtime framework](javascript:call_link\('abenabap_timer_manager_glosry.htm'\) "Glossary Entry"). An ABAP Timer is started using a [timer manager](javascript:call_link\('abenabap_timer_manager_glosry.htm'\) "Glossary Entry") for a given runtime and at the end a timeout event is raised, which can be responded to with a [timer handler](javascript:call_link\('abenabap_timer_handler_glosry.htm'\) "Glossary Entry").
