ABAP timer

A timer for wait times in [non-blocking mode](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abennon_blocking_model_glosry.htm "Glossary Entry") integrated into the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_timer_manager_glosry.htm "Glossary Entry"). An ABAP Timer is started using a [timer manager](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_timer_manager_glosry.htm "Glossary Entry") for a given runtime and at the end a timeout event is raised, which can be responded to with a [timer handler](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_timer_handler_glosry.htm "Glossary Entry").
