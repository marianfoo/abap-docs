ABAP timer handler

[Class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry") that implements the [interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenoo_intf_glosry.htm "Glossary Entry") IF\_ABAP\_TIMER\_HANDLER. The interface method ON\_TIMEOUT of the timer handler, which is specified with a [ABAP Timer Manager](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_timer_manager_glosry.htm "Glossary Entry") when an [ABAP Timer](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_timer_glosry.htm "Glossary Entry") is started. It handles the timeout event of the timer.
