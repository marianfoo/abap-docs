append structure

[Structure](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstructure_glosry.htm "Glossary Entry") of the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") that is appended to a different DDIC structure or [DDIC database table](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenddic_db_table_glosry.htm "Glossary Entry") thus adding its [components](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomponent_glosry.htm "Glossary Entry") to them. Append structures and database tables can be used to add components to structures and database tables delivered by SAP without making modifications to them.
