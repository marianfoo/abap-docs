archiving parameter

Parameter that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenspool_request_glosry.htm "Glossary Entry") if its data is to be archived using [ArchiveLink](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenarchivelink_glosry.htm "Glossary Entry").
