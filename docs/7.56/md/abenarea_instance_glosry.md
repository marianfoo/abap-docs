area instance

Sum of all [area instance versions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenarea_instance_version_glosry.htm "Glossary Entry") with the same area instance name. An area instance is a unique variant of an [area](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenarea_glosry.htm "Glossary Entry") and is divided into versions.
