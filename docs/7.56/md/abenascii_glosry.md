ASCII

Abbreviation for American Standard Code for Information Interchange. 7-bit [character set](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencharacter_set_glosry.htm "Glossary Entry") in accordance with the ISO-646 standard. Expanded to 8-bit character sets in [ISO-8859](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeniso-8859_glosry.htm "Glossary Entry").
