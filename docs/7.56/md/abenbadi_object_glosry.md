BAdI object

Instance of a [BAdI](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbadi_glosry.htm "Glossary Entry"). BAdI objects are addressed exclusively using [BAdI reference variables](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbadi_reference_variable_glosry.htm "Glossary Entry") in which a [BAdI reference](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbadi_reference_glosry.htm "Glossary Entry") was set using [GET BADI](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapget_badi.htm). A BAdI object acts as a handle for the calls of [BAdI methods](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbadi_method_glosry.htm "Glossary Entry") and contains references to the [object plug-ins](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_plugin_glosry.htm "Glossary Entry") created based on filter specifications. From a technical point of view, BAdI objects are [instances](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninstance_glosry.htm "Glossary Entry") of [BAdI classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbadi_class_glosry.htm "Glossary Entry").
