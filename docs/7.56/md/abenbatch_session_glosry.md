background session

[User session](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenuser_session_glosry.htm "Glossary Entry") for [background processing](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbackround_processing_glosry.htm "Glossary Entry"). A background session is opened when a program is executed using [SUBMIT VIA JOB](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapsubmit_via_job.htm).
