BCD

Abbreviation of Binary Coded Decimals. Code for representing decimal numbers that stores each decimal digit in 4 bits. See also [packed number](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpacked_number_glosry.htm "Glossary Entry").
