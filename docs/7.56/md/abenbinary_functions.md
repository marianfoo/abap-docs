Byte String Functions (bit\_func)

Byte string functions are one of the [built-in functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbuilt_in_functions.htm). They are divided into:

-   [Description function for byte-like data types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendescriptive_functions_binary.htm)
-   [Bit functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbit_functions.htm)
