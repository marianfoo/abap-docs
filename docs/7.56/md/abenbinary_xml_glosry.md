binary XML

SAP-specific format for the binary storage of [XML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxml_glosry.htm "Glossary Entry") data. In Binary XML, redundant information is suppressed, and [UTF-8](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenutf8_glosry.htm "Glossary Entry") encoding is used, which leads to performance benefits when storing and transporting data. In [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry"), Binary XML occurs as one of the supported [formats](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sxml_lib_formats.htm) in the [sXML Library](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sxml_lib.htm).
