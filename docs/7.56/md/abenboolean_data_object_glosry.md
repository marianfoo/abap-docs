Boolean data object

[Data object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_object_glosry.htm "Glossary Entry") used to register [truth values](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentruth_value_glosry.htm "Glossary Entry"). Boolean data objects are not supported in ABAP. Instead, a truth value can be evaluated as the return value of a [Boolean function](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenboole_function_glosry.htm "Glossary Entry").
