byte chain

Sequence of bytes, also called byte sequence. Content of each data object. However, direct access to the bytes of a byte chain is possible only in [byte-like data objects](javascript:call_link\('abenbyte_like_data_object_glosry.htm'\) "Glossary Entry"), or by using appropriately typed [field symbols](javascript:call_link\('abenfield_symbol_glosry.htm'\) "Glossary Entry").
