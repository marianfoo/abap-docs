catchable runtime error

Exception situation in the runtime framework that can still be handled by the obsolete statement [CATCH SYSTEM-EXCEPTIONS](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapcatch_sys.htm). Replaced by [class-based exceptions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_based_exception_glosry.htm "Glossary Entry").
