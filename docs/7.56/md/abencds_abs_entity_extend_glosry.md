CDS abstract entity extension

Transportable extension of a [CDS abstract entity](javascript:call_link\('abencds_abstract_entity_glosry.htm'\) "Glossary Entry"). A CDS abstract entity extension can be used to add elements and/or clauses to an abstract entity without making any modifications to the original entity. It is defined using the statement EXTEND ABSTRACT ENTITY.

See also: [CDS entity extension](javascript:call_link\('abencds_entity_extend_glosry.htm'\) "Glossary Entry").
