CDS behavior definition header

BDEF header short. A BDEF header is the part of a [CDS behavior definition](javascript:call_link\('abencds_behavior_definition_glosry.htm'\) "Glossary Entry") that defines the [implementation type](javascript:call_link\('abencds_bdef_impl_type_glosry.htm'\) "Glossary Entry") and some global attributes that apply to the complete [business object](javascript:call_link\('abenbusiness_object_glosry.htm'\) "Glossary Entry"). It is the topmost part and includes all statements that are listed before the [entity behavior definition](javascript:call_link\('abencds_entity_bdef_glosry.htm'\) "Glossary Entry"), define behavior for.
