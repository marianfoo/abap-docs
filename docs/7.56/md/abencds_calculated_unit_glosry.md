CDS calculated unit

Component of a [CDS view entity](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencds_v2_view_glosry.htm "Glossary Entry") of data type CHAR that specifies a self-defined unit which is not contained in any list of predefined values. A CDS calculated unit can be [CDS calculated quantity](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencds_calculated_quantity.htm) of the same CDS view entity.
