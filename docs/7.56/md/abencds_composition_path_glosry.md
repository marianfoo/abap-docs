CDS composition path

Sequence of [CDS compositions](javascript:call_link\('abencds_composition_glosry.htm'\) "Glossary Entry") that link the [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") of a [CDS composition tree](javascript:call_link\('abencds_composition_tree_glosry.htm'\) "Glossary Entry") of a [RAP business object](javascript:call_link\('abenrap_bo_glosry.htm'\) "Glossary Entry") with each other. CDS composition paths can be specified like [CDS association paths](javascript:call_link\('abencds_association_path_glosry.htm'\) "Glossary Entry") in [SQL path expressions](javascript:call_link\('abensql_path_expression_glosry.htm'\) "Glossary Entry").
