CDS role

A [CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") defined using the statement [DEFINE ROLE](javascript:call_link\('abencds_f1_define_role.htm'\)) in a [DCL source code](javascript:call_link\('abendcl_source_code_glosry.htm'\) "Glossary Entry") in the [CDS DCL](javascript:call_link\('abencds_dcl_glosry.htm'\) "Glossary Entry") of [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"). This object applies [CDS access control](javascript:call_link\('abencds_access_control_glosry.htm'\) "Glossary Entry") based on [access conditions](javascript:call_link\('abenaccess_condition_glosry.htm'\) "Glossary Entry") when non-abstract [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") are accessed using [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"). A CDS role currently is assigned to every [ABAP user](javascript:call_link\('abenabap_user_glosry.htm'\) "Glossary Entry"). The access conditions of a CDS role evaluate the classic [authorizations](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry") of the current user.
