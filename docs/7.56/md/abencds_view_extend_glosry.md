CDS view extension

Transportable extension of a [CDS DDIC-based view](javascript:call_link\('abencds_v1_view_glosry.htm'\) "Glossary Entry"). A CDS view extension can be used to add elements and/or clauses to a view without making any modifications to the original view. It is defined using the statement EXTEND VIEW.

See also: [CDS entity extension](javascript:call_link\('abencds_entity_extend_glosry.htm'\) "Glossary Entry").
