character set

Defined set of characters, for example, [ASCII](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenascii_glosry.htm "Glossary Entry"), [EBCDIC](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenebcdic_glosry.htm "Glossary Entry") or [Unicode](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenunicode_glosry.htm "Glossary Entry"). Character sets are mapped onto [character representations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenchar_representation_glosry.htm "Glossary Entry") via [code pages](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencodepage_glosry.htm "Glossary Entry").
