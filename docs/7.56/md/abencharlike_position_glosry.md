character-like operand position

[Read position](javascript:call_link\('abenread_position_glosry.htm'\) "Glossary Entry") with character-like [operand type](javascript:call_link\('abenoperand_type_glosry.htm'\) "Glossary Entry") in which a [character-like data object](javascript:call_link\('abencharlike_data_object_glosry.htm'\) "Glossary Entry") that can be converted into the operand type can be specified. See also [character-like expression position](javascript:call_link\('abencharlike_expr_position_glosry.htm'\) "Glossary Entry").
