child node

[Hierarchy node](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") that inherits a [parent-child relationship](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpcr_glosry.htm "Glossary Entry") from a [parent node](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenparent_node_glosry.htm "Glossary Entry"). A child node can have multiple parent nodes. A child node is a [descendant node](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendescendant_node_glosry.htm "Glossary Entry") of its parent node, this parent's node, and so on. A hierarchy in which each child node has only one parent node is a tree-like hierarchy, where each [root node](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenroot_node_glosry.htm "Glossary Entry") can be the origin of a tree.
