Lists - System Class

The methods of the class CL\_ABAP\_LIST\_UTILITIES can be used for the calculation of output lengths, the conversion of values from the list buffer, and the calculation of required output lengths on lists.

The return values of the methods of this class can be used to program a correct column alignment in ABAP lists, even for output of Eastern Asian characters.
