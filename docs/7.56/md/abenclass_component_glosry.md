class component

[Component](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomponent_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry"). Declared in the [declaration part](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendeclaration_part_glosry.htm "Glossary Entry") of the class. Possible class components are: [Attributes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenattribute_glosry.htm "Glossary Entry"), [methods](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmethod_glosry.htm "Glossary Entry"), and [events](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenevent_glosry.htm "Glossary Entry").
