class library

The class library consists of all [global](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenglobal_type_glosry.htm "Glossary Entry") [object types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_type_glosry.htm "Glossary Entry"), namely of all [global classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenglobal_class_glosry.htm "Glossary Entry") and [interfaces](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenglobal_interface_glosry.htm "Glossary Entry").
