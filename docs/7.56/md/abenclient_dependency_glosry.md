client dependency

Property of a [SQL data source](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensql_data_source_glosry.htm "Glossary Entry"). A SQL data source can be [client-dependent](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclient_dependent_glosry.htm "Glossary Entry") or [client-independent](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclient_independent_glosry.htm "Glossary Entry"). When accessing a client-dependent SQL data source in [ABAP SQL](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sql_glosry.htm "Glossary Entry"), an implicit [client handling](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclient_handling_glosry.htm "Glossary Entry") takes place.
