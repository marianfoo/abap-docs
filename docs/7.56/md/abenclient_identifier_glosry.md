client ID

Three-digit ID of a [client](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclient_glosry.htm "Glossary Entry") that is specified when [logging on](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlogon_glosry.htm "Glossary Entry") to [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry"). In [application tables](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenapplication_table_glosry.htm "Glossary Entry"), client-dependent data is indicated by the client ID stored in the [client column](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclient_column_glosry.htm "Glossary Entry").
