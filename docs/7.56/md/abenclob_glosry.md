CLOB

Short for Character Large Object. Name for a [database field](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_field_glosry.htm "Glossary Entry") with the data type [STRING](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenddic_builtin_types.htm) or the result of an [SQL expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapsql_expr.htm) that returns the data type [STRING](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenddic_builtin_types.htm).
