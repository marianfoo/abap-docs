Code Inspector

CI for short. Tool for checking [repository objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry") in terms of performance, security, syntax, and adherence to naming conventions. Called using [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") SCI.
