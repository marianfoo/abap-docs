command field

Input field in the [standard toolbar](javascript:call_link\('abenstandard_toolbar_glosry.htm'\) "Glossary Entry"). This input field accepts [function codes](javascript:call_link\('abenfunction_code_glosry.htm'\) "Glossary Entry"), [transaction codes](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry"), and other shortcuts such as "/h" for starting the [ABAP Debugger](javascript:call_link\('abenabap_debugger_glosry.htm'\) "Glossary Entry").
