correlated subquery

[Subquery](javascript:call_link\('abensubquery_glosry.htm'\) "Glossary Entry") that uses columns of the surrounding [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") in its [SQL conditions](javascript:call_link\('abensql_cond_glosry.htm'\) "Glossary Entry"). It must be possible to assign the column to a [SQL data source](javascript:call_link\('abensql_data_source_glosry.htm'\) "Glossary Entry") in the surrounding query. A correlated subquery is evaluated for each individual row of the result set of the surrounding query.
