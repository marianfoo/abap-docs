CTE association

[Association](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenassociation_glosry.htm "Glossary Entry") that is defined for a [common table expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencommon_table_expression_glosry.htm "Glossary Entry") ([CTE](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencte_glosry.htm "Glossary Entry")) and can be used in the [queries](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenquery_glosry.htm "Glossary Entry") of its [WITH](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapwith.htm) statement in [SQL path expressions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensql_path_expression_glosry.htm "Glossary Entry"). A CTE association is defined in the definition of the common table expression using the addition [WITH ASSOCIATIONS JOIN](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapwith_associations_defining.htm).
