CTE hierarchy

[Common table expression](javascript:call_link\('abencommon_table_expression_glosry.htm'\) "Glossary Entry") ([CTE](javascript:call_link\('abencte_glosry.htm'\) "Glossary Entry")) that can be used as an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry") in the subsequent [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") of its [WITH](javascript:call_link\('abapwith.htm'\)) statement. A CTE hierarchy is created when the common table expression is defined using the addition [WITH HIERARCHY](javascript:call_link\('abapwith_hierarchy.htm'\)).
