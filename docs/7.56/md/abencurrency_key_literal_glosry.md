currency key literal

[Literal](javascript:call_link\('abenabap_literal_glosry.htm'\) "Glossary Entry") of the data type [CUKY](javascript:call_link\('abenddic_builtin_types.htm'\)) that contains a [currency code](javascript:call_link\('abencurrency_id_glosry.htm'\) "Glossary Entry"). Only [typed](javascript:call_link\('abentyped_literal_glosry.htm'\) "Glossary Entry") currency key literals are available.
