data aging

A [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") concept where current data is distinguished from outdated data by entries in a [temperature column](javascript:call_link\('abentemperature_column_glosry.htm'\) "Glossary Entry"). Only current data exists in the memory.
