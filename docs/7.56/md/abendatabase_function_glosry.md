database function

[SQLScript function](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensql_script_function_glosry.htm "Glossary Entry") in the [SAP HANA database](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenhana_database_glosry.htm "Glossary Entry"). Database functions must have a return value defined with RETURN. There are [table functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentable_function_glosry.htm "Glossary Entry") with a tabular return value and [scalar functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscalar_function_glosry.htm "Glossary Entry") with a scalar return value. Unlike [database procedures](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_procedure_glosry.htm "Glossary Entry"), database functions cannot be called using CALL, but can be used directly in read positions of statements. Database functions can be managed as [AMDP functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenamdp_function_glosry.htm "Glossary Entry") in the [ABAP Managed Database Procedures](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_managed_db_proc_glosry.htm "Glossary Entry") (AMDP).
