database hint

Extension of the SQL standard that tells a [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") how to execute an access. Database hints are usually dependent on the database system and its release. In [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), database hints can be specified using the addition [%\_HINTS](javascript:call_link\('abenabap_sql_db_hints.htm'\)).
