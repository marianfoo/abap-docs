database server

Computer in the [database layer](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_layer_glosry.htm "Glossary Entry") on which the [database system](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_system_glosry.htm "Glossary Entry") is installed.
