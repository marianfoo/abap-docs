database system

System for storing large amounts of data in a [database](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_glosry.htm "Glossary Entry"). Besides the database, a database system consists of a database management system that manages the database and offers access to the data via a programming interface ([SQL](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensql_glosry.htm "Glossary Entry") for [relational databases](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrelational_database_glosry.htm "Glossary Entry")).
