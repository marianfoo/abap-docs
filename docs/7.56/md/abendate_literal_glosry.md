date literal

[Literal](javascript:call_link\('abenabap_literal_glosry.htm'\) "Glossary Entry") that contains a date. It has the data type [DATN](javascript:call_link\('abenddic_builtin_types.htm'\)) or [DATS](javascript:call_link\('abenddic_builtin_types.htm'\)). Only [typed](javascript:call_link\('abentyped_literal_glosry.htm'\) "Glossary Entry") date literals are available.
