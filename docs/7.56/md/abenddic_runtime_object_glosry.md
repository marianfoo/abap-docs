DDIC runtime object

Summary of all relevant properties of a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") defined in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). A DDIC runtime object is created when a dictionary object is activated and is stored as an entry in the [name table](javascript:call_link\('abenname_table_glosry.htm'\) "Glossary Entry") ([nametab](javascript:call_link\('abenname_tab_glosry.htm'\) "Glossary Entry")). The [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") accesses the DDIC runtime object when types from the ABAP Dictionary are used.
