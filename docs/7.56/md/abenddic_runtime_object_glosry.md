DDIC runtime object

Summary of all relevant properties of a [data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_type_glosry.htm "Glossary Entry") defined in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). A DDIC runtime object is created when a dictionary object is activated and is stored as an entry in the [name table](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenname_table_glosry.htm "Glossary Entry") ([nametab](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenname_tab_glosry.htm "Glossary Entry")). The [ABAP runtime framework](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") accesses the DDIC runtime object when types from the ABAP Dictionary are used.
