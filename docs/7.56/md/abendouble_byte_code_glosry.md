double-byte code

Representation of a character in two bytes. Was used before the introduction of [Unicode](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenunicode_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry") for representing Asian characters.
