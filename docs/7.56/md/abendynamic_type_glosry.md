dynamic type

For a [reference variable](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenreference_variable_glosry.htm "Glossary Entry"), the [data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_type_glosry.htm "Glossary Entry") or the [class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry") of the [object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_glosry.htm "Glossary Entry"), currently pointed to by the reference variable points. The dynamic type is always more specific than or the same as the [static type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstatic_type_glosry.htm "Glossary Entry").
