edit mask

Template for formatting the output of a [data object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_object_glosry.htm "Glossary Entry") in a [list](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclassic_list_glosry.htm "Glossary Entry"). An edit mask is a character string consisting of placeholders for the characters of the data object to be output, and specific characters for formatting the output.
