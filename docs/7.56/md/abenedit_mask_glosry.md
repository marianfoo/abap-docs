edit mask

Template for formatting the output of a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") in a [list](javascript:call_link\('abenclassic_list_glosry.htm'\) "Glossary Entry"). An edit mask is a character string consisting of placeholders for the characters of the data object to be output, and specific characters for formatting the output.
