enhancement Information System

Tool of the [ABAP Workbench](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for the search for [enhancements](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenhancement_glosry.htm "Glossary Entry") of the [enhancement concept](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenhancement_concept_glosry.htm "Glossary Entry"). Called in the [Object Navigator](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_navigator_glosry.htm "Glossary Entry") or via [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") SPAU\_ENH.
