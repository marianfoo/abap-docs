enhancement implementation element

Individual element of an [enhancement implementation](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenhancement_impl_glosry.htm "Glossary Entry"). One enhancement implementation can contain multiple enhancement implementation elements, for example, different [source code plug-ins](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensource_code_plugin_glosry.htm "Glossary Entry").
