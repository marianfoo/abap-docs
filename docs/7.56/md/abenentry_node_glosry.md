entry node

A [mesh node](javascript:call_link\('abenmesh_node_glosry.htm'\) "Glossary Entry") from which the evaluation of a [mesh association](javascript:call_link\('abenmesh_association_glosry.htm'\) "Glossary Entry") starts. The entry node is not always the [start node](javascript:call_link\('abenstart_node_glosry.htm'\) "Glossary Entry") of the association.
