EPP connection counter

Component of an [extended passport (EPP)](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenepp_glosry.htm "Glossary Entry"). Counts connections between clients and servers with the same [EPP connection ID](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenepp_connection_id_glosry.htm "Glossary Entry").
