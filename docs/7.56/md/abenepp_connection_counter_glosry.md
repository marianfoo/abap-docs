EPP connection counter

Component of an [extended passport (EPP)](javascript:call_link\('abenepp_glosry.htm'\) "Glossary Entry"). Counts connections between clients and servers with the same [EPP connection ID](javascript:call_link\('abenepp_connection_id_glosry.htm'\) "Glossary Entry").
