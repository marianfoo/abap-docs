EPP connection ID

Component of an [extended passport (EPP)](javascript:call_link\('abenepp_glosry.htm'\) "Glossary Entry"). ID of a connection that is created by calls from [ABAP sessions](javascript:call_link\('abenabap_session_glosry.htm'\) "Glossary Entry") or external connections in ABAP sessions.
