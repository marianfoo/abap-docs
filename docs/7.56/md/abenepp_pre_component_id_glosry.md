EPP precomponent ID

Component of an [extended passport (EPP)](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenepp_glosry.htm "Glossary Entry"). Contains the [system ID](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensystem_id_glosry.htm "Glossary Entry") and the name of the [application server instance](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_instance_glosry.htm "Glossary Entry") of the [ABAP session](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_session_glosry.htm "Glossary Entry") on which the current connection is based.
