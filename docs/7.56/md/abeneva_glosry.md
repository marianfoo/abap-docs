IPO

Short for Input, Processing, Output. Possible process flow in program execution. The IPO principle implies that a program is executed in the order Input, Processing, and then Output of the results. See also [reporting](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenreporting_glosry.htm "Glossary Entry").
