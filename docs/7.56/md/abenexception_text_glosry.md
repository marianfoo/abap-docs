exception text

Text that describes an exception and is specified in an [exception class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenexception_class_glosry.htm "Glossary Entry"). An exception text can be given parameters using the attributes of the exception class. Exception texts are usually defined with reference to [messages](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmessage_glosry.htm "Glossary Entry") and in [system classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensystem_class_glosry.htm "Glossary Entry") with reference to the Online Text Repository (OTR).
