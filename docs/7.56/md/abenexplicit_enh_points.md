Explicit enhancement options

Explicit [enhancement options](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenhancement_point_glosry.htm "Glossary Entry") are created in ABAP programs by the following statements:

-   [ENHANCEMENT-POINT](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapenhancement-point.htm)
    
    Defines a position in an ABAP program as an enhancement option, at which one or more source code plug-ins can be inserted.
    
-   [ENHANCEMENT-SECTION](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapenhancement-section.htm)
    
    Defines a section of an ABAP program as an enhancement option, which can be replaced by one or more source code plug-ins.
