ABAP file interface

The ABAP file interface makes it possible to process [files](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenfile_glosry.htm "Glossary Entry") on the [host computers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenhost_computer_glosry.htm "Glossary Entry") of [AS instances](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_instance_glosry.htm "Glossary Entry") using ABAP statements.
