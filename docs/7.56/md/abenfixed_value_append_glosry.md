fixed value append

A fixed value append defines [fixed values](javascript:call_link\('abenfixed_value_glosry.htm'\) "Glossary Entry") and/or intervals that are appended to an existing [DDIC domain](javascript:call_link\('abendomain_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). Domains delivered by SAP can be enhanced in customer systems by means of fixed value appends without modification.
