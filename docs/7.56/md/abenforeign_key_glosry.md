foreign key

One or more columns of a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") ([foreign key table](javascript:call_link\('abenforeign_key_table_glosry.htm'\) "Glossary Entry")) that are [primary keys](javascript:call_link\('abenprimary_key_glosry.htm'\) "Glossary Entry") of another database table ([check table](javascript:call_link\('abencheck_table_glosry.htm'\) "Glossary Entry")).
