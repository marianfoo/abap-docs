global class

[Class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that is defined as a [global type](javascript:call_link\('abenglobal_type_glosry.htm'\) "Glossary Entry"). Global classes are declared in [class pools](javascript:call_link\('abenclass_pool_glosry.htm'\) "Glossary Entry") and are marked as such there using the addition [PUBLIC](javascript:call_link\('abapclass_options.htm'\)) of the statement [CLASS](javascript:call_link\('abapclass.htm'\)). A class pool can only contain one global class.
