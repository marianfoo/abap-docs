global interface

[Interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenoo_intf_glosry.htm "Glossary Entry") that is defined as a [global type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenglobal_type_glosry.htm "Glossary Entry"). Global interfaces are declared in the [interface pools](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninterface_pool_glosry.htm "Glossary Entry") and are marked as such there using the addition PUBLIC of the statement [INTERFACE](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapinterface.htm). An interface pool cannot contain anything except for a global interface.
