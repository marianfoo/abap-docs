GUI

Short for Graphical User Interface. Graphical part of the [user interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenuser_interface_glosry.htm "Glossary Entry"). The GUI is displayed in a [window](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenwindow_glosry.htm "Glossary Entry").
