host expression

ABAP-[expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenexpression_glosry.htm "Glossary Entry") specified in an operand position of an [ABAP SQL](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sql_host_expressions.htm) statement. Host expressions are always specified in parentheses preceded by the [escape character](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenescape_character_glosry.htm "Glossary Entry") @: @( ... ).
