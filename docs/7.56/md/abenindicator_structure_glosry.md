indicator structure

[Substructure](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensubstructure_glosry.htm "Glossary Entry") of a [structured type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstructured_type_glosry.htm "Glossary Entry"). The indicator structure is the last component of the structured type and mirrors the preceding components as indicators of type x of length 1. Indicator structures can be defined with the addition [WITH INDICATORS](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abaptypes_indicators.htm) of the statement [TYPES](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abaptypes_indicators.htm) and can be used as [ABAP SQL indicators](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sql_indicator_glosry.htm "Glossary Entry").
