initial dynpro

First [dynpro](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_glosry.htm "Glossary Entry") in a [dynpro sequence](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_sequence_glosry.htm "Glossary Entry"). In a [dialog transaction](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendialog_transaction_glosry.htm "Glossary Entry"), the dynpro that is linked to the [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry"). For dynpro sequences called using CALL SCREEN, the initial dynpro is the dynpro specified in this command.
