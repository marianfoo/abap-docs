initial value sharing

The status of a [boxed component](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenboxed_component_glosry.htm "Glossary Entry") as long as it is processed by read access only. The [initial value](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninitial_value_glosry.htm "Glossary Entry") of all type-compliant boxed components of the same type is only stored once in each [AS instance](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_instance_glosry.htm "Glossary Entry") in the [PXA](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpxa_glosry.htm "Glossary Entry").
