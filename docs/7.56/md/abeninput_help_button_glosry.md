input help button

One-column [pushbutton](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpushbutton_glosry.htm "Glossary Entry") with an [icon](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenicon_glosry.htm "Glossary Entry") that is displayed to the right of an [input field](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninput_output_field_glosry.htm "Glossary Entry") on a [dynpro](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_glosry.htm "Glossary Entry") if an [input help](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninput_help_glosry.htm "Glossary Entry") is defined for that field. Selecting the pushbutton has the same effect as pressing F4 for the field.
