input/output field

[Screen element](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscreen_element_glosry.htm "Glossary Entry") that is linked to a [dynpro field](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_field_glosry.htm "Glossary Entry"). Displays the content of the dynpro field and enables it to be changed using keys of the keyboard or on other input device that are not [function keys](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenfunction_key_glosry.htm "Glossary Entry").
