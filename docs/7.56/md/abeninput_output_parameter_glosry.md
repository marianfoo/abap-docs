input/output parameter

[Formal parameter](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenformal_parameter_glosry.htm "Glossary Entry") of a [procedure](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenprocedure_glosry.htm "Glossary Entry"). When a call is made, the value of an [actual parameter](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenactual_parameter_glosry.htm "Glossary Entry") is passed to an input/output parameter. Its value can be set in the procedure and is available in the actual parameter at the end of the procedure. In all procedures, input/output parameters are defined using the addition CHANGING and are therefore also referred to as CHANGING parameters.
