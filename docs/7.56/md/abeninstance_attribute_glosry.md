instance attribute

[Attribute](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that is declared using [DATA](javascript:call_link\('abapdata.htm'\)). Instance attributes are only valid in the [context](javascript:call_link\('abenobj_context_glosry.htm'\) "Glossary Entry") of an [instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of the class. The content of the instance attributes determines the state of the [objects](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of a class.
