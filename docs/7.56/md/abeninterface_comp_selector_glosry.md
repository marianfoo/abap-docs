interface component selector

Character ~. A [component](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomponent_glosry.htm "Glossary Entry") [comp](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_ifac_components.htm) of an [interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenoo_intf_glosry.htm "Glossary Entry") intf can be addressed in an implementing [class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry") or a [compound interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencompound_interface_glosry.htm "Glossary Entry") via intf~comp.
