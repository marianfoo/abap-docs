iXML Library

Global [classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenglobal_class_glosry.htm "Glossary Entry") that are accessible via [interfaces](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenoo_intf_glosry.htm "Glossary Entry") for handling [XML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxml_glosry.htm "Glossary Entry") documents in [DOM](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendom_glosry.htm "Glossary Entry") representation. The "integrated" iXML Library provides services such as an XML parser, an XML renderer, and reads and writes to the DOM. The interfaces of the iXML Library start with the prefix IF\_IXML.
