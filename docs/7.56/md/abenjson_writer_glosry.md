JSON writer

[XML writer](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxml_writer_glosry.htm "Glossary Entry") that writes [JSON](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenjson_glosry.htm "Glossary Entry") data. A JSON writer expects input for [JSON-XML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenjson_xml_glosry.htm "Glossary Entry") and uses it to create the JSON data.
