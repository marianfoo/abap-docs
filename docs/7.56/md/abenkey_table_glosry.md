key table

Superordinate term for [sorted tables](javascript:call_link\('abensorted_table_glosry.htm'\) "Glossary Entry") and [hashed tables](javascript:call_link\('abenhashed_table_glosry.htm'\) "Glossary Entry"), in which the [primary table key](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry"), unlike [standard tables](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry"), has its own stable administration that allows optimized access.
