[Formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") or other parameter to which an [actual parameter](javascript:call_link\('abenactual_parameter_glosry.htm'\) "Glossary Entry") is passed by specifying the formal parameter name in the calling statement.
