list processor

System program for [list processing](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlist_processing_glosry.htm "Glossary Entry") that contains the [list dynpro](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlist_dynpro_glosry.htm "Glossary Entry") for displaying [screen lists](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscreen_list_glosry.htm "Glossary Entry").
