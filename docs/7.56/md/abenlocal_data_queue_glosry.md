local data queue

Persistence layer into which data can be placed and read only in the order in which it was stored. The local data queue triggers the no-send scenario of the [qRFC](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenqrfc_glosry.htm "Glossary Entry").
