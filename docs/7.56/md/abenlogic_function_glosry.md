logical function

[Built-in function](javascript:call_link\('abenbuiltin_function_glosry.htm'\) "Glossary Entry"). Either a [Boolean function](javascript:call_link\('abenboole_function_glosry.htm'\) "Glossary Entry") that evaluates a [logical expression](javascript:call_link\('abenlogical_expression_glosry.htm'\) "Glossary Entry") or a [predicate function](javascript:call_link\('abenpredicate_function_glosry.htm'\) "Glossary Entry") that returns a [logical value](javascript:call_link\('abentruth_value_glosry.htm'\) "Glossary Entry").
