logon language

Language that an [ABAP user](javascript:call_link\('abenabap_user_glosry.htm'\) "Glossary Entry") specifies when logging on to an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). The logon language is persistent for the entire [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry"). The logon language affects [messages](javascript:call_link\('abenmessage_glosry.htm'\) "Glossary Entry"), [text elements](javascript:call_link\('abentext_element_glosry.htm'\) "Glossary Entry"), and the [text environment](javascript:call_link\('abentext_environment_glosry.htm'\) "Glossary Entry"). It can be determined using the method GET\_LOGON\_LANGUAGE of the class CL\_ABAP\_SYST.
