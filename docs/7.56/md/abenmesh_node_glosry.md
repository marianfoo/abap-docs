mesh node

Component of a [mesh](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_glosry.htm "Glossary Entry"). Mesh nodes are always [internal tables](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninternal_table_glosry.htm "Glossary Entry") with a structured [line type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrow_type_glosry.htm "Glossary Entry"). Mesh nodes can have [mesh associations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_association_glosry.htm "Glossary Entry") that they can address in [mesh paths](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_path_glosry.htm "Glossary Entry").
