mesh path

Chaining of multiple [mesh nodes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_node_glosry.htm "Glossary Entry") using their [mesh associations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_association_glosry.htm "Glossary Entry"). A mesh path describes a set of lines in its final path node. These lines are accessed when a mesh path is used.
