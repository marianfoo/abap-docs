mesh path

Chaining of multiple [mesh nodes](javascript:call_link\('abenmesh_node_glosry.htm'\) "Glossary Entry") using their [mesh associations](javascript:call_link\('abenmesh_association_glosry.htm'\) "Glossary Entry"). A mesh path describes a set of lines in its final path node. These lines are accessed when a mesh path is used.
