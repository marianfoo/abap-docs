message server

Component of the [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry") that implements the communication between the [AS instances](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_instance_glosry.htm "Glossary Entry"), and which determines the AS instance used when an [ABAP user](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_user_glosry.htm "Glossary Entry") logs on to an [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry").
