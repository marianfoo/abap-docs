Assignments in Release 7.50

Modification

System Class for Dynamic Mapping

The new system class [CL\_ABAP\_CORRESPONDING](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencl_abap_corresponding.htm) makes it possible to specify dynamic mapping rules for the component-by-component assignment of structures and internal tables.
