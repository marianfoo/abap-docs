numeric data type

[Data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_type_glosry.htm "Glossary Entry") whose [data objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_object_glosry.htm "Glossary Entry") contain numeric values in platform-dependent encoding. These data types are [integers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninteger_glosry.htm "Glossary Entry"), [packed numbers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpacked_number_glosry.htm "Glossary Entry"), [decimal floating pointing numbers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendecfloat_glosry.htm "Glossary Entry") and [binary floating pointing numbers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbinfloat_glosry.htm "Glossary Entry"). The corresponding [generic data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [numeric](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbuilt_in_types_generic.htm).
