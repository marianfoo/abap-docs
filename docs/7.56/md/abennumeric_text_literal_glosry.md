numeric text literal

[Literal](javascript:call_link\('abenabap_literal_glosry.htm'\) "Glossary Entry") that contains [numeric text](javascript:call_link\('abennumeric_text_glosry.htm'\) "Glossary Entry") and has the length of the specified characters. Only [typed](javascript:call_link\('abentyped_literal_glosry.htm'\) "Glossary Entry") numeric text literals are available.
