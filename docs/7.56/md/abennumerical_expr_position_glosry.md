numeric expression position

[Read position](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenread_position_glosry.htm "Glossary Entry") in which not only a [numeric data object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abennumeric_data_object_glosry.htm "Glossary Entry") that can be converted into the operand type can be specified, but also a [numeric expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abennumerical_expression_glosry.htm "Glossary Entry").
