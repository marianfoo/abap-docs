Object Navigator

Tool of the [ABAP Workbench](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for the central maintenance of [repository objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry"). The Object Navigator displays repository objects as nodes of different tree representations (browsers), such as [Repository Browser](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_browser_glosry.htm "Glossary Entry"), [Repository Information System](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenris_glosry.htm "Glossary Entry"), or [Enhancement Information System](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenh_info_system_glosry.htm "Glossary Entry"), in the navigation area. From here, they can be accessed via [forward navigation](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenforward_navigation_glosry.htm "Glossary Entry") in the tool area. Called using [transaction](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE80.
