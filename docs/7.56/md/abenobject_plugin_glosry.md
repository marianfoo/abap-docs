object plug-in

Object plug-ins are used to make modification-free functional [enhancements](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") to ABAP programs. They are instances of [BAdI implementation classes](javascript:call_link\('abenbadi_implement_class_glosry.htm'\) "Glossary Entry") and are administered by [BAdI objects](javascript:call_link\('abenbadi_object_glosry.htm'\) "Glossary Entry"). Object plug-ins are created when the [GET BADI](javascript:call_link\('abapget_badi.htm'\)) statement is executed and in accordance with the filter specifications.
