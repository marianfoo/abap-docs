obsolete language element

[ABAP language element](javascript:call_link\('abenabap_language_element_glosry.htm'\) "Glossary Entry") that still exists in the language for reasons of downward compatibility, but should no longer be used. Most obsolete language elements are allowed only outside [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry").
