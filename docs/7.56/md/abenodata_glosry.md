OData

Short for Open Data Protocol. OData is a standardized protocol for defining and consuming [RESTful](javascript:call_link\('abenrestful_glosry.htm'\) "Glossary Entry") APIs. [SAP Gateway](javascript:call_link\('abensap_gateway_glosry.htm'\) "Glossary Entry") provides such APIs for [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry").
