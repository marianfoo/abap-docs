OO transaction

The [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") of an OO transaction is linked with a [method](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmethod_glosry.htm "Glossary Entry") of a local or global [class](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry"). When the transaction is called, the associated program is loaded, for [instance methods](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninstance_method_glosry.htm "Glossary Entry") an [object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_glosry.htm "Glossary Entry") of the class is created, and the method is executed. OO transactions can be connected to the Transaction Service of the [Object services](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_services_glosry.htm "Glossary Entry").
