parallel Remote Function Call

pRFC for short. Extension of the [asynchronous remote function call](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenasynchronous_rfc_glosry.htm "Glossary Entry") that enables parallelization to be controlled.
