polymorphism

In object-oriented programming, polymorphism means the ability to address differently implemented [methods](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmethod_glosry.htm "Glossary Entry") that belong to different [objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_glosry.htm "Glossary Entry") of different [classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry") with the same name and via the same interface.
