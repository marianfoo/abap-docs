prefix namespace

Namespace for [repository objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry") of SAP departments, SAP partners, and SAP customers that can be requested from SAP. The names of the repository objects of a prefix namespace are introduced with a [namespace prefix](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenname_space_prefix_glosry.htm "Glossary Entry").
