spool parameter

Parameters that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenspool_request_glosry.htm "Glossary Entry"). [Archiving parameters](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenarchiving_parameter_glosry.htm "Glossary Entry") can be added to spool parameters.
