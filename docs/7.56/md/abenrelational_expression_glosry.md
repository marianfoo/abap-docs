relational expression

[Expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenexpression_glosry.htm "Glossary Entry") whose result is a [truth value](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentruth_value_glosry.htm "Glossary Entry"). Relational expressions include [comparison expressions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomparison_expression_glosry.htm "Glossary Entry") and [predicates](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpredicate_glosry.htm "Glossary Entry"). A single relational expression or multiple relational expressions combined using [Boolean operators](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenboolean_operator_glosry.htm "Glossary Entry") form a [logical expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlogical_expression_glosry.htm "Glossary Entry") in ABAP.
