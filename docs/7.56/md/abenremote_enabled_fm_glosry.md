remote-enabled function module

[RFM](javascript:call_link\('abenrfm_glosry.htm'\) "Glossary Entry") for short. [Function module](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") that is marked as remote-enabled in its properties and can be called using [remote function call](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry").
