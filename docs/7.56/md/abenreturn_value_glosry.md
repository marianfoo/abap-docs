return value

Result of [functional methods](javascript:call_link\('abenfunctional_method_glosry.htm'\) "Glossary Entry") and [built-in functions](javascript:call_link\('abenbuiltin_function_glosry.htm'\) "Glossary Entry"). The result of functional methods is the [formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") defined by the addition [RETURNING](javascript:call_link\('abapmethods_functional.htm'\)) of the statement [*\[*CLASS-*\]*](javascript:call_link\('abapclass-methods.htm'\))[METHODS](javascript:call_link\('abapmethods.htm'\)).
