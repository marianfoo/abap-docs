Repository Info System

Tool in the [ABAP Workbench](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for searching for [repository objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry"). Called in the [Object Navigator](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_navigator_glosry.htm "Glossary Entry") or via [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE15.
