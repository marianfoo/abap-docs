session memory

Memory for the modifiable parts of a program in an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry"). From the perspective of an ABAP program, the session memory can be divided into a [heap](javascript:call_link\('abenheap_glosry.htm'\) "Glossary Entry") area and a [stack](javascript:call_link\('abenstack_glosry.htm'\) "Glossary Entry") area. The non-modifiable parts of a program are stored in [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry"), which is referred to in the session memory.
