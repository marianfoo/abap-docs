session memory

Memory for the modifiable parts of a program in an [internal session](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninternal_session_glosry.htm "Glossary Entry"). From the perspective of an ABAP program, the session memory can be divided into a [heap](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenheap_glosry.htm "Glossary Entry") area and a [stack](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstack_glosry.htm "Glossary Entry") area. The non-modifiable parts of a program are stored in [PXA](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpxa_glosry.htm "Glossary Entry"), which is referred to in the session memory.
