runtime monitor

Tool for recording events raised by [system classes](javascript:call_link\('abensystem_class_glosry.htm'\) "Glossary Entry") when a program is executed. Called using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SRTM.
