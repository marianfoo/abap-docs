runtime error

Caused by uncatchable exceptions when a program is executed, or it can be forced by the language elements [RAISE SHORTDUMP](javascript:call_link\('abapraise_shortdump.htm'\)), [THROW SHORTDUMP](javascript:call_link\('abenconditional_expression_result.htm'\)), or [ASSERT](javascript:call_link\('abapassert.htm'\)) or by [exit messages](javascript:call_link\('abenexit_message_glosry.htm'\) "Glossary Entry"). Every runtime error terminates the program, which in turn raises a [database rollback](javascript:call_link\('abendatabase_rollback_glosry.htm'\) "Glossary Entry") and is documented by default in a [short dump](javascript:call_link\('abenshort_dump_glosry.htm'\) "Glossary Entry").
