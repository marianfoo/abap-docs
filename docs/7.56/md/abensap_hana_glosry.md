SAP HANA

SAP HANA is a platform for SAP In-Memory Computing based on the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). If an SAP HANA database is used as the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"), ABAP programs can use [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") to access SAP HANA directly. In addition, [secondary connections](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") can also be used to access SAP HANA from ABAP.
