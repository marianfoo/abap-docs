search help

A [repository object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry") maintained in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") to create value lists. It supplies [input fields](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninput_output_field_glosry.htm "Glossary Entry") on [dynpros](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_glosry.htm "Glossary Entry") with single or multi-column [input helps](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninput_help_glosry.htm "Glossary Entry"). Search helps can be linked in the ABAP Dictionary with [components](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencomponent_glosry.htm "Glossary Entry") of [structures](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenstructure_glosry.htm "Glossary Entry"), [DDIC data elements](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_element_glosry.htm "Glossary Entry"), and [check tables](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencheck_table_glosry.htm "Glossary Entry"). A search help enables searches for input values using associated data, without the need to know the exact spelling of the value.
