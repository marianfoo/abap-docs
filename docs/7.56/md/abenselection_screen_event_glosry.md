selection screen event

[Event](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenevent_glosry.htm "Glossary Entry") of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") that occurs during the [selection screen processing](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenselscreen_processing_glosry.htm "Glossary Entry"). The corresponding [event keyword](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenevent_keyword_glosry.htm "Glossary Entry") is [AT SELECTION-SCREEN](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abapat_selection-screen.htm).
