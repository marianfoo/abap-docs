selection screen version

Version of the [standard selection screen](javascript:call_link\('abenstandard_selscreen_glosry.htm'\) "Glossary Entry") of a [logical database](javascript:call_link\('abenlogical_data_base_glosry.htm'\) "Glossary Entry"). Selection screen versions allow certain elements of the standard selection screen to be suppressed. A selection screen version is created using [SELECTION-SCREEN - VERSION](javascript:call_link\('abapselection-screen_ldb_version.htm'\)) in combination with [SELECTION-SCREEN EXCLUDE](javascript:call_link\('abapselection-screen_ldb_version.htm'\)) and can be selected when linking an [executable program](javascript:call_link\('abenexecutable_program_glosry.htm'\) "Glossary Entry") with a logical database.
