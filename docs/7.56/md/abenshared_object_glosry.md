shared object

[Instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of a [shared memory-enabled class](javascript:call_link\('abenshm_class_glosry.htm'\) "Glossary Entry") or [anonymous data object](javascript:call_link\('abenanonymous_data_object_glosry.htm'\) "Glossary Entry") that is stored in an [area instance version](javascript:call_link\('abenarea_instance_version_glosry.htm'\) "Glossary Entry") of the [shared memory](javascript:call_link\('abenshared_memory_glosry.htm'\) "Glossary Entry"). A shared object is created using the addition AREA HANDLE of statement [CREATE OBJECT](javascript:call_link\('abapcreate_object.htm'\)) or [CREATE DATA](javascript:call_link\('abapcreate_data.htm'\)). [References](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") to [shared objects](javascript:call_link\('abenshared_objects_glosry.htm'\) "Glossary Entry") are possible in other shared objects of the same area instance version and during the connection to the respective area instance version via an [area handle](javascript:call_link\('abenarea_handle_glosry.htm'\) "Glossary Entry").
