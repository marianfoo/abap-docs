short dump

Error log that is displayed and stored after a [runtime error](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenruntime_error_glosry.htm "Glossary Entry") or by a [runtime warning](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenruntime_warning_glosry.htm "Glossary Entry"). Stored short dumps can be called using the [ABAP dump analysis](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_dump_analsyis_glosry.htm "Glossary Entry") tool.
