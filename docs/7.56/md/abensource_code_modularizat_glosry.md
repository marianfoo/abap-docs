source code modularization

Splitting of the [source code](javascript:call_link\('abensource_code_glosry.htm'\) "Glossary Entry") of a program into its individual parts. The source code modularization methods applied in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") are [include programs](javascript:call_link\('abeninclude_program_glosry.htm'\) "Glossary Entry") and [macros](javascript:call_link\('abenmacro_glosry.htm'\) "Glossary Entry").
