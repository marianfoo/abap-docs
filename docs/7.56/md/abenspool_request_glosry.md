spool request

Output request to the [SAP spool system](javascript:call_link\('abensap_spool_system_glosry.htm'\) "Glossary Entry"). In a spool request, the pages of a [spool list](javascript:call_link\('abenspool_list_glosry.htm'\) "Glossary Entry") are transferred to the SAP spool system. [Spool parameters](javascript:call_link\('abenprint_parameter_glosry.htm'\) "Glossary Entry") are defined when a spool request is opened and cannot be changed subsequently. A spool request is always assigned to the current [spool list level](javascript:call_link\('abenspool_list_level_glosry.htm'\) "Glossary Entry"). The spool requests of stacked spool list levels are also stacked.
