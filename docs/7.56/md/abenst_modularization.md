ST - Modularization with Subtemplates

An ST program can contain any number of templates that are defined with tt:template. All templates but the [main template](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenst_tt_template_main.htm) are subtemplates that can be called in the main template.
