ST - option, Mapping Rules

Syntax

... option="...,..."

Effect

The attribute option can be used to override the associated mapping rule, which depends on the data type, from and to [asXML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_xslt_asxml.htm) for certain data types. The possible options can be specified as the content of the attribute, in a list separated by commas. Any options that are mutually exclusive produce a syntax check warning.

-   [Mapping Rules for Elementary Types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenst_option_format.htm)
-   [Mapping Rules for XML Fragments](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenst_option_xsdany.htm)
