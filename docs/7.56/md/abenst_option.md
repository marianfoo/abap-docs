ST - option, Mapping Rules

Syntax

... option="...,..."

Effect

The attribute option can be used to override the associated mapping rule, which depends on the data type, from and to [asXML](javascript:call_link\('abenabap_xslt_asxml.htm'\)) for certain data types. The possible options can be specified as the content of the attribute, in a list separated by commas. Any options that are mutually exclusive produce a syntax check warning.
