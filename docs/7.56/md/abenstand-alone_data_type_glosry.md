standalone data type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that is defined using the statement [TYPES](javascript:call_link\('abaptypes.htm'\)) in an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"), as a data type of the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") or as an [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry").
