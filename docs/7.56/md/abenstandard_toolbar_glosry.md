standard toolbar

Part of a [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry"). Contains predefined pushbuttons and [icon](javascript:call_link\('abenicon_glosry.htm'\) "Glossary Entry") [push buttons](javascript:call_link\('abenpushbutton_glosry.htm'\) "Glossary Entry") to handle the program, as well as a special [command field](javascript:call_link\('abencommand_field_glosry.htm'\) "Glossary Entry") for the direct input of [function codes](javascript:call_link\('abenfunction_code_glosry.htm'\) "Glossary Entry"). Part of the [GUI status](javascript:call_link\('abengui_status_glosry.htm'\) "Glossary Entry") defined by [SET PF-STATUS](javascript:call_link\('abapset_pf-status_dynpro.htm'\)). Created using the [Menu Painter](javascript:call_link\('abenmenu_painter_glosry.htm'\) "Glossary Entry") tool. It is referred to as the [symbol toolbar](javascript:call_link\('abensymbol_toolbar_glosry.htm'\) "Glossary Entry") in the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry").
