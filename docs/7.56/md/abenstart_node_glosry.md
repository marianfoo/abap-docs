start node

-   In a [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry"), a [hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") from which a [hierarchy node navigator](javascript:call_link\('abenhierarchy_node_navi_glosry.htm'\) "Glossary Entry") determines further hierarchy nodes and adds them to its result set.
-   In a [mesh](javascript:call_link\('abenmesh_glosry.htm'\) "Glossary Entry"), a [mesh node](javascript:call_link\('abenmesh_node_glosry.htm'\) "Glossary Entry") for which a [mesh association](javascript:call_link\('abenmesh_association_glosry.htm'\) "Glossary Entry") is defined is the association's start node.
