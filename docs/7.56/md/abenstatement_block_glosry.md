statement block

Structural unit within [control structures](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencontrol_structure_glosry.htm "Glossary Entry") that consists of one or more successive statements. The statements of a statement block are processed sequentially. Statement blocks can again contain control structures.
