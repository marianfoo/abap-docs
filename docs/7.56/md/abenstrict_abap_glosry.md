strict ABAP

ABAP development in a [strict ABAP development environment](javascript:call_link\('abenstrict_abap_dev_envir_glosry.htm'\) "Glossary Entry"). Compared to [classic ABAP](javascript:call_link\('abenclassic_abap_glosry.htm'\) "Glossary Entry"), there are the following restrictions:

-   Only [restricted ABAP language versions](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry") are allowed.
-   Access to other repository objects is restricted by [released APIs](javascript:call_link\('abenreleased_api_glosry.htm'\) "Glossary Entry").
-   Usage of tools is limited for example to [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry").
