character string expression

String expression for short. Formulation of an operation with character-like operands in a [calculation expression](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencalculation_expression_glosry.htm "Glossary Entry"), the result of which is a [character string](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencharacter_string_glosry.htm "Glossary Entry").
