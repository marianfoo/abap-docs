string literal

[Literal](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_literal_glosry.htm "Glossary Entry") of data type string. Only [text string literals](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentext_string_literal_glosry.htm "Glossary Entry") can be created as string literals in the current release.
