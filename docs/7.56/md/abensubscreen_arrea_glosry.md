subscreen area

Area within a [screen layout](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry") in which other screen layouts can be displayed as [subscreens](javascript:call_link\('abensubscreen_glosry.htm'\) "Glossary Entry").
