Switch Framework

SFW for short. Framework for the externally controlling the visibility of [repository objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrepository_object_glosry.htm "Glossary Entry"), their components, or [enhancements](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenenhancement_glosry.htm "Glossary Entry") using [switches](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenswitch_glosry.htm "Glossary Entry").
