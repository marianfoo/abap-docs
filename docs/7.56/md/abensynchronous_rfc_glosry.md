synchronous Remote Function Call

sRFC for short. [Remote function call](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenremote_function_call_glosry.htm "Glossary Entry") in which the calling program waits for the end of processing of the remotely called function.
