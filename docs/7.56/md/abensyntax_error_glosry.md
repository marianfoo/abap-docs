syntax error

Violation of the ABAP syntax rules. A program that has syntax errors cannot be executed. Syntax errors are returned after the [syntax check](javascript:call_link\('abensyntax_check_glosry.htm'\) "Glossary Entry") that can be called in [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry").
