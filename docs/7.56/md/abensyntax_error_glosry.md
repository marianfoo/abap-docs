syntax error

Violation of the ABAP syntax rules. A program that has syntax errors cannot be executed. Syntax errors are returned after the [syntax check](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensyntax_check_glosry.htm "Glossary Entry") that can be called in [ABAP Editor](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_editor_glosry.htm "Glossary Entry").
