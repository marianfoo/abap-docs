system interface

[Interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_glosry.htm "Glossary Entry") of the [class library](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_library_glosry.htm "Glossary Entry") that is included in [system classes](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensystem_class_glosry.htm "Glossary Entry") and is used to implement system functions in separate classes or is required as a [tag interface](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentag_interface_glosry.htm "Glossary Entry"). The name usually begins with IF\_ABAP\_ or IF\_SYSTEM\_. Using the [input help](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninput_help_glosry.htm "Glossary Entry") for IF\_ABAP\_\* or IF\_SYSTEM\_\* in the [Class Builder](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenclass_builder_glosry.htm "Glossary Entry") can give an overview of all system interfaces.
