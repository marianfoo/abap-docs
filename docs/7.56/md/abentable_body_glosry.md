table body

All lines in an [internal table](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninternal_table_glosry.htm "Glossary Entry"). Name of the actual internal table for internal tables with a [header line](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenheader_line_glosry.htm "Glossary Entry"). The table body can be addressed by attaching \[\] to the table name. Internal tables and table bodies are identical for internal tables without a header line.
