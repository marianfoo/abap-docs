itab - Expressions and Functions

Expressions and functions for reading internal tables can be listed at many [operand positions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenexpression_positions.htm).

-   [Table Expressions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentable_expressions.htm)
-   [Table Functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentable_functions.htm)
-   [FOR - Table Iterations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenfor_itab.htm)
-   [FILTER - Filter Operator](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenconstructor_expression_filter.htm)

[Mesh path expressions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmesh_path_expression.htm) are a special form of table expressions.
