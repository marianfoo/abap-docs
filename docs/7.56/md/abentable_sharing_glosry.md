table sharing

[Sharing](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensharing_glosry.htm "Glossary Entry") of [internal tables](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abeninternal_table_glosry.htm "Glossary Entry") in [assignments](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenassignment_glosry.htm "Glossary Entry") and [pass by value](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenpass_by_value_glosry.htm "Glossary Entry"). Table sharing is used only between tables of the same type if their [line types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrow_type_glosry.htm "Glossary Entry") do not themselves contain any [table types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentable_type_glosry.htm "Glossary Entry").
