termination message

[Message](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmessage_glosry.htm "Glossary Entry") of [message type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenmessage_type_glosry.htm "Glossary Entry") A. Termination messages are displayed in a [dialog box](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendialog_box_glosry.htm "Glossary Entry"), create a [database rollback](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendatabase_rollback_glosry.htm "Glossary Entry"), and terminate the running program.
