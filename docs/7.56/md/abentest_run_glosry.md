test run

Execution of a [test task](javascript:call_link\('abentest_task_glosry.htm'\) "Glossary Entry") in [ABAP Unit](javascript:call_link\('abenabap_unit_glosry.htm'\) "Glossary Entry"). Test runs for test tasks of an individual program can be started directly in the corresponding editor via Program or Function Module → Test → Unit Test or Class → Unit Test. More extensive test runs can be executed using [Code Inspector](javascript:call_link\('abencode_inspector_glosry.htm'\) "Glossary Entry"). The results of a test run are displayed in the ABAP Unit interface.
