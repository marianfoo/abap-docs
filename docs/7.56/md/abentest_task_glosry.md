test task

Combination of tests of one or more ABAP programs in [ABAP Unit](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_unit_glosry.htm "Glossary Entry") for which a [test run](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentest_run_glosry.htm "Glossary Entry") can be executed.
