text field literal

[Character literal](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abencharacter_literal_glosry.htm "Glossary Entry") of the type [text field](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentext_field_glosry.htm "Glossary Entry") with the length of the specified characters. A text field literal can be linked to a [text symbol](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentext_symbol_glosry.htm "Glossary Entry").
