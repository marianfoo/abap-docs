time stamp

Combined [date](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendate_glosry.htm "Glossary Entry")/ [time](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentime_glosry.htm "Glossary Entry") specification for determining a point in time. Time stamps of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") are [POSIX time stamps](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenposix_timestamp_glosry.htm "Glossary Entry") independent of time zone. They are represented in a [time stamp field](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentimestamp_field_glosry.htm "Glossary Entry") with the [time stamp type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentimestamp_field_glosry.htm "Glossary Entry") [utclong](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbuiltin_types_date_time.htm) or as [time stamps in packed numbers](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentime_stamps_packed.htm) whose data type is defined using the [DDIC data elements](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_element_glosry.htm "Glossary Entry") TIMESTAMP or TIMESTAMPL from the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_dictionary_glosry.htm "Glossary Entry").
