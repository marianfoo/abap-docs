time zone

Geographical area in which the same [local time](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlocal_time_glosry.htm "Glossary Entry") and [local date](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenlocal_date_glosry.htm "Glossary Entry") apply based on a set of [time zone rules](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentime_zone_rules.htm) at a given time.
