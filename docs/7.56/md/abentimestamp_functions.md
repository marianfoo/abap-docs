Time Stamp Functions (ts\_func)

The following [built-in functions](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbuiltin_function_glosry.htm "Glossary Entry") for processing time stamps in time stamp fields are available:

-   [utclong\_current](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenutclong_current.htm) creates a current time stamp
-   [utclong\_add](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenutclong_add.htm) adds values to a time stamp
-   [utclong\_diff](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenutclong_diff.htm) determines the difference between two time stamps
