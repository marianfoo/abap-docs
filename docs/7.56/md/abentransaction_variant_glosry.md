transaction variant

Transaction variants are allowed to preassign default values to the input fields of several [dynpros](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendynpro_glosry.htm "Glossary Entry") of a [transaction](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_glosry.htm "Glossary Entry"), to change the attributes of [screen elements](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscreen_element_glosry.htm "Glossary Entry"), and to suppress entire [screens](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenscreen_glosry.htm "Glossary Entry"). Transaction variants are maintained and administrated using [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") SHD0.
