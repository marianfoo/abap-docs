Transformation Editor

Tool in the [ABAP Workbench](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for creating [XSLT programs](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxslt_program_glosry.htm "Glossary Entry") or [Simple Transformations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abensimple_transformation_glosry.htm "Glossary Entry") in [transformation source code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxslt_program_glosry.htm "Glossary Entry"). Called using the [transaction code](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentransaction_code_glosry.htm "Glossary Entry") STRANS.
