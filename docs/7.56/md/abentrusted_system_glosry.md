trusted system

[AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") that is classified as a trusted caller of [remote function calls](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry") in a [trusting system](javascript:call_link\('abentrusting_system_glosry.htm'\) "Glossary Entry"). The trusted system provides the [RFC client](javascript:call_link\('abenrfc_client_glosry.htm'\) "Glossary Entry"). The system is classified in [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SMT1. Trusted systems log on to the trusting system without passing a password. User-specific logon data is checked in the trusting system.
