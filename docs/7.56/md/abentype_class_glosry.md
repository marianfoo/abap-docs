type description class

[System class](javascript:call_link\('abensystem_class_glosry.htm'\) "Glossary Entry") of the [runtime type services](javascript:call_link\('abenrun_time_type_services_glosry.htm'\) "Glossary Entry"). The [static methods](javascript:call_link\('abenstatic_method_glosry.htm'\) "Glossary Entry") of a type description classes create [type description objects](javascript:call_link\('abentype_object_glosry.htm'\) "Glossary Entry") from existing types or construct new types.
