type conversion

Converts the content of a [data object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_object_glosry.htm "Glossary Entry") according to the associated [conversion rules](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenconversion_rule_glosry.htm "Glossary Entry"). Type conversions occur implicitly in [assignments](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenassignment_glosry.htm "Glossary Entry") between data objects with different [data types](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_type_glosry.htm "Glossary Entry") or can be performed explicitly using the [conversion operator](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenconversion_operator_glosry.htm "Glossary Entry") [CONV](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenconstructor_expression_value.htm).
