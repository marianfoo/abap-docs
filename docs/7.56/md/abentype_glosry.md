type

Abstract description of an [object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_glosry.htm "Glossary Entry"). Either a [data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_type_glosry.htm "Glossary Entry") for a [data object](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abendata_object_glosry.htm "Glossary Entry") or an [object type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_type_glosry.htm "Glossary Entry") for an object in [ABAP Objects](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_objects_glosry.htm "Glossary Entry"). The associated [generic data type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [any](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenbuilt_in_types_generic.htm). All types share the same [namespace](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abennaming_conventions.htm).
