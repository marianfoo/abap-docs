type name

Name of a [type](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abentype_glosry.htm "Glossary Entry"). In a program, only the [relative type name](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenrelative_type_name_glosry.htm "Glossary Entry") is known and statically usable. The [absolute type name](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabsolute_typename_glosry.htm "Glossary Entry") can also be used to specify dynamic types.
