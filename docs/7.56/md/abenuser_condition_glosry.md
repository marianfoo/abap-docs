user condition

Part of an [access condition](javascript:call_link\('abenaccess_condition_glosry.htm'\) "Glossary Entry") in a [CDS role](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry") that compares an element of a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") with the current [user name](javascript:call_link\('abenuser_name_glosry.htm'\) "Glossary Entry").
