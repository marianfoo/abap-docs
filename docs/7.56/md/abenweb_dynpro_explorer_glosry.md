Web Dynpro Explorer

Tool in the [ABAP Workbench](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for creating [Web Dynpros](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenweb_dynpro_glosry.htm "Glossary Entry"). Called via Create → Web Dynpro → ... in [Object Navigator](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenobject_navigator_glosry.htm "Glossary Entry").
