work area

Description for a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") that is particularly useful when working with [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") or [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") as a source for changing operations or as a target for reading operations.
