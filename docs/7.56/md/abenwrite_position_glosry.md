write position

[Operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry") in which the [operand](javascript:call_link\('abenabap_operand_glosry.htm'\) "Glossary Entry") is written to. There are write-only positions and positions where an operand is also read-accessed.
