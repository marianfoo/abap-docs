write position

[Operand position](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenoperand_position_glosry.htm "Glossary Entry") in which the [operand](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_operand_glosry.htm "Glossary Entry") is written to. There are write-only positions and positions where an operand is also read-accessed.
