WebSocket protocol

WSP for short. A TCP based protocol for the bidirectional communication between a client and a server. Using [ABAP Push Channels](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_push_channels_glosry.htm "Glossary Entry"), an [AS ABAP](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenas_abap_glosry.htm "Glossary Entry") can be used as a server and as a client for the WebSocket protocol.
