XOP

Short for XML-binary Optimized Packaging (see [http://www.w3.org/TR/xop10/](http://www.w3.org/TR/xop10/)). Representation of [XML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxml_glosry.htm "Glossary Entry") data in which any binary raw data is stored separately from the data represented by characters. In ABAP, XOP is supported by a [format](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sxml_lib_formats.htm) of the [sXML Library](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_sxml_lib.htm).
