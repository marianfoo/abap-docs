XSLT

Short for Extensible Stylesheet Language Transformation, see [www.w3.org/TR/xslt](http://www.w3.org/TR/xslt). XSLT allows the conversion of [XML](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxml_glosry.htm "Glossary Entry") formats into any other XML formats. The [ABAP runtime framework](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") contains the [SAP XSLT processor](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxslt_processor_glosry.htm "Glossary Entry") for the execution of [XSL transformations](https://help.sap.com/doc/abapdocu_756_index_htm/7.56/en-US/abenxsl_transformation_glosry.htm "Glossary Entry").
