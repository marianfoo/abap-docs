AT PF - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapat_pfnn.htm)

Syntax

AT PFnn.

Effect

Obsolete: Introduces an event block whose event is selected when a list is displayed by selecting a function using the function code PFnn. Here. nn is a number between 01 and 24.
