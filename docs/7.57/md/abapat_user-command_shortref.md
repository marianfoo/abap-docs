AT USER-COMMAND - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapat_user-command.htm)

Syntax

AT USER-COMMAND.

Effect

Introduces an event block whose result is raised by a function with a self-defined function code on a list.
