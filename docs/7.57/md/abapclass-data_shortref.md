CLASS-DATA - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapclass-data.htm)

Syntax

CLASS-DATA attr options.

Effect

Declares a [static attribute](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstatic_attribute_glosry.htm "Glossary Entry") attr of a class or an interface.

Addition  

-   options
    As in [DATA](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapdata_shortref.htm), except that the addition WITH HEADER LINE cannot be specified.
