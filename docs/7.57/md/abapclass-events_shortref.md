CLASS-EVENTS - Short Reference

[Reference](javascript:call_link\('abapclass-events.htm'\))

Syntax

Effect

Declares a [static event](javascript:call_link\('abenstatic_event_glosry.htm'\) "Glossary Entry") evt of a class or an interface.
