CLASS-METHODS - Short Reference

[Reference](javascript:call_link\('abapclass-methods.htm'\))

Syntax

CLASS-METHODS meth options
                  *|**{*FOR TABLE FUNCTION cds\_entity*}*.

Effect

Declares a [static method](javascript:call_link\('abenstatic_method_glosry.htm'\) "Glossary Entry") meth of a class or interface.

Additions   

-   options
    As in [METHODS](javascript:call_link\('abapmethods_shortref.htm'\)).
-   [FOR TABLE FUNCTION cds\_tabfunc](javascript:call_link\('abapclass-methods_for_tabfunc.htm'\))
    Declares an [AMDP function implementation](javascript:call_link\('abenamdp_function_method_glosry.htm'\) "Glossary Entry") for a [CDS table function](javascript:call_link\('abencds_table_function_glosry.htm'\) "Glossary Entry").
