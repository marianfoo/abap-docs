COMPUTE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapcompute.htm)

Syntax

COMPUTE *\[*EXACT*\]* lhs =*|*?= rhs.

Effect

Obsolete: Assigns the value of a right side rhs to a left side lhs.

Addition  

-   EXACT
    For certain arithmetic expressions on the right side, forces a [lossless calculation](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlossless_calculation_glosry.htm "Glossary Entry") without rounding.
