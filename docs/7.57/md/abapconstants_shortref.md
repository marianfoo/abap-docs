CONSTANTS - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapconstants.htm)

Syntax

CONSTANTS const options.

Effect

Declares a constant const.

Addition  

-   options
    As in [DATA](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapdata_shortref.htm), except that the addition VALUE must be specified and the addition READ-ONLY cannot be specified.
