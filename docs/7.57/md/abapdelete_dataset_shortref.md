DELETE DATASET - Short Reference

[Reference](javascript:call_link\('abapdelete_dataset.htm'\))

Syntax

DELETE DATASET dset.

Effect

Deletes the file specified in dset on the [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") of the current [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry").
