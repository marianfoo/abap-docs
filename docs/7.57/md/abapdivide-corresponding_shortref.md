DIVIDE-CORRESPONDING - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapadd-corresponding.htm)

Syntax

DIVIDE-CORRESPONDING struc1 BY struc2.

Effect

Obsolete: Divides the components of a structure struc1 by identically named components of a structure struc2.
