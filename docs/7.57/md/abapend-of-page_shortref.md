END-OF-PAGE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapend-of-page.htm)

Syntax

END-OF-PAGE.

Effect

Initiates an event block whose event is raised when the [page footer](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpage_footer_glosry.htm "Glossary Entry") is reached when a basic list is created.
