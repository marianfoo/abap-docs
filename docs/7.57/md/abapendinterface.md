ENDINTERFACE

[Short Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapinterface_shortref.htm)

Syntax

ENDINTERFACE.

Effect

The statement ENDINTERFACE closes an interface definition introduced using [INTERFACE](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapinterface.htm).
