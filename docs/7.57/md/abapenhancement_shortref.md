ENHANCEMENT - Short Reference

[Reference](javascript:call_link\('abapenhancement.htm'\))

Syntax

Effect

Implements a [source code enhancement](javascript:call_link\('abensource_code_enhancement_glosry.htm'\) "Glossary Entry") as a [source code plug-in](javascript:call_link\('abensource_code_plugin_glosry.htm'\) "Glossary Entry") with the ID id.
