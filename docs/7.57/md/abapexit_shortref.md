EXIT - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapexit_loop.htm)

Syntax

EXIT.

Effect

Exits a [loop](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapexit_loop.htm) or a [processing block](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapexit_processing_blocks.htm). Within a loop, the entire loop processing is terminated and the program resumes after the loop. Outside of a loop, the current processing block is terminated.
