GET PARAMETER - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapget_parameter.htm)

Syntax

GET PARAMETER ID pid FIELD dobj.

Effect

Returns the value of the [SPA/GPA parameters](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenspa_gpa_parameter_glosry.htm "Glossary Entry") specified in pid in dobj.
