GET RUN TIME - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapget_run_time.htm)

Syntax

GET RUN TIME FIELD rtime.

Effect

Returns the program runtime, measured in microseconds, since the first execution of the statement in rtime.
