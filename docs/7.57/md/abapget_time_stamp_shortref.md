GET TIME STAMP - Short Reference

[Reference](javascript:call_link\('abapget_time-stamp.htm'\))

Syntax

GET TIME STAMP FIELD time\_stamp.

Effect

Returns the current [time stamp](javascript:call_link\('abentime_stamp_glosry.htm'\) "Glossary Entry") in time\_stamp of the type TIMESTAMP or TIMESTAMPL.
