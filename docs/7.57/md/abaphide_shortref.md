HIDE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abaphide.htm)

Syntax

HIDE dobj.

Effect

Saves the content of the variable dobj with the current list line in the current list level.
