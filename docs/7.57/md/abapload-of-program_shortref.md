LOAD-OF-PROGRAM - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapload-of-program.htm)

Syntax

LOAD-OF-PROGRAM.

Effect

Initializes an event block whose event is raised in an [internal session](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninternal_session_glosry.htm "Glossary Entry") when the program is loaded.
