LOCAL - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abaplocal.htm)

Syntax

LOCAL dobj.

Effect

Obsolete: Saves the content of the data object dobj to an internal cache in a [procedure](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenprocedure_glosry.htm "Glossary Entry").
