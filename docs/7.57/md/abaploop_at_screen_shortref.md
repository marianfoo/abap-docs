LOOP AT SCREEN - Short Reference

[Reference](javascript:call_link\('abaploop_at_screen.htm'\))

Syntax

LOOP AT SCREEN INTO wa.
  ...
ENDLOOP.

Effect

Reads the properties of the screen elements of the current [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") in a loop and assigns the results to a work area wa of the structure [SCREEN](javascript:call_link\('abenscreen.htm'\)).
