NODES - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapnodes.htm)

Syntax

NODES node *\[*TYPE type*\]*.

Effect

Declares an [interface work area](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninterface_work_area_glosry.htm "Glossary Entry") node for data transfer in [logical databases](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlogical_data_base_glosry.htm "Glossary Entry").

Addition  

-   TYPE type
    Specifies the data type type of the work area from a predefined list.
