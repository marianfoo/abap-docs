SET BIT - Short Reference

[Reference](javascript:call_link\('abapset_bit.htm'\))

Syntax

SET BIT bitpos OF byte\_string *\[*TO val*\]*.

Effect

Sets the bit to the bit position bitpos of data object byte\_string to the value 1.

Addition  

-   TO val
    Sets the bit to the value val.
