SET HOLD DATA - Short Reference

[Reference](javascript:call_link\('abapset_hold_data.htm'\))

Syntax

SET HOLD DATA *{*ON*|*OFF*}*.

Effect

Enables or disables the standard menu entries Hold Data, Set Data, and Delete Data in the [GUI status](javascript:call_link\('abengui_status_glosry.htm'\) "Glossary Entry") of the current [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry").
