SET LOCALE LANGUAGE - Short Reference

[Reference](javascript:call_link\('abapset_locale.htm'\))

Syntax

SET LOCALE LANGUAGE lang *\[*COUNTRY cntry*\]* *\[*MODIFIER mod*\]*.

Effect

Sets the [text environment](javascript:call_link\('abentext_environment_glosry.htm'\) "Glossary Entry").

Additions   

-   LANGUAGE lang
    Specifies the language in lang.
-   [COUNTRY cntry](javascript:call_link\('abapset_locale_obsolete.htm'\))
    Obsolete: Specifies the country key in cntry.
-   [MODIFIER mod](javascript:call_link\('abapset_locale_obsolete.htm'\))
    Obsolete: Specifies the ID for a specific locale in mod.
