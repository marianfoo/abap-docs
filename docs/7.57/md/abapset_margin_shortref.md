SET MARGIN - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapset_margin.htm)

Syntax

SET MARGIN macol *\[*marow*\]*.

Effect

Defines the left margin of a spool list as the columns specified in macol and the upper margin as the rows specified in marow.
