SET RUN TIME ANALYZER - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapset_run_time_analyzer.htm)

Syntax

SET RUN TIME ANALYZER *{*ON*|*OFF*}*.

Effect

Defines which units of a program are measured with the [runtime analysis](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenruntime_analysis_glosry.htm "Glossary Entry") tool.
