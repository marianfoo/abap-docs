SET RUN TIME ANALYZER - Short Reference

[Reference](javascript:call_link\('abapset_run_time_analyzer.htm'\))

Syntax

SET RUN TIME ANALYZER *{*ON*|*OFF*}*.

Effect

Defines which units of a program are measured with the [runtime analysis](javascript:call_link\('abenruntime_analysis_glosry.htm'\) "Glossary Entry") tool.
