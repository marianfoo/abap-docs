START-OF-EDITING

[Short Reference](javascript:call_link\('abapstart-of-editing_shortref.htm'\))

Obsolete Syntax

START-OF-EDITING.

Effect

The statement START-OF-EDITING defines an [event block](javascript:call_link\('abenevent_block_glosry.htm'\) "Glossary Entry") in an [executable program](javascript:call_link\('abenexecutable_program_glosry.htm'\) "Glossary Entry"). The event of the program is raised from the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") in the [executable program flow](javascript:call_link\('abenreporting_process.htm'\)) directly after [START-OF-SELECTION](javascript:call_link\('abapstart-of-selection.htm'\)).

Hint

An implementation of the event block START-OF-EDITING is unnecessary.
