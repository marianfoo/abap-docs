SUPPLY - Short Reference

[Reference](javascript:call_link\('abapsupply.htm'\))

Syntax

SUPPLY key1 = f1 key2 = f2 ... TO CONTEXT context\_ref.

Effect

Obsolete: Fills the key fields key1, key2, ... of a [context](javascript:call_link\('abencontext_glosry.htm'\) "Glossary Entry") instance referenced by context\_ref with the values of data objects f1, f2, ...
