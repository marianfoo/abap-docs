ABAP Compiler

ABAP Compiler creates [byte code](javascript:call_link\('abenbytecode_glosry.htm'\) "Glossary Entry") as intermediate code when generating a program from [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") that is stored in the database as a program [load](javascript:call_link\('abenload_glosry.htm'\) "Glossary Entry") and is loaded into the [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry") if required.
