ABAP daemon class

Global subclass of CL\_ABAP\_DAEMON\_EXT\_BASE. The instances of an ABAP daemon class are [ABAP daemons](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") created by the [ABAP daemon framework](javascript:call_link\('abenabap_daemon_framework_glosry.htm'\) "Glossary Entry"), each of which are executed in a separate [ABAP daemon session](javascript:call_link\('abenabap_daemon_session_glosry.htm'\) "Glossary Entry"). An ABAP daemon class implements interface methods inherited from the superclass to react to [ABAP daemon events](javascript:call_link\('abenabap_daemon_event_glosry.htm'\) "Glossary Entry").
