ABAP daemon manager

The class CL\_ABAP\_DAEMON\_CLIENT\_MANAGER. The static methods of the ABAP daemon manager are used to create and use [ABAP daemons](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry").
