ABAP daemon memory

Part of the [user memory](javascript:call_link\('abenuser_memory_glosry.htm'\) "Glossary Entry") in which an [ABAP daemon](javascript:call_link\('abenabap_daemon_glosry.htm'\) "Glossary Entry") can store data in [PCP](javascript:call_link\('abenpcp_glosry.htm'\) "Glossary Entry") format. An ABAP daemon can use approximately 1 megabyte of the ABAP daemon memory.
