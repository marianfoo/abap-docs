ABAP Database Connectivity

ADBC for short. Class-based API for the [Native SQL interface](javascript:call_link\('abennative_sql_interface_glosry.htm'\) "Glossary Entry"). ADBC enables object-oriented execution of dynamically specified [Native SQL](javascript:call_link\('abennative_sql_glosry.htm'\) "Glossary Entry").
