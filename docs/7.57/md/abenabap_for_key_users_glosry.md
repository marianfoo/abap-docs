ABAP for Key Users

[Restricted ABAP language version](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry") with restricted language scope and restricted access to [released APIs](javascript:call_link\('abenreleased_api_glosry.htm'\) "Glossary Entry") for [enhancements](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") made by key users at dedicated enhancement points. ABAP for Key Users is available in the special enhancement tools for key users.
