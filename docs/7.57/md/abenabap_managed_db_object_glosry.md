ABAP-managed database object

[Database object](javascript:call_link\('abendb_object_glosry.htm'\) "Glossary Entry") in the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") that is both managed and used by AS ABAP. Only ABAP should be used to access an ABAP-managed database object.
