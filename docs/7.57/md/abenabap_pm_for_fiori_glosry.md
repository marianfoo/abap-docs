ABAP programming model for SAP Fiori

Obsolete programming model for SAP HANA-optimized Fiori apps based on the [AS ABAP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_abap_glosry.htm "Glossary Entry").
