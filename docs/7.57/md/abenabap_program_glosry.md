ABAP program

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that contains [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") and has a [program type](javascript:call_link\('abenprogram_type_glosry.htm'\) "Glossary Entry").
