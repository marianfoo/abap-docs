ABAP push channel

[Repository object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrepository_object_glosry.htm "Glossary Entry") that defines the [APC handler class](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenapc_handler_class_glosry.htm "Glossary Entry"), a node in the [ICF](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenicf_glosry.htm "Glossary Entry") service tree, the protocol ([Web socket](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenwsp_glosry.htm "Glossary Entry") or TCP), an optional subprotocol ([PCP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpcp_glosry.htm "Glossary Entry")), and other properties for an [APC server](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenapc_server_glosry.htm "Glossary Entry").
