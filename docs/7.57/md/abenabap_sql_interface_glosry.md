ABAP SQL Interface

Part of the [database interface](javascript:call_link\('abendatabase_interface_glosry.htm'\) "Glossary Entry") responsible for [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") statements. The ABAP SQL interface transforms all ABAP SQL statements that access the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") to vendor-specific [SQL](javascript:call_link\('abensql_glosry.htm'\) "Glossary Entry") and forwards the results to the [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry").
