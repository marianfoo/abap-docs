application program

[ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that is part of the business applications executed on [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Application programs generally only work with the data of the current [application client](javascript:call_link\('abenapplication_client_glosry.htm'\) "Glossary Entry") that is stored in [application tables](javascript:call_link\('abenapplication_table_glosry.htm'\) "Glossary Entry").
