area instance

Sum of all [area instance versions](javascript:call_link\('abenarea_instance_version_glosry.htm'\) "Glossary Entry") with the same area instance name. An area instance is a unique variant of an [area](javascript:call_link\('abenarea_glosry.htm'\) "Glossary Entry") and is divided into versions.
