authorization

Entry in the [user master record](javascript:call_link\('abenuser_master_record_glosry.htm'\) "Glossary Entry") as part of an [authorization profile](javascript:call_link\('abenauthorization_profile_glosry.htm'\) "Glossary Entry"). An authorization consists of complete or generic values for the [authorization fields](javascript:call_link\('abenauthorization_field_glosry.htm'\) "Glossary Entry") of an [authorization object](javascript:call_link\('abenauthorization_object_glosry.htm'\) "Glossary Entry"). The combination specifies which activities an [ABAP user](javascript:call_link\('abenabap_user_glosry.htm'\) "Glossary Entry") can use to access which data. Authorizations are generated with the profile generator from role management tool (transaction PFCG) and can be displayed using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SU03.
