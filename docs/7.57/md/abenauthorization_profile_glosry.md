authorization profile

A summary of multiple [authorizations](javascript:call_link\('abenauthorization_glosry.htm'\) "Glossary Entry"). An authorization can be assigned to multiple authorization profiles. Entering authorization profiles in the [user master record](javascript:call_link\('abenuser_master_record_glosry.htm'\) "Glossary Entry") assigns [user names](javascript:call_link\('abenuser_name_glosry.htm'\) "Glossary Entry") to authorizations. Authorization profiles can be generated by the profile generator from role maintenance (transaction PFCG) or be displayed using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SU02.
