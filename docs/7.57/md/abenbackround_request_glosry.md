background request

[Job](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenjob_glosry.htm "Glossary Entry") used to start one or more [ABAP programs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_program_glosry.htm "Glossary Entry") in [background processing](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbackround_processing_glosry.htm "Glossary Entry"). A background request consists of [background tasks](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbackround_task_glosry.htm "Glossary Entry").
