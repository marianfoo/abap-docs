BAdI interface

A BAdI interface is a global [interface](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenoo_intf_glosry.htm "Glossary Entry") that includes the [tag interface](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentag_interface_glosry.htm "Glossary Entry") IF\_BADI\_INTERFACE. A BAdI interface can be used as part of the definition of one or more [BAdIs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbadi_glosry.htm "Glossary Entry") and may only contain [methods](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmethod_glosry.htm "Glossary Entry") ([BAdI methods](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbadi_method_glosry.htm "Glossary Entry")) and [events](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenevent_glosry.htm "Glossary Entry") but not variable [attributes](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenattribute_glosry.htm "Glossary Entry"). Furthermore, a BAdI interface cannot contain any [component interfaces](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencomponent_interface_glosry.htm "Glossary Entry") other than other BAdI interfaces. An interface that includes a BAdI interface is also a BAdI interface.
