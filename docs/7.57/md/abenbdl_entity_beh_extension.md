Effect

Defines the [behavior extension body](javascript:call_link\('abenrap_behavior_ext_body_glosry.htm'\) "Glossary Entry") to extend an existing [RAP BO entity](javascript:call_link\('abenrap_bo_entity_glosry.htm'\) "Glossary Entry") in the [CDS BDL](javascript:call_link\('abencds_bdl_glosry.htm'\) "Glossary Entry").

-   [Adding Elements](javascript:call_link\('abenbdl_new_elem_ext.htm'\)): Elements can be added to an existing [entity behavior definition](javascript:call_link\('abencds_entity_bdef_glosry.htm'\) "Glossary Entry").
-   [Extending Elements](javascript:call_link\('abenbdl_ext_elem_ext.htm'\)): Existing elements of existing entity behavior definitions can be extended.
