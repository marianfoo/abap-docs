ABAP behavior implementation

An ABAP behavior implementation implements the [unmanaged RAP BO provider](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenunmanaged_rap_bo_prov_glosry.htm "Glossary Entry"). The implementation takes place in local classes of an [ABAP behavior pool (ABP)](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbehavior_pool_glosry.htm "Glossary Entry").
