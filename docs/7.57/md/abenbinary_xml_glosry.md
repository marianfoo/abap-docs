binary XML

SAP-specific format for the binary storage of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data. In Binary XML, redundant information is suppressed, and [UTF-8](javascript:call_link\('abenutf8_glosry.htm'\) "Glossary Entry") encoding is used, which leads to performance benefits when storing and transporting data. In [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"), Binary XML occurs as one of the supported [formats](javascript:call_link\('abenabap_sxml_lib_formats.htm'\)) in the [sXML Library](javascript:call_link\('abenabap_sxml_lib.htm'\)).
