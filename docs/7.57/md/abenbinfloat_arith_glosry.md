binary floating point arithmetic

Calculation with [binary floating point numbers](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbinfloat_glosry.htm "Glossary Entry"). Can produce rounding errors because floating point numbers are not represented internally using the decimal system.
