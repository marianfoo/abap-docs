Built-In Data Objects

ABAP programs have access to a range of built-in data objects:

-   [ABAP System Fields](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensystem_fields.htm)
-   [The Constant space](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenspace.htm)
-   [The Self-Reference me](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenme.htm)

The obsolete built-in structure [screen](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenscreen_structure_obsolete.htm) also exists.
