built-in function

A predefined function in ABAP that can be used in [certain operand positions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenoperands_expressions.htm). [Numeric functions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmathematical_funktion_glosry.htm "Glossary Entry"), [string functions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstring_function_glosry.htm "Glossary Entry"), [table functions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentable_function_glosry.htm "Glossary Entry"), and [byte string functions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbinary_function_glosry.htm "Glossary Entry") that have a return value of the corresponding data type can be distinguished. [Description functions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendescription_function_glosry.htm "Glossary Entry") that provide information on data objects, are also available. The names of the built-in functions are a subset of the [reserved names](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_words.htm). Built-in functions are also available in [ABAP SQL](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_sql_functions.htm) and in the [CDS DDL](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_builtin_functions_v2.htm) in ABAP [CDS](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_cds_glosry.htm "Glossary Entry").
