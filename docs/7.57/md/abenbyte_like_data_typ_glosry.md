byte-like data type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that allows access to the unencoded bytes of [byte chains](javascript:call_link\('abenbyte_chain_glosry.htm'\) "Glossary Entry"). These data types are [byte fields](javascript:call_link\('abenbyte_field_glosry.htm'\) "Glossary Entry") and [byte strings](javascript:call_link\('abenbyte_string_glosry.htm'\) "Glossary Entry"). The associated [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [xsequence](javascript:call_link\('abenbuilt_in_types_generic.htm'\)).
