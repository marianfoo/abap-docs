Obsolete Access to CDS Entities

-   Access to and typing with [CDS-managed DDIC views](javascript:call_link\('abencds_mngdddic_view_glosry.htm'\) "Glossary Entry") is obsolete
    
    A CDS-managed DDIC view (obsolete), specified using the annotation [@AbapCatalog.sqlViewName](javascript:call_link\('abencds_define_view_v1.htm'\)) in a [CDS DDIC-based view (obsolete)](javascript:call_link\('abencds_v1_view_glosry.htm'\) "Glossary Entry"), must be considered as purely technical helper construct and should not be used at all.
    
    In particular, [accessing a CDS-managed DDIC view from ABAP SQL](javascript:call_link\('abenabap_sql_cds_obsolete.htm'\)) is obsolete.
