CDS access policy

Framework defined in the [DCL](javascript:call_link\('abendcl_glosry.htm'\) "Glossary Entry") of the [ABAP Core Data Services](javascript:call_link\('abenabap_core_data_services_glosry.htm'\) "Glossary Entry") using [DEFINE ACCESSPOLICY](javascript:call_link\('abencds_f1_define_accesspolicy.htm'\)). Currently, only one CDS access policy is used as a frame for the definition of exactly one [PFCG mapping](javascript:call_link\('abencds_pfcg_mapping_glosry.htm'\) "Glossary Entry").
