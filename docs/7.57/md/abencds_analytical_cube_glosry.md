analytical cube view

[CDS view entity](javascript:call_link\('abencds_v2_view_glosry.htm'\) "Glossary Entry") or [CDS DDIC-based view (obsolete)](javascript:call_link\('abencds_v1_view_glosry.htm'\) "Glossary Entry") that serves as analytical interface view. It selects from facts and is used as data source for analytical queries. A cube view must be annotated with [@Analytics.dataCategory: #CUBE](javascript:call_link\('abencds_annotations_frmwrk_tables.htm'\)).
