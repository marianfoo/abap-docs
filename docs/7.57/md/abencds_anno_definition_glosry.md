CDS annotation definition

Defines a [CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") as a [CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry"). The definition is made using the statement [define annotation](javascript:call_link\('abencds_f1_define_annotation.htm'\)) in [DDLA source code](javascript:call_link\('abenddla_source_code_glosry.htm'\) "Glossary Entry") in [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"). A CDS annotation definition specifies the [CDS annotation syntax](javascript:call_link\('abencds_annotation_syntax_glosry.htm'\) "Glossary Entry") used to specify the annotation in [CDS source code](javascript:call_link\('abencds_source_code_glosry.htm'\) "Glossary Entry").
