RAP authorization context

A container for [authorization objects](javascript:call_link\('abenauthorization_object_glosry.htm'\) "Glossary Entry") that lists one or more authorization objects. Specified using the syntax [define authorization context](javascript:call_link\('abenbdl_def_auth_context.htm'\)).

Alongside the [CDS behavior definition header](javascript:call_link\('abencds_bdef_header_glosry.htm'\) "Glossary Entry") and [entity behavior definition](javascript:call_link\('abencds_entity_bdef_glosry.htm'\) "Glossary Entry"), authorization contexts are defined in a [CDS behavior definition](javascript:call_link\('abencds_behavior_definition_glosry.htm'\) "Glossary Entry").

See also: [RAP own authorization context](javascript:call_link\('abencds_own_ac_glosry.htm'\) "Glossary Entry").
