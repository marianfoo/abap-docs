CDS calculated unit

Component of a [CDS view entity](javascript:call_link\('abencds_v2_view_glosry.htm'\) "Glossary Entry") of data type CHAR that specifies a self-defined unit which is not contained in any list of predefined values. A CDS calculated unit can be [CDS calculated quantity](javascript:call_link\('abencds_calculated_quantity.htm'\)) of the same CDS view entity.
