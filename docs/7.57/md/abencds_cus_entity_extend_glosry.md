CDS custom entity extension

Transportable extension of a [CDS custom entity](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_custom_entity_glosry.htm "Glossary Entry"). A CDS custom entity extension can be used to add elements and/or clauses to a custom entity without making any modifications to the original entity. It is defined using the statement EXTEND CUSTOM ENTITY.

See also: [CDS entity extension](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_entity_extend_glosry.htm "Glossary Entry").
