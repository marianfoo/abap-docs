CDS custom entity extension

Transportable extension of a [CDS custom entity](javascript:call_link\('abencds_custom_entity_glosry.htm'\) "Glossary Entry"). A CDS custom entity extension can be used to add elements and/or clauses to a custom entity without making any modifications to the original entity. It is defined using the statement EXTEND CUSTOM ENTITY.

See also: [CDS entity extension](javascript:call_link\('abencds_entity_extend_glosry.htm'\) "Glossary Entry").
