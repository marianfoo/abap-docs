CDS custom entity

[CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") defined using [DEFINE CUSTOM ENTITY](javascript:call_link\('abencds_f1_define_custom_entity.htm'\)) in a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). A CDS custom entity defines data. An [ABAP class](javascript:call_link\('abapclass.htm'\)) that is referenced in a CDS annotation can be used to implement data provisioning and data retrieval. In [RAP](javascript:call_link\('abenrap_glosry.htm'\) "Glossary Entry"), custom entities are used as the data model basis for [unmanaged queries](javascript:call_link\('abenrap_unmanged_query_glosry.htm'\) "Glossary Entry").
