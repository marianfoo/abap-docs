CDS element

Element in a [CDS entity](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_entity_glosry.htm "Glossary Entry"). A CDS element is defined in an element list of a [CDS data definition](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_data_definition_glosry.htm "Glossary Entry") and is a [component](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencomponent_glosry.htm "Glossary Entry") of the [structured type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstructured_type_glosry.htm "Glossary Entry") defined by the CDS entity.
