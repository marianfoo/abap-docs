Obsolete CDS Entities

-   [CDS DDIC-based Entities](javascript:call_link\('abencds_ddic_entity.htm'\)): CDS DDIC-based entities are obsolete and as a consequence, CDS DDIC-based views (obsolete), representing a sub-category of CDS DDIC-based entities, are obsolete. Accessing a [CDS-managed DDIC view](javascript:call_link\('abencds_mngdddic_view_glosry.htm'\) "Glossary Entry"), which is created for each CDS DDIC-based view (obsolete), is [obsolete](javascript:call_link\('abencds_access_obsolete.htm'\)) as well.
