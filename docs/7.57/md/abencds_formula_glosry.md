analytical formula

An analytical formula is an element of a [CDS analytical projection view](javascript:call_link\('abencds_analytical_pv_glosry.htm'\) "Glossary Entry") that is added as a new element in the projection layer. A formula can be an [arithmetic expression](javascript:call_link\('abencds_arithmetic_expression_v2.htm'\)), a currency conversion using the built-in function [CURRENCY\_CONVERSION](javascript:call_link\('abencds_conv_func_unit_curr_v2.htm'\)), or a [formula-related case expression](javascript:call_link\('abencds_case_expression_apv.htm'\)).

The annotation Aggregation.default: #FORMULA is mandatory.
