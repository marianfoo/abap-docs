non-SQL CDS entity

[CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") for which no [ABAP-managed database object](javascript:call_link\('abenabap_managed_db_object_glosry.htm'\) "Glossary Entry") is created and which cannot be accessed with [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"). A non-SQL CDS entity cannot be used as a data source of SELECT statements or used by [path expressions](javascript:call_link\('abenpath_expression_glosry.htm'\) "Glossary Entry").

Currently, the following non-SQL entities are available:

-   [CDS custom entities](javascript:call_link\('abencds_custom_entity_glosry.htm'\) "Glossary Entry")
-   [CDS abstract entities](javascript:call_link\('abencds_abstract_entity_glosry.htm'\) "Glossary Entry")
