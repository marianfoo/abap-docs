CDS schema notation

CSN for short. Notation for compact representations of data models, such as [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") data models. SAP standard format for metadata exchange between data models of different technology stacks, for example:

-   HANA CDS on [SAP HANA](javascript:call_link\('abensap_hana_glosry.htm'\) "Glossary Entry") and [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") on [ABAP Platform](javascript:call_link\('abenabap_platform_glosry.htm'\) "Glossary Entry")
-   [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") and SAP Analytics Cloud (SAC).
