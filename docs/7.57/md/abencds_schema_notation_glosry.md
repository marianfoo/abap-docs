CDS schema notation

CSN for short. Notation for compact representations of data models, such as [ABAP CDS](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_cds_glosry.htm "Glossary Entry") data models. SAP standard format for metadata exchange between data models of different technology stacks, for example:

-   HANA CDS on [SAP HANA](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensap_hana_glosry.htm "Glossary Entry") and [ABAP CDS](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_cds_glosry.htm "Glossary Entry") on [ABAP Platform](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_platform_glosry.htm "Glossary Entry")
-   [ABAP CDS](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_cds_glosry.htm "Glossary Entry") and SAP Analytics Cloud (SAC).
