CDS service definition extension

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that extends a [CDS service definition](javascript:call_link\('abencds_service_definition_glosry.htm'\) "Glossary Entry"). This is done using the [CDS SDL](javascript:call_link\('abencds_sdl_glosry.htm'\) "Glossary Entry") language elements of [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") in [SDL source code](javascript:call_link\('abensdl_source_code_glosry.htm'\) "Glossary Entry").
