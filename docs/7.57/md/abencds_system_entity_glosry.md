CDS system entity

[CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") delivered with an [ABAP Platform](javascript:call_link\('abenabap_platform_glosry.htm'\) "Glossary Entry") that implements basic functionality.
