CDS tuning object

[Tuning object](javascript:call_link\('abentuning_object_glosry.htm'\) "Glossary Entry") for [CDS objects](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry"). A CDS tuning object is a CDS object defined with [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") in [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry"). For the time being, there are [CDS entity buffers](javascript:call_link\('abencds_entity_buffer_glosry.htm'\) "Glossary Entry").
