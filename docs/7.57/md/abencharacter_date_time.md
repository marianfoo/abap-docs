Date Fields and Time Fields

The ABAP types [d](javascript:call_link\('abenbuiltin_types_date_time.htm'\)) and [t](javascript:call_link\('abenbuiltin_types_date_time.htm'\)) are built-in types for data objects that specify dates and times. The date type d describes a character-like local date in the format yyyymmdd. The date type t describes a character-like local time in the format "hhmmss".
