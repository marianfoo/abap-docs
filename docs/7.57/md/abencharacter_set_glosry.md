character set

Defined set of characters, for example, [ASCII](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenascii_glosry.htm "Glossary Entry"), [EBCDIC](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenebcdic_glosry.htm "Glossary Entry") or [Unicode](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenunicode_glosry.htm "Glossary Entry"). Character sets are mapped onto [character representations](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenchar_representation_glosry.htm "Glossary Entry") via [code pages](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencodepage_glosry.htm "Glossary Entry").
