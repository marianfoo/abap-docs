child node

[Hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") that inherits a [parent-child relationship](javascript:call_link\('abenpcr_glosry.htm'\) "Glossary Entry") from a [parent node](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry"). A child node can have multiple parent nodes. A child node is a [descendant node](javascript:call_link\('abendescendant_node_glosry.htm'\) "Glossary Entry") of its parent node, this parent's node, and so on. A hierarchy in which each child node has only one parent node is a tree-like hierarchy, where each [root node](javascript:call_link\('abenroot_node_glosry.htm'\) "Glossary Entry") can be the origin of a tree.
