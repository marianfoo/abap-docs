client dependency

Property of a [SQL data source](javascript:call_link\('abensql_data_source_glosry.htm'\) "Glossary Entry"). A SQL data source can be [client-dependent](javascript:call_link\('abenclient_dependent_glosry.htm'\) "Glossary Entry") or [client-independent](javascript:call_link\('abenclient_independent_glosry.htm'\) "Glossary Entry"). When accessing a client-dependent SQL data source in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), an implicit [client handling](javascript:call_link\('abenclient_handling_glosry.htm'\) "Glossary Entry") takes place.
