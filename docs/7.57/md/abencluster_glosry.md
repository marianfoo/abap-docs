cluster

Grouping of different data in shared storage. Currently only used as [data clusters](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_cluster_glosry.htm "Glossary Entry") in different storage media for [data objects](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_object_glosry.htm "Glossary Entry"). Before Release 7.53, also used as a [table cluster](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentable_cluster_glosry.htm "Glossary Entry") for special [DDIC database tables](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddic_db_table_glosry.htm "Glossary Entry") ([cluster tables](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencluster_table_glosry.htm "Glossary Entry")).
