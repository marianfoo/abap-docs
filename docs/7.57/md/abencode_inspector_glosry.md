Code Inspector

CI for short. Tool for checking [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in terms of performance, security, syntax, and adherence to naming conventions. Called using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SCI.
