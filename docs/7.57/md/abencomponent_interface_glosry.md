component interface

[Interface](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenoo_intf_glosry.htm "Glossary Entry") included in a [compound interface](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencompound_interface_glosry.htm "Glossary Entry") by an [INTERFACES](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapinterfaces.htm) statement. A component interface can itself be a compound interface.
