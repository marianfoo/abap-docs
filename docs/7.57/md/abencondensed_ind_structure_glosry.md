condensed indicator structure

Special [indicator structure](javascript:call_link\('abenindicator_structure_glosry.htm'\) "Glossary Entry") that is not a [substructure](javascript:call_link\('abensubstructure_glosry.htm'\) "Glossary Entry") but the last component of a structure with a byte field type x. Here, the individual bits of the [byte field](javascript:call_link\('abenbyte_field_glosry.htm'\) "Glossary Entry") serve as indicators for the preceding components.
