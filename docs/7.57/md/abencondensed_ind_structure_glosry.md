condensed indicator structure

Special [indicator structure](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenindicator_structure_glosry.htm "Glossary Entry") that is not a [substructure](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensubstructure_glosry.htm "Glossary Entry") but the last component of a structure with a byte field type x. Here, the individual bits of the [byte field](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbyte_field_glosry.htm "Glossary Entry") serve as indicators for the preceding components.
