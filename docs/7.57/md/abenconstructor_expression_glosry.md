constructor expression

A constructor expression uses a [constructor operator](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenconstructor_operator_glosry.htm "Glossary Entry") to create a result for a specified [type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentype_glosry.htm "Glossary Entry") and constructs its content.
