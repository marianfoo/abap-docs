Core Data Services

Infrastructure for defining and consuming semantically rich data models. Currently, there are two versions:

-   [ABAP CDS](javascript:call_link\('abenabap_core_data_services_glosry.htm'\) "Glossary Entry") for [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") and
-   HANA CDS for [SAP HANA](javascript:call_link\('abensap_hana_glosry.htm'\) "Glossary Entry").

[CDS schema notation (CSN)](javascript:call_link\('abencds_schema_notation_glosry.htm'\) "Glossary Entry") is a data format for data exchange that can be used, among others, to exchange metadata between ABAP CDS and HANA CDS.
