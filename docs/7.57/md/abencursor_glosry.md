cursor

A cursor determines a position. First, the term "cursor" means the position of a pointer in a display window. Second, a [list cursor](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlist_cursor_glosry.htm "Glossary Entry") determines the output position in a [list](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenclassic_list_glosry.htm "Glossary Entry"), and there is also a [database cursor](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendatabase_cursor_glosry.htm "Glossary Entry") to read data from [database tables](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendatabase_table_glosry.htm "Glossary Entry"). In this sense, the [file pointer](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfile_pointer_glosry.htm "Glossary Entry") can be seen as a cursor as well.
