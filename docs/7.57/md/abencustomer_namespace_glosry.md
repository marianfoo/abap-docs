customer namespace

Predefined [namespace](javascript:call_link\('abennamespace_glosry.htm'\) "Glossary Entry") for [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") created in non-SAP and non-partner development systems. Names have Y or Z as the first character. These names have to be used in non-SAP and non-partner systems, if no [prefix namespace](javascript:call_link\('abenprefix_name_space_glosry.htm'\) "Glossary Entry") is available.
