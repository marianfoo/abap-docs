DDIC data element

[Repository object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrepository_object_glosry.htm "Glossary Entry") in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") that defines an [elementary data type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenelementary_data_type_glosry.htm "Glossary Entry") or a [reference type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenreference_type_glosry.htm "Glossary Entry"). In addition to [technical type properties](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentechnical_type_prpt_glosry.htm "Glossary Entry"), it specifies the semantic meaning of a ([database](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendatabase_field_glosry.htm "Glossary Entry")) [field](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfield_glosry.htm "Glossary Entry") (e.g., the header for tabular representations or documentation text). Properties such as [data type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_type_glosry.htm "Glossary Entry"), length, and so on, are defined directly in the data element or are copied from a [DDIC domain](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendomain_glosry.htm "Glossary Entry").
