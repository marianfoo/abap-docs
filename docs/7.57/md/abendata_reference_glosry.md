data reference

[Reference](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") that points to a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") or a part of a data object. A non-initial data reference can be a [heap reference](javascript:call_link\('abenheap_reference_glosry.htm'\) "Glossary Entry") or a [stack reference](javascript:call_link\('abenstack_reference_glosry.htm'\) "Glossary Entry").
