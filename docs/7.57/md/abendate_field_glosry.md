date field

[Data object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_object_glosry.htm "Glossary Entry") of the [date type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendate_type_glosry.htm "Glossary Entry") [t](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbuiltin_types_date_time.htm) for storing a character-like [calendar date](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencalendar_date_glosry.htm "Glossary Entry"). Valid values have the format yyyymmdd. The internal representation takes place in the [character representation](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenchar_representation_glosry.htm "Glossary Entry") of the current [code page](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencodepage_glosry.htm "Glossary Entry"). When assignments are made to data objects of [numeric data types](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abennumeric_data_type_glosry.htm "Glossary Entry") and in calculations, valid values are converted to the number of days since 01.01.0001.
