DDIC projection view

Special [DDIC table view](javascript:call_link\('abentable_view_glosry.htm'\) "Glossary Entry") of the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary.htm'\)) for a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry"). A DDIC projection view is not instantiated as an [SQL view](javascript:call_link\('abensql_view_glosry.htm'\) "Glossary Entry") on the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry").
