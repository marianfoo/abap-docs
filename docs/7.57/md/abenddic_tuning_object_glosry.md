DDIC tuning object

[Tuning object](javascript:call_link\('abentuning_object_glosry.htm'\) "Glossary Entry") for [DDIC objects](javascript:call_link\('abendictionary_object_glosry.htm'\) "Glossary Entry"). A DDIC tuning object is a [dictionary object](javascript:call_link\('abendictionary_object_glosry.htm'\) "Glossary Entry") defined with [DDIC DDL](javascript:call_link\('abendictionary_ddl_glosry.htm'\) "Glossary Entry") in [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry"). There are [dependency rules](javascript:call_link\('abendependency_rule_glosry.htm'\) "Glossary Entry") and [dynamic caches](javascript:call_link\('abendynamic_cache_glosry.htm'\) "Glossary Entry").
