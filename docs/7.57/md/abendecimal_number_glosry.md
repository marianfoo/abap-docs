decimal number

Number in the decimal numbering system. It consists of [places](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenplace_glosry.htm "Glossary Entry"), which can contain the decimal numbers 0-9, and can also contain a [decimal separator](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendecimal_point_separat_glosry.htm "Glossary Entry") and a leading plus/minus sign.
