deserialization

Transformation of [XML](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxml_glosry.htm "Glossary Entry") data or [JSON](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenjson_glosry.htm "Glossary Entry") data to ABAP data objects. Performed either by calling an [XSLT program](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxsl_transformation_glosry.htm "Glossary Entry") or a [Simple Transformation](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensimple_transformation_glosry.htm "Glossary Entry"). If use [XSLT](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxslt_glosry.htm "Glossary Entry") is used, the result of the transformation (in XML) must be a [canonical XML representation](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencanonical_xml_represent_glosry.htm "Glossary Entry") (([asXML](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenasxml_glosry.htm "Glossary Entry")).
