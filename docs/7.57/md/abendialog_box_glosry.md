dialog box

Also referred to as a modal dialog box. Display of the [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry") of a [dynpro sequence](javascript:call_link\('abendynpro_sequence_glosry.htm'\) "Glossary Entry") called by [CALL SCREEN](javascript:call_link\('abapcall_screen.htm'\)), where the previous GUI window remains visible although it is inactive. Up to nine dialog boxes can be stacked over a GUI window (see [popup level](javascript:call_link\('abenpop-up_level_glosry.htm'\) "Glossary Entry")). Modeless dialog boxes, where the previous GUI window remains active, can be implemented in [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry") only as [GUI controls](javascript:call_link\('abengui_control_glosry.htm'\) "Glossary Entry").
