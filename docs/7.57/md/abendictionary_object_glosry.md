dictionary object

DDIC object for short. [Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") defined in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). Examples of dictionary objects are global [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry"), [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry"), and [DDIC views](javascript:call_link\('abenddic_view_glosry.htm'\) "Glossary Entry").
