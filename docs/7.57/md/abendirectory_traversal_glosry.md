directory traversal

Unauthorized access to files or directories. Directory traversals are possible in ABAP if physical file names or file paths that are transferred from outside ABAP to programs are then passed on to the [ABAP file interface](javascript:call_link\('abenfile_interface_glosry.htm'\) "Glossary Entry") without being checked or escaped. Directory traversals must be prevented by [validating file names](javascript:call_link\('abendataset_auth_self.htm'\)).
