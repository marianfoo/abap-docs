dynamic cache

[DDIC tuning object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddic_tuning_object_glosry.htm "Glossary Entry") defined using the statement [DEFINE DYNAMIC CACHE](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddicddl_define_dynamic_cache.htm). A dynamic cache can improve performance when a view is accessed repeatedly. At the same time, a dynamic cache guarantees up-to-date data.

Equivalent to a SAP HANA dynamic result cache, integrated into [ABAP Dictionary](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_dictionary_glosry.htm "Glossary Entry").
