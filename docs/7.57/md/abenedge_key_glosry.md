edge key

An edge key represents one [edge attribute](javascript:call_link\('abenedge_attr_glosry.htm'\) "Glossary Entry") or a set of edge attributes that must uniquely identify edges in an [edge table](javascript:call_link\('abenedge_table_glosry.htm'\) "Glossary Entry").
