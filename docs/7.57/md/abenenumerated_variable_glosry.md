enumerated variable

[Variable](javascript:call_link\('abenvariable_glosry.htm'\) "Glossary Entry") with an [enumerated type](javascript:call_link\('abenenumerated_type_glosry.htm'\) "Glossary Entry"). An enumerated variable is a variable [enumerated object](javascript:call_link\('abenenumerated_object_glosry.htm'\) "Glossary Entry") and can only contain the corresponding [enumerated values](javascript:call_link\('abenenumerated_value_glosry.htm'\) "Glossary Entry"). The data type of an enumerated variable is the enumerated type. The technical data type of the content is the [base type](javascript:call_link\('abenbase_type_glosry.htm'\) "Glossary Entry") of the enumerated type.
