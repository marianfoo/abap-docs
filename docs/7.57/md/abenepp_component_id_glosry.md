EPP component ID

Component of an [extended passport (EPP)](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenepp_glosry.htm "Glossary Entry"). Contains the [system ID](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensystem_id_glosry.htm "Glossary Entry") and the name of the [application server instance](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_instance_glosry.htm "Glossary Entry") of the original [ABAP session](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_session_glosry.htm "Glossary Entry").
