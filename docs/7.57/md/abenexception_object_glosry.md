exception object

[Instance](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninstance_glosry.htm "Glossary Entry") of an [exception class](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenexception_class_glosry.htm "Glossary Entry"). Is created when a [catchable exception](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencatchable_exception_glosry.htm "Glossary Entry") is raised and contains information on the exception situation. Can be propagated across call levels.
