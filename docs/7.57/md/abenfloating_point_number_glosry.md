floating point number

Either a [decimal floating point number](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendecfloat_glosry.htm "Glossary Entry") or a [binary floating point number](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbinfloat_glosry.htm "Glossary Entry"). In floating point numbers, the number of [decimal places](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendecimal_place_glosry.htm "Glossary Entry") is part of the value and not part of the data type, as is the case in [packed numbers](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpacked_number_glosry.htm "Glossary Entry"). [Floating point arithmetic](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfloating_point_arith_glosry.htm "Glossary Entry") is used for calculations with floating point numbers.
