group level processing

The evaluation of [group levels](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abengroup_level_glosry.htm "Glossary Entry") in [internal tables](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninternal_table_glosry.htm "Glossary Entry") or the obsolete [extracts](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenextract_glosry.htm "Glossary Entry"). Group level processing takes place in [LOOP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abaploop_at_itab.htm)\- [loops](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenloop_glosry.htm "Glossary Entry") using [control structures](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencontrol_structure_glosry.htm "Glossary Entry") defined using the keyword [AT](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapat_itab.htm).
