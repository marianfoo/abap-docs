grouping function

[Aggregate function](javascript:call_link\('abenaggregate_function_glosry.htm'\) "Glossary Entry") for [GROUPING SETS](javascript:call_link\('abapgrouping_sets_clause.htm'\)) in a [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry"). The result of the grouping function GROUPING indicates whether a specific column was part of the aggregation. The return value of the grouping function is 0 for the column that was not involved in the aggregation or 1 for the column that was involved in the aggregation.
