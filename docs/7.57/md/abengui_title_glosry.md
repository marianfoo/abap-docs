GUI title

Text that can be displayed in the [title bar](javascript:call_link\('abentitle_bar_glosry.htm'\) "Glossary Entry") of a [GUI window](javascript:call_link\('abengui_window_glosry.htm'\) "Glossary Entry"). A GUI title is set using the statement [SET TITLEBAR](javascript:call_link\('abapset_titlebar_dynpro.htm'\)) and created using the [Menu Painter](javascript:call_link\('abenmenu_painter_glosry.htm'\) "Glossary Entry") tool.
