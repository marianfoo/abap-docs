SAP HANA database

The SAP HANA database is the in-memory database of [SAP HANA](javascript:call_link\('abensap_hana_glosry.htm'\) "Glossary Entry"). It can be used as the database for an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). In this case, [HANA views](javascript:call_link\('abenhana_view_glosry.htm'\) "Glossary Entry") can be accessed in ABAP programs with [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), provided that these views are defined as [DDIC external views](javascript:call_link\('abenexternal_view_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). Furthermore, [AMDP](javascript:call_link\('abenamdp_glosry.htm'\) "Glossary Entry") can be used to manage and call [database procedures](javascript:call_link\('abendatabase_procedure_glosry.htm'\) "Glossary Entry") written in [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry").
