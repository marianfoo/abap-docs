SAP HANA view

An SAP HANA view is used to select, analyze, or perform calculations with data from the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). Attribute views, analytic views and calculation views can be distinguished. SAP HANA views can be defined as objects of [SAP HANA XS](javascript:call_link\('abenxs_glosry.htm'\) "Glossary Entry") for which there are special [methods for accessing](javascript:call_link\('abensap_hana_access.htm'\)) these views in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry").
