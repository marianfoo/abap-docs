SAP HANA view

An SAP HANA view is used to select, analyze, or perform calculations with data from the [SAP HANA database](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhana_database_glosry.htm "Glossary Entry"). Attribute views, analytic views and calculation views can be distinguished. SAP HANA views can be defined as objects of [SAP HANA XS](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxs_glosry.htm "Glossary Entry") for which there are special [methods for accessing](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensap_hana_access.htm) these views in [ABAP programs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_program_glosry.htm "Glossary Entry").
