Obsolete Access to Objects in HANA XS

[SAP HANA XSA](javascript:call_link\('abenxsa_glosry.htm'\) "Glossary Entry") is the successor of [SAP HANA XSC](javascript:call_link\('abenxsc_glosry.htm'\) "Glossary Entry"). The access methods designed specifically for XSC objects are not suitable for XSA objects.

The following access methods for XSC objects are obsolete:

-   [DDIC external views](javascript:call_link\('abenexternal_view_glosry.htm'\) "Glossary Entry")
-   [Database procedure proxies](javascript:call_link\('abendatabase_proc_proxy_glosry.htm'\) "Glossary Entry")

This is also described in SAP Notes [3116165](https://launchpad.support.sap.com/#/notes/3116165) and [2465027](https://launchpad.support.sap.com/#/notes/2465027).
