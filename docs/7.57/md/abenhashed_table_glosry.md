hashed table

[Table category](javascript:call_link\('abentable_category_glosry.htm'\) "Glossary Entry") of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") whose lines are stored internally using a [hash algorithm](javascript:call_link\('abenhash_algorithm_glosry.htm'\) "Glossary Entry") and can be accessed using a unique [hash key](javascript:call_link\('abenhash_key_glosry.htm'\) "Glossary Entry"). A hashed table does not have a [primary table index](javascript:call_link\('abenprimary_table_index_glosry.htm'\) "Glossary Entry"), but it can be assigned a [secondary table index](javascript:call_link\('abensecondary_table_index_glosry.htm'\) "Glossary Entry") using a [secondary table key](javascript:call_link\('abensecondary_table_key_glosry.htm'\) "Glossary Entry"). The corresponding [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [hashed table](javascript:call_link\('abenbuilt_in_types_generic.htm'\)).
