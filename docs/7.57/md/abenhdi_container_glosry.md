HDI container

An HDI container is a [database schema](javascript:call_link\('abendatabase_schema_glosry.htm'\) "Glossary Entry") of the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") for [HDI objects](javascript:call_link\('abenhdi_object_glosry.htm'\) "Glossary Entry"). Special [ABAP Managed HDI Containers (AMHC)](javascript:call_link\('abenamhc_glosry.htm'\) "Glossary Entry") can be created for HDI objects accessed in ABAP.
