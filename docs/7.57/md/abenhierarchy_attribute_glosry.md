hierarchy attribute

Component of the tabular result set of a built-in database hierarchy function that contains hierarchy-specific information for each row. When an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry") is accessed in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), the hierarchy attributes can be included in the result set as [hierarchy columns](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") of [hierarchy nodes](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry"). The hierarchy attributes can be used to define elements of the CDS hierarchy can be used in the definition of a [CDS hierarchy](javascript:call_link\('abencds_hierarchy_glosry.htm'\) "Glossary Entry").
