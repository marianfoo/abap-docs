immutable variable

[Variable](javascript:call_link\('abenvariable_glosry.htm'\) "Glossary Entry") that is declared with the [declaration operator](javascript:call_link\('abendeclaration_operator_glosry.htm'\) "Glossary Entry") [FINAL](javascript:call_link\('abenfinal_inline.htm'\)). This [inline declaration](javascript:call_link\('abeninline_declaration_glosry.htm'\) "Glossary Entry") is the only [write position](javascript:call_link\('abenwrite_position_glosry.htm'\) "Glossary Entry") within a [context](javascript:call_link\('abencontext_glosry.htm'\) "Glossary Entry") where a value can be assigned to an immutable variable.
