initial memory requirement

Property of a [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry") that affects the memory area allocated by the ABAP runtime framework for an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") before the table is filled with lines. The initial memory requirement is not a [technical type property](javascript:call_link\('abentechnical_type_prpt_glosry.htm'\) "Glossary Entry") and does not affect the [compatibility](javascript:call_link\('abencompatible_glosry.htm'\) "Glossary Entry") of the table type.
