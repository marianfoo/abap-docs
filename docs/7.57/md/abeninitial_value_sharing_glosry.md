initial value sharing

The status of a [boxed component](javascript:call_link\('abenboxed_component_glosry.htm'\) "Glossary Entry") as long as it is processed by read access only. The [initial value](javascript:call_link\('abeninitial_value_glosry.htm'\) "Glossary Entry") of all type-compliant boxed components of the same type is only stored once in each [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry") in the [PXA](javascript:call_link\('abenpxa_glosry.htm'\) "Glossary Entry").
