input/output parameter

[Formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") of a [procedure](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry"). When a call is made, the value of an [actual parameter](javascript:call_link\('abenactual_parameter_glosry.htm'\) "Glossary Entry") is passed to an input/output parameter. Its value can be set in the procedure and is available in the actual parameter at the end of the procedure. In all procedures, input/output parameters are defined using the addition CHANGING and are therefore also referred to as CHANGING parameters.
