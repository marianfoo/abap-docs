instance component

Superordinate term for [instance attributes](javascript:call_link\('abeninstance_attribute_glosry.htm'\) "Glossary Entry"), [instance events](javascript:call_link\('abeninstance_event_glosry.htm'\) "Glossary Entry"), and [instance methods](javascript:call_link\('abeninstance_method_glosry.htm'\) "Glossary Entry"). The instance components of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") can only be addressed in [instances](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of the class.
