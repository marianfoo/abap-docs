instance method

[Method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") declared using [METHODS](javascript:call_link\('abapmethods.htm'\)), which can only be used in an [instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of the class. Instance methods can access all [attributes](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry") and [events](javascript:call_link\('abenevent_glosry.htm'\) "Glossary Entry") of their class.
