Internet Communication Framework

ICF for short. A software layer in [AS ABAP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_abap_glosry.htm "Glossary Entry") that provides an ABAP interface for HTTP, HTTPS, and SMTP requests. Using the ICF, [ABAP programs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_program_glosry.htm "Glossary Entry") can communicate with the internet in both the sever and the client role. The ICF receives and sends web-based ABAP calls using the [Internet Communication Manager](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninternet_con_man_glosry.htm "Glossary Entry").
