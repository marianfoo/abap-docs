iXML Library

Global [classes](javascript:call_link\('abenglobal_class_glosry.htm'\) "Glossary Entry") that are accessible via [interfaces](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") for handling [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") documents in [DOM](javascript:call_link\('abendom_glosry.htm'\) "Glossary Entry") representation. The "integrated" iXML Library provides services such as an XML parser, an XML renderer, and reads and writes to the DOM. The interfaces of the iXML Library start with the prefix IF\_IXML.
