JSON

JavaScript Object Notation ( [http://www.json.org/](http://www.json.org/)). Data format in text form for data exchange between applications. In ABAP, the [sXML Library](javascript:call_link\('abensxml_library_glosry.htm'\) "Glossary Entry") and [transformations](javascript:call_link\('abenabap_json_trafos.htm'\)) can be used to access JSON, for which a special JSON-XML supported by [sXML](javascript:call_link\('abenjson_xml_glosry.htm'\) "Glossary Entry") Library is used.
