ABAP kernel

Part of the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") which is primarily programmed in C/C++. In particular, [ABAP statements](javascript:call_link\('abenabap_statement_glosry.htm'\) "Glossary Entry") are implemented by C functions in the ABAP kernel. When an ABAP statement is executed, the corresponding C functions or the ABAP kernel are called.
