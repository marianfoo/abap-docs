line element

Predefined output element in a [list](javascript:call_link\('abenclassic_list_glosry.htm'\) "Glossary Entry"). Line elements can be corners, crosses, horizontal and vertical lines, and T sections. Line elements are either output explicitly using the addition [AS LINE](javascript:call_link\('abapwrite_list_elements.htm'\)) of the WRITE statement, or set by the system during automatic [write lines](javascript:call_link\('abenwrite_lines.htm'\)).
