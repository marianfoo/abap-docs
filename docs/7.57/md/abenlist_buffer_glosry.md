list buffer

Memory area for storing [screen lists](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenscreen_list_glosry.htm "Glossary Entry"). Subdivided into [list levels](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlist_level_glosry.htm "Glossary Entry"). For each [dynpro sequence](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendynpro_sequence_glosry.htm "Glossary Entry"), the list buffer manages a hierarchy consisting of a [basic list](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbasic_list_glosry.htm "Glossary Entry") and up to 20 [details lists](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendetails_list_glosry.htm "Glossary Entry").
