list index

Identifier of a [list](javascript:call_link\('abenclassic_list_glosry.htm'\) "Glossary Entry") in the [list buffer](javascript:call_link\('abenlist_buffer_glosry.htm'\) "Glossary Entry"). The value of the list index is 0 for the [basic list](javascript:call_link\('abenbasic_list_glosry.htm'\) "Glossary Entry") and 1 to 20 for the [details lists](javascript:call_link\('abendetails_list_glosry.htm'\) "Glossary Entry").
