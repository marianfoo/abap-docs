local update

[Update](javascript:call_link\('abenupdate_glosry.htm'\) "Glossary Entry") in the current [work process](javascript:call_link\('abenwork_process_glosry.htm'\) "Glossary Entry"). It is not executed in a separate [update session](javascript:call_link\('abenupdate_session_glosry.htm'\) "Glossary Entry") but in a separate [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") of the current [ABAP session](javascript:call_link\('abenabap_session_glosry.htm'\) "Glossary Entry"). The local update can be activated for an [SAP LUW](javascript:call_link\('abensap_luw_glosry.htm'\) "Glossary Entry") by using the statement [SET UPDATE TASK LOCAL](javascript:call_link\('abapset_update_task_local.htm'\)).
