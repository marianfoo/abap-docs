logical file name

Platform-independent name that can be assigned to [physical file names](javascript:call_link\('abenphysical_filename_glosry.htm'\) "Glossary Entry") using the transactions FILE and SF01. Logical file names are used for platform-independent programming to [validate file names](javascript:call_link\('abendataset_auth_self.htm'\)).
