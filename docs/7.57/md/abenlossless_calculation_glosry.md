lossless calculation

[Arithmetic](javascript:call_link\('abenarithmetic_expression_glosry.htm'\) "Glossary Entry") calculation where no rounding occurs. The [lossless operator](javascript:call_link\('abenlossless_operator_glosry.htm'\) "Glossary Entry") [EXACT](javascript:call_link\('abenconstructor_expression_exact.htm'\)) or the addition EXACT of the obsolete statement [COMPUTE](javascript:call_link\('abapcompute.htm'\)) can be used to check whether a calculation is lossless.
