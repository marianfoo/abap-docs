multitenancy

Principle of software architecture in which a single software instance is run for different [tenants](javascript:call_link\('abentenant_glosry.htm'\) "Glossary Entry") on a single server. In a multitenancy [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"), [application clients](javascript:call_link\('abenapplication_client_glosry.htm'\) "Glossary Entry") assume the role of tenants. Multitenancy is not supported by the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") in the current release.
