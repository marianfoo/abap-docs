name table

Nametab for short. Stores the [DDIC runtime objects](javascript:call_link\('abenddic_runtime_object_glosry.htm'\) "Glossary Entry") of [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). The name table is persisted as the native [database table](javascript:call_link\('abendatabase_table_glosry.htm'\) "Glossary Entry") DDNTT and contains [nametab headers](javascript:call_link\('abenname_tab_header_glosry.htm'\) "Glossary Entry") and [nametab field descriptions](javascript:call_link\('abenname_tab_fields_glosry.htm'\) "Glossary Entry"). This database table is not defined as a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") in the ABAP Dictionary. The name table is accessed internally in the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") or by using APIs.
