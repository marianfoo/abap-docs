Assignments in Release 7.52

Modification

System Class for Dynamic Mapping

The system class [CL\_ABAP\_CORRESPONDING](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencl_abap_corresponding.htm) now has a new method [CREATE\_WITH\_VALUE](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencl_abap_corresponding_3.htm), which allows the values of any suitable data objects to be assigned to the components of the target structure or target table.
