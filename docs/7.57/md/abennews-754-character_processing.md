Character String Processing in Release 7.54

Modification

Decimal Places in Time Stamps

In [embedded expressions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstring_templates_expressions.htm) of [string templates](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstring_template_glosry.htm "Glossary Entry") and for the statements [WRITE](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapwrite-.htm) *\[* [TO](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapwrite_to.htm)*\]*, the formatting option DECIMALS can now be combined with TIMESTAMP and TIME ZONE to define the number of decimal places in time stamps in packed numbers.
