Data References in Release 7.56

Modification

Dereferencing Data References

The [dereferencing operator\->\*](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendereferencing_operator.htm) can be used for generically typed data reference variables in almost all operand positions now. Before, that was possible in the ASSIGN statement only.
