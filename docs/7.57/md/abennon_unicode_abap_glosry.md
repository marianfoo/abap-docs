Non-Unicode ABAP

Obsolete [language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") in which the [Unicode checks](javascript:call_link\('abenunicode_check_glosry.htm'\) "Glossary Entry") are not effective. A program with this language version cannot be executed in [Unicode systems](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry"). In the current [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") release, only Unicode systems are supported. Therefore, Non-Unicode ABAP can no longer be used.
