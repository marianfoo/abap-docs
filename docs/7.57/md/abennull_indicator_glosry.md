null indicator

[ABAP SQL indicator](javascript:call_link\('abenabap_sql_indicator_glosry.htm'\) "Glossary Entry") in a structured target area of an [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry"). The components or bits of a null indicator indicate which columns of the result set contain the [null value](javascript:call_link\('abennull_value_glosry.htm'\) "Glossary Entry") and which do not.
