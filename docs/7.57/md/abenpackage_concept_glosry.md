package concept

Rule set that is based on [packages](javascript:call_link\('abenpackage_glosry.htm'\) "Glossary Entry") and specifies the mutual usability of [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). Currently, only some of the rules are checked and only by the [ABAP Test Cockpit (ATC)](javascript:call_link\('abenabap_test_cockpit_glosry.htm'\) "Glossary Entry"). There are no checks at runtime.
