parallel Remote Function Call

pRFC for short. Extension of the [asynchronous remote function call](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenasynchronous_rfc_glosry.htm "Glossary Entry") that enables parallelization to be controlled.
