CDS parent entity

A [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") of a [CDS composition tree](javascript:call_link\('abencds_composition_tree_glosry.htm'\) "Glossary Entry") that uses a [CDS composition](javascript:call_link\('abencds_composition_glosry.htm'\) "Glossary Entry") to define a [child entity](javascript:call_link\('abenchild_entity_glosry.htm'\) "Glossary Entry"). A parent entity can have multiple child entities. Each child entity must define and expose a [to-parent association](javascript:call_link\('abento_parent_association_glosry.htm'\) "Glossary Entry") with its parent entity.
