password

Sequence of characters that is only known to a specific [ABAP user](javascript:call_link\('abenabap_user_glosry.htm'\) "Glossary Entry") and is required from this user to [log on](javascript:call_link\('abenlogon_glosry.htm'\) "Glossary Entry") to [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Used together with the [user name](javascript:call_link\('abenuser_name_glosry.htm'\) "Glossary Entry"), the password enables authorized use of an AS ABAP.
