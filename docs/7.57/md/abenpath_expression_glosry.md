path expression

A sequence of [associations](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenassociation_glosry.htm "Glossary Entry") specified at an operand position. Either an [SQL path expression](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensql_path_expression_glosry.htm "Glossary Entry") or a [mesh path expression](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmesh_path_expression_glosry.htm "Glossary Entry").
