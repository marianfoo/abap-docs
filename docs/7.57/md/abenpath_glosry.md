path

Sequence of relationships that link objects. In ABAP, sequences of [associations](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenassociation_glosry.htm "Glossary Entry") have a special meaning. Paths like these can be specified using [path expressions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpath_expression_glosry.htm "Glossary Entry"). See [CDS association path](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_association_path_glosry.htm "Glossary Entry"), [CDS composition path](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_composition_path_glosry.htm "Glossary Entry"), and [mesh path](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmesh_path_glosry.htm "Glossary Entry").
