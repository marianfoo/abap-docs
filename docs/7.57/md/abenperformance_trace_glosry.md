performance trace

Test tool combining multiple trace functions that allow system behavior to be analyzed in [database access](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendb_access.htm) ([SQL trace](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensql_trace_glosry.htm "Glossary Entry")), access to the [table buffer](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentable_buffer_glosry.htm "Glossary Entry"), access to [lock management](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensap_lock.htm) reads, and for external read access using [RFC](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrfc_glosry.htm "Glossary Entry") and [ICF](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenicf_glosry.htm "Glossary Entry"). Called using [transaction](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentransaction_code_glosry.htm "Glossary Entry") ST05.
