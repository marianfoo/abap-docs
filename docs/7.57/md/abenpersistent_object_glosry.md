persistent object

[Object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") in a [persistent class](javascript:call_link\('abenpersistent_class_glosry.htm'\) "Glossary Entry") whose [attributes](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry") are saved as database content after the runtime of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"). Persistent objects are managed by [object services](javascript:call_link\('abenobject_services_glosry.htm'\) "Glossary Entry").
