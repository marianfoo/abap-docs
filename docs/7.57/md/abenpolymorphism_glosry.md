polymorphism

In object-oriented programming, polymorphism means the ability to address differently implemented [methods](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") that belong to different [objects](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of different [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") with the same name and via the same interface.
