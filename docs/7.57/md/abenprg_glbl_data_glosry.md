program global data

Global data for short. [Data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") that is declared in an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and is known in the entire program behind its declaration. Global data is declared in the [global declaration part](javascript:call_link\('abenglobal_declaration_sect_glosry.htm'\) "Glossary Entry") of a program. See also [local data](javascript:call_link\('abenlocal_data_glosry.htm'\) "Glossary Entry").
