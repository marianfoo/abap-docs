program name

Unique identification of an ABAP program in the [repository](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrepository_glosry.htm "Glossary Entry"). A program name must consist of at least one character and can have a maximum of 30 alphanumeric characters, including underscores.
