program property

Property of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"). Program properties are used to administrate the program and affect how it is handled in [syntax checks](javascript:call_link\('abensyntax_check_glosry.htm'\) "Glossary Entry") and in the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry"). Examples of important program properties are [program type](javascript:call_link\('abenprogram_type_glosry.htm'\) "Glossary Entry") or the [ABAP language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry"), which specifies whether or not a program has the ABAP language version [Standard ABAP](javascript:call_link\('abenstandard_abap_glosry.htm'\) "Glossary Entry").
