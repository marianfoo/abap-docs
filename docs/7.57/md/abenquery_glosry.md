query

Database query that is implemented in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") with the language element SELECT. The [clauses and additions](javascript:call_link\('abenselect_clauses.htm'\)) of SELECT define the result set returned by the database. ‎See [main query](javascript:call_link\('abenmainquery_glosry.htm'\) "Glossary Entry") and [subquery](javascript:call_link\('abensubquery_glosry.htm'\) "Glossary Entry").
