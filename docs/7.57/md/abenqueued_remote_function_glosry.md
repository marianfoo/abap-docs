queued Remote Function Call

qRFC for short. Obsolete extension of the [transactional remote function call](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentrfc_1_glosry.htm "Glossary Entry") enabling a call sequence to be determined.
