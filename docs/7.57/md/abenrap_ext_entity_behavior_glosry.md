RAP extension entity behavior

Node extension for short. Component of a [BDEF extension](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_extension_glosry.htm "Glossary Entry") that defines the behavior for an extension node. Extension entity behavior is defined using [define behavior for](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbdl_define_beh_extend.htm). The extension node must be added to the current RAP BO's data model via [RAP data model extensions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_data_model_ext_glosry.htm "Glossary Entry").
