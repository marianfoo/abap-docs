RAP instance feature control

[Feature control](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_feature_control_glosry.htm "Glossary Entry") that defines access restrictions depending on the state of a [BO entity instance](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_entity_inst_glosry.htm "Glossary Entry"). Instance feature control can be defined for [fields](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_field_glosry.htm "Glossary Entry") and [operations](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_operation_glosry.htm "Glossary Entry").
