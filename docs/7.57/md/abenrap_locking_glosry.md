RAP locking

Locking mechanism that is based on the [SAP lock](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensap_lock_glosry.htm "Glossary Entry") concept. It uses [lock function modules](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlock_function_module_glosry.htm "Glossary Entry") for [pessimistic concurrency control](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpessimist_conc_control_glosry.htm "Glossary Entry").
