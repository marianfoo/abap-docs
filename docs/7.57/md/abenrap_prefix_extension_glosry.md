prefix extension

Used in the context of [RAP BO extensions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_extension_glosry.htm "Glossary Entry") to refer to an element or [repository object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrepository_object_glosry.htm "Glossary Entry") from a RAP BO extension. Examples: extension behavior, extension node, extension field.
