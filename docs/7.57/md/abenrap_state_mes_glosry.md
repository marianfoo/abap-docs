RAP state message

A [message](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_message_glosry.htm "Glossary Entry") in the [reported](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapeml_response.htm) structure of a [RAP business object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_glosry.htm "Glossary Entry") that refers to the state of an [entity instance](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrap_bo_entity_inst_glosry.htm "Glossary Entry"). A state message is defined with the parameter [%state\_area](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapderived_types_state_area.htm), which assigns it to the corresponding state area. A state message is persisted with the instance it refers to in [draft-enabled BOs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendraft_rap_bo_glosry.htm "Glossary Entry") in the [draft table](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendraft_table_glosry.htm "Glossary Entry").
