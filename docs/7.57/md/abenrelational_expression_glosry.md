relational expression

[Expression](javascript:call_link\('abenexpression_glosry.htm'\) "Glossary Entry") whose result is a [truth value](javascript:call_link\('abentruth_value_glosry.htm'\) "Glossary Entry"). Relational expressions include [comparison expressions](javascript:call_link\('abencomparison_expression_glosry.htm'\) "Glossary Entry") and [predicates](javascript:call_link\('abenpredicate_glosry.htm'\) "Glossary Entry"). A single relational expression or multiple relational expressions combined using [Boolean operators](javascript:call_link\('abenboolean_operator_glosry.htm'\) "Glossary Entry") form a [logical expression](javascript:call_link\('abenlogical_expression_glosry.htm'\) "Glossary Entry") in ABAP.
