reporting event

[Event](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenevent_glosry.htm "Glossary Entry") of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") that occurs in [executable programs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenexecutable_program_glosry.htm "Glossary Entry") when these are started via a [SUBMIT](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapsubmit.htm) or via a [report transaction](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenreport_transaction_glosry.htm "Glossary Entry"). Reporting events map the [IPO](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeneva_glosry.htm "Glossary Entry") principle of [reporting](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenreporting_glosry.htm "Glossary Entry") when used with a [logical database](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenlogical_data_base_glosry.htm "Glossary Entry"). The [event key words](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenevent_keyword_glosry.htm "Glossary Entry") assigned to reporting events are [INITIALIZATION](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapinitialization.htm), [START-OF-SELECTION](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapstart-of-selection.htm), [GET](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapget-.htm), and [END-OF-SELECTION](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapend-of-selection.htm).
