RFC - Authorizations

If the [profile parameter](javascript:call_link\('abenprofile_parameter_glosry.htm'\) "Glossary Entry") auth/rfc\_authority\_check is set to 1, an automatic authorization check is executed when a function module is called by remote call.

More Information   

For detailed information about RFC authorizations, see the documentation RFC in SAP Help Portal.
