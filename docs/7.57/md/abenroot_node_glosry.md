root node

[Hierarchy node](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") that does not have [parent nodes](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenparent_node_glosry.htm "Glossary Entry"). In [SQL hierarchies](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensql_hierarchy_glosry.htm "Glossary Entry"), root nodes are part of the [root node set](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenroot_node_set_glosry.htm "Glossary Entry"). The [hierarchy column](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhierarchy_column_glosry.htm "Glossary Entry") [HIERARCHY\_PARENT\_RANK](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddddl_hierarchy.htm) of a root node of an SQL hierarchy contains the value 0 and the hierarchy column [HIERARCHY\_LEVEL](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddddl_hierarchy.htm) contains the value 1.
