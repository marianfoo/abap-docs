root object

Special [shared object](javascript:call_link\('abenshared_object_glosry.htm'\) "Glossary Entry"). [Instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of the [area root class](javascript:call_link\('abenroot_data_class_glosry.htm'\) "Glossary Entry"). A non-empty [area instance version](javascript:call_link\('abenarea_instance_version_glosry.htm'\) "Glossary Entry") must contain a root object. The [attribute](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry") ROOT of an associated [area handle](javascript:call_link\('abenarea_handle_glosry.htm'\) "Glossary Entry") points to the root object, which can contain references to the other shared objects of the area instance version.
