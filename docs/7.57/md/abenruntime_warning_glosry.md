runtime warning

A runtime warning indicates critical situations that may cause a [runtime error](javascript:call_link\('abenruntime_error_glosry.htm'\) "Glossary Entry") during program execution, such as high memory consumption. A runtime warning creates a [short dump](javascript:call_link\('abenshort_dump_glosry.htm'\) "Glossary Entry"), but does not terminate the current program.
