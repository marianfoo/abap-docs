SAP HANA Cloud database

Database of [SAP HANA Cloud](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensap_hana_cloud_glosry.htm "Glossary Entry") that offers restricted functionality compared with a general [SAP HANA database](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhana_database_glosry.htm "Glossary Entry"). An SAP HANA Cloud database does not support [SAP HANA XSC](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxsc_glosry.htm "Glossary Entry") and there is no [full text index](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfull_text_index_glosry.htm "Glossary Entry").
