SAP HANA Cloud database

Database of [SAP HANA Cloud](javascript:call_link\('abensap_hana_cloud_glosry.htm'\) "Glossary Entry") that offers restricted functionality compared with a general [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). An SAP HANA Cloud database does not support [SAP HANA XSC](javascript:call_link\('abenxsc_glosry.htm'\) "Glossary Entry") and there is no [full text index](javascript:call_link\('abenfull_text_index_glosry.htm'\) "Glossary Entry").
