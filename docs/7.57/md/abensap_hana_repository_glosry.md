SAP HANA Repository

Repository of objects in the [SAP HANA database](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhana_database_glosry.htm "Glossary Entry"). More specifically, [XSC](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxsc_glosry.htm "Glossary Entry") objects are stored in the SAP HANA Repository. [XSA](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenxsa_glosry.htm "Glossary Entry") objects, on the other hand, are not stored in the SAP HANA Repository but are usually deployed using [HDI](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenhdi_glosry.htm "Glossary Entry").
