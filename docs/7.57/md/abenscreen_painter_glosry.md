screen painter

[ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") tool to create [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry"). Contains a text editor for writing the [dynpro flow logic](javascript:call_link\('abendynpro_flow_logic_glosry.htm'\) "Glossary Entry") and a [Layout Editor](javascript:call_link\('abenlayout_editor_glosry.htm'\) "Glossary Entry") for configuring the [screen layout](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry"). Called using [transaction](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SE51.
