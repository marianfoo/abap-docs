secondary database

Target database of a [secondary connection](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") defined in the database table DBCON. The database system must be a system supported by SAP and the software required for access using ABAP must be installed as a [database shared library (DBSL)](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") on the current [AS ABAP](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry"), if the database is different to its standard database of the AS ABAP. Only [SAP HANA databases](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") should be used as secondary databases of systems that have such a database as their [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry").
