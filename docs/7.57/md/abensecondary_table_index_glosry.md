secondary table index

[Table index](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentable_index_glosry.htm "Glossary Entry") for the administration of [sorted](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensorted_key_glosry.htm "Glossary Entry") [secondary table keys](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensecondary_table_key_glosry.htm "Glossary Entry"). A secondary table index of a non-unique key is not updated immediately for each line insertion or deletion in an [internal table](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninternal_table_glosry.htm "Glossary Entry"); it is only updated when the internal table is accessed via the associated secondary key.
