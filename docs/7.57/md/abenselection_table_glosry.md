selection table

Global [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") with a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry"), the name of a [selection criterion](javascript:call_link\('abenselection_criterion_glosry.htm'\) "Glossary Entry"), and the same layout as a [ranges table](javascript:call_link\('abenranges_table_glosry.htm'\) "Glossary Entry"). Declared using the statement [SELECT-OPTIONS](javascript:call_link\('abapselect-options.htm'\)). A selection table can be used in the same way as a ranges table.
