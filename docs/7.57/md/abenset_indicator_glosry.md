set indicator

[ABAP SQL indicator](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_sql_indicator_glosry.htm "Glossary Entry") in a [work area](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenwork_area_glosry.htm "Glossary Entry") or an [internal table](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninternal_table_glosry.htm "Glossary Entry"). The components of a set indicator indicate the columns of a [DDIC database table](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddic_db_table_glosry.htm "Glossary Entry") to be changed using the [UPDATE](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapupdate.htm) statement.
