sibling node

Sibling nodes are all [child nodes](javascript:call_link\('abenchild_node_glosry.htm'\) "Glossary Entry") of a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry") that have the same [parent node](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry"). In an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry"), sibling nodes have the same values in the [hierarchy column](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") [HIERARCHY\_PARENT\_RANK](javascript:call_link\('abenddddl_hierarchy.htm'\)).
