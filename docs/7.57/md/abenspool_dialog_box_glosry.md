spool dialog box

Also referred to as print dialog box. Standard [dialog box](javascript:call_link\('abendialog_box_glosry.htm'\) "Glossary Entry") for entering [spool parameters](javascript:call_link\('abenprint_parameter_glosry.htm'\) "Glossary Entry"). The spool dialog box is either called automatically or using the [function module](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") GET\_PRINT\_PARAMETERS.
