spool list level

Hierarchical level of a stacked [spool list](javascript:call_link\('abenspool_list_glosry.htm'\) "Glossary Entry"). A spool list level has exactly one [spool request](javascript:call_link\('abenspool_request_glosry.htm'\) "Glossary Entry") assigned to it.
