SQLScript procedure

A [database procedure](javascript:call_link\('abendatabase_procedure_glosry.htm'\) "Glossary Entry") of the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") programmed in the language [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry"). SQLScript procedures can be created using tools such as [SAP Web IDE for SAP HANA](javascript:call_link\('abensap_web_ide_for_sap_han_glosry.htm'\) "Glossary Entry") or [Native SQL](javascript:call_link\('abennative_sql_glosry.htm'\) "Glossary Entry") in programs. [ABAP Managed Database Procedures (AMDP)](javascript:call_link\('abenabap_managed_db_proc_glosry.htm'\) "Glossary Entry") enable SQLScript procedures to be implemented in [AMDP procedure implementations](javascript:call_link\('abenamdp_procedure_method_glosry.htm'\) "Glossary Entry") of [AMDP classes](javascript:call_link\('abenamdp_class_glosry.htm'\) "Glossary Entry"). [Database procedure proxies](javascript:call_link\('abendatabase_proc_proxy_glosry.htm'\) "Glossary Entry") can be created to call SQLScript procedures in the [SAP HANA Repository](javascript:call_link\('abensap_hana_repository_glosry.htm'\) "Glossary Entry"). See also [SQLScript Function](javascript:call_link\('abensql_script_function_glosry.htm'\) "Glossary Entry").
