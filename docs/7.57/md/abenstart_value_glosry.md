start value

Value specified when a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") is declared. The data object is filled with this value when it is created. The start value is set using the [VALUE](javascript:call_link\('abapdata_options.htm'\)) addition of the [DATA](javascript:call_link\('abapdata.htm'\)) statement. If the VALUE addition is not used, the system uses the type-specific [initial value](javascript:call_link\('abeninitial_value_glosry.htm'\) "Glossary Entry").
