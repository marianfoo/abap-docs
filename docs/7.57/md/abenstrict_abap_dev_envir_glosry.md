strict ABAP development environment

Strict ABAP DE for short. [ABAP development environment](javascript:call_link\('abenabap_dev_envir_glosry.htm'\) "Glossary Entry") for [strict ABAP](javascript:call_link\('abenstrict_abap_glosry.htm'\) "Glossary Entry") development:

-   The [ABAP language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") is restricted to one of the [restricted ABAP language versions](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry").
-   Access to other [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") is restricted to [released APIs](javascript:call_link\('abenreleased_api_glosry.htm'\) "Glossary Entry").
-   Only [ABAP Development Tools (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") can be used. There is no access to [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry").

Strict ABAP development environments are provided by [ABAP Platform Strict](javascript:call_link\('abenabap_platform_strict_glosry.htm'\) "Glossary Entry").
