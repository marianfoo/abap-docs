string\_exp - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapcompute_string.htm)

Syntax

... *{* |string\_template| *}*
  *|* *{* operand1 && operand2 *\[*&&  operand3 ... *\]* *}* ...

Effect

String expression for creating a character string

Additions   

-   [string\_template](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstring_templates.htm)
    String template that represents a character string
-   [&&](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenstring_operators.htm)
    String operator for concatenating two character strings
