system ID

SID for short. Identification or name of an [AS ABAP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_abap_glosry.htm "Glossary Entry") and therefore also the ID or name of the [ABAP system](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_system_glosry.htm "Glossary Entry"). A system ID consists of three characters and is the content of the [system field](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensystem_field_glosry.htm "Glossary Entry") [sy-sysid](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abensystem_fields.htm).
