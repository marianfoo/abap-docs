system ID

SID for short. Identification or name of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") and therefore also the ID or name of the [ABAP system](javascript:call_link\('abenabap_system_glosry.htm'\) "Glossary Entry"). A system ID consists of three characters and is the content of the [system field](javascript:call_link\('abensystem_field_glosry.htm'\) "Glossary Entry") [sy-sysid](javascript:call_link\('abensystem_fields.htm'\)).
