system interface

[Interface](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") of the [class library](javascript:call_link\('abenclass_library_glosry.htm'\) "Glossary Entry") that is included in [system classes](javascript:call_link\('abensystem_class_glosry.htm'\) "Glossary Entry") and is used to implement system functions in separate classes or is required as a [tag interface](javascript:call_link\('abentag_interface_glosry.htm'\) "Glossary Entry"). The name usually begins with IF\_ABAP\_ or IF\_SYSTEM\_. Using the [input help](javascript:call_link\('abeninput_help_glosry.htm'\) "Glossary Entry") for IF\_ABAP\_\* or IF\_SYSTEM\_\* in the [Class Builder](javascript:call_link\('abenclass_builder_glosry.htm'\) "Glossary Entry") can give an overview of all system interfaces.
