system table

[Client-independent](javascript:call_link\('abenclient_independent_glosry.htm'\) "Glossary Entry") [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") without a [client column](javascript:call_link\('abenclient_column_glosry.htm'\) "Glossary Entry"). A system table contains data that is not specific for a client and that is generally only processed by [system programs](javascript:call_link\('abensystem_program_glosry.htm'\) "Glossary Entry").
