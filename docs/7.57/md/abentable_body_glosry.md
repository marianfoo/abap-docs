table body

All lines in an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry"). Internal table and table body are identical for internal tables without a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry"). For internal tables with a header line, the table body denotes the actual tabular data and can be addressed by attaching \[\] to the table name.
