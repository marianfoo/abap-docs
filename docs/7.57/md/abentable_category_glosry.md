table category

[Technical type property](javascript:call_link\('abentechnical_type_prpt_glosry.htm'\) "Glossary Entry") of a [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry") that specifies the storage and primary access method. Table categories include [standard tables](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry"), [sorted tables](javascript:call_link\('abensorted_table_glosry.htm'\) "Glossary Entry"), and [hashed tables](javascript:call_link\('abenhashed_table_glosry.htm'\) "Glossary Entry").
