table key

Columns of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") or [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") whose content identifies table lines. For internal tables, distinction is made between [primary table keys](javascript:call_link\('abenprimary_table_key_glosry.htm'\) "Glossary Entry") and [secondary table keys](javascript:call_link\('abensecondary_table_key_glosry.htm'\) "Glossary Entry"), which are [technical type properties](javascript:call_link\('abentechnical_type_prpt_glosry.htm'\) "Glossary Entry") of the [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry").
