table parameter

Obsolete [formal parameter](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") of [function modules](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") or [subroutines](javascript:call_link\('abensubroutine_glosry.htm'\) "Glossary Entry") typed as a [standard table](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry") with a [header line](javascript:call_link\('abenheader_line_glosry.htm'\) "Glossary Entry"). A table parameter is defined using the addition TABLES to the statements [FUNCTION](javascript:call_link\('abenfunction.htm'\)) and [FORM](javascript:call_link\('abapform.htm'\)).
