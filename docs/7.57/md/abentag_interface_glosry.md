tag interface

Special predefined global interface. By implementing or including a tag interface, [classes](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenclass_glosry.htm "Glossary Entry") or other [interfaces](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenoo_intf_glosry.htm "Glossary Entry") are particularly tagged for the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry"). A tag interface generally does not contain its own [interface components](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abeninterface_component_glosry.htm "Glossary Entry") but instead assigns a particular task to the integrating classes or interfaces and changes the way they are handled by the [ABAP Compiler](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_compiler_glosry.htm "Glossary Entry").
