test relation

Relation between a [test class](javascript:call_link\('abentest_class_glosry.htm'\) "Glossary Entry") or a [test method](javascript:call_link\('abentest_method_glosry.htm'\) "Glossary Entry") and another [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). A test relation is defined by the special [ABAP Doc comment](javascript:call_link\('abenabap_doc_comment_glosry.htm'\) "Glossary Entry") ["! @testing](javascript:call_link\('abentest_relations.htm'\)) and allows repository objects to be tested with [ABAP Unit](javascript:call_link\('abenabap_unit_glosry.htm'\) "Glossary Entry") in [ADT](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") that do not support test classes themselves.
