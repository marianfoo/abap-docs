test run

Execution of a [test task](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentest_task_glosry.htm "Glossary Entry") in [ABAP Unit](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_unit_glosry.htm "Glossary Entry"). Test runs for test tasks of an individual program can be started directly in the corresponding editor via Program or Function Module → Test → Unit Test or Class → Unit Test. More extensive test runs can be executed using [Code Inspector](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencode_inspector_glosry.htm "Glossary Entry"). The results of a test run are displayed in the ABAP Unit interface.
