text pool

[Component](javascript:call_link\('abencomponent_glosry.htm'\) "Glossary Entry") of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that contains the [text elements](javascript:call_link\('abentext_element_glosry.htm'\) "Glossary Entry") of the program. Multiple text pools can be defined in various languages for a program.
