text table

Special [foreign key table](javascript:call_link\('abenforeign_key_table_glosry.htm'\) "Glossary Entry") whose [primary key](javascript:call_link\('abenprimary_key_glosry.htm'\) "Glossary Entry") consists of the [foreign key](javascript:call_link\('abenforeign_key_glosry.htm'\) "Glossary Entry"), that is, the primary key of the [check table](javascript:call_link\('abencheck_table_glosry.htm'\) "Glossary Entry") and a language key of the type [LANG](javascript:call_link\('abenddic_builtin_types.htm'\)). A text table contains language-dependent texts that are assigned to the lines of the check table.
