time field

[Data object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_object_glosry.htm "Glossary Entry") of [time type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentime_type_glosry.htm "Glossary Entry") [t](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenbuiltin_types_date_time.htm) for storing a character-like [time](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenday_time_glosry.htm "Glossary Entry"). Valid values have the format hhmmss. The internal representation takes place in the [character representation](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenchar_representation_glosry.htm "Glossary Entry") of the current [code page](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencodepage_glosry.htm "Glossary Entry"). For assignments, data objects with [numeric data types](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abennumeric_data_type_glosry.htm "Glossary Entry") and for calculations, the content of the time field is converted to the number of seconds since 00:00:00.
