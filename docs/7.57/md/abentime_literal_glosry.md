time literal

[Literal](javascript:call_link\('abenabap_literal_glosry.htm'\) "Glossary Entry") that contains a time. It has the data type [TIMN](javascript:call_link\('abenddic_builtin_types.htm'\)) or [TIMS](javascript:call_link\('abenddic_builtin_types.htm'\)). Only [typed](javascript:call_link\('abentyped_literal_glosry.htm'\) "Glossary Entry") time literals are available.
