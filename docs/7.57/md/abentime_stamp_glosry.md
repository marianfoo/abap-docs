time stamp

Combined [date](javascript:call_link\('abendate_glosry.htm'\) "Glossary Entry")/ [time](javascript:call_link\('abentime_glosry.htm'\) "Glossary Entry") specification for determining a point in time. Time stamps of the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") are [POSIX time stamps](javascript:call_link\('abenposix_timestamp_glosry.htm'\) "Glossary Entry") independent of time zone. They are represented in a [time stamp field](javascript:call_link\('abentimestamp_field_glosry.htm'\) "Glossary Entry") with the [time stamp type](javascript:call_link\('abentimestamp_field_glosry.htm'\) "Glossary Entry") [utclong](javascript:call_link\('abenbuiltin_types_date_time.htm'\)) or as [time stamps in packed numbers](javascript:call_link\('abentime_stamps_packed.htm'\)) whose data type is defined using the [DDIC data elements](javascript:call_link\('abendata_element_glosry.htm'\) "Glossary Entry") TIMESTAMP or TIMESTAMPL from the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry").
