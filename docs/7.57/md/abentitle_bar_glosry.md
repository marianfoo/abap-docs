title bar

Component of the [GUI](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abengui_glosry.htm "Glossary Entry") that contains the [GUI title](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abengui_title_glosry.htm "Glossary Entry") of a [GUI window](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abengui_window_glosry.htm "Glossary Entry"). The title bar is set using the [SET TITLEBAR](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abapset_titlebar_dynpro.htm) statement.
