transaction code

Twenty-character name that is connected with a [dynpro](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendynpro_glosry.htm "Glossary Entry"), another transaction code, or a [method](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmethod_glosry.htm "Glossary Entry") of an [ABAP program](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_program_glosry.htm "Glossary Entry") and is used to execute programs. Transaction codes linked with dynpros are possible for [executable programs](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenexecutable_program_glosry.htm "Glossary Entry"), [module pools](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenmodul_pool_glosry.htm "Glossary Entry"), and [function pools](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfunction_pool_glosry.htm "Glossary Entry"). [Parameter transactions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenparameter_transaction_glosry.htm "Glossary Entry") and [variant transactions](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenvariant_transaction_glosry.htm "Glossary Entry") are linked with other transaction codes. Transaction codes linked with methods are allowed for all [program types](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenprogram_type_glosry.htm "Glossary Entry") that contain methods. Transaction codes are maintained and administrated using transaction code SE93.
