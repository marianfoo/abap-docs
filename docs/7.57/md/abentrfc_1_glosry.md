transactional Remote Function Call

tRFC for short. Obsolete [remote function call](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry") in which the function called remotely is registered for execution and started using the statement [COMMIT WORK](javascript:call_link\('abapcommit.htm'\)).
