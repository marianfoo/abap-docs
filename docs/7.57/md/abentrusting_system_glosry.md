trusting system

[AS ABAP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_abap_glosry.htm "Glossary Entry") that trusts a [trusted system](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentrusted_system_glosry.htm "Glossary Entry") in calls using [Remote Function Call](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenremote_function_call_glosry.htm "Glossary Entry"). The trusting system provides the [RFC server](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrfc_server_glosry.htm "Glossary Entry"). The trusting systems can be listed in a trusted system using [transaction code](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abentransaction_code_glosry.htm "Glossary Entry") SMT2.
