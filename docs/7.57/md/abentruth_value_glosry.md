truth value

A truth value is either true or false. In ABAP, truth values only occur as the result of [logical expressions](javascript:call_link\('abenlogical_expression_glosry.htm'\) "Glossary Entry"). [Data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") for [Boolean data objects](javascript:call_link\('abenboolean_data_object_glosry.htm'\) "Glossary Entry") that can accept truth values are not supported. Instead, a truth value can be evaluated as the result of a [Boolean function](javascript:call_link\('abenboole_function_glosry.htm'\) "Glossary Entry").
