tuning object

[Repository object](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenrepository_object_glosry.htm "Glossary Entry") that defines technical settings or tunings for another repository object. A tuning object can be defined and transported separately from the repository object for which it is defined. There are [DDIC tuning objects](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenddic_tuning_object_glosry.htm "Glossary Entry") and [CDS tuning objects](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencds_tuning_object_glosry.htm "Glossary Entry").
