tuning object

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that defines technical settings or tunings for another repository object. A tuning object can be defined and transported separately from the repository object for which it is defined. There are [DDIC tuning objects](javascript:call_link\('abenddic_tuning_object_glosry.htm'\) "Glossary Entry") and [CDS tuning objects](javascript:call_link\('abencds_tuning_object_glosry.htm'\) "Glossary Entry").
