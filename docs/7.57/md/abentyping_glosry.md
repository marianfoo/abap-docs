typing

[Generic](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abengeneric_typing_glosry.htm "Glossary Entry") or [complete](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencomplete_typing_glosry.htm "Glossary Entry") definition of a [data type](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendata_type_glosry.htm "Glossary Entry") for [formal parameters](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenformal_parameter_glosry.htm "Glossary Entry") or [field symbols](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfield_symbol_glosry.htm "Glossary Entry"). Specifies in which operand positions a formal parameter or field symbol can be used and which type properties are used. When parameters are passed, it is checked whether the data type of the assigned [actual parameter](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenactual_parameter_glosry.htm "Glossary Entry"), or memory area is [compatible](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencompatible_glosry.htm "Glossary Entry") with the typing.
