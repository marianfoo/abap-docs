UCS

Short for Universal Character Set. A 4-byte [character set](javascript:call_link\('abencharacter_set_glosry.htm'\) "Glossary Entry"), standardized in ISO-10646, that contains all other character sets. The UCS character set is currently identical to the [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") character set; that is, no other characters than those contained in Unicode are currently defined. [Character representations](javascript:call_link\('abenchar_representation_glosry.htm'\) "Glossary Entry") for the UCS character set are the [UTF](javascript:call_link\('abenutf_glosry.htm'\) "Glossary Entry") representations [UCS-2](javascript:call_link\('abenucs2_glosry.htm'\) "Glossary Entry") and UCS-4.
