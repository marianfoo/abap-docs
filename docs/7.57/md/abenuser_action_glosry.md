user action

Action performed by a user on a [control element](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencontrol_element_glosry.htm "Glossary Entry") on the [user interface](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenuser_interface_glosry.htm "Glossary Entry"). Triggers the [dynpro event](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abendynpro_event_glosry.htm "Glossary Entry") [PAI](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpai_glosry.htm "Glossary Entry") if the selected control element is linked with a [function code](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenfunction_code_glosry.htm "Glossary Entry").
