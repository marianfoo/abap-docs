user name

Name of an [ABAP user](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenabap_user_glosry.htm "Glossary Entry") that is required for logging on to a [user session](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenuser_session_glosry.htm "Glossary Entry") in the [AS ABAP](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenas_abap_glosry.htm "Glossary Entry"). Together with the [password](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenpassword_glosry.htm "Glossary Entry"), the user name allows authorized use of an AS ABAP.
