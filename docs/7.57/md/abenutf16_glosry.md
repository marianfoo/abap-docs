UTF-16

16-bit [Unicode character representation](javascript:call_link\('abenunicode_char_represent_glosry.htm'\) "Glossary Entry"). In this representation, all characters are represented using 2 bytes or 4 bytes. UTF-16 is the [system code page](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry") of a [Unicode system](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry"). UTF-16 uses the same 2-byte encoding as [UCS-2](javascript:call_link\('abenucs2_glosry.htm'\) "Glossary Entry") for the Basic Multilingual Plane (BMP) of the Unicode standard and a 4-byte encoding for the other Unicode planes in the [surrogate area](javascript:call_link\('abensurrogate_area_glosry.htm'\) "Glossary Entry") that are addressed by [surrogates](javascript:call_link\('abensurrogates_glosry.htm'\) "Glossary Entry"). The ABAP programming language supports UCS-2 as a subset of UTF-16 without surrogates.
