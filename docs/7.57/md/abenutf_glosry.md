UTF

Short for Universal Transformation Format. Different representations of the [Unicode](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenunicode_glosry.htm "Glossary Entry") [character set](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abencharacter_set_glosry.htm "Glossary Entry"). All UTF character representations objectively refer to the characters defined in Unicode and can be transformed into each other. The representations differ only in the type of addressing. See [UTF-8](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenutf8_glosry.htm "Glossary Entry"), [UTF-16](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenutf16_glosry.htm "Glossary Entry"), and [UTF-32](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenutf32_glosry.htm "Glossary Entry") plus [UCS-2](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenucs2_glosry.htm "Glossary Entry").
