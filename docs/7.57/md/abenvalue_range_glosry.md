value range

Set of valid values for a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). The values that are valid for a data object are determined by its [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry"). In [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"), the term value range indicates a [semantic property](javascript:call_link\('abenddic_domains_sema.htm'\)) of a [DDIC domain](javascript:call_link\('abendomain_glosry.htm'\) "Glossary Entry").
