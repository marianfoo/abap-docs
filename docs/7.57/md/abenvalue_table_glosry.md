value table

Semantic property of a [DDIC domain](javascript:call_link\('abendomain_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). The value table is the default value for the [check table](javascript:call_link\('abencheck_table_glosry.htm'\) "Glossary Entry") of table fields defined with reference to the domain.
