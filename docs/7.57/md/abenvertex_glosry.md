vertex

Stores [graph](javascript:call_link\('abengraph_glosry.htm'\) "Glossary Entry") data in combination with [edges](javascript:call_link\('abenedge_glosry.htm'\) "Glossary Entry"). A vertex can be considered as a node in a network. One vertex is denoted as the source and the other as the target. An edge connects two vertices. Vertices can have any number of [attributes](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry"). An attribute consists of a name that is associated with a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") and a value.
