vertex key

A vertex key represents one [vertex](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenvertex_glosry.htm "Glossary Entry") attribute or a set of [vertex attributes](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenvertex_attr_glosry.htm "Glossary Entry") that must uniquely identify vertices. If the vertex key consists of multiple [attributes](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenvertex_attr_glosry.htm "Glossary Entry"), then the [edge table's](https://help.sap.com/doc/abapdocu_757_index_htm/7.57/en-US/abenedge_table_glosry.htm "Glossary Entry") source and target must consist of a corresponding set of attributes.
