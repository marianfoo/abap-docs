window function

Function that is specified in front of the definition of a [window](javascript:call_link\('abenwindow_glosry.htm'\) "Glossary Entry") in an [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") [window expression](javascript:call_link\('abenwindow_expression_glosry.htm'\) "Glossary Entry") and that evaluates the rows of the window. A window function is either an [aggregate function](javascript:call_link\('abenaggregate_function_glosry.htm'\) "Glossary Entry"), a [ranking function](javascript:call_link\('abenranking_function_glosry.htm'\) "Glossary Entry"), or a [value function](javascript:call_link\('abenvalue_function_glosry.htm'\) "Glossary Entry").
