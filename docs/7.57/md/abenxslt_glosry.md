XSLT

Short for Extensible Stylesheet Language Transformation, see [www.w3.org/TR/xslt](http://www.w3.org/TR/xslt). XSLT allows the conversion of [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") formats into any other XML formats. The [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") contains the [SAP XSLT processor](javascript:call_link\('abenxslt_processor_glosry.htm'\) "Glossary Entry") for the execution of [XSL transformations](javascript:call_link\('abenxsl_transformation_glosry.htm'\) "Glossary Entry").
