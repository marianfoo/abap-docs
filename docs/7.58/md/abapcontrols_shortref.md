CONTROLS - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcontrols.htm)

Syntax

CONTROLS contrl TYPE *{* TABLEVIEW USING SCREEN dynnr *}*
                   *|* *{* TABSTRIP *}*.

Effect

Declares the use of a [control](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencontrol_glosry.htm "Glossary Entry") contrl on a [dynpro](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendynpro_glosry.htm "Glossary Entry") of the program. An identically named structure is created for controlling the control.

Additions   

-   [TABLEVIEW USING SCREEN dynnr](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcontrols_tableview.htm)
    Declares the use of a [table control](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentable_control_glosry.htm "Glossary Entry") on the dynpro specified in dynnr.
-   [TABSTRIP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcontrols_tabstrip.htm)
    Declares the use of a [tabstrip control](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentabstrip_control_glosry.htm "Glossary Entry").
