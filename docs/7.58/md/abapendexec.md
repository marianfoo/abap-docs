ENDEXEC

[Short Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapexec_sql_shortref.htm)

Syntax

ENDEXEC.

Effect

The statement ENDEXEC closes an embedded [Native SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennative_sql_glosry.htm "Glossary Entry") statement introduced using [EXEC SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapexec.htm).
