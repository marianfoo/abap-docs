ENDFUNCTION

[Short Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapfunction_shortref.htm)

Syntax

ENDFUNCTION.

Effect

The statement ENDFUNCTION closes a function module introduced using [FUNCTION](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapfunction.htm).
