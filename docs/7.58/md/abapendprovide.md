ENDPROVIDE

[Short Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapprovide_shortref.htm)

Syntax

ENDPROVIDE.

Effect

The statement ENDPROVIDE closes a loop started using [PROVIDE](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapprovide.htm).
