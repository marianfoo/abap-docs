EXTRACT - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapextract.htm)

Syntax

EXTRACT *\[* header *|* field\_group *\]*.

Effect

Obsolete: Attaches a [field group](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfield_group_glosry.htm "Glossary Entry"), declared using FIELD-GROUPS, to the [extract dataset](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenextract_dataset_glosry.htm "Glossary Entry").
