INITIALIZATION - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapinitialization.htm)

Syntax

INITIALIZATION.

Effect

Initializes an event block whose event is raised in an [executable program](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexecutable_program_glosry.htm "Glossary Entry") between LOAD-OF-PROGRAM and [selection screen processing](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenselscreen_processing_glosry.htm "Glossary Entry").
