LEAVE PROGRAM - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapleave_program.htm)

Syntax

LEAVE PROGRAM.

Effect

Exits the current program and continues processing as specified by the program call after the call position of the program or the [call sequence](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencall_sequence_glosry.htm "Glossary Entry").
