SELECTION-SCREEN, LINE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapselection-screen_line.htm)

Syntax

SELECTION-SCREEN BEGIN OF LINE.
...
SELECTION-SCREEN END OF LINE.

Effect

Defines a line on the current [selection screen](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenselection_screen_glosry.htm "Glossary Entry") in which screen elements can be positioned using [SELECTION-SCREEN POSITION](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapselection-screen_shortref.htm).
