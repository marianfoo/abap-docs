SET PARAMETER - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapset_parameter.htm)

Syntax

SET PARAMETER ID pid FIELD dobj.

Effect

Sets the [SPA/GPA parameter](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenspa_gpa_parameter_glosry.htm "Glossary Entry") specified in pid to the value of dobj.
