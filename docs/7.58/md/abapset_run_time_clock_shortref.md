SET RUN TIME CLOCK - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapset_run_time_clock_resolution.htm)

Syntax

SET RUN TIME CLOCK RESOLUTION *{*HIGH*|*LOW*}*.

Effect

Sets the measurement accuracy for the statement GET RUN TIME.
