[Reference](javascript:call_link\('abapsuppress.htm'\))

Syntax

Effect

If specified, processes the current [dynpro](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") at [PBO](javascript:call_link\('abenpbo_glosry.htm'\) "Glossary Entry") without displaying the [screen layout](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry").
