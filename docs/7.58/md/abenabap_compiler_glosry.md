ABAP Compiler

ABAP Compiler creates [byte code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbytecode_glosry.htm "Glossary Entry") as intermediate code when generating a program from [ABAP source code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_source_code_glosry.htm "Glossary Entry") that is stored in the database as a program [load](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenload_glosry.htm "Glossary Entry") and is loaded into the [PXA](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpxa_glosry.htm "Glossary Entry") if required.
