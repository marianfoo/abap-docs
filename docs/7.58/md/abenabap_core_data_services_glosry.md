ABAP Core Data Services

ABAP CDS for short. An infrastructure for defining and consuming semantically rich data models on [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). ABAP CDS provides different languages for defining metadata realized as [CDS objects](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry").
