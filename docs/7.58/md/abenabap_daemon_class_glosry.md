ABAP daemon class

Global subclass of CL\_ABAP\_DAEMON\_EXT\_BASE. The instances of an ABAP daemon class are [ABAP daemons](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_daemon_glosry.htm "Glossary Entry") created by the [ABAP daemon framework](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_daemon_framework_glosry.htm "Glossary Entry"), each of which are executed in a separate [ABAP daemon session](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_daemon_session_glosry.htm "Glossary Entry"). An ABAP daemon class implements interface methods inherited from the superclass to react to [ABAP daemon events](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_daemon_event_glosry.htm "Glossary Entry").
