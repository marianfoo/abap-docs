Character String and Byte String Processing

This section describes the language elements, expressions, and functions designed especially for the processing of [character strings](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencharacter_string_glosry.htm "Glossary Entry") and [byte strings](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_chain_glosry.htm "Glossary Entry") in character-like and byte-like data objects.

-   [Character String and Byte String Processing - Overview](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_processing_oview.htm)
-   [Statements for Character String and Byte String Processing](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_processing_statements.htm)
-   [Expressions and Functions for String Processing](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_processing_expr_func.htm)
