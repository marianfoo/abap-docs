ABAP Database Connectivity

ADBC for short. Class-based API for the [Native SQL interface](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennative_sql_interface_glosry.htm "Glossary Entry"). ADBC enables object-oriented execution of dynamically specified [Native SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennative_sql_glosry.htm "Glossary Entry").
