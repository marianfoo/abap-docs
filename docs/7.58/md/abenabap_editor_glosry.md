ABAP Editor

[ABAP Workbench](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_workbench_glosry.htm "Glossary Entry") tool for creating [ABAP programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry"). It is either called directly using [transaction code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentransaction_code_glosry.htm "Glossary Entry") SE38 or it is integrated into other tools, such as [Object Navigator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenobject_navigator_glosry.htm "Glossary Entry"), [Class Builder](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_builder_glosry.htm "Glossary Entry"), or [Function Builder](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfunction_builder_glosry.htm "Glossary Entry").
