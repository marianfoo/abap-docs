ABAP Editor

[ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") tool for creating [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"). It is either called directly using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SE38 or it is integrated into other tools, such as [Object Navigator](javascript:call_link\('abenobject_navigator_glosry.htm'\) "Glossary Entry"), [Class Builder](javascript:call_link\('abenclass_builder_glosry.htm'\) "Glossary Entry"), or [Function Builder](javascript:call_link\('abenfunction_builder_glosry.htm'\) "Glossary Entry").
