Hints

-   iXML documents and their nodes can also be specified as an XML source for XSL transformations called using [CALL TRANSFORMATION](javascript:call_link\('abapcall_transformation.htm'\)) so that they are also read.
-   If the content of elements or attributes contain the notations &lt;, &gt;, &amp;, &quot;, or &apos;, they are transformed automatically to the corresponding [characters](javascript:call_link\('abenxml_oview.htm'\)) when read.
