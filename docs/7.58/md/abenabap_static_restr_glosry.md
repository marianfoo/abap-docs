Static ABAP with restricted object use

Obsolete [ABAP language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") with restricted use of [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in which all language elements of [Standard ABAP](javascript:call_link\('abenstandard_abap_glosry.htm'\) "Glossary Entry") except dynamic language elements are allowed.
