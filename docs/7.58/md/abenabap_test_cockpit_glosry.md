ABAP Test Cockpit

ATC for short. Framework for executing and evaluating various tests for [repository objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrepository_object_glosry.htm "Glossary Entry") integrated into [ABAP Workbench](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_workbench_glosry.htm "Glossary Entry") or [ABAP development tools for Eclipse (ADT)](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenadt_glosry.htm "Glossary Entry") and the [Change and Transport System (CTS)](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencts_glosry.htm "Glossary Entry").
