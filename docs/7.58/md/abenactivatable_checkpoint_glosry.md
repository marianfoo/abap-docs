activatable checkpoint

[Checkpoint](javascript:call_link\('abencheckpoint_glosry.htm'\) "Glossary Entry") that is assigned to a [checkpoint group](javascript:call_link\('abencheckpoint_group_glosry.htm'\) "Glossary Entry") using the addition ID of the statements [ASSERT](javascript:call_link\('abapassert.htm'\)), [BREAK-POINT](javascript:call_link\('abapbreak-point.htm'\)), or [LOG-POINT](javascript:call_link\('abaplog-point.htm'\)). The behavior of an activatable checkpoint is controlled by either group-specific or program-specific activation settings, where program-specific settings apply to all activatable checkpoints of a program and overwrite the settings of the relevant checkpoint group.
