annotation

An annotation can be used to add metadata to a [repository object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrepository_object_glosry.htm "Glossary Entry"). The following types of annotations are available:

→ [CDS annotation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_annotation_glosry.htm "Glossary Entry")

→ [DDIC annotation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_annotation_glosry.htm "Glossary Entry")
