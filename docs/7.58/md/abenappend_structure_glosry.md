append structure

[Structure](javascript:call_link\('abenstructure_glosry.htm'\) "Glossary Entry") of the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that is appended to a different DDIC structure or [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") thus adding its [structure components](javascript:call_link\('abenstructure_component_glosry.htm'\) "Glossary Entry") to them. Append structures and database tables can be used to add components to structures and database tables delivered by SAP without making modifications to them.
