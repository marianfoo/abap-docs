application layer

Software layer of a [client-server system](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_server_system_glosry.htm "Glossary Entry") that contains the application logic. [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry") represents the application layer of an [ABAP system](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_system_glosry.htm "Glossary Entry") in which [ABAP programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry") are executed.
