application table

[DDIC database table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_db_table_glosry.htm "Glossary Entry") with [client column](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_column_glosry.htm "Glossary Entry"). An application table contains client-dependent data that is processed by [application programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenapplication_program_glosry.htm "Glossary Entry").
