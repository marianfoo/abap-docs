archiving

Storing of [spool lists](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenspool_list_glosry.htm "Glossary Entry") using the [ArchiveLink](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenarchivelink_glosry.htm "Glossary Entry") tool. Spool lists stored in the [SAP spool system](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensap_spool_system_glosry.htm "Glossary Entry") can be archived. During archiving, spool lists passed from the spool system are saved in the same format that was used there.
