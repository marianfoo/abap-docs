archiving parameter

Parameter that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenspool_request_glosry.htm "Glossary Entry") if its data is to be archived using [ArchiveLink](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenarchivelink_glosry.htm "Glossary Entry").
