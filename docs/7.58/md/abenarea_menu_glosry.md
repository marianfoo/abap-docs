area menu

Tree structure whose nodes can be linked with [transaction codes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentransaction_code_glosry.htm "Glossary Entry"), [executable programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexecutable_program_glosry.htm "Glossary Entry"), and other area menus. Area menus are maintained using transaction code SE43 and displayed via [SAP Easy Access](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensap_easy_access_glosry.htm "Glossary Entry").
