ASCII

Abbreviation for American Standard Code for Information Interchange. 7-bit [character set](javascript:call_link\('abencharacter_set_glosry.htm'\) "Glossary Entry") in accordance with the ISO-646 standard. Expanded to 8-bit character sets in [ISO-8859](javascript:call_link\('abeniso-8859_glosry.htm'\) "Glossary Entry").
