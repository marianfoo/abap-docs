background request

[Job](javascript:call_link\('abenjob_glosry.htm'\) "Glossary Entry") used to start one or more [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") in [background processing](javascript:call_link\('abenbackround_processing_glosry.htm'\) "Glossary Entry").
