BAdI class

Specific global class that is created when a [BAdI](javascript:call_link\('abenbadi_glosry.htm'\) "Glossary Entry") is defined in the [Enhancement Builder](javascript:call_link\('abenenhancement_builder_glosry.htm'\) "Glossary Entry") and given the name of the BAdI as a [subclass](javascript:call_link\('abensubclass_glosry.htm'\) "Glossary Entry") of the [abstract](javascript:call_link\('abenabstract_glosry.htm'\) "Glossary Entry") global [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") CL\_BADI\_BASE. BAdI classes are a part of the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") and, apart from being in the namespace of all global [types](javascript:call_link\('abentype_glosry.htm'\) "Glossary Entry"), they are not visible externally.
