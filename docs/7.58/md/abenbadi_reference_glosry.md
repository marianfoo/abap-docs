BAdI reference

Reference to a [BAdI object](javascript:call_link\('abenbadi_object_glosry.htm'\) "Glossary Entry"). BAdI references occur only as content of [BAdI reference variables](javascript:call_link\('abenbadi_reference_variable_glosry.htm'\) "Glossary Entry"). Technically, BAdI references are [object references](javascript:call_link\('abenobject_reference_glosry.htm'\) "Glossary Entry") to [instances](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of [BAdI classes](javascript:call_link\('abenbadi_class_glosry.htm'\) "Glossary Entry").
