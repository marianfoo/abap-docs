batch input session

A summary of a series of transaction calls including input data and [user actions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenuser_action_glosry.htm "Glossary Entry"). A batch input session can be used to execute [dialog transactions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendialog_transaction_glosry.htm "Glossary Entry") in [batch input](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbatch_input_glosry.htm "Glossary Entry"), whereby some or all of the [screens](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenscreen_glosry.htm "Glossary Entry") are handled by the session. Batch input sessions are stored persistently and are managed by the batch input administration in transaction SM35. See also [batch input tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbatch_input_table_glosry.htm "Glossary Entry").
