Effect

Defines the [behavior extension body](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_behavior_ext_body_glosry.htm "Glossary Entry") to extend an existing [RAP BO entity](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_entity_glosry.htm "Glossary Entry") in the [RAP BDL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_bdl_glosry.htm "Glossary Entry").

-   [Adding Elements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbdl_new_elem_ext.htm): Elements can be added to an existing [entity behavior definition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_entity_bdef_glosry.htm "Glossary Entry").
-   [Extending Elements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbdl_ext_elem_ext.htm): Existing elements of existing entity behavior definitions can be extended.
