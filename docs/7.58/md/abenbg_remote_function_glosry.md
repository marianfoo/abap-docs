background Remote Function Call

bgRFC for short. Asynchronous remote function call for which the called [RFMs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrfm_glosry.htm "Glossary Entry") can be executed transactionally and in a predefined call sequence. bgRFC replaces [tRFC](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentrfc_2_glosry.htm "Glossary Entry") and [qRFC](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenqrfc_glosry.htm "Glossary Entry") apart from the no-send scenario of qRFC. The latter is replaced by the [local data queue (LDQ)](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenlocal_data_queue_glosry.htm "Glossary Entry").
