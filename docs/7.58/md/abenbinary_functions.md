Byte String Functions (bit\_func)

Byte string functions are one of the [built-in functions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_functions.htm). They are divided into:

-   [Description function for byte-like data types](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendescriptive_functions_binary.htm)
-   [Bit functions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbit_functions.htm)
