binary floating point arithmetic

Calculation with [binary floating point numbers](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbinfloat_glosry.htm "Glossary Entry"). Can produce rounding errors because floating point numbers are not represented internally using the decimal system.
