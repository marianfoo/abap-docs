bound data type

[Data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") that only exists as a property of a [data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry"). A bound data type has a [technical type name](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentechnical_type_name_glosry.htm "Glossary Entry"), that can be retrieved as part of an [absolute type name](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabsolute_typename_glosry.htm "Glossary Entry") by [RTTI](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrun_time_type_identific_glosry.htm "Glossary Entry"). See also [standalone data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstand-alone_data_type_glosry.htm "Glossary Entry").
