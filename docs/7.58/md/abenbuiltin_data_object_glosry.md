built-in data object

[Data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") in an [ABAP program](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry") by the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") that is either permanently active or only in specific [contexts](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenobj_context_glosry.htm "Glossary Entry"). The built-in data objects are [system fields](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_field_glosry.htm "Glossary Entry"), the [constant](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconstant_glosry.htm "Glossary Entry") [space](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenspace.htm), and the [self-reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenself_reference_glosry.htm "Glossary Entry") [me](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenme.htm). The [structure](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstructure_glosry.htm "Glossary Entry") [screen](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenscreen_structure_obsolete.htm) is obsolete.
