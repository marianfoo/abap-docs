byte field type

[Byte-like data type](javascript:call_link\('abenbyte_like_data_typ_glosry.htm'\) "Glossary Entry") for storing [byte strings](javascript:call_link\('abenbyte_string_glosry.htm'\) "Glossary Entry") of fixed length. The associated [built-in ABAP type](javascript:call_link\('abenbuiltin_abap_type_glosry.htm'\) "Glossary Entry") is [x](javascript:call_link\('abenbuiltin_types_byte.htm'\)). The [built-in DDIC types](javascript:call_link\('abenbuiltin_ddic_type_glosry.htm'\) "Glossary Entry") are [RAW](abenddic_builtin_types.htm#@@ITOC@@ABENDDIC_BUILTIN_TYPES_4) and [LRAW](abenddic_builtin_types.htm#@@ITOC@@ABENDDIC_BUILTIN_TYPES_4). A [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") of a byte field type is a [byte field](javascript:call_link\('abenbyte_field_glosry.htm'\) "Glossary Entry").
