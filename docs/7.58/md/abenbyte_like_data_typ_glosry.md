byte-like data type

[Data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") that allows access to the unencoded bytes of [byte chains](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_chain_glosry.htm "Glossary Entry"). These data types are [byte field types](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_field_type_glosry.htm "Glossary Entry") and [byte string types](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_string_type_glosry.htm "Glossary Entry"). The associated [generic data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [xsequence](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_types_generic.htm).
