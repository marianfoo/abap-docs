byte string type

[Byte-like data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_like_data_typ_glosry.htm "Glossary Entry") for storing [byte strings](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_string_glosry.htm "Glossary Entry") of variable length. The associated [built-in ABAP type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuiltin_abap_type_glosry.htm "Glossary Entry") is [xstring](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuiltin_types_byte.htm). The [built-in DDIC type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuiltin_ddic_type_glosry.htm "Glossary Entry") is [RAWSTRING](abenddic_builtin_types.htm#@@ITOC@@ABENDDIC_BUILTIN_TYPES_2). Internal administration takes place using an internal [reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenreference_glosry.htm "Glossary Entry") and the data type is [deep](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendeep_glosry.htm "Glossary Entry"). A [data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") of a byte field type is a [byte string](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_string_glosry.htm "Glossary Entry").
