calculation assignment operator

Operator for a [calculation assignment](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignment_glosry.htm "Glossary Entry"). The following calculation assignment operators are available: [addition assignment operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenadd_assignment_op_glosry.htm "Glossary Entry") ([+=](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignments.htm)), a [subtraction assignment operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensub_assignment_op_glosry.htm "Glossary Entry") ([\-=](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignments.htm)), a [multiplication assignment operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmult_assignment_op_glosry.htm "Glossary Entry") ([\*=](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignments.htm)), a [division assignment operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendiv_assignment_op_glosry.htm "Glossary Entry") ([/=](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignments.htm)), or a [concatenation assignment operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconcat_assignment_op_glosry.htm "Glossary Entry") ([&&=](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencalculation_assignment_string.htm)).
