canonical JSON representation

Format of [JSON](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenjson_glosry.htm "Glossary Entry") data that is created during the [serialization](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenserialization_glosry.htm "Glossary Entry") of ABAP data by the [identity transformation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenid_trafo_glosry.htm "Glossary Entry") and that is a prerequisite for a [deserialization](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendeserialization_glosry.htm "Glossary Entry") using identity transformations. The short name is [asJSON](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenasjson_glosry.htm "Glossary Entry").
