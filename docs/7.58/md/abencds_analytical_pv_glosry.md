CDS analytical projection view

Synonym for [CDS analytical query](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_analytic_query_glosry.htm "Glossary Entry"). [CDS projection view](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_projection_view_glosry.htm "Glossary Entry") that is intended for modeling analytical queries within a CDS data model. A CDS analytical projection view is defined using the [CDS DDL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_ddl_glosry.htm "Glossary Entry") statement [DEFINE TRANSIENT VIEW ENTITY AS PROJECTION ON](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_define_view_as_analytical.htm) and it must have the [provider contract](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_pv_provider_contract.htm) set to ANALYTICAL\_QUERY.
