RAP behavior definition header

BDEF header short. A BDEF header is the part of a [RAP behavior definition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_behavior_definition_glosry.htm "Glossary Entry") that defines the [implementation type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_bdef_impl_type_glosry.htm "Glossary Entry") and some global attributes that apply to the complete [business object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbusiness_object_glosry.htm "Glossary Entry"). It is the topmost part and includes all statements that are listed before the [entity behavior definition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_entity_bdef_glosry.htm "Glossary Entry"), define behavior for.
