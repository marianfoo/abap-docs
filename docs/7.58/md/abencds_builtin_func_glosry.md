CDS built-in function

A predefined function in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") that can be used in operand positions of [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry"). ABAP CDS offers [numeric functions](javascript:call_link\('abencds_sql_functions_numeric_v2.htm'\)), [string functions](javascript:call_link\('abencds_sql_functions_character_v2.htm'\)), and [date and time functions](javascript:call_link\('abencds_date_time_functions_v2.htm'\)) that have a return value of the corresponding data type. [Conversion functions](javascript:call_link\('abencds_conversion_functions_v2.htm'\)) are available to perform conversions between data types.
