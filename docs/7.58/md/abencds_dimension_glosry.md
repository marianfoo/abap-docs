CDS analytical dimension field

An analytical dimension field is an element of an [analytical cube view](javascript:call_link\('abencds_analytical_cube_glosry.htm'\) "Glossary Entry") that describes a characteristic of the view's subject and that can be used for [grouping](javascript:call_link\('abencds_group_by_v2.htm'\)) or filtering. For example if a view describes a sales order, the following columns can be dimensions: customer, date, region.
