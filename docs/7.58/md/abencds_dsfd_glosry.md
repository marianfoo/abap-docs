CDS scalar function definition

[CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") that defines a [CDS scalar function](javascript:call_link\('abencds_scalar_function_glosry.htm'\) "Glossary Entry") using language elements of the [CDS FDL](javascript:call_link\('abencds_fdl_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") in [FDL source code](javascript:call_link\('abenfdl_source_code_glosry.htm'\) "Glossary Entry"). A CDS scalar function definition is defined using the statement [DEFINE SCALAR FUNCTION](javascript:call_link\('abencds_define_scalar_function.htm'\)).
