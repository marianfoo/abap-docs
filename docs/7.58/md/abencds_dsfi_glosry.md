CDS scalar function implementation reference

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that binds a [CDS scalar function](javascript:call_link\('abencds_scalar_function_glosry.htm'\) "Glossary Entry") to a runtime environment. In case of an [SQL-based scalar function](javascript:call_link\('abencds_sql_scalar_glosry.htm'\) "Glossary Entry"), the scalar function implementation reference also specifies the [AMDP function implementation](javascript:call_link\('abenamdp_function_method_glosry.htm'\) "Glossary Entry") that implements the scalar function in question.
