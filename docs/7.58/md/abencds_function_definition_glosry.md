CDS function definition

A [CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") that defines a [CDS function](javascript:call_link\('abencds_function_glosry.htm'\) "Glossary Entry") using language elements of the [CDS FDL](javascript:call_link\('abencds_fdl_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") in [FDL source code](javascript:call_link\('abenfdl_source_code_glosry.htm'\) "Glossary Entry"). Currently, [CDS scalar function definitions](javascript:call_link\('abencds_dsfd_glosry.htm'\) "Glossary Entry") are the only type of CDS function definition available.
