CDS function

Function in [ABAP CDS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_cds_glosry.htm "Glossary Entry") that can be used in operand positions of [CDS entities](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_entity_glosry.htm "Glossary Entry"). A CDS function can be a [user-defined function](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_user_func_glosry.htm "Glossary Entry"), a [built-in function](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_builtin_func_glosry.htm "Glossary Entry"), or a [system function](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_system_func_glosry.htm "Glossary Entry").
