CDS-managed DDIC view

An obsolete [DDIC database view](javascript:call_link\('abendatabase_view_glosry.htm'\) "Glossary Entry") of the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that is created in addition to the [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") when a [CDS DDIC-based view (obsolete)](javascript:call_link\('abencds_v1_view_glosry.htm'\) "Glossary Entry") is activated. CDS-managed DDIC views are used as technical foundations for a CDS view of this type, but should not be used directly in ABAP or ABAP CDS.
