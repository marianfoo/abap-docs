Specifies a literal [literal](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_literal_v2.htm), a field [field](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_field_v2.htm), an [enumerated constant](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenenumerated_constant_glosry.htm "Glossary Entry") [EnumType.#EnumConstant](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_enum_field_v2.htm), a parameter [$parameters.pname](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_parameter_v2.htm), or a session variable [$session.vname](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_session_variable_v2.htm) as an elementary [CDS operand](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_operand_glosry.htm "Glossary Entry") in a CDS DDL operand position.
