CDS schema notation

CSN for short. Notation for compact representations of data models, such as data models defined with [ABAP CDS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_cds_glosry.htm "Glossary Entry"). SAP standard format for metadata exchange between data models of different technology stacks, for example between [ABAP CDS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_cds_glosry.htm "Glossary Entry") and SAP Analytics Cloud (SAC).
