CDS service definition extension

A transportable extension of a [CDS service definition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_service_definition_glosry.htm "Glossary Entry") using the [CDS SDL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_sdl_glosry.htm "Glossary Entry") statement [EXTEND SERVICE](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensrvd_extend_service.htm). A CDS service definition extension can add further entities to an existing CDS service definition without modifying the original service definition.
