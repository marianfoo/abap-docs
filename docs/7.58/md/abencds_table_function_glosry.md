CDS table function

A [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") defined using [DEFINE TABLE FUNCTION](javascript:call_link\('abencds_f1_define_table_function.htm'\)) in a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). Each CDS table function is linked with an [AMDP table function](javascript:call_link\('abenamdp_table_function_glosry.htm'\) "Glossary Entry") in which it is implemented using [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry"). A CDS table function returns a tabular result set.
