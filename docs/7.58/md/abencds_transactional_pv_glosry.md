CDS transactional query

A [CDS projection view](javascript:call_link\('abencds_projection_view_glosry.htm'\) "Glossary Entry") that is intended for modeling transactional queries. A transactional query is typically used in the context of the [ABAP RESTful Application Programming Model](javascript:call_link\('abenarap_glosry.htm'\) "Glossary Entry") to adapt a CDS data model for service-specific use-cases. A transactional query is defined using the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") statement [DEFINE VIEW ENTITY AS PROJECTION ON](javascript:call_link\('abencds_define_view_as_projection.htm'\)) and it must have the [provider contract](javascript:call_link\('abencds_pv_provider_contract.htm'\)) set to TRANSACTIONAL\_QUERY.
