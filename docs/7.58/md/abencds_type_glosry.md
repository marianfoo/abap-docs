CDS type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") defined as a [global type](javascript:call_link\('abenglobal_type_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"). CDS types are [user-defined types](javascript:call_link\('abencds_user_defined_type_glosry.htm'\) "Glossary Entry") or [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry"). Both can be used as CDS types in ABAP programs.
