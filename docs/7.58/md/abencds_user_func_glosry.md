CDS user-defined function

Custom function in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") that allows developers to define and implement their own custom logic and reuse it in operand positions of [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry"). [SQL-based scalar functions](javascript:call_link\('abencds_sql_scalar_glosry.htm'\) "Glossary Entry") are the only type of CDS user-defined functions available.
