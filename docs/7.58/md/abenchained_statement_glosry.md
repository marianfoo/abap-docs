chained statement

Abbreviated way of writing multiple [ABAP statements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_statement_glosry.htm "Glossary Entry") with the same initial part. The identical initial part is specified once and closed using a colon (:). This is followed by the remaining parts separated by commas.
