character-like data type

[Data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") whose [data objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") contain [characters](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencharacter_glosry.htm "Glossary Entry") encoded according to the current [code page](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencodepage_glosry.htm "Glossary Entry"). The actual character-like data types are [text field type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentext_field_type_glosry.htm "Glossary Entry"), [numeric text field type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennumeric_text_field_type_glosry.htm "Glossary Entry"), and [text string type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentext_string_type_glosry.htm "Glossary Entry"). [Date type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendate_type_glosry.htm "Glossary Entry") and [time type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentime_type_glosry.htm "Glossary Entry") are special character-like data types. The associated [generic data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [clike](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_types_generic.htm).
