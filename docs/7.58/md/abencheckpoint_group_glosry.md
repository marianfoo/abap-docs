checkpoint group

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that can be used to group and activate [activatable checkpoints](javascript:call_link\('abenactivatable_checkpoint_glosry.htm'\) "Glossary Entry"). To create checkpoint groups and maintain the activation settings, use [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SAAB. The activation settings of a checkpoint group are valid for all checkpoints assigned to the group. Settings for a set of checkpoint groups can be created using [activation variants](javascript:call_link\('abenactivation_variant_glosry.htm'\) "Glossary Entry").
