ABAP SQL - System Classes

The following system classes are available for extending the scope of the ABAP SQL language:

-   [CL\_OSQL\_EXTENDED\_RESULT](javascript:call_link\('abencl_osql_extended_result.htm'\)) handles extended results
-   [CL\_OSQL\_REPLACE](javascript:call_link\('abencl_osql_replace.htm'\)) provides a replacement service for executing [unit tests](javascript:call_link\('abenunit_test_glosry.htm'\) "Glossary Entry")
-   [CL\_ABAP\_DBFEATURES](javascript:call_link\('abencl_abap_dbfeatures.htm'\)) returns features of the current database
