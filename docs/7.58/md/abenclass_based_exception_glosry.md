class-based exception

[Catchable exception](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencatchable_exception_glosry.htm "Glossary Entry"), which is represented by an [exception object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexception_object_glosry.htm "Glossary Entry") of an [exception class](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexception_class_glosry.htm "Glossary Entry"). Class-based exceptions replace the obsolete [catchable runtime errors](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencatchable_runtime_error_glosry.htm "Glossary Entry") and non-class-based exceptions. A class-based exception can either terminate the current context or be a [resumable exception](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenresumable_exception_glosry.htm "Glossary Entry").
