client-independent

A [SQL data source](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensql_data_source_glosry.htm "Glossary Entry") that is not [client-dependent](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_dependent_glosry.htm "Glossary Entry"). Client-independent SQL data sources contain data that is not specific to a [client](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_glosry.htm "Glossary Entry") and should be accessed by [system programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_program_glosry.htm "Glossary Entry"). See also [system table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_table_glosry.htm "Glossary Entry").
