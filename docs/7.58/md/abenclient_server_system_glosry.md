client-server system

Software system that consists of multiple software layers. These are usually the [presentation layer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpresentation_layer_glosry.htm "Glossary Entry"), the [application layer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenapplication_layer_glosry.htm "Glossary Entry"), and the [database layer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendatabase_layer_glosry.htm "Glossary Entry"). [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry") represents the application layer of a client-server system ([ABAP system](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_system_glosry.htm "Glossary Entry")) in which [ABAP programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry") are executed.
