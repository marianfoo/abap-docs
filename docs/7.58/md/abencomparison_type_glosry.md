comparison type

Property of [comparison expressions](javascript:call_link\('abencomparison_expression_glosry.htm'\) "Glossary Entry") whose [operands](javascript:call_link\('abenoperand_glosry.htm'\) "Glossary Entry") have [elementary data types](javascript:call_link\('abenelementary_data_type_glosry.htm'\) "Glossary Entry"), including [calculation expressions](javascript:call_link\('abencalculation_expression_glosry.htm'\) "Glossary Entry"). The comparison type is determined from the operands and defines how the comparison is executed. The comparison type can be of any [built-in ABAP type](javascript:call_link\('abenbuiltin_abap_type_glosry.htm'\) "Glossary Entry"). Operands of another type are converted to the comparison type before the comparison.
