complex data type

A [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that is made up of other data types. A complex data type is either a [structured data type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry"), a [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry"), or a [mesh type](javascript:call_link\('abenmesh_type_glosry.htm'\) "Glossary Entry").
