complex data type

A [data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") that is made up of other data types. A complex data type is either a [structured data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstructured_type_glosry.htm "Glossary Entry"), a [table type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentable_type_glosry.htm "Glossary Entry"), or a [mesh type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_type_glosry.htm "Glossary Entry").
