component interface

[Interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") included in a [compound interface](javascript:call_link\('abencompound_interface_glosry.htm'\) "Glossary Entry") by an [INTERFACES](javascript:call_link\('abapinterfaces.htm'\)) statement. A component interface can itself be a compound interface.
