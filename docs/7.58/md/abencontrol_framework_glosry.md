Control Framework

CFW for short. Class hierarchy whose [classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_glosry.htm "Glossary Entry") begin with the prefix CL\_GUI\_. The common [superclass](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensuperclass_glosry.htm "Glossary Entry") is CL\_GUI\_OBJECT. The classes in Control Framework wrap [GUI controls](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengui_control_glosry.htm "Glossary Entry") and screen containers for these controls.
