component operator

[Constructor operator](javascript:call_link\('abenconstructor_operator_glosry.htm'\) "Glossary Entry") [CORRESPONDING](javascript:call_link\('abenconstructor_expr_corresponding.htm'\)) for component by component assignments between [structures](javascript:call_link\('abenstructure_glosry.htm'\) "Glossary Entry") or between [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") in [operand positions](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry"). By default, the mapping between the [structure components](javascript:call_link\('abenstructure_component_glosry.htm'\) "Glossary Entry") is based on identical names and can also be defined using mapping rules.
