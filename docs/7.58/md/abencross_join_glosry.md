cross join

A [join type](javascript:call_link\('abenjoin_type_glosry.htm'\) "Glossary Entry") for which the rows of the result set are determined by a cross product of the result sets of both [SQL data sources](javascript:call_link\('abensql_data_source_glosry.htm'\) "Glossary Entry"). The result set contains all possible combinations of rows.
