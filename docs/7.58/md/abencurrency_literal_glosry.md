currency field literal

[Literal](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_literal_glosry.htm "Glossary Entry") of the data type [CURR](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_builtin_types.htm) that contains a [currency field](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencurrency_field_glosry.htm "Glossary Entry"). Only [typed](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentyped_literal_glosry.htm "Glossary Entry") currency field literals are available.
