data cluster

Grouping of [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") for transient and persistent storage in a storage medium. A data cluster can be processed using the statements [IMPORT](javascript:call_link\('abapimport_data_cluster.htm'\)), [EXPORT](javascript:call_link\('abapexport_data_cluster.htm'\)), and [DELETE FROM](javascript:call_link\('abapdelete_cluster.htm'\)).
