database procedure

General synonym for [stored procedure](javascript:call_link\('abenstored_procedure_glosry.htm'\) "Glossary Entry"). In [SAP HANA databases](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") it is a name for an [SQLScript procedure](javascript:call_link\('abensql_script_procedure_glosry.htm'\) "Glossary Entry") programmed in [SQLScript](javascript:call_link\('abensql_script_glosry.htm'\) "Glossary Entry") or an [L procedure](javascript:call_link\('abenl_procedure_glosry.htm'\) "Glossary Entry") programmed in [L](javascript:call_link\('abenllang_glosry.htm'\) "Glossary Entry"). Database procedures can be managed as [AMDP procedures](javascript:call_link\('abenamdp_procedure_glosry.htm'\) "Glossary Entry") in the [ABAP Managed Database Procedures](javascript:call_link\('abenabap_managed_db_proc_glosry.htm'\) "Glossary Entry") (AMDP). See also [database function](javascript:call_link\('abendatabase_function_glosry.htm'\) "Glossary Entry").
