database user

Person or system logged on to a [database system](javascript:call_link\('abendatabase_system_glosry.htm'\) "Glossary Entry") using a [database user name](javascript:call_link\('abendatabase_user_name_glosry.htm'\) "Glossary Entry"). When the database is accessed using the [database connections](javascript:call_link\('abendatabase_connection_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"), the database user is a [work process](javascript:call_link\('abenwork_process_glosry.htm'\) "Glossary Entry") that implicitly accesses the [database schema](javascript:call_link\('abendatabase_schema_glosry.htm'\) "Glossary Entry") assigned to the user name of the connection.
