DDIC annotation

A DDIC annotation adds metadata to a DDIC object. It can be used, for example, to add technical or semantic properties to a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") or a [DDIC structure](javascript:call_link\('abenddic_structure_glosry.htm'\) "Glossary Entry").
