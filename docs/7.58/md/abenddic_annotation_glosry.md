DDIC annotation

A DDIC annotation adds metadata to a DDIC object. It can be used, for example, to add technical or semantic properties to a [DDIC database table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_db_table_glosry.htm "Glossary Entry") or a [DDIC structure](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_structure_glosry.htm "Glossary Entry").
