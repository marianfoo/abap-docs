DDL

Short for Data Definition Language. A subset of [SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensql_glosry.htm "Glossary Entry"). The statements in the DDL create and delete the objects of a [relational database](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrelational_database_glosry.htm "Glossary Entry"). [ABAP-managed database objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_managed_db_object_glosry.htm "Glossary Entry") are edited using tools and frameworks in [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry"), such as [ABAP Dictionary](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"), [ABAP CDS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_cds_glosry.htm "Glossary Entry"), and [AMDP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenamdp_glosry.htm "Glossary Entry"), where [DDIC DDL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_dcl_glosry.htm "Glossary Entry"), [CDS DDL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_ddl_glosry.htm "Glossary Entry"), or [Native SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennative_sql_glosry.htm "Glossary Entry") can be used. [ABAP SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_sql_glosry.htm "Glossary Entry") does not support DDL.
