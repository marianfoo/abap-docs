developer extensibility

An extensibility approach that allows ABAP developers to create upgrade-stable custom ABAP code in cloud and on-premise environments. It delivers flexibility beyond key user extensibility and can be done directly on the software stack using the [ABAP development tools for Eclipse](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") (ADT) via local [released APIs](javascript:call_link\('abenreleased_api_glosry.htm'\) "Glossary Entry").

One use case for developer extensibility is [RAP extensibility](javascript:call_link\('abenrap_extensibility_glosry.htm'\) "Glossary Entry"). Using [ABAP for Cloud Development](javascript:call_link\('abenabap_for_cloud_dev_glosry.htm'\) "Glossary Entry"), a RAP application can be extended from the database to the service definition directly in ADT with custom ABAP code.
