DDIC domain

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that describes the properties of [DDIC data elements](javascript:call_link\('abendata_element_glosry.htm'\) "Glossary Entry"), such as [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") or value range. A domain can be linked with any number of data elements.
