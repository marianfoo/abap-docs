double-byte code

Representation of a character in two bytes. Was used before the introduction of [Unicode](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenunicode_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry") for representing Asian characters.
