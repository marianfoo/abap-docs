dynamic cache

[HANA tuning object](javascript:call_link\('abenhana_tuning_object_glosry.htm'\) "Glossary Entry") defined using the statement [DEFINE DYNAMIC CACHE](javascript:call_link\('abenddicddl_define_dynamic_cache.htm'\)). A dynamic cache can improve performance when a view is accessed repeatedly. At the same time, a dynamic cache guarantees up-to-date data.

Equivalent to a SAP HANA dynamic result cache, integrated into [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry").
