dynpro event

An [event](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenevent_glosry.htm "Glossary Entry") during [dynpro](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendynpro_glosry.htm "Glossary Entry") processing. Either [PAI](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpai_glosry.htm "Glossary Entry"), [PBO](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpbo_glosry.htm "Glossary Entry"), [POH](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpoh_glosry.htm "Glossary Entry"), or [POV](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpov_glosry.htm "Glossary Entry").
