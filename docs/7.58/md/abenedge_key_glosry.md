edge key

An edge key represents one [edge attribute](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenedge_attr_glosry.htm "Glossary Entry") or a set of edge attributes that must uniquely identify edges in an [edge table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenedge_table_glosry.htm "Glossary Entry").
