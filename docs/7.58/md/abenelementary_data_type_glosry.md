elementary data type

A scalar [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") of fixed or variable length that is neither [structured](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry"), nor a [table type](javascript:call_link\('abentable_type_glosry.htm'\) "Glossary Entry") or a [reference type](javascript:call_link\('abenreference_type_glosry.htm'\) "Glossary Entry").

-   In ABAP, user-defined elementary data types are based on the [built-in ABAP types](javascript:call_link\('abenbuiltin_abap_type_glosry.htm'\) "Glossary Entry").
-   In [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"), user-defined elementary data types are represented as [DDIC data elements](javascript:call_link\('abendata_element_glosry.htm'\) "Glossary Entry") that are based on elementary built-in dictionary types.
-   In [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"), user-defined elementary data types are represented as [CDS simple types](javascript:call_link\('abencds_simple_type_glosry.htm'\) "Glossary Entry") that are based on elementary built-in dictionary types.

An [enumerated type](javascript:call_link\('abenenum_type_glosry.htm'\) "Glossary Entry") is a special kind of an elementary data type that can be defined by users in ABAP and in ABAP CDS.
