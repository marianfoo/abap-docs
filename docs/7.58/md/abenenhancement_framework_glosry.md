ABAP Enhancement Framework

Framework for [enhancing](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). The ABAP Enhancement Framework comprises the [Enhancement Builder](javascript:call_link\('abenenhancement_builder_glosry.htm'\) "Glossary Entry") tool of the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry") and specific [ABAP statements](javascript:call_link\('abenenhancement_framework.htm'\)) for enhancing [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry").
