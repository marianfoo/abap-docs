enhancement spot element invocation

Part of the definition of an explicit [enhancement option](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenenhancement_point_glosry.htm "Glossary Entry"); the calling position of an [enhancement spot element definition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenenhancement_spot_def_glosry.htm "Glossary Entry"). An enhancement spot element definition can be called at multiple positions besides [source code enhancements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensource_code_enhancement_glosry.htm "Glossary Entry").
