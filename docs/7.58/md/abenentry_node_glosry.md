entry node

A [mesh node](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_node_glosry.htm "Glossary Entry") from which the evaluation of a [mesh association](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_association_glosry.htm "Glossary Entry") starts. The entry node is not always the [start node](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstart_node_glosry.htm "Glossary Entry") of the association.
