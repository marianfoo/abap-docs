enumerated value

An enumerated value is an allowed value of an [enumerated variable](javascript:call_link\('abenenumerated_variable_glosry.htm'\) "Glossary Entry") or a [CDS enumerated element](javascript:call_link\('abencds_enum_element_glosry.htm'\) "Glossary Entry"). The possible enumerated values are defined in the definition of the respective [enumerated type](javascript:call_link\('abenenum_type_glosry.htm'\) "Glossary Entry") as the values of [enumerated constants](javascript:call_link\('abenenumerated_constant_glosry.htm'\) "Glossary Entry") or of components of an [enumerated structure](javascript:call_link\('abenenumerated_structure_glosry.htm'\) "Glossary Entry"). The technical data type of an enumerated value is the [base type](javascript:call_link\('abenbase_type_glosry.htm'\) "Glossary Entry") of the enumerated type.
