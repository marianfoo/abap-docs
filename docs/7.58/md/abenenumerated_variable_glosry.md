enumerated variable

[Variable](javascript:call_link\('abenvariable_glosry.htm'\) "Glossary Entry") with an [enumerated type](javascript:call_link\('abenenum_type_glosry.htm'\) "Glossary Entry"). An enumerated variable is a variable [enumerated object](javascript:call_link\('abenenumerated_object_glosry.htm'\) "Glossary Entry") and can only contain the corresponding [enumerated values](javascript:call_link\('abenenumerated_value_glosry.htm'\) "Glossary Entry").
