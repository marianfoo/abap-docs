EPP connection ID

Component of an [extended passport (EPP)](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenepp_glosry.htm "Glossary Entry"). ID of a connection that is created by calls from [ABAP sessions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_session_glosry.htm "Glossary Entry") or external connections in ABAP sessions.
