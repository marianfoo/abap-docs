event

An event can trigger the processing of a [processing block](javascript:call_link\('abenprocessing_block_glosry.htm'\) "Glossary Entry"), without actually knowing the processing block.

Either a component of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") or an [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry") declared using [*\[*CLASS-*\]*](javascript:call_link\('abapclass-events.htm'\))[EVENTS](javascript:call_link\('abapevents.htm'\)), or an event of the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry"). Events in classes trigger [event handlers](javascript:call_link\('abenevent_handler_glosry.htm'\) "Glossary Entry").

Events of the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") trigger the processing of [event blocks](javascript:call_link\('abenevent_block_glosry.htm'\) "Glossary Entry").
