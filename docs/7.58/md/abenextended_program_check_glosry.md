extended program check

Complete check of the [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") for all statically recognizable errors. Programs with errors in the extended program check can be executed, but usually result in an [exception](javascript:call_link\('abenexception_glosry.htm'\) "Glossary Entry"). Called in the [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry") via Program → Check → Extended Program Check or via the [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SLIN or using the [ABAP Test Cockpit](javascript:call_link\('abenabap_test_cockpit_glosry.htm'\) "Glossary Entry"). Settings for the extended program check can be made in the administration transaction SLIN\_ADMIN.
