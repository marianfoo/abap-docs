fixed point number

Number with a fixed number of [decimal places](javascript:call_link\('abendecimal_place_glosry.htm'\) "Glossary Entry"). In ABAP, fixed point numbers are implemented using [packed numbers](javascript:call_link\('abenpacked_number_glosry.htm'\) "Glossary Entry") where a sequence of digits contains a decimal separator at a position defined by a [packed number type](javascript:call_link\('abenpacked_number_type_glosry.htm'\) "Glossary Entry").
