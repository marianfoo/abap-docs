function

A function calculates a [return value](javascript:call_link\('abenreturn_value_glosry.htm'\) "Glossary Entry") from one or more arguments. In ABAP, functions are either predefined as [built-in functions](javascript:call_link\('abenbuiltin_function_glosry.htm'\) "Glossary Entry") or can be defined as [functional methods](javascript:call_link\('abenfunctional_method_glosry.htm'\) "Glossary Entry").
