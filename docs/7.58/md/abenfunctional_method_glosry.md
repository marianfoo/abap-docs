functional method

[Method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") with exactly one [return value](javascript:call_link\('abenreturn_value_glosry.htm'\) "Glossary Entry") declared using [RETURNING](javascript:call_link\('abapmethods_functional.htm'\)) and any number of [formal parameters](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry"). Can be used in [functional method calls](javascript:call_link\('abenfunctional_method_call_glosry.htm'\) "Glossary Entry").
