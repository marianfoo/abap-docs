global temporary table

GTT for short. [Table category](javascript:call_link\('abenddic_database_tables_tab_cat.htm'\)) of a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). A GTT is used to store temporary data that is deleted at the end of each [database LUW](javascript:call_link\('abendatabase_luw_glosry.htm'\) "Glossary Entry"). Only [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") should be used to access global temporary tables defined in the ABAP Dictionary.
