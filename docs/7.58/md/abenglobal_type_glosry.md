global type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") or [object type](javascript:call_link\('abenobject_type_glosry.htm'\) "Glossary Entry") that is visible as a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") in the entire [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Global data types are defined as [DDIC types](javascript:call_link\('abenddic_type_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") or as [CDS types](javascript:call_link\('abencds_type_glosry.htm'\) "Glossary Entry") in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry"). Global object types are defined as [global classes](javascript:call_link\('abenglobal_class_glosry.htm'\) "Glossary Entry") or as [global interfaces](javascript:call_link\('abenglobal_interface_glosry.htm'\) "Glossary Entry") in the [class library](javascript:call_link\('abenclass_library_glosry.htm'\) "Glossary Entry"). All these global types share the same namespace. Besides these global types, data types defined in the public [visibility section](javascript:call_link\('abenvisibility_section_glosry.htm'\) "Glossary Entry") of global classes or in global interfaces can also be used globally.
