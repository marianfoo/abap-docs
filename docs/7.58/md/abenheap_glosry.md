heap

Also referred to as free or dynamic memory. Part of the [program memory](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenroll_area_glosry.htm "Glossary Entry"), where [anonymous data objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenanonymous_data_object_glosry.htm "Glossary Entry") created dynamically using [CREATE DATA](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcreate_data.htm), [CREATE OBJECT](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcreate_object.htm), or [NEW](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconstructor_expression_new.htm) or [instances](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninstance_glosry.htm "Glossary Entry") of [classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_glosry.htm "Glossary Entry") are stored. The occupied memory can be released again by the [garbage collector](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengarbage_collector_glosry.htm "Glossary Entry"). See also [stack](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstack_glosry.htm "Glossary Entry").
