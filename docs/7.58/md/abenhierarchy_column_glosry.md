hierarchy column

Column of the result set when a [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") [query](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") is made to an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry") or to a [hierarchy navigator](javascript:call_link\('abenhierarchy_navigator_glosry.htm'\) "Glossary Entry") containing a [hierarchy attribute](javascript:call_link\('abenhierarchy_attribute_glosry.htm'\) "Glossary Entry"). A hierarchy column must first be specified explicitly in the [SELECT](javascript:call_link\('abapselect_list.htm'\)) list before it can be included in the result set.
