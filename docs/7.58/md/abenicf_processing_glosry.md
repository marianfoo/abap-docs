ICF processing

Processing of an HTTP/HTTPS query or SMTP query using [ICF](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenicf_glosry.htm "Glossary Entry") in an appropriate handler class and in a separate [ICF session](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenicf_session_glosry.htm "Glossary Entry").
