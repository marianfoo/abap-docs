Information Access protocol

[InA](javascript:call_link\('abenina_glosry.htm'\) "Glossary Entry") for short. InA is a standardized protocol for defining and consuming multidimensional analytical queries. InA can be specified as a type of [service binding](javascript:call_link\('abenservice_binding_glosry.htm'\) "Glossary Entry") and it is interpreted by the [ABAP Analytical Engine](javascript:call_link\('abenabap_ae_glosry.htm'\) "Glossary Entry").
