index table

General term for [internal tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninternal_table_glosry.htm "Glossary Entry") that are managed using a [primary table index](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenprimary_table_index_glosry.htm "Glossary Entry") ([standard tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstandard_table_glosry.htm "Glossary Entry") and [sorted tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensorted_table_glosry.htm "Glossary Entry")). A [secondary table key](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensecondary_table_key_glosry.htm "Glossary Entry") can be assigned to enable access using a [hash algorithm](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhash_algorithm_glosry.htm "Glossary Entry"). The corresponding [generic data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [index table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_types_generic.htm).
