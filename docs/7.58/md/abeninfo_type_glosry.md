infotype

Special [structure](javascript:call_link\('abenstructure_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). The statement [INFOTYPES](javascript:call_link\('abapinfotypes.htm'\)) creates special [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") of this structure, which can be processed in a [PROVIDE](javascript:call_link\('abapprovide.htm'\)) [loop](javascript:call_link\('abenloop_glosry.htm'\) "Glossary Entry").
