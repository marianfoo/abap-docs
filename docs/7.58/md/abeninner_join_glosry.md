inner join

A [join type](javascript:call_link\('abenjoin_type_glosry.htm'\) "Glossary Entry") for which each row of the result set of the [join](javascript:call_link\('abenjoin_glosry.htm'\) "Glossary Entry") fulfills the [join conditions](javascript:call_link\('abenjoin_condition_glosry.htm'\) "Glossary Entry").
