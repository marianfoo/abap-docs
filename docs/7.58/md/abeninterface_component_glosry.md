interface component

[Component](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencomponent_glosry.htm "Glossary Entry") of an [interface](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenoo_intf_glosry.htm "Glossary Entry"). Declared in the [declaration part](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendeclaration_part_glosry.htm "Glossary Entry") of an interface. Possible interface components are: [Attributes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenattribute_glosry.htm "Glossary Entry"), [methods](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmethod_glosry.htm "Glossary Entry"), and [events](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenevent_glosry.htm "Glossary Entry").
