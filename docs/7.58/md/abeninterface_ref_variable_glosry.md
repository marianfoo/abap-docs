interface reference variable

[Object reference variable](javascript:call_link\('abenobject_refer_variable_glosry.htm'\) "Glossary Entry") of the [static type](javascript:call_link\('abenstatic_type_glosry.htm'\) "Glossary Entry") of an [interface](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry"). An interface reference variable can be used to access the visible [interface components](javascript:call_link\('abeninterface_component_glosry.htm'\) "Glossary Entry") of an object that exists in the corresponding interface. This applies to both dynamic and static access.
