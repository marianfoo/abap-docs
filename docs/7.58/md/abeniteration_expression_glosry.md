iteration expression

Expression that executes an [iteration](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeniteration_glosry.htm "Glossary Entry"). An iteration expression is a subexpression of a [constructor expression](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconstructor_expression_glosry.htm "Glossary Entry") introduced using the [iteration operator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeniteration_operator_glosry.htm "Glossary Entry") [FOR](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfor.htm) and executes [conditional iterations](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfor_conditional.htm) or [table iterations](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentable_iteration_glosry.htm "Glossary Entry").
