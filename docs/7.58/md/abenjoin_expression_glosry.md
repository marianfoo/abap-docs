join expression

The implementation of a [join](javascript:call_link\('abenjoin_glosry.htm'\) "Glossary Entry") in [SQL](javascript:call_link\('abensql_glosry.htm'\) "Glossary Entry") using the addition JOIN. Join expressions can be defined in [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") defined by the [SELECT](javascript:call_link\('abapselect.htm'\)) statement of [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), the [SELECT statement in CDS view entities](javascript:call_link\('abencds_select_statement_v2.htm'\)), or the [SELECT statement in CDS DDIC-based views (obsolete)](javascript:call_link\('abencds_select_statement_v2.htm'\)).
