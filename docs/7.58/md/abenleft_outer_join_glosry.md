left outer join

[Join type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenjoin_type_glosry.htm "Glossary Entry") that determines the same rows as an [inner join](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninner_join_glosry.htm "Glossary Entry") and an additional row for each row of the left [SQL data source](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensql_data_source_glosry.htm "Glossary Entry") that is not included by the [join conditions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenjoin_condition_glosry.htm "Glossary Entry"). In these rows, the columns coming from the right data source contain [null values](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennull_value_glosry.htm "Glossary Entry").
