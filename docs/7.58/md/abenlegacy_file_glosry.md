legacy file

[File](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfile_glosry.htm "Glossary Entry") on a [host computer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhost_computer_glosry.htm "Glossary Entry") of an [AS instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_instance_glosry.htm "Glossary Entry") that was opened using the addition [LEGACY](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapopen_dataset_mode.htm) of the statement [OPEN DATASET](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapopen_dataset_mode.htm). In a legacy file, the [byte order](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_order_glosry.htm "Glossary Entry") and the [code page](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencodepage_glosry.htm "Glossary Entry") for handling the content of the file can be specified.
