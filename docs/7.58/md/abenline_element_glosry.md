line element

Predefined output element in a [list](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclassic_list_glosry.htm "Glossary Entry"). Line elements can be corners, crosses, horizontal and vertical lines, and T sections. Line elements are either output explicitly using the addition [AS LINE](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapwrite_list_elements.htm) of the WRITE statement, or set by the system during automatic [write lines](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenwrite_lines.htm).
