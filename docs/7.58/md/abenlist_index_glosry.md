list index

Identifier of a [list](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclassic_list_glosry.htm "Glossary Entry") in the [list buffer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenlist_buffer_glosry.htm "Glossary Entry"). The value of the list index is 0 for the [basic list](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbasic_list_glosry.htm "Glossary Entry") and 1 to 20 for the [details lists](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendetails_list_glosry.htm "Glossary Entry").
