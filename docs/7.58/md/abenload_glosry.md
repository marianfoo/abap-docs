load

In ABAP, a load is a binary representation of a [repository object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrepository_object_glosry.htm "Glossary Entry"), which is optimized for fast access, in the memory or on the database. Examples of loads are program load that is made up of the [byte code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbytecode_glosry.htm "Glossary Entry") of the existing program or the package load.
