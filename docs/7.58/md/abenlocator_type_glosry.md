locator type

Property of a [locator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenlocator_glosry.htm "Glossary Entry"). Either a binary locator for [BLOBs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenblob_glosry.htm "Glossary Entry") or a character-like locator for [CLOBs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclob_glosry.htm "Glossary Entry").
