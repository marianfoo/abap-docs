logical file name

Platform-independent name that can be assigned to [physical file names](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenphysical_filename_glosry.htm "Glossary Entry") using the transactions FILE and SF01. Logical file names are used for platform-independent programming to [validate file names](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendataset_auth_self.htm).
