MDMP system

Short for multidisplay, multiprocessing system. An MDMP system contains multiple non- [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") [system code pages](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry"). The environment [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry") of an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") depends on its current [text environment](javascript:call_link\('abentext_environment_glosry.htm'\) "Glossary Entry"). MDMP systems are no longer supported from [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") 7.0.
