mesh

Instance of a [mesh type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_type_glosry.htm "Glossary Entry"). Special [structure](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstructure_glosry.htm "Glossary Entry") whose [components](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstructure_component_glosry.htm "Glossary Entry") described as [mesh nodes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_node_glosry.htm "Glossary Entry") are exclusively [internal tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninternal_table_glosry.htm "Glossary Entry") that can be joined using [mesh associations](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_association_glosry.htm "Glossary Entry").
