named data object

[Data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") that can be identified via a name. Named data objects are [constants](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconstant_glosry.htm "Glossary Entry"), [variables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenvariable_glosry.htm "Glossary Entry"), and [text symbols](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentext_symbol_glosry.htm "Glossary Entry") and can be stored in the [stack](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstack_glosry.htm "Glossary Entry") or as [instance attributes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninstance_attribute_glosry.htm "Glossary Entry") in the [heap](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenheap_glosry.htm "Glossary Entry"). See also [unnamed data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenunnamed_data_object_glosry.htm "Glossary Entry").
