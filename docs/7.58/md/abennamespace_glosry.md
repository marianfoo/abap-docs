namespace

A namespace is a set of unique names that identify a set of objects. In ABAP, we can distinguish [namespaces for program-internal objects](javascript:call_link\('abennaming_conventions.htm'\)) and a namespace for [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). The latter can be divided into a [SAP namespace](javascript:call_link\('abensap_namespace_glosry.htm'\) "Glossary Entry"), a [customer namespace](javascript:call_link\('abencustomer_namespace_glosry.htm'\) "Glossary Entry"), a [partner namespace](javascript:call_link\('abenpartner_namespace_glosry.htm'\) "Glossary Entry") and [prefix namespaces](javascript:call_link\('abenprefix_name_space_glosry.htm'\) "Glossary Entry").
