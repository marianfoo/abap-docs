Unicode in ABAP Release 6.10

ABAP release 6.10 is the first [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") to support Unicode. Extensive changes were made here as part of modifications made to the ABAP programming language.
