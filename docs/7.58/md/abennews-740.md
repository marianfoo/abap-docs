News for ABAP Release 7.40 and its SPs

[ABAP release](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_release_glosry.htm "Glossary Entry") 7.40 is based on [7.0, EhP3 or 7.3, EhP1](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennews-703.htm). Therefore, apart from a few enhancements, this release is based almost in full on 7.0, [EhP2](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abennews-71.htm). The changes in ABAP release 7.40 are delivered in support packages that are coupled with kernel releases.
