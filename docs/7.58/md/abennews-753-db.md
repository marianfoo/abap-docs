Standard Database in ABAP Release 7.53

Modification

Only SAP HANA Database as Standard Database

As of this release, only an [SAP HANA database](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhana_database_glosry.htm "Glossary Entry") is supported as the [standard database](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstandard_db_glosry.htm "Glossary Entry") of an [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry"). For this reason, Open SQL was renamed to ABAP SQL (see [ABAP SQL in ABAP Release 7.53](abennews-753-abap_sql.htm#!ABAP_MODIFICATION_1@1@)). Note: Using a [secondary connection](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensecondary_db_connection_glosry.htm "Glossary Entry") in [ABAP SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_sql_glosry.htm "Glossary Entry"), it is still possible to connect to other databases.
