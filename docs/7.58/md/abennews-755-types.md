Data Types in ABAP Release 7.55

Modification

Indicator Structures

The new addition [INDICATORS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abaptypes_indicators.htm) of the statement [TYPES](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abaptypes.htm) defines an [indicator structure](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenindicator_structure_glosry.htm "Glossary Entry") as a [substructure](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensubstructure_glosry.htm "Glossary Entry") of a given [structured type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstructured_type_glosry.htm "Glossary Entry"). An indicator structure can be used as a [ABAP SQL indicator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_sql_indicator_glosry.htm "Glossary Entry") in ABAP SQL read and write statements.
