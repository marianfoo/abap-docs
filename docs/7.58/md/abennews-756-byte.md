Byte String Processing in ABAP Release 7.56

Modification

Writable Expression in SET BIT

For the operand byte\_string of statement [SET BIT](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapset_bit.htm), a [writable expression](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenwritable_expression_glosry.htm "Glossary Entry") can be specified now.
