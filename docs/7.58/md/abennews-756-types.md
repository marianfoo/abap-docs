Data Types in ABAP Release 7.56

Modification

Condensed Indicator Structures

The new addition [AS BITFIELD](abaptypes_indicators.htm#!ABAP_ADDITION_2@2@) behind addition [INDICATORS](javascript:call_link\('abaptypes_indicators.htm'\)) of the statement [TYPES](javascript:call_link\('abaptypes.htm'\)) allows a [condensed indicator structure](javascript:call_link\('abencondensed_ind_structure_glosry.htm'\) "Glossary Entry") to be defined as a [byte field type](javascript:call_link\('abenbyte_field_type_glosry.htm'\) "Glossary Entry") component of a given [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry"). An condensed indicator structure can be used as a [null indicator](javascript:call_link\('abennull_indicator_glosry.htm'\) "Glossary Entry") in ABAP SQL statements.
