Modification 3   

IF ... THEN ... ELSE ... Control Structure Available in Access Conditions

Control structures [IF ... THEN ... ELSE ...](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_f1_dcl_ifthenelse.htm) can now be used as part of an [access condition](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenaccess_condition_glosry.htm "Glossary Entry") after the DEFINE ROLE statement.

Modification 4   

DEFINE ROLE Statement Has New Addition SWITCHABLE

The new addition [SWITCHABLE](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_f1_define_role.htm) is now available for the [DEFINE ROLE](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_f1_define_role.htm) statement. Using this addition, the content of the role reacts to status changes in the Switch Framework with respect to the package of the access control.
