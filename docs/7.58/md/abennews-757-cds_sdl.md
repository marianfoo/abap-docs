CDS SDL in ABAP Release 7.57

[1\. CDS Service Definition Extension](#!ABAP_MODIFICATION_1@1@)
[2\. CDS Service Definition, Provider Contract](#!ABAP_MODIFICATION_2@2@)

Modification 1   

CDS Service Definition Extension

It is now possible to define [service definition extensions](javascript:call_link\('abencds_service_def_ext_glosry.htm'\) "Glossary Entry") in [CDS SDL](javascript:call_link\('abencds_sdl_glosry.htm'\) "Glossary Entry") using the statement [EXTEND SERVICE](javascript:call_link\('abensrvd_extend_service.htm'\)).

Modification 2   

CDS Service Definition, Provider Contract

The new statement [PROVIDER CONTRACTS](javascript:call_link\('abensrvd_provider_contract.htm'\)) is now available for [CDS service definitions](javascript:call_link\('abencds_service_definition_glosry.htm'\) "Glossary Entry"). A provider contract defines the type of [service binding](javascript:call_link\('abenservice_binding_glosry.htm'\) "Glossary Entry") that is to be used for a service definition. The effect is that stricter syntax checks are applied.
