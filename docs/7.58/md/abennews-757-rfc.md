Remote Function Call in ABAP Release 7.57

Modification

Parameter Tables in sRFC

Using the additions [PARAMETER-TABLE](abapcall_function_destination_para.htm#!ABAP_ADDITION_6@6@) and [EXCEPTION-TABLE](abapcall_function_destination_para.htm#!ABAP_ADDITION_6@6@), values for parameters and non-class-based exceptions can now be passed dynamically in [sRFC](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensrfc_glosry.htm "Glossary Entry").
