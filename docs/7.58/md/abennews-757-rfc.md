Remote Function Call in ABAP Release 7.57

Modification

Parameter Tables in sRFC

Using the additions [PARAMETER-TABLE](abapcall_function_destination_para.htm#!ABAP_ADDITION_6@6@) and [EXCEPTION-TABLE](abapcall_function_destination_para.htm#!ABAP_ADDITION_6@6@), values for parameters and non-class-based exceptions can now be passed dynamically in [sRFC](javascript:call_link\('abensrfc_glosry.htm'\) "Glossary Entry").
