Data Types in ABAP Release 7.58

Modification

Absolute Type Names for Line Types

The new specification \\LINE allows the line type of an internal table to be specified in an [absolute type name](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentype_names.htm). \\LINE can be followed by \-comp to specify the type of a component of the line type.
