Remote Function Call in ABAP Release 7.56

Modification

Predefined Destination SELF

The new predefined [RFC destination](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrfc_destination.htm) SELF works as NONE but uses the [RFC protocol](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrfc_protocol.htm) fast serialization.
