node cycle

A node cycle occurs in an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry") for a [child node](javascript:call_link\('abenchild_node_glosry.htm'\) "Glossary Entry") that is also one of the preceding [parent nodes](javascript:call_link\('abenparent_node_glosry.htm'\) "Glossary Entry") or its own parent node. For a child node of this type, the [hierarchy column](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") [HIERARCHY\_IS\_CYCLE](javascript:call_link\('abenddddl_hierarchy.htm'\)) contains the value 1.
