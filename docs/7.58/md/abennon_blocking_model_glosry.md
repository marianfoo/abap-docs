non-blocking mode

Any statements that block the program flow are not allowed in an [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that is processed in non-blocking mode. There are values of the non-blocking mode for [APC](javascript:call_link\('abenapc_non_blocking_model.htm'\)) and [ADF](javascript:call_link\('abenadf_non_blocking_model.htm'\)).
