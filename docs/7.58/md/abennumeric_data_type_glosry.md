numeric data type

[Data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") whose [data objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") contain numeric values in platform-dependent encoding. These data types are [integer type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninteger_type_glosry.htm "Glossary Entry"), [packed number type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpacked_number_type_glosry.htm "Glossary Entry"), [decimal floating pointing number type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendecfloat_type_glosry.htm "Glossary Entry") and [binary floating pointing number type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbinfloat_type_glosry.htm "Glossary Entry"). The corresponding [generic data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [numeric](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_types_generic.htm).
