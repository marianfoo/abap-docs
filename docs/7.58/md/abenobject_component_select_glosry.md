object component selector

Character \->. Also referred to as [instance component selector](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninstance_comp_selector_glosry.htm "Glossary Entry"). Each visible [component](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencomponent_glosry.htm "Glossary Entry") comp of a [data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") or an [instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninstance_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_glosry.htm "Glossary Entry") can be addressed using ref->comp if ref is a [reference variable](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenreference_variable_glosry.htm "Glossary Entry") that refers to the object.
