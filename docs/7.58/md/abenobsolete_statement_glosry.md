obsolete language element

[ABAP language element](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_language_element_glosry.htm "Glossary Entry") that still exists in the language for reasons of downward compatibility, but should no longer be used. Most obsolete language elements are allowed only outside [classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_glosry.htm "Glossary Entry").
