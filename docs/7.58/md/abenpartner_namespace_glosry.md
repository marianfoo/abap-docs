partner namespace

Predefined [namespace](javascript:call_link\('abennamespace_glosry.htm'\) "Glossary Entry") for [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") created in partner development systems. Names have J as the first character. These names can be used in partner systems, if no [prefix namespace](javascript:call_link\('abenprefix_name_space_glosry.htm'\) "Glossary Entry") is available.
