Perl Compatible Regular Expression

PCRE for short. [Regular expressions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenregular_expression_glosry.htm "Glossary Entry") using the same syntax and semantics as [Perl 5](https://www.perl.org/). In general, ABAP supports [PCRE syntax](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenregex_pcre_syntax.htm) for regular expressions by implementing the [PCRE2 library](https://www.pcre.org/) in the [ABAP kernel](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenkernel_glosry.htm "Glossary Entry"). [ABAP SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_sql_glosry.htm "Glossary Entry") supports PCRE by using the PCRE1 library implemented in the [SAP HANA database](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhana_database_glosry.htm "Glossary Entry").
