persistent class

Special [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") whose [attributes](javascript:call_link\('abenattribute_glosry.htm'\) "Glossary Entry") are linked to [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") via object-relational mapping. Can be created using the Mapping Assistant in [Class Builder](javascript:call_link\('abenclass_builder_glosry.htm'\) "Glossary Entry"). The [objects](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of persistent classes are managed by [object services](javascript:call_link\('abenobject_services_glosry.htm'\) "Glossary Entry").
