persistent object

[Object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenobject_glosry.htm "Glossary Entry") in a [persistent class](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpersistent_class_glosry.htm "Glossary Entry") whose [attributes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenattribute_glosry.htm "Glossary Entry") are saved as database content after the runtime of an [ABAP program](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry"). Persistent objects are managed by [object services](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenobject_services_glosry.htm "Glossary Entry").
