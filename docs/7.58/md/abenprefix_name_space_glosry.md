prefix namespace

[Namespace](javascript:call_link\('abennamespace_glosry.htm'\) "Glossary Entry") for [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") of SAP departments, SAP partners, and SAP customers that can be requested from SAP. The names of the repository objects of a prefix namespace are introduced with a [namespace prefix](javascript:call_link\('abenname_space_prefix_glosry.htm'\) "Glossary Entry").
