processing block

Indivisible and non-nestable modularization unit of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"). Possible processing blocks are [procedures](javascript:call_link\('abenprocedure_glosry.htm'\) "Glossary Entry"), [dialog modules](javascript:call_link\('abendialog_module_glosry.htm'\) "Glossary Entry"), and [event blocks](javascript:call_link\('abenevent_block_glosry.htm'\) "Glossary Entry"). They contain statements that are structured into [statement blocks](javascript:call_link\('abenstatement_block_glosry.htm'\) "Glossary Entry") by [control structures](javascript:call_link\('abencontrol_structure_glosry.htm'\) "Glossary Entry"). Each non-declarative statement in an ABAP program is part of a processing block.
