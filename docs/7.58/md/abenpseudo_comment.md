Pseudo Comments

Pseudo comments are [program directives](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenprogram_directive_glosry.htm "Glossary Entry") for managing checks and test runs. Pseudo comments have mostly become [obsolete](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpseudo_comments_obsolete.htm) and have been replaced by [pragmas](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpragma.htm) or real additions.

Non-Obsolete Pseudo Comments   

-   [Pseudo Comments for Code Inspector](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpseudo_comment_ci.htm)

Obsolete Pseudo Comments   

-   [Pseudo Comments for the Extended Program Check](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpseudo_comment_slin.htm)
-   [Pseudo Comments for Test Classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenpseudo_comment_test_class.htm)

Hint

Only one pseudo comment can be specified per program line. To specify multiple pseudo comments for a statement, they must be split into multiple lines.
