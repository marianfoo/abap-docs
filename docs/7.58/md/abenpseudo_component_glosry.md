pseudo component

Expression table\_line that can be specified in accesses to [internal tables](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") instead of a [structure component](javascript:call_link\('abenstructure_component_glosry.htm'\) "Glossary Entry"). The entire line of the internal table is then interpreted as the only component. The name table\_line is a reserved name. In ABAP and in [ABAP Dictionary](javascript:call_link\('abenabap_dictionary.htm'\)), no structure components called table\_line can be defined or created.
