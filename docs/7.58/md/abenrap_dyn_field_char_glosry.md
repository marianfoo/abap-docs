RAP dynamic field characteristic

[Field characteristic](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_field_char_glosry.htm "Glossary Entry") that defines usage restrictions for a [RAP BO field](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_field_glosry.htm "Glossary Entry") depending on the state of a [RAP BO entity instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_entity_inst_glosry.htm "Glossary Entry") or the [operation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_operation_glosry.htm "Glossary Entry") in which the field is used. For example, a field can be dynamically defined as [read-only](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenread_only_rap_bo_field_glosry.htm "Glossary Entry"), [mandatory](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmandatory_rap_bo_field_glosry.htm "Glossary Entry"), or without restrictions.
