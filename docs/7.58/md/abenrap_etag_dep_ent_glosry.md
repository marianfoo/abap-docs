RAP ETag dependent entity

[RAP BO entity](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_entity_glosry.htm "Glossary Entry") that uses the [ETag](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_etag_glosry.htm "Glossary Entry") field of another entity for [optimistic concurrency control](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenoptimistic_conc_control_glosry.htm "Glossary Entry"). Defined using the keyword [etag dependent by \_Assoc](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbdl_etag.htm).
