RAP runtime engine

Processor that handles the consumption of [RAP queries](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_query_glosry.htm "Glossary Entry") and [RAP BOs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_glosry.htm "Glossary Entry"). There is the [RAP query engine](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_query_engine_glosry.htm "Glossary Entry") for non-transactional consumption and the [RAP transactional engine](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_transac_engine_glosry.htm "Glossary Entry") for transactional consumption.
