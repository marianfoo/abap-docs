static RAP BO operation

A [RAP BO operation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_operation_glosry.htm "Glossary Entry") that is not bound to any [RAP BO entity instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_entity_inst_glosry.htm "Glossary Entry") but relates to the complete [RAP BO entity](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_entity_glosry.htm "Glossary Entry"). Defined using the optional addition [static](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbdl_static.htm).
