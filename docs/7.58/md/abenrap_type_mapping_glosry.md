RAP type mapping

Mapping between data types of the data model that is defined in CDS for a [RAP business object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_bo_glosry.htm "Glossary Entry") to other data types that do not match a [BDEF derived type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrap_derived_type_glosry.htm "Glossary Entry") component pattern.

For example, it is used in the behavior definition to map database field names to CDS element names.
