reference variable

[Data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") that contains a [reference](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenreference_glosry.htm "Glossary Entry"). Reference variables are either [data reference variables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_reference_variable_glosry.htm "Glossary Entry") or [object reference variables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenobject_refer_variable_glosry.htm "Glossary Entry"). Reference variables have a length of 8 bytes and are opaque, which means the reference cannot be accessed directly. They are typed with the addition [REF TO](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abaptypes_references.htm) that defines their [static type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstatic_type_glosry.htm "Glossary Entry"). The static type is always more general than or the same as the [dynamic type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendynamic_type_glosry.htm "Glossary Entry"). For assignments between reference variables, [reference semantics](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenreference_semantics_glosry.htm "Glossary Entry") apply.
