report

The process of reading and displaying data, for example in a [list](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclassic_list_glosry.htm "Glossary Entry"). Outdated term for an [executable program](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexecutable_program_glosry.htm "Glossary Entry") that only executes [reporting](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenreporting_glosry.htm "Glossary Entry").
