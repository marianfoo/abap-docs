return code

Numeric value that is placed in the [system field](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_field_glosry.htm "Glossary Entry") [sy-subrc](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_fields.htm) for many [ABAP statements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_statement_glosry.htm "Glossary Entry") when the statement is executed. The value 0 usually indicates a successful completion.
