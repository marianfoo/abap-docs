return code

Numeric value that is placed in the [system field](javascript:call_link\('abensystem_field_glosry.htm'\) "Glossary Entry") [sy-subrc](javascript:call_link\('abensystem_fields.htm'\)) for many [ABAP statements](javascript:call_link\('abenabap_statement_glosry.htm'\) "Glossary Entry") when the statement is executed. The value 0 usually indicates a successful completion.
