return value

Result of [functional methods](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfunctional_method_glosry.htm "Glossary Entry") and [built-in functions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuiltin_function_glosry.htm "Glossary Entry"). The result of functional methods is the [formal parameter](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenformal_parameter_glosry.htm "Glossary Entry") defined by the addition [RETURNING](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapmethods_functional.htm) of the statement [*\[*CLASS-*\]*](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapclass-methods.htm)[METHODS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapmethods.htm).
