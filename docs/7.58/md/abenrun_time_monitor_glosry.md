runtime monitor

Tool for recording events raised by [system classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_class_glosry.htm "Glossary Entry") when a program is executed. Called using [transaction code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentransaction_code_glosry.htm "Glossary Entry") SRTM.
