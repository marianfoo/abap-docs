Runtime Type Creation

RTTC for short. Creation of [data types](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") during program runtime. Implemented using methods of the [type description classes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentype_class_glosry.htm "Glossary Entry") of [RTTS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrun_time_type_services_glosry.htm "Glossary Entry").
