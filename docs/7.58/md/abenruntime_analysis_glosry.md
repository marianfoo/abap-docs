runtime analysis

Tool for analyzing the execution of program parts or individual statements and for measuring their runtime in SAP GUI. Called using [transaction code](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentransaction_code_glosry.htm "Glossary Entry") SAT. Replaced by the ABAP Profiler in the [ABAP development tools for Eclipse (ADT)](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenadt_glosry.htm "Glossary Entry").
