SAP annotation

[CDS annotation](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") delivered by SAP as a [CDS annotation definition](javascript:call_link\('abencds_anno_definition_glosry.htm'\) "Glossary Entry"). There are the following categories of SAP annotations:

-   [ABAP annotations](javascript:call_link\('abenabap_annotation_glosry.htm'\) "Glossary Entry"), which are evaluated by the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry") for every [CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry").
-   [Framework-specific annotations](javascript:call_link\('abenfrmwrk_annotation_glosry.htm'\) "Glossary Entry"), which are evaluated by the frameworks of other software components.
