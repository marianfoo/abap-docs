SAP namespace

[Namespace](javascript:call_link\('abennamespace_glosry.htm'\) "Glossary Entry") for [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") created in SAP's own development systems. There are no real restrictions for the names. To avoid namespace conflicts [prefix namespace](javascript:call_link\('abenprefix_name_space_glosry.htm'\) "Glossary Entry") can be used. Internally at SAP, all names are checked using a cross-system table to ensure their uniqueness.
