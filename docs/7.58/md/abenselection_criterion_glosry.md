selection criterion

Component of a [selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry") defined using [SELECT-OPTIONS](javascript:call_link\('abapselect-options.htm'\)). Displayed on the selection screen via two [input fields](javascript:call_link\('abeninput_output_field_glosry.htm'\) "Glossary Entry") for interval boundaries and a pushbutton for [multiple selection](javascript:call_link\('abenmultiple_selection_glosry.htm'\) "Glossary Entry"). It is represented internally by a [selection table](javascript:call_link\('abenselection_table_glosry.htm'\) "Glossary Entry").
