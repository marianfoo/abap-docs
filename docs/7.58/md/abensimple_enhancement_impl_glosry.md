simple enhancement implementation

Semantically interrelated enhancement of a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). A simple enhancement implementation manages one or more [enhancement implementation elements](javascript:call_link\('abenenhancement_impl_elem_glosry.htm'\) "Glossary Entry"). A simple enhancement implementation can occur independently or as a component of a [composite enhancement implementation](javascript:call_link\('abencomp_enhancement_impl_glosry.htm'\) "Glossary Entry").
