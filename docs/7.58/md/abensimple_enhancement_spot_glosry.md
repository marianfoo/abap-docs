simple enhancement spot

Reference to a set of [enhancement spot element definitions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenenhancement_spot_def_glosry.htm "Glossary Entry"). A simple enhancement spot can occur independently or as a component of a [composite enhancement spot](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencomp_enhancement_spot_glosry.htm "Glossary Entry").
