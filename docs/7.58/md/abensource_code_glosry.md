source code

Textual representation of programs or other [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). For example, [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") for [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry"), [CDS source code](javascript:call_link\('abencds_source_code_glosry.htm'\) "Glossary Entry") for [CDS objects](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry"), or [transformation source code](javascript:call_link\('abentrafo_source_code_glosry.htm'\) "Glossary Entry") for [XSLT programs](javascript:call_link\('abenxslt_program_glosry.htm'\) "Glossary Entry") and [Simple Transformations](javascript:call_link\('abensimple_transformation_glosry.htm'\) "Glossary Entry").
