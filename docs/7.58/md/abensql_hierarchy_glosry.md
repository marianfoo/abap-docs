SQL hierarchy

Tabular [SQL data source](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensql_data_source_glosry.htm "Glossary Entry") in [ABAP SQL](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_sql_glosry.htm "Glossary Entry") whose rows form the [hierarchy nodes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") of a [hierarchy](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_glosry.htm "Glossary Entry"). Special [hierarchy columns](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_column_glosry.htm "Glossary Entry") contain [hierarchy attributes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_attribute_glosry.htm "Glossary Entry") of the individual hierarchy nodes in the SQL hierarchy. SQL hierarchies are either created using a [hierarchy generator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_generator_glosry.htm "Glossary Entry") or an existing [CDS hierarchy](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencds_hierarchy_glosry.htm "Glossary Entry") or a [CTE hierarchy](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencte_hierarchy_glosry.htm "Glossary Entry") is used. SQL hierarchies can be used in the [FROM](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapfrom_clause.htm) clause and as the source of special [hierarchy navigators](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_navigator_glosry.htm "Glossary Entry").
