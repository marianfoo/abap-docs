SQL trace

Trace function used to analyze the [SQL](javascript:call_link\('abensql_glosry.htm'\) "Glossary Entry") statements passed from the [database interface](javascript:call_link\('abendatabase_interface_glosry.htm'\) "Glossary Entry") to the [database system](javascript:call_link\('abendatabase_interface_glosry.htm'\) "Glossary Entry"). It can be called from the tools [ABAP Debugger](javascript:call_link\('abenabap_debugger_glosry.htm'\) "Glossary Entry") or [performance trace](javascript:call_link\('abenperformance_trace_glosry.htm'\) "Glossary Entry") (transaction ST05).
