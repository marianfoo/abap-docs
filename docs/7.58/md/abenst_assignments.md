Hint

When values are assigned to the nodes of [data roots](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenst_tt_root.htm), it must be noted that the content of a data root cannot be modified by serialization. Only writes can be performed on a data root in deserializations.
