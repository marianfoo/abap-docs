standard database

Central database of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). A [standard connection](javascript:call_link\('abenstandard_db_connection_glosry.htm'\) "Glossary Entry") to the standard database always exists for each [work process](javascript:call_link\('abenwork_process_glosry.htm'\) "Glossary Entry") of an AS ABAP. Only a [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") is supported as a standard database.
