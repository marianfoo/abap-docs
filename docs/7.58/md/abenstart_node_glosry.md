start node

-   In a [SQL hierarchy](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensql_hierarchy_glosry.htm "Glossary Entry"), a [hierarchy node](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_node_glosry.htm "Glossary Entry") from which a [hierarchy node navigator](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhierarchy_node_navi_glosry.htm "Glossary Entry") determines further hierarchy nodes and adds them to its result set.
-   In a [mesh](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_glosry.htm "Glossary Entry"), a [mesh node](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_node_glosry.htm "Glossary Entry") for which a [mesh association](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_association_glosry.htm "Glossary Entry") is defined is the association's start node.
