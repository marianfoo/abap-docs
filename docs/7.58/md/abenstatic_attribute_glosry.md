static attribute

[Attribute](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenattribute_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclass_glosry.htm "Glossary Entry") declared using [CLASS-DATA](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapclass-data.htm) that is valid independent of the class [instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninstance_glosry.htm "Glossary Entry"). The content of the static attributes determines the instance-dependent state of a class.
