static component

Superordinate term for [static attribute](javascript:call_link\('abenstatic_attribute_glosry.htm'\) "Glossary Entry"), [static event](javascript:call_link\('abenstatic_event_glosry.htm'\) "Glossary Entry"), and [static method](javascript:call_link\('abenstatic_method_glosry.htm'\) "Glossary Entry"). The static components of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") can be addressed using the name of the class. A class that only contains static components is referred to as a [static class](javascript:call_link\('abenstatic_class_glosry.htm'\) "Glossary Entry").
