static cache

A static cache intends to improve performance when a view is repeatedly accessed. Called [Result Cache](https://help.sap.com/docs/SAP_HANA_PLATFORM/bed8c14f9f024763b0777aa72b5436f6/8ea43b3ff2e541e398e27d7e5caefab7) on SAP HANA. The cache is kept stable for a predefined retention time, and invalidated when this time is over.
