streaming data source

Repository from which a reader [data stream](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_stream_glosry.htm "Glossary Entry") takes its data in [streaming](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstreaming_glosry.htm "Glossary Entry"). A streaming data source can be an external resource such as a [LOB](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenlob_glosry.htm "Glossary Entry") column of a [DDIC database table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_db_table_glosry.htm "Glossary Entry") or a [file](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfile_glosry.htm "Glossary Entry"), or an internal resource such as a [string](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_glosry.htm "Glossary Entry") or an [internal table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninternal_table_glosry.htm "Glossary Entry"). [Streaming](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstreaming_glosry.htm "Glossary Entry") is currently implemented for [DDIC database tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_db_table_glosry.htm "Glossary Entry"), [strings](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_glosry.htm "Glossary Entry"), and [internal tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninternal_table_glosry.htm "Glossary Entry").
