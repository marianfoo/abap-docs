Either [text string literal](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentext_string_literal_glosry.htm "Glossary Entry") or [byte string literal](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbyte_string_literal_glosry.htm "Glossary Entry").
