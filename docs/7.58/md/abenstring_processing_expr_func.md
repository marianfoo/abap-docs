Expressions and Functions for String Processing

Expressions and functions for character string processing can be listed at many [operand positions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexpression_positions.htm).

-   [String expressions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapcompute_string.htm)
-   [String functions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_functions.htm)
-   [Regular expressions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenregular_expressions.htm)

Hint

An advantage of the expressions and functions for string processing over the [statements for character and byte string processing](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenstring_processing_statements.htm) is that expressions can be specified for all operands, which is only possible to a very limited extent in the statements.
