Expressions and Functions for String Processing

Expressions and functions for character string processing can be listed at many [operand positions](javascript:call_link\('abenexpression_positions.htm'\)).

-   [String expressions](javascript:call_link\('abapcompute_string.htm'\))
-   [String functions](javascript:call_link\('abenstring_functions.htm'\))
-   [Regular expressions](javascript:call_link\('abenregular_expressions.htm'\))

Hint

An advantage of the expressions and functions for string processing over the [statements for character and byte string processing](javascript:call_link\('abenstring_processing_statements.htm'\)) is that expressions can be specified for all operands, which is only possible to a very limited extent in the statements.
