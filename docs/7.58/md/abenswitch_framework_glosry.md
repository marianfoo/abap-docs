Switch Framework

SFW for short. Framework for the externally controlling the visibility of [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"), their components, or [enhancements](javascript:call_link\('abenenhancement_glosry.htm'\) "Glossary Entry") using [switches](javascript:call_link\('abenswitch_glosry.htm'\) "Glossary Entry").
