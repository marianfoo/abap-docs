sXML library

Global [interfaces](javascript:call_link\('abenglobal_interface_glosry.htm'\) "Glossary Entry") and [classes](javascript:call_link\('abenglobal_class_glosry.htm'\) "Glossary Entry") for handling [XML](javascript:call_link\('abenxml_glosry.htm'\) "Glossary Entry") data in different [formats](javascript:call_link\('abenabap_sxml_lib_formats.htm'\)). The serial XML Library provides XML readers and XML writers for various sources and targets. The classes and interfaces in sXML Library start with the prefixes CL\_SXML or IF\_SXML.
