SXPG framework

A framework for calling operating system statements of the current [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry") [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") from ABAP programs, based on the [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") SM69 and [function modules](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") in the [function pool](javascript:call_link\('abenfunction_pool_glosry.htm'\) "Glossary Entry") SXPG
