system client

Special [client](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_abap_glosry.htm "Glossary Entry") with the [client ID](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenclient_identifier_glosry.htm "Glossary Entry") 000. The system client is used to execute system-specific [system programs](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abensystem_program_glosry.htm "Glossary Entry").
