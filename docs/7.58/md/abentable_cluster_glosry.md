table cluster

Before [ABAP release](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_release_glosry.htm "Glossary Entry") 7.53, [DDIC database table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_db_table_glosry.htm "Glossary Entry") in the [database](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendatabase_glosry.htm "Glossary Entry") that contains the data of multiple [cluster tables](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencluster_table_glosry.htm "Glossary Entry"). Cluster tables are no longer supported and all table clusters have been removed.
