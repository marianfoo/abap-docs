table cluster

Before [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") 7.53, [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") in the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry") that contains the data of multiple [cluster tables](javascript:call_link\('abencluster_table_glosry.htm'\) "Glossary Entry"). Cluster tables are no longer supported and all table clusters have been removed.
