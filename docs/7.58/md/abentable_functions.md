itab - Table Functions (table\_func)

Table functions are part of the [built-in functions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenbuilt_in_functions.htm). The following table functions are possible:

-   [lines - Line Function](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendescriptive_functions_table.htm)
-   [line\_index - Index Function](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenline_index_function.htm)

Furthermore, the string function [concat\_lines\_of](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenconcatenation_functions.htm) expects an internal table as an argument and the predicate function [line\_exists](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenline_exists_function.htm) expects a [table expression](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentable_expression_glosry.htm "Glossary Entry") or a [mesh path expression](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenmesh_path_expression_glosry.htm "Glossary Entry").

Hint

Table functions can be used in suitable [operand positions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenexpression_positions.htm), for example directly as an operand of [arithmetic](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenarithmetic_expression_glosry.htm "Glossary Entry") or [relational expressions](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenrelational_expression_glosry.htm "Glossary Entry").
