table pool

Before [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") 7.53, [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") in the [database](javascript:call_link\('abendatabase_glosry.htm'\) "Glossary Entry") that contains the data of multiple [pooled tables](javascript:call_link\('abenpooled_table_glosry.htm'\) "Glossary Entry"). Pooled tables are no longer supported and all table pools have been removed.
