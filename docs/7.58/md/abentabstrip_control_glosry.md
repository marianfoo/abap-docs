tabstrip control

Tabstrip controls allow tab pages to be displayed on dynpros. A tabstrip control is a [screen element](javascript:call_link\('abenscreen_element_glosry.htm'\) "Glossary Entry") built from several [tab pages](javascript:call_link\('abentab_page_glosry.htm'\) "Glossary Entry") or tabs.
