target cardinality

The [cardinality](javascript:call_link\('abencardinality_glosry.htm'\) "Glossary Entry") of a [join](javascript:call_link\('abenjoin_glosry.htm'\) "Glossary Entry") or [association](javascript:call_link\('abenassociation_glosry.htm'\) "Glossary Entry") can consist of a [source cardinality](javascript:call_link\('abensource_cardinality_glosry.htm'\) "Glossary Entry") and a target cardinality. The target cardinality describes the number of matching entries for the entries of the original data source. For example, a target cardinality of many means that each record of the original data source has multiple matching entries in the joined data source.
