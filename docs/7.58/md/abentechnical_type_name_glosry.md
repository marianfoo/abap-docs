technical type name

Internal name of a of a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that only exists as a [bound data type](javascript:call_link\('abenbound_data_type_glosry.htm'\) "Glossary Entry") of a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). A technical type name can be part of an [absolute type name](javascript:call_link\('abenabsolute_typename_glosry.htm'\) "Glossary Entry") and retrieved by [RTTI](javascript:call_link\('abenrun_time_type_identific_glosry.htm'\) "Glossary Entry").
