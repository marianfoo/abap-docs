text file

[File](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenfile_glosry.htm "Glossary Entry") on the [host computer](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenhost_computer_glosry.htm "Glossary Entry") of an [AS instance](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenas_instance_glosry.htm "Glossary Entry") that has been opened using the addition [TEXT](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapopen_dataset_mode.htm) of the statement [OPEN DATASET](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapopen_dataset_mode.htm). In a text file, the content is handled as text in the [character representation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenchar_representation_glosry.htm "Glossary Entry") that was chosen when the file was opened.
