text pool

Component of an [ABAP program](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_program_glosry.htm "Glossary Entry") that contains the [text elements](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentext_element_glosry.htm "Glossary Entry") of the program. Multiple text pools can be defined in various languages for a program.
