type description object

Instance of a [type description class](javascript:call_link\('abentype_class_glosry.htm'\) "Glossary Entry"). Type description objects describe a [type](javascript:call_link\('abentype_glosry.htm'\) "Glossary Entry"). Type description objects are created using the static methods of the type description classes. The type description class and possible attributes of type description objects determine the [technical properties](javascript:call_link\('abentechnical_type_prpt_glosry.htm'\) "Glossary Entry") of the type. [References](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") to type description objects can be used, for example, after the addition HANDLE of the statements [CREATE DATA](javascript:call_link\('abapcreate_data_handle.htm'\)) and [ASSIGN](javascript:call_link\('abapassign_casting.htm'\)).
