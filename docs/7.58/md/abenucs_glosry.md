UCS

Short for Universal Character Set. A 4-byte [character set](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencharacter_set_glosry.htm "Glossary Entry"), standardized in ISO-10646, that contains all other character sets. The UCS character set is currently identical to the [Unicode](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenunicode_glosry.htm "Glossary Entry") character set; that is, no other characters than those contained in Unicode are currently defined. [Character representations](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenchar_representation_glosry.htm "Glossary Entry") for the UCS character set are the [UTF](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenutf_glosry.htm "Glossary Entry") representations [UCS-2](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenucs2_glosry.htm "Glossary Entry") and UCS-4.
