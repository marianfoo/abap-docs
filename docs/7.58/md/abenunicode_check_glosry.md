Unicode check

[Syntax check](javascript:call_link\('abensyntax_check_glosry.htm'\) "Glossary Entry") for [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") that are to be executed in a [Unicode system](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry"). The Unicode check is carried out for [ABAP language versions](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") [Standard ABAP](javascript:call_link\('abenstandard_abap_glosry.htm'\) "Glossary Entry") or higher. Only Unicode systems are supported in the current [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") release and therefore no programs of the obsolete ABAP language version [Obsolete ABAP (Non-Unicode)](javascript:call_link\('abennon_unicode_abap_glosry.htm'\) "Glossary Entry") are supported.
