Unicode

Language-independent [character set](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencharacter_set_glosry.htm "Glossary Entry") for international data processing ([www.unicode.org](http://www.unicode.org)). Subset of the [UCS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenucs_glosry.htm "Glossary Entry") character set according to ISO-10646. Unicode can represent 1114111 different characters numbered hexadecimally from U+0000 to U+10FFFF. The Unicode character set is normally accessed using a [UTF](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenutf_glosry.htm "Glossary Entry") or [UCS](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenucs2_glosry.htm "Glossary Entry") [character representation](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenchar_representation_glosry.htm "Glossary Entry").
