[Literal](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_literal_glosry.htm "Glossary Entry") of type [UNIT](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenddic_builtin_types.htm) that contains a unit. Only [typed](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abentyped_literal_glosry.htm "Glossary Entry") unit key literals are available.
