unnamed data object

[Data object](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_object_glosry.htm "Glossary Entry") that cannot be addressed by a name. Unnamed data objects are [literals](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_literal_glosry.htm "Glossary Entry") and [anonymous data objects](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenanonymous_data_object_glosry.htm "Glossary Entry").
