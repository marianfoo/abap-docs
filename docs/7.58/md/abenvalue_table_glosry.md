value table

Semantic property of a [DDIC domain](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendomain_glosry.htm "Glossary Entry") in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenabap_dictionary_glosry.htm "Glossary Entry"). The value table is the default value for the [check table](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abencheck_table_glosry.htm "Glossary Entry") of table fields defined with reference to the domain.
