selection screen variant

Set of input values for [selection screens](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenselection_screen_glosry.htm "Glossary Entry"). They can be used when calling selection screens to enter values in the [input fields](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abeninput_output_field_glosry.htm "Glossary Entry"). Variants on a selection screen can be created by choosing the menu path Goto → Variants and be passed, for example, by calling the statement [SUBMIT](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abapsubmit_selscreen_parameters.htm).
