vertex

Stores [graph](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abengraph_glosry.htm "Glossary Entry") data in combination with [edges](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenedge_glosry.htm "Glossary Entry"). A vertex can be considered as a node in a network. One vertex is denoted as the source and the other as the target. An edge connects two vertices. Vertices can have any number of [attributes](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abenattribute_glosry.htm "Glossary Entry"). An attribute consists of a name that is associated with a [data type](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendata_type_glosry.htm "Glossary Entry") and a value.
