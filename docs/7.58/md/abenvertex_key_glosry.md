vertex key

A vertex key represents one [vertex](javascript:call_link\('abenvertex_glosry.htm'\) "Glossary Entry") attribute or a set of [vertex attributes](javascript:call_link\('abenvertex_attr_glosry.htm'\) "Glossary Entry") that must uniquely identify vertices. If the vertex key consists of multiple [attributes](javascript:call_link\('abenvertex_attr_glosry.htm'\) "Glossary Entry"), then the [edge table's](javascript:call_link\('abenedge_table_glosry.htm'\) "Glossary Entry") source and target must consist of a corresponding set of attributes.
