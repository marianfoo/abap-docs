writable expression

Expression representing an operand that can be written to. Writable expressions are certain [constructor expressions](javascript:call_link\('abenconstructor_expression_glosry.htm'\) "Glossary Entry") using [NEW](javascript:call_link\('abennew_constructor_params_class.htm'\)) and [CAST](javascript:call_link\('abenconstructor_expression_cast.htm'\)) or [table expressions](javascript:call_link\('abentable_expression_glosry.htm'\) "Glossary Entry"). A writable expression can be specified in a [result position](javascript:call_link\('abenresult_position_glosry.htm'\) "Glossary Entry").
