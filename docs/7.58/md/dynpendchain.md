dynpro - ENDCHAIN

Syntax

ENDCHAIN.

Effect

Closes a processing chain defined in the [dynpro flow logic](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/abendynpro_flow_logic_glosry.htm "Glossary Entry"). See [CHAIN](https://help.sap.com/doc/abapdocu_758_index_htm/7.58/en-US/dynpchain.htm) for more information.
