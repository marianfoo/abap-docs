BACK - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapback.htm)

Syntax

BACK.

Effect

Positions the [list cursor](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlist_cursor_glosry.htm "Glossary Entry") at the first position of a line block defined with RESERVE, a [page header](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpage_header_glosry.htm "Glossary Entry") defined after TOP-OF-PAGE, or the first line underneath the page header.
