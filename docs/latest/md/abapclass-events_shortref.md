CLASS-EVENTS - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclass-events.htm)

Syntax

CLASS-EVENTS evt options.

Effect

Declares a [static event](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstatic_event_glosry.htm "Glossary Entry") evt of a class or an interface.
