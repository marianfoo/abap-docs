DESCRIBE TABLE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapdescribe_table.htm)

Syntax

DESCRIBE TABLE itab *\[*KIND knd*\]* *\[*LINES lin*\]* *\[*OCCURS n*\]*.

Effect

Defines the properties of an internal table itab.

Additions   

-   KIND knd
    Returns the table category in knd.
-   LINES lin
    Returns the number of lines in lin.
-   OCCURS n
    Returns the [initial memory requirement](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninitial_mem_req_glosry.htm "Glossary Entry") in n.
