ENDENHANCEMENT

[Short Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapenhancement_shortref.htm)

Syntax

ENDENHANCEMENT.

Effect

The statement ENDENHANCEMENT closes a [source code plug-in](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensource_code_plugin_glosry.htm "Glossary Entry") introduced using [ENHANCEMENT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapenhancement.htm).
