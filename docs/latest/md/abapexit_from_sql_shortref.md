EXIT FROM SQL - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapexit_sql.htm)

Syntax

EXIT FROM SQL.

Effect

Obsolete: Cancels implicit cursor processing in EXEC SQL. Processing is resumed after ENDEXEC.
