FUNCTION - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapfunction.htm)

Syntax

FUNCTION func.
  ...
ENDFUNCTION.

Effect

Implements a function module func. A function module is defined in [Function Builder](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_builder_glosry.htm "Glossary Entry") and implemented between FUNCTION and ENDFUNCTION.
