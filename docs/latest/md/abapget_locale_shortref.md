GET LOCALE LANGUAGE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapget_locale.htm)

Syntax

GET LOCALE LANGUAGE lang COUNTRY cntry MODIFIER mod.

Effect

Determines the current [text environment](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentext_environment_glosry.htm "Glossary Entry").

Additions   

-   LANGUAGE lang
    Returns the language key in lang.
-   [COUNTRY cntry](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapget_locale_obsolete.htm)
    Obsolete: Content of cntry is undefined.
-   [MODIFIER mod](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapget_locale_obsolete.htm)
    Obsolete: Initializes mod.
