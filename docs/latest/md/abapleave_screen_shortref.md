LEAVE SCREEN - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapleave_screen.htm)

Syntax

LEAVE *{* SCREEN *|* *{*TO SCREEN dynnr*}* *}*.

Effect

Exits the current [dynpro](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_glosry.htm "Glossary Entry") and continues processing using the [next dynpro](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abennext_dynpro_glosry.htm "Glossary Entry").

Addition  

-   TO SCREEN dynnr
    Specifies the next dynpro in dynnr.
