MODIFY SCREEN - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapmodify_screen.htm)

Syntax

MODIFY SCREEN FROM wa.

Effect

In LOOP AT SCREEN, overwrites the properties of the current screen element with the content of the predefined work area wa of the type [SCREEN](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen.htm).
