REFRESH CONTROL - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abaprefresh_control.htm)

Syntax

REFRESH CONTROL contrl FROM SCREEN dynnr.

Effect

Assigns the values predefined in the [dynpro](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_glosry.htm "Glossary Entry") dynnr to components of the structure contrl defined for a [table control](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_control_glosry.htm "Glossary Entry") using CONTROLS.
