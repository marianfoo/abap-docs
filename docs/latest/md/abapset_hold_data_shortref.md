SET HOLD DATA - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_hold_data.htm)

Syntax

SET HOLD DATA *{*ON*|*OFF*}*.

Effect

Enables or disables the standard menu entries Hold Data, Set Data, and Delete Data in the [GUI status](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengui_status_glosry.htm "Glossary Entry") of the current [dynpro](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_glosry.htm "Glossary Entry").
