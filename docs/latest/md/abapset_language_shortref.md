SET LANGUAGE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_language.htm)

Syntax

SET LANGUAGE lang.

Effect

Loads the [list headers](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlist_header_glosry.htm "Glossary Entry") and text symbols for the [text pool](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentext_pool_glosry.htm "Glossary Entry") of the language specified in lang.
