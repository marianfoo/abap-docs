SET LOCALE LANGUAGE - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_locale.htm)

Syntax

SET LOCALE LANGUAGE lang *\[*COUNTRY cntry*\]* *\[*MODIFIER mod*\]*.

Effect

Sets the [text environment](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentext_environment_glosry.htm "Glossary Entry").

Additions   

-   LANGUAGE lang
    Specifies the language in lang.
-   [COUNTRY cntry](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_locale_obsolete.htm)
    Obsolete: Specifies the country key in cntry.
-   [MODIFIER mod](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_locale_obsolete.htm)
    Obsolete: Specifies the ID for a specific locale in mod.
