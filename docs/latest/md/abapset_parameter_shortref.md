SET PARAMETER - Short Reference

[Reference](javascript:call_link\('abapset_parameter.htm'\))

Syntax

SET PARAMETER ID pid FIELD dobj.

Effect

Sets the [SPA/GPA parameter](javascript:call_link\('abenspa_gpa_parameter_glosry.htm'\) "Glossary Entry") specified in pid to the value of dobj.
