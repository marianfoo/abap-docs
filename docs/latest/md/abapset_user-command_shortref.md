SET USER-COMMAND - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_user-command.htm)

Syntax

SET USER-COMMAND fcode.

Effect

Raises a list event with a [function code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_code_glosry.htm "Glossary Entry") specified in fcode.
