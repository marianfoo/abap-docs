TEST-INJECTION - Short Reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abaptest-injection.htm)

Syntax

TEST-INJECTION seam.
  *\[*statement\_block*\]*
END-TEST-INJECTION.

Effect

Defines an [injection](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninjection_glosry.htm "Glossary Entry") for a [test seam](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_seam_glosry.htm "Glossary Entry").
