ABAP Cloud

ABAP development model for state-of-the-art, cloud-ready and upgrade-stable solutions. ABAP Cloud is based on a usage type of the [ABAP Platform](javascript:call_link\('abenabap_platform_glosry.htm'\) "Glossary Entry"), where development is done with the following restrictions:

-   The [ABAP language version](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry") is restricted to the [restricted ABAP language version](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry") [ABAP for Cloud Development](javascript:call_link\('abenabap_for_cloud_dev_glosry.htm'\) "Glossary Entry").
-   Access to other [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") is restricted to APIs that are [released](javascript:call_link\('abenreleased_api_glosry.htm'\) "Glossary Entry") for [ABAP for Cloud Development](javascript:call_link\('abenabap_for_cloud_dev_glosry.htm'\) "Glossary Entry").
-   Only [ABAP development tools for Eclipse (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry") can be used. There is no access to [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry").

The transactional programming model for ABAP Cloud is [RAP](javascript:call_link\('abenrap_glosry.htm'\) "Glossary Entry").

ABAP Cloud is supported in

-   [SAP BTP ABAP Environment](javascript:call_link\('abensap_btp_abap_env_glosry.htm'\) "Glossary Entry")
-   [ABAP Platform Cloud](javascript:call_link\('abenabap_platform_cloud_glosry.htm'\) "Glossary Entry")

See also [classic ABAP](javascript:call_link\('abenclassic_abap_glosry.htm'\) "Glossary Entry").
