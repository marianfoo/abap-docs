ABAP Core Data Services

ABAP CDS for short. An infrastructure for defining and consuming semantically rich data models on [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"). ABAP CDS provides different languages for defining metadata realized as [CDS objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_object_glosry.htm "Glossary Entry").
