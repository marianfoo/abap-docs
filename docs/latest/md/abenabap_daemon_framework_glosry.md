ABAP daemon framework

ADF for short. A framework based on [classes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") and [interfaces](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoo_intf_glosry.htm "Glossary Entry") for using [ABAP daemons](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_daemon_glosry.htm "Glossary Entry"). ADF includes APIs both for communication with ABAP daemons in [ABAP programs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") and for handling events using ABAP daemons in [ABAP daemon sessions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_daemon_session_glosry.htm "Glossary Entry").
