ABAP daemon manager

The class CL\_ABAP\_DAEMON\_CLIENT\_MANAGER. The static methods of the ABAP daemon manager are used to create and use [ABAP daemons](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_daemon_glosry.htm "Glossary Entry") in [ABAP programs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry").
