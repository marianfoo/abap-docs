ABAP Debugger

Tool used to execute [ABAP programs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") line by line or section by section. This makes it possible to edit the content of [data objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry") and to check the program logic. See also [breakpoint](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbreakpoint_glosry.htm "Glossary Entry").
