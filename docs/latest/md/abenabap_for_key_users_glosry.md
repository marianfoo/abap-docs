ABAP for Key Users

[Restricted ABAP language version](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrestricted_version_glosry.htm "Glossary Entry") with restricted language scope and restricted access to [released APIs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenreleased_api_glosry.htm "Glossary Entry") for [enhancements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_glosry.htm "Glossary Entry") made by key users at dedicated enhancement points. ABAP for Key Users is available in the special enhancement tools for key users.
