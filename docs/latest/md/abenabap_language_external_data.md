Processing External Data

External data generally covers all data that is stored outside of the [internal session](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninternal_session_glosry.htm "Glossary Entry"). The following sections describe the processing of external data.

-   [ABAP Database Access](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendb_access.htm)
-   [Data Clusters](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_cluster.htm)
-   [ABAP File Interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_language_files.htm)

Hint

In addition to the statements described here, there is also [obsolete processing of external data](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_storage_obsolete.htm).
