Lists - Editing List Buffers

The following statements process lists that are already saved as [screen lists](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_list_glosry.htm "Glossary Entry") in the list buffers:

-   [READ LINE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapread_line.htm),
-   [MODIFY LINE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapmodify_line.htm),
-   [SCROLL LIST](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapscroll.htm), and
-   [DESCRIBE LIST](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapdescribe_list.htm)

[Spool lists](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_list_glosry.htm "Glossary Entry") that are sent page by page to the SAP spool system cannot be processed in this manner.
