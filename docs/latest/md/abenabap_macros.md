Macros

Macros enable [source code modularization](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensource_code_modularizat_glosry.htm "Glossary Entry") within an ABAP program. They have the following properties:

-   Defined between the statements [DEFINE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapdefine.htm) and [END-OF-DEFINITION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapend-of-definition.htm) or in database table TRMAC.
-   They are [included](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmacro_include.htm) by specifying their name.

Programming Guideline

[Only use macros in exceptional cases.](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmacros_guidl.htm "Guideline")
