Messages - Control Processing

When [Control Framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_framework_glosry.htm "Glossary Entry") events are handled, messages of types W and E are converted to type A. All other messages are processed as specified in [dialog processing](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_message_dialog.htm).
