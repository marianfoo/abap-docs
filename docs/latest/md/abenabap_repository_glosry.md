ABAP Repository

Part of the [standard database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstandard_db_glosry.htm "Glossary Entry") of [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") that contains client-independent [repository objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry"). The repository can be searched using the Repository [Information System](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenris_glosry.htm "Glossary Entry").
