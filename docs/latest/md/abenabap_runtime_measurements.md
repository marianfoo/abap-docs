Runtime Measurements

The statements in this section enable the runtime of a program to be measured and the corresponding settings to be made.

-   [GET RUN TIME](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapget_run_time.htm)
-   [SET RUN TIME CLOCK RESOLUTION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_run_time_clock_resolution.htm)
-   [SET RUN TIME ANALYZER](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_run_time_analyzer.htm)

Alternatively, a [class for runtime measurements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencl_abap_runtime.htm) CL\_ABAP\_RUNTIME can be used.
