ABAP SQL engine

Part of the [ABAP SQL Interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_interface_glosry.htm "Glossary Entry") for processing [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry") statements that [read](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_reading.htm) tabular data in the memory of an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"). The ABAP SQL engine is used by ABAP SQL when accessing the table buffer or when internal tables are specified in the [FROM](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapselect_itab.htm) clause. The ABAP SQL engine supports a large subset of ABAP SQL but there are also some restrictions.
