ABAP SQL engine

Part of the [ABAP SQL Interface](javascript:call_link\('abenabap_sql_interface_glosry.htm'\) "Glossary Entry") for processing [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") statements that [read](javascript:call_link\('abenabap_sql_reading.htm'\)) tabular data in the memory of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). The ABAP SQL engine is used by ABAP SQL when accessing the table buffer or when internal tables are specified in the [FROM](javascript:call_link\('abapselect_itab.htm'\)) clause. The ABAP SQL engine supports a large subset of ABAP SQL but there are also some restrictions.
