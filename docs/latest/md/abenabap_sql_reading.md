ABAP SQL - Read Access

The ABAP SQL language elements for performing reads on the database are:

-   [SELECT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapselect.htm)
-   [WITH](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapwith.htm)
-   [OPEN CURSOR](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapopen_cursor.htm)
    -   [FETCH NEXT CURSOR](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapfetch.htm)
    -   [CLOSE CURSOR](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclose_cursor.htm)
-   [UNION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapunion.htm), [INTERSECT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapunion.htm), [EXCEPT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapunion.htm)
