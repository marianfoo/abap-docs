ABAP tuning object

Synonym for app-server tuning object. [Tuning object](javascript:call_link\('abentuning_object_glosry.htm'\) "Glossary Entry") that defines technical settings for a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") on [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Currently, there are the following ABAP tuning objects:

-   [CDS entity buffers](javascript:call_link\('abencds_entity_buffer_glosry.htm'\) "Glossary Entry").
