ABAP language version

[Program property](javascript:call_link\('abenprogram_property_glosry.htm'\) "Glossary Entry") that determines the language scope and the syntax rules of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") and other [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). See also [unrestricted](javascript:call_link\('abenunrestricted_version_glosry.htm'\) "Glossary Entry") and [restricted ABAP language versions](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry").
