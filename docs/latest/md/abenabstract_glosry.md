abstract

Term in [ABAP Objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_objects_glosry.htm "Glossary Entry"). An abstract [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") cannot be instantiated. An abstract [method](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmethod_glosry.htm "Glossary Entry") can only be implemented in [subclasses](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensubclass_glosry.htm "Glossary Entry") of its class. A non-abstract class or method is [concrete](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenconcrete_glosry.htm "Glossary Entry").
