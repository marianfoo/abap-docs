CDS access condition

Condition in a [CDS role](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_role_glosry.htm "Glossary Entry") defined in an [access rule](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenaccess_rule_glosry.htm "Glossary Entry") of the statement [DEFINE ROLE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_f1_define_role.htm) for a [CDS entity](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_entity_glosry.htm "Glossary Entry"). [CDS access control](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_access_control_glosry.htm "Glossary Entry") evaluates the access conditions of a CDS entity by default when [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry") is used to access objects. Access conditions can include for example [PFCG conditions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpfcg_condition_glosry.htm "Glossary Entry"), [literal conditions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenliteral_condition_glosry.htm "Glossary Entry"), [user conditions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_condition_glosry.htm "Glossary Entry"), and other conditions.
