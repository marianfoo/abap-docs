CDS access rule

An access rule formulates an [access condition](javascript:call_link\('abenaccess_condition_glosry.htm'\) "Glossary Entry") for a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") or guarantees open access in a [CDS role](javascript:call_link\('abencds_role_glosry.htm'\) "Glossary Entry").
