aggregate

A summary of the data of multiple rows of an [SQL source](javascript:call_link\('abensql_data_source_glosry.htm'\) "Glossary Entry"). Can be determined using [aggregate functions](javascript:call_link\('abenaggregate_function_glosry.htm'\) "Glossary Entry") in the [SELECT](javascript:call_link\('abapselect.htm'\)) statement.
