AMDP method

[Method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of an [AMDP class](javascript:call_link\('abenamdp_class_glosry.htm'\) "Glossary Entry") that is flagged as an [AMDP procedure implementation](javascript:call_link\('abenamdp_procedure_method_glosry.htm'\) "Glossary Entry") using [BY DATABASE PROCEDURE](javascript:call_link\('abapmethod_by_db_proc.htm'\)) or as an [AMDP function implementation](javascript:call_link\('abenamdp_function_method_glosry.htm'\) "Glossary Entry") using [BY DATABASE FUNCTION](javascript:call_link\('abapmethod_by_db_proc.htm'\)) in the [implementation part](javascript:call_link\('abenimplementation_part_glosry.htm'\) "Glossary Entry") of the class.
