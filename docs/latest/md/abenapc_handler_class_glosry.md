APC handler class

[Global class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenglobal_class_glosry.htm "Glossary Entry") in which the handling of a query sent to an [ABAP push channel](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_push_channel_glosry.htm "Glossary Entry") in [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") is implemented. The request is handled as [APC processing](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_processing_glosry.htm "Glossary Entry") in a separate [APC session](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_session_glosry.htm "Glossary Entry").
