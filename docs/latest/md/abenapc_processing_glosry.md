APC processing

Processing of a WebSocket or TCP request using the [APC](javascript:call_link\('abenapc_glosry.htm'\) "Glossary Entry") framework in an [APC handler class](javascript:call_link\('abenapc_handler_class_glosry.htm'\) "Glossary Entry"). APC processing takes place in a dedicated [APC session](javascript:call_link\('abenapc_session_glosry.htm'\) "Glossary Entry"). APC processing can be stateless or stateful. Stateful APC processing must take place in [non-blocking mode](javascript:call_link\('abennon_blocking_model_glosry.htm'\) "Glossary Entry"), where blocking statements are forbidden.
