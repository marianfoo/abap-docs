APC session

[User session](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_session_glosry.htm "Glossary Entry") for [APC processing](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_processing_glosry.htm "Glossary Entry"). An APC session is opened by the [APC framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_glosry.htm "Glossary Entry") for the handling of a WebSocket or TCP queries in an [APC handler class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_handler_class_glosry.htm "Glossary Entry").
