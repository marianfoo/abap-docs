area lock

[Read lock](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenread_lock_glosry.htm "Glossary Entry") or [change lock](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenchange_lock_glosry.htm "Glossary Entry") ([write lock](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwrite_lock_glosry.htm "Glossary Entry"), [update lock](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenupdate_lock_glosry.htm "Glossary Entry")) on an [area instance version](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenarea_instance_version_glosry.htm "Glossary Entry") in the [shared objects memory](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenshared_objects_memory_glosry.htm "Glossary Entry"). Area locks are set by binding [area handles](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenarea_handle_glosry.htm "Glossary Entry") to area instance versions.
