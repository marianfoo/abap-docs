arithmetic expression

Formulation of an arithmetic calculation in a [calculation expression](javascript:call_link\('abencalculation_expression_glosry.htm'\) "Glossary Entry"). An arithmetic expression is assigned a [calculation type](javascript:call_link\('abencalculation_type_glosry.htm'\) "Glossary Entry") that is used to execute the calculation. The result of an arithmetic expression is a numeric value that exists in the calculation type. Arithmetic expressions occur in the [read positions of specific statements](javascript:call_link\('abenoperands_expressions.htm'\)).
