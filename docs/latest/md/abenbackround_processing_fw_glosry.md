background processing framework

The background processing framework (bgPF) is a framework that asynchronously and reliably executes methods of applications that develop background processes. It is based on the background remote function call and supports the inbound scenario. Background processes are executed transactional and with a service quality of exactly once. The bgPF can ensure transactional consistency within the controlled SAP LUW.
