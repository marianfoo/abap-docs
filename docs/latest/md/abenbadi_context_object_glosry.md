BAdI context object

[Instance](javascript:call_link\('abeninstance_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") that implements the [tag interface](javascript:call_link\('abentag_interface_glosry.htm'\) "Glossary Entry") IF\_BADI\_CONTEXT. This tag interface can be implemented by any class. [References](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") to BAdI context objects can be used in the statement [GET BADI](javascript:call_link\('abapget_badi.htm'\)) to specify a context for [object plug-ins](javascript:call_link\('abenobject_plugin_glosry.htm'\) "Glossary Entry").
