BAdI context object

[Instance](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninstance_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") that implements the [tag interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentag_interface_glosry.htm "Glossary Entry") IF\_BADI\_CONTEXT. This tag interface can be implemented by any class. [References](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenreference_glosry.htm "Glossary Entry") to BAdI context objects can be used in the statement [GET BADI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapget_badi.htm) to specify a context for [object plug-ins](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_plugin_glosry.htm "Glossary Entry").
