Base64

Method for coding binary data into printable characters. Each byte is converted into a short character string whose characters are obtained from a record of 64 [code page](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencodepage_glosry.htm "Glossary Entry")\-independent [ASCII](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenascii_glosry.htm "Glossary Entry") characters.
