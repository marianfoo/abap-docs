batch input table

[Internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") of [line type](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry") BDCDATA, which contains the data for a [dialog transaction](javascript:call_link\('abendialog_transaction_glosry.htm'\) "Glossary Entry") in the same format as in a [batch input session](javascript:call_link\('abenbatch_input_session_glosry.htm'\) "Glossary Entry"). Batch input tables can be specified after the addition USING of the statement [CALL TRANSACTION](javascript:call_link\('abapcall_transaction.htm'\)) when calling [transactions](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry").
