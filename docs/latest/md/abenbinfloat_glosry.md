binary floating point number

[Numeric data object](javascript:call_link\('abennumeric_data_object_glosry.htm'\) "Glossary Entry") of ABAP or elementary object of the database with a [binary floating point number type](javascript:call_link\('abenbinfloat_type_glosry.htm'\) "Glossary Entry"). Due to the internal representation as binary fractions, not every decimal number within the type's value range can be represented exactly by a binary floating point number.
