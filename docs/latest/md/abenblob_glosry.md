BLOB

Abbreviation for Binary Large Object. Term for a [database field](javascript:call_link\('abendatabase_field_glosry.htm'\) "Glossary Entry") of the data type [RAWSTRING](javascript:call_link\('abenddic_builtin_types.htm'\)) or the result of an [SQL expression](javascript:call_link\('abapsql_expr.htm'\)) that returns the data type [RAWSTRING](javascript:call_link\('abenddic_builtin_types.htm'\)).
