Business Object Processing Framework

[BOPF](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbopf_glosry.htm "Glossary Entry") for short. An ABAP-based framework that provides a set of generic services and functionalities to create [business objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbusiness_object_glosry.htm "Glossary Entry") (BOs), such as [CDS-based BOPF BOs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_bopf_bo_glosry.htm "Glossary Entry").
