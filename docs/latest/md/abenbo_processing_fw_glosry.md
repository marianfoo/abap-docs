Business Object Processing Framework

[BOPF](javascript:call_link\('abenbopf_glosry.htm'\) "Glossary Entry") for short. An ABAP-based framework that provides a set of generic services and functionalities to create [business objects](javascript:call_link\('abenbusiness_object_glosry.htm'\) "Glossary Entry") (BOs), such as [CDS-based BOPF BOs](javascript:call_link\('abencds_bopf_bo_glosry.htm'\) "Glossary Entry").
