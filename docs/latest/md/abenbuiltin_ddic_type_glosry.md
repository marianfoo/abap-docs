built-in DDIC type

[Built-in](javascript:call_link\('abenbuiltin_data_type_glosry.htm'\) "Glossary Entry") [elementary data type](javascript:call_link\('abenelementary_data_type_glosry.htm'\) "Glossary Entry") of the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). Built-in DDIC types can be used to define types in ABAP Dictionary and [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") but cannot be used in [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") directly.
