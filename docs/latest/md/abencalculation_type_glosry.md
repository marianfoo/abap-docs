calculation type

Property of an [arithmetic expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenarithmetic_expression_glosry.htm "Glossary Entry") depending on the [numeric data types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abennumeric_data_type_glosry.htm "Glossary Entry") involved. The calculation type determines the calculation rule used to evaluate an arithmetic expression. There are different rules for the possible calculation types i, int8, p, decfloat34, and f.
