canonical XML representation

Format of [XML](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenxml_glosry.htm "Glossary Entry") data that is created during the [serialization](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenserialization_glosry.htm "Glossary Entry") of ABAP data with the [identity transformation](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenid_trafo_glosry.htm "Glossary Entry") and that is a prerequisite for a [deserialization](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendeserialization_glosry.htm "Glossary Entry") using identity transformations and is used as an intermediate format when using [XSLT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenxslt_glosry.htm "Glossary Entry") to access ABAP data. The short name is [asXML](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenasxml_glosry.htm "Glossary Entry"); see also [basXML](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbasxml_glosry.htm "Glossary Entry")
