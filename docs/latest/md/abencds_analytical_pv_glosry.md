CDS analytical projection view

Synonym for [CDS analytical query](javascript:call_link\('abencds_analytic_query_glosry.htm'\) "Glossary Entry"). [CDS projection view](javascript:call_link\('abencds_projection_view_glosry.htm'\) "Glossary Entry") that is intended for modeling analytical queries within a CDS data model. A CDS analytical projection view is defined using the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") statement [DEFINE TRANSIENT VIEW ENTITY AS PROJECTION ON](javascript:call_link\('abencds_define_view_as_analytical.htm'\)) and it must have the [provider contract](javascript:call_link\('abencds_pv_provider_contract.htm'\)) set to ANALYTICAL\_QUERY.
