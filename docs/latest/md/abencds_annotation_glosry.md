CDS annotation

A CDS annotation adds metadata to a [CDS object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_object_glosry.htm "Glossary Entry") that goes beyond the syntax features of [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_glosry.htm "Glossary Entry"). An annotation is specified using the [annotation syntax](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_annotation_syntax_glosry.htm "Glossary Entry") in the [CDS source code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_source_code_glosry.htm "Glossary Entry") of the CDS object. User-defined CDS annotations can be specified using [CDS annotation definitions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_anno_definition_glosry.htm "Glossary Entry"). SAP delivers a predefined set of [SAP annotations](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_annotation_glosry.htm "Glossary Entry").
