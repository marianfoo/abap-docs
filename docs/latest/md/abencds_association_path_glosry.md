CDS association path

A string of [CDS associations](javascript:call_link\('abencds_association_glosry.htm'\) "Glossary Entry"). CDS association paths can be specified in the form of [SQL path expressions](javascript:call_link\('abensql_path_expression_glosry.htm'\) "Glossary Entry") in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry") and in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry").
