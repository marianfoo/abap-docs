CDS custom entity extension

[CDS entity extension](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_entity_extend_glosry.htm "Glossary Entry") of a [CDS custom entity](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_custom_entity_glosry.htm "Glossary Entry"). A CDS custom entity extension can be used to add elements to a custom entity without making any modifications to the original entity. It is defined using the statement [EXTEND CUSTOM ENTITY](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_extend_custom_entity.htm).
