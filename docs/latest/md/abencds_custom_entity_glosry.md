CDS custom entity

A [non-SQL CDS entity](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_non_sql_entity_glosry.htm "Glossary Entry") defined using [DEFINE CUSTOM ENTITY](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_f1_define_custom_entity.htm) in a [CDS data definition](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_data_definition_glosry.htm "Glossary Entry"). A CDS custom entity allows developers to implement their own data retrieval using [ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_glosry.htm "Glossary Entry"). The data retrieval is implemented by a method of an [ABAP class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclass.htm) that is called when accessing the custom entity in a framework such as the [RAP query engine](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrap_query_engine_glosry.htm "Glossary Entry"). Custom entities cannot be used in [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry"). In [RAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrap_glosry.htm "Glossary Entry"), custom entities are used as the data model basis for [unmanaged queries](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrap_unmanged_query_glosry.htm "Glossary Entry").
