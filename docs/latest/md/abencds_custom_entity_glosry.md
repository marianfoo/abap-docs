CDS custom entity

A [non-SQL CDS entity](javascript:call_link\('abencds_non_sql_entity_glosry.htm'\) "Glossary Entry") defined using [DEFINE CUSTOM ENTITY](javascript:call_link\('abencds_f1_define_custom_entity.htm'\)) in a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). A CDS custom entity allows developers to implement their own data retrieval using [ABAP](javascript:call_link\('abenabap_glosry.htm'\) "Glossary Entry"). The data retrieval is implemented by a method of an [ABAP class](javascript:call_link\('abapclass.htm'\)) that is called when accessing the custom entity in a framework such as the [RAP query engine](javascript:call_link\('abenrap_query_engine_glosry.htm'\) "Glossary Entry"). Custom entities cannot be used in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"). In [RAP](javascript:call_link\('abenrap_glosry.htm'\) "Glossary Entry"), custom entities are used as the data model basis for [unmanaged queries](javascript:call_link\('abenrap_unmanged_query_glosry.htm'\) "Glossary Entry").
