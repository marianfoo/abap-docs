CDS element

An element in a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry"). A CDS element is defined in an element list of a [CDS data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry") and is a [structure component](javascript:call_link\('abenstructure_component_glosry.htm'\) "Glossary Entry") of the [structured type](javascript:call_link\('abenstructured_type_glosry.htm'\) "Glossary Entry") defined by the CDS entity. CDS elements can be, for example, [CDS source fields](javascript:call_link\('abencds_field_glosry.htm'\) "Glossary Entry"), [CDS literals](javascript:call_link\('abencds_literal_glosry.htm'\) "Glossary Entry"), or [CDS DDL expressions](javascript:call_link\('abencds_expression_glosry.htm'\) "Glossary Entry").
