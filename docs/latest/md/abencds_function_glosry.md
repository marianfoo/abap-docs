CDS function

Function in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") that can be used in operand positions of [CDS entities](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry"). A CDS function can be a [user-defined function](javascript:call_link\('abencds_user_func_glosry.htm'\) "Glossary Entry"), a [built-in function](javascript:call_link\('abencds_builtin_func_glosry.htm'\) "Glossary Entry"), or a [system function](javascript:call_link\('abencds_system_func_glosry.htm'\) "Glossary Entry").
