CDS metadata extension

Short MDE. A [CDS object](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") of the [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") is defined in [DDLX source code](javascript:call_link\('abenddlx_source_code_glosry.htm'\) "Glossary Entry") with [language elements](javascript:call_link\('abencds_f1_ddlx_syntax.htm'\)) of the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry"). In a CDS metadata extension, [CDS annotations](javascript:call_link\('abencds_annotation_glosry.htm'\) "Glossary Entry") are specified for a [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") outside its [data definition](javascript:call_link\('abencds_data_definition_glosry.htm'\) "Glossary Entry"). A CDS metadata extension is always assigned to a layer such as industry, partner or customer.
