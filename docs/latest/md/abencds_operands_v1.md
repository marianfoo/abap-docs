CDS DDL - DDIC-Based View, Elementary CDS Operands

Syntax

... [literal](javascript:call_link\('abencds_literal_v1.htm'\))
  *|* [field](javascript:call_link\('abencds_field_v1.htm'\))
  *|* [$parameters.pname](javascript:call_link\('abencds_parameter_v1.htm'\))
  *|* [$session.vname](javascript:call_link\('abencds_session_variable_v1.htm'\)) ...

Effect

Specifies a literal [literal](javascript:call_link\('abencds_literal_v1.htm'\)), a field [field](javascript:call_link\('abencds_field_v1.htm'\)), a parameter [$parameters.pname](javascript:call_link\('abencds_parameter_v1.htm'\)), or a session variable [$session.vname](javascript:call_link\('abencds_session_variable_v1.htm'\)) as an elementary [CDS operand](javascript:call_link\('abencds_operand_glosry.htm'\) "Glossary Entry") in a CDS DDL operand position.
