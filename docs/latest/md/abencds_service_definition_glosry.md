CDS service definition

A CDS service definition is a [RAP object](javascript:call_link\('abencds_rap_object_glosry.htm'\) "Glossary Entry") that exposes [CDS entities](javascript:call_link\('abencds_object_glosry.htm'\) "Glossary Entry") for a [business service](javascript:call_link\('abenbusiness_service_glosry.htm'\) "Glossary Entry"). This is done using the [CDS SDL](javascript:call_link\('abencds_sdl_glosry.htm'\) "Glossary Entry") language elements of [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") in [SDL source code](javascript:call_link\('abensdl_source_code_glosry.htm'\) "Glossary Entry").
