CDS system entity

A [CDS entity](javascript:call_link\('abencds_entity_glosry.htm'\) "Glossary Entry") delivered with an [ABAP Platform](javascript:call_link\('abenabap_platform_glosry.htm'\) "Glossary Entry") that implements basic functionality, such as generating a series of integers (SERIES\_GENERATE\_INTEGER). It can be used as data source of other CDS entities or in [queries](javascript:call_link\('abenquery_glosry.htm'\) "Glossary Entry") of [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry").
