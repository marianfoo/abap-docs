CDS DDIC-based view

An obsolete type of [CDS view](javascript:call_link\('abencds_view_glosry.htm'\) "Glossary Entry") that is technically based on a [CDS-managed DDIC view (obsolete)](javascript:call_link\('abencds_mngdddic_view_glosry.htm'\) "Glossary Entry") in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry"). A CDS DDIC-based view (obsolete) is defined with the [CDS DDL](javascript:call_link\('abencds_ddl_glosry.htm'\) "Glossary Entry") statement [DEFINE VIEW ddic\_based\_view](javascript:call_link\('abencds_define_view_v1.htm'\)).
