character set

Defined set of characters, for example, [ASCII](javascript:call_link\('abenascii_glosry.htm'\) "Glossary Entry"), [EBCDIC](javascript:call_link\('abenebcdic_glosry.htm'\) "Glossary Entry") or [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry"). Character sets are mapped onto [character representations](javascript:call_link\('abenchar_representation_glosry.htm'\) "Glossary Entry") via [code pages](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry").
