character-like expression

Represents a [string](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencharacter_string_glosry.htm "Glossary Entry"). A character-like expression is a [constructor expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenconstructor_glosry.htm "Glossary Entry"), a [string expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstring_expression_glosry.htm "Glossary Entry"), a [table expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_expression_glosry.htm "Glossary Entry"), or a [function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_glosry.htm "Glossary Entry"), the return value of which has a [character-like data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencharlike_data_type_glosry.htm "Glossary Entry").
