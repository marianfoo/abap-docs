class

Template for [objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") in [ABAP Objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_objects_glosry.htm "Glossary Entry"). Defined with [CLASS - ENDCLASS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclass.htm), either as a [global class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenglobal_class_glosry.htm "Glossary Entry") in a [class pool](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_pool_glosry.htm "Glossary Entry") or as a [local class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlocal_class_glosry.htm "Glossary Entry") in almost any [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry"). The definition of a class comprises a [declaration part](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendeclaration_part_glosry.htm "Glossary Entry") for the declaration of the [class components](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_component_glosry.htm "Glossary Entry") and an [implementation part](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenimplementation_part_glosry.htm "Glossary Entry") for the implementation of the [methods](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmethod_glosry.htm "Glossary Entry").
