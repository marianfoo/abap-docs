classic ABAP

ABAP development model for legacy solutions. Classic ABAP is based on a [ABAP Platform](javascript:call_link\('abenabap_platform_glosry.htm'\) "Glossary Entry") without restrictions regarding [ABAP language versions](javascript:call_link\('abenabap_version_glosry.htm'\) "Glossary Entry"), usage of tools or access to [repository objects](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry"). Classic ABAP is supported in [SAP S/4HANA](javascript:call_link\('abensap_s4hana_glosry.htm'\) "Glossary Entry"). See also [ABAP Cloud](javascript:call_link\('abenabap_cloud_glosry.htm'\) "Glossary Entry").
