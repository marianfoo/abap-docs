client ID

Three-digit ID of a [client](javascript:call_link\('abenclient_glosry.htm'\) "Glossary Entry") that is specified when [logging on](javascript:call_link\('abenlogon_glosry.htm'\) "Glossary Entry") to [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). In [application tables](javascript:call_link\('abenapplication_table_glosry.htm'\) "Glossary Entry"), client-dependent data is indicated by the client ID stored in the [client column](javascript:call_link\('abenclient_column_glosry.htm'\) "Glossary Entry").
