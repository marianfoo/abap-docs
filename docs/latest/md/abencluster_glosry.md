cluster

Grouping of different data in shared storage. Currently only used as [data clusters](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_cluster_glosry.htm "Glossary Entry") in different storage media for [data objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry"). Before [ABAP release](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_release_glosry.htm "Glossary Entry") 7.53, also used as a [table cluster](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_cluster_glosry.htm "Glossary Entry") for special [DDIC database tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenddic_db_table_glosry.htm "Glossary Entry") ([cluster tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencluster_table_glosry.htm "Glossary Entry")).
