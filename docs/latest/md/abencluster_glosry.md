cluster

Grouping of different data in shared storage. Currently only used as [data clusters](javascript:call_link\('abendata_cluster_glosry.htm'\) "Glossary Entry") in different storage media for [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). Before [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") 7.53, also used as a [table cluster](javascript:call_link\('abentable_cluster_glosry.htm'\) "Glossary Entry") for special [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") ([cluster tables](javascript:call_link\('abencluster_table_glosry.htm'\) "Glossary Entry")).
