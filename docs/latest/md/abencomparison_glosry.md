comparison

[Relational expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrelational_expression_glosry.htm "Glossary Entry") that compares one or more [operands](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoperand_glosry.htm "Glossary Entry"). Implemented in ABAP using [comparison expressions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencomparison_expression_glosry.htm "Glossary Entry").
