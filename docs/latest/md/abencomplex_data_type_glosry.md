complex data type

A [data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_type_glosry.htm "Glossary Entry") that is made up of other data types. A complex data type is either a [structured data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstructured_type_glosry.htm "Glossary Entry"), a [table type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_type_glosry.htm "Glossary Entry"), or a [mesh type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmesh_type_glosry.htm "Glossary Entry").
