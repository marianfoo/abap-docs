compound interface

Interface in which at least one [component interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencomponent_interface_glosry.htm "Glossary Entry") is included in its declaration with [INTERFACES](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapinterfaces_ifac.htm). Also referred to as a nested interface.
