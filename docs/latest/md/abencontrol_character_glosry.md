control character

Also non-printing character. Element of a [character set](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencharacter_set_glosry.htm "Glossary Entry") that does not represent a written symbol but causes an effect such as a line feed or a tabulator.
