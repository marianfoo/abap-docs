control

Software component of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") used to process complex [screen elements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_element_glosry.htm "Glossary Entry"). [Table controls](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_control_glosry.htm "Glossary Entry"), [tabstrip controls](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentabstrip_control_glosry.htm "Glossary Entry"), and [splitter controls](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensplitter_control_glosry.htm "Glossary Entry") are platform-independent controls that processed directly using dynpro statements and [ABAP statements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_statement_glosry.htm "Glossary Entry"). In addition, there are platform-independent [GUI controls](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengui_control_glosry.htm "Glossary Entry") that are accessed in ABAP using the [classes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") of the [Control Framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_framework_glosry.htm "Glossary Entry").
