conversion routine

A conversion routine is a [function module](javascript:call_link\('abenfunction_module_glosry.htm'\) "Glossary Entry") that implements a part of a [conversion exit](javascript:call_link\('abenconversion_exit_glosry.htm'\) "Glossary Entry"). For each conversion exit, there is a function module for input and a function module for output using the naming convention CONVERSION\_EXIT\_.....\_INPUT*|*OUTPUT where the dots are the name of the conversion exit.
