component operator

[Constructor operator](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenconstructor_operator_glosry.htm "Glossary Entry") [CORRESPONDING](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenconstructor_expr_corresponding.htm) for component by component assignments between [structures](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstructure_glosry.htm "Glossary Entry") or between [internal tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninternal_table_glosry.htm "Glossary Entry") in [operand positions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoperand_position_glosry.htm "Glossary Entry"). By default, the mapping between the [structure components](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstructure_component_glosry.htm "Glossary Entry") is based on identical names and can also be defined using mapping rules.
