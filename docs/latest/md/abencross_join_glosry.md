cross join

A [join type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenjoin_type_glosry.htm "Glossary Entry") for which the rows of the result set are determined by a cross product of the result sets of both [SQL data sources](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_data_source_glosry.htm "Glossary Entry"). The result set contains all possible combinations of rows.
