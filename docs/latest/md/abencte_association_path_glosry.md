CTE association path

Sequence of [CTE associations](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencte_association_glosry.htm "Glossary Entry") that link [common table expressions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencommon_table_expression_glosry.htm "Glossary Entry") with [association targets](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenassociation_target_glosry.htm "Glossary Entry"). CTE association paths can be specified in the form of [SQL path expressions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_path_expression_glosry.htm "Glossary Entry") in [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry").
