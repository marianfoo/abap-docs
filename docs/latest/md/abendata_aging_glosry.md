data aging

A [SAP HANA database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_database_glosry.htm "Glossary Entry") concept where current data is distinguished from outdated data by entries in a [temperature column](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentemperature_column_glosry.htm "Glossary Entry"). Only current data exists in the memory.
