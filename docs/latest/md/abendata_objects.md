Data Objects

Data objects are instances of [data types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_types.htm) and contain the concrete data that a program uses at runtime. This is done by specifying [data objects in operand positions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoperands_data_objects.htm).
