database rollback

End of a [database LUW](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_luw_glosry.htm "Glossary Entry") in which all modifying database operations are undone until the beginning of the LUW. In an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"), database rollbacks are triggered implicitly, as well as by explicit requests.
