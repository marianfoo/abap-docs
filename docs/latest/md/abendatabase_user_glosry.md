database user

Person or system logged on to a [database system](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_system_glosry.htm "Glossary Entry") using a [database user name](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_user_name_glosry.htm "Glossary Entry"). When the database is accessed using the [database connections](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_connection_glosry.htm "Glossary Entry") of an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"), the database user is a [work process](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwork_process_glosry.htm "Glossary Entry") that implicitly accesses the [database schema](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_schema_glosry.htm "Glossary Entry") assigned to the user name of the connection.
