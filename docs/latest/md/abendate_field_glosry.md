date field

[Data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") of the [date type](javascript:call_link\('abendate_type_glosry.htm'\) "Glossary Entry") [d](javascript:call_link\('abenbuiltin_types_date_time.htm'\)) for storing a character-like [calendar date](javascript:call_link\('abencalendar_date_glosry.htm'\) "Glossary Entry"). Valid values have the format yyyymmdd. The internal representation takes place in the [character representation](javascript:call_link\('abenchar_representation_glosry.htm'\) "Glossary Entry") of the current [code page](javascript:call_link\('abencodepage_glosry.htm'\) "Glossary Entry"). When assignments are made to data objects of [numeric data types](javascript:call_link\('abennumeric_data_type_glosry.htm'\) "Glossary Entry") and in calculations, valid values are converted to the number of days since 01.01.0001.
