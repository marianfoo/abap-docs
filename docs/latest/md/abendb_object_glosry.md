database object

Object saved in a [database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_glosry.htm "Glossary Entry"), such as a [database table](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_table_glosry.htm "Glossary Entry"), an [SQL view](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_view_glosry.htm "Glossary Entry"), or a [stored procedure](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstored_procedure_glosry.htm "Glossary Entry"). [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") mainly uses [ABAP-managed database objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_managed_db_object_glosry.htm "Glossary Entry").
