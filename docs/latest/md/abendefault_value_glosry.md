default value

A value that appears in an [input field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninput_output_field_glosry.htm "Glossary Entry") when a [screen layout](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_glosry.htm "Glossary Entry") is called and can be overwritten by the user. A default value is usually set in the corresponding [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") before the screen layout is called.
