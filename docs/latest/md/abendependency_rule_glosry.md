dependency rule

[Database object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendb_object_glosry.htm "Glossary Entry") of the [SAP HANA database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_database_glosry.htm "Glossary Entry"). A dependency rule can be defined for [DDIC database tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenddic_db_table_glosry.htm "Glossary Entry") and derives additional conditions from a selection condition, used by the optimizer of the [database system](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_system_glosry.htm "Glossary Entry") to extend the read access to these database tables. On an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"), a dependency rules can be defined as a [HANA tuning object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_tuning_object_glosry.htm "Glossary Entry") in the [ABAP Dictionary](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_dictionary_glosry.htm "Glossary Entry").
