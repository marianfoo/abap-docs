description function

[Built-in function](javascript:call_link\('abenbuiltin_function_glosry.htm'\) "Glossary Entry") that returns properties of [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). There are [description functions for character-like](javascript:call_link\('abendescriptive_functions.htm'\)), [table-like](javascript:call_link\('abendescriptive_functions_table.htm'\)), and [byte-like](javascript:call_link\('abendescriptive_functions_binary.htm'\)) arguments.
