details list

[List](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclassic_list_glosry.htm "Glossary Entry") with a [list level](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlist_level_glosry.htm "Glossary Entry") greater than 0. List output in [event blocks](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenevent_block_glosry.htm "Glossary Entry") of the [interactive list processing](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninteractive_list_proces_glosry.htm "Glossary Entry") writes to details lists.
