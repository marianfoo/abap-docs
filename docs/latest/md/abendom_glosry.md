DOM

Short for Document Object Model. Interface for accessing [XML](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenxml_glosry.htm "Glossary Entry") documents that are based on an object-oriented tree-like representation of the documents in the memory, where each node is represented by an object. In ABAP, [iXML Library](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenixml_library_glosry.htm "Glossary Entry") can be used to process XML documents in DOM representation.
