double-click

Selecting a [screen element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_element_glosry.htm "Glossary Entry") by double-clicking with the mouse. In the [SAP GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_gui_glosry.htm "Glossary Entry"), double-clicking with the mouse always has the same effect as selecting the [function key](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_key_glosry.htm "Glossary Entry") F2.
