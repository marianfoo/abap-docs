edge table

A table that stores a set of different [edges](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenedge_glosry.htm "Glossary Entry"). Among them, there are at least two additional columns that reference the key column of the [vertex table](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenvertex_table_glosry.htm "Glossary Entry"). One of the columns identifies the source [vertex](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenvertex_glosry.htm "Glossary Entry"), and the other identifies the target vertex of an [edge](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenedge_glosry.htm "Glossary Entry").
