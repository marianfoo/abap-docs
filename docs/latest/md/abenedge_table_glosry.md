edge table

A table that stores a set of different [edges](javascript:call_link\('abenedge_glosry.htm'\) "Glossary Entry"). Among them, there are at least two additional columns that reference the key column of the [vertex table](javascript:call_link\('abenvertex_table_glosry.htm'\) "Glossary Entry"). One of the columns identifies the source [vertex](javascript:call_link\('abenvertex_glosry.htm'\) "Glossary Entry"), and the other identifies the target vertex of an [edge](javascript:call_link\('abenedge_glosry.htm'\) "Glossary Entry").
