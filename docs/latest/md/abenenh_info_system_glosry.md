Enhancement Information System

Tool of the [ABAP Workbench](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_workbench_glosry.htm "Glossary Entry") for the search for [enhancements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_glosry.htm "Glossary Entry") made with the [ABAP Enhancement Framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_framework_glosry.htm "Glossary Entry"). Called in the [Object Navigator](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_navigator_glosry.htm "Glossary Entry") or via [transaction code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentransaction_code_glosry.htm "Glossary Entry") SPAU\_ENH.
