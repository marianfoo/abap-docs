enhancement

Addition to a [repository object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry") that is managed by the [ABAP Enhancement Framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_framework_glosry.htm "Glossary Entry") and changes its behavior without modification. An enhancement can be made by creating [enhancement implementation elements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_impl_elem_glosry.htm "Glossary Entry") for explicitly defined or implicitly available [enhancement options](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_point_glosry.htm "Glossary Entry"). Enhancements are switchable using the [Switch Framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenswitch_framework_glosry.htm "Glossary Entry").
