enumerated type

Enum for short. An enumerated type is a [data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") that specifies a value set in addition to the actual type properties. Enumerated types can be defined in ABAP with [TYPES BEGIN OF ENUM](javascript:call_link\('abaptypes_enum.htm'\)) or in [ABAP CDS](javascript:call_link\('abenabap_cds_glosry.htm'\) "Glossary Entry") with [DEFINE TYPE ENUM](javascript:call_link\('abencds_define_enum_type.htm'\)).
