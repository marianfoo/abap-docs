enumerated value

An enumerated value is an allowed value of an [enumerated variable](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenumerated_variable_glosry.htm "Glossary Entry") or a [CDS enumerated element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_enum_element_glosry.htm "Glossary Entry"). The possible enumerated values are defined in the definition of the respective [enumerated type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenum_type_glosry.htm "Glossary Entry") as the values of [enumerated constants](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenumerated_constant_glosry.htm "Glossary Entry") or of components of an [enumerated structure](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenumerated_structure_glosry.htm "Glossary Entry"). The technical data type of an enumerated value is the [base type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbase_type_glosry.htm "Glossary Entry") of the enumerated type.
