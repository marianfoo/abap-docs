IPO

Short for Input, Processing, Output. Possible process flow in program execution. The IPO principle implies that a program is executed in the order Input, Processing, and then Output of the results. See also [reporting](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenreporting_glosry.htm "Glossary Entry").
