event

An event can trigger the processing of a [processing block](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprocessing_block_glosry.htm "Glossary Entry"), without actually knowing the processing block.

Either a component of a [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") or an [interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoo_intf_glosry.htm "Glossary Entry") declared using [*\[*CLASS-*\]*](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclass-events.htm)[EVENTS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapevents.htm), or an event of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry"). Events in classes trigger [event handlers](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenevent_handler_glosry.htm "Glossary Entry").

Events of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") trigger the processing of [event blocks](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenevent_block_glosry.htm "Glossary Entry").
