extract dataset

Structured unnamed dataset of an [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry"). Consists of lines that are organized as [field groups](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfield_group_glosry.htm "Glossary Entry"). Not part of the ABAP type concept for [data types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_type_glosry.htm "Glossary Entry") and [data objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry").
