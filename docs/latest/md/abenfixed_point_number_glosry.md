fixed point number

Number with a fixed number of [decimal places](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendecimal_place_glosry.htm "Glossary Entry"). In ABAP, fixed point numbers are implemented using [packed numbers](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpacked_number_glosry.htm "Glossary Entry") where a sequence of digits contains a decimal separator at a position defined by a [packed number type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpacked_number_type_glosry.htm "Glossary Entry").
