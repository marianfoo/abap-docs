floating point arithmetic

Calculation with [floating point numbers](javascript:call_link\('abenfloating_point_number_glosry.htm'\) "Glossary Entry"). It is executed directly on the [host computer](javascript:call_link\('abenhost_computer_glosry.htm'\) "Glossary Entry") of the current [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry"). Either [binary floating point arithmetic](javascript:call_link\('abenbinfloat_arith_glosry.htm'\) "Glossary Entry") or [decimal floating point arithmetic](javascript:call_link\('abendecfloat_arith_glosry.htm'\) "Glossary Entry").
