floating point arithmetic

Calculation with [floating point numbers](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfloating_point_number_glosry.htm "Glossary Entry"). It is executed directly on the [host computer](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhost_computer_glosry.htm "Glossary Entry") of the current [AS instance](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_instance_glosry.htm "Glossary Entry"). Either [binary floating point arithmetic](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbinfloat_arith_glosry.htm "Glossary Entry") or [decimal floating point arithmetic](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendecfloat_arith_glosry.htm "Glossary Entry").
