foreign key dependency

Relationship between a [foreign key table](javascript:call_link\('abenforeign_key_table_glosry.htm'\) "Glossary Entry") and a [check table](javascript:call_link\('abencheck_table_glosry.htm'\) "Glossary Entry"). A foreign key table usually only contains entries for which the content of the [foreign key](javascript:call_link\('abenforeign_key_glosry.htm'\) "Glossary Entry") also occurs as content of the [primary key](javascript:call_link\('abenprimary_key_glosry.htm'\) "Glossary Entry") in the check table.
