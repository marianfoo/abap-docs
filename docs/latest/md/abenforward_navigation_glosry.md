forward navigation

Method used to navigate to the associated tool of a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") within the [ABAP Workbench](javascript:call_link\('abenabap_workbench_glosry.htm'\) "Glossary Entry"). To do this, the [cursor](javascript:call_link\('abencursor_glosry.htm'\) "Glossary Entry") must be positioned on the object name and the [object](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") must be selected.
