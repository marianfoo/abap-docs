forward navigation

Method used to navigate to the associated tool of a [repository object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry") within the [ABAP Workbench](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_workbench_glosry.htm "Glossary Entry"). To do this, the [cursor](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencursor_glosry.htm "Glossary Entry") must be positioned on the object name and the [object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") must be selected.
