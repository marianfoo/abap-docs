frame

[Screen element](javascript:call_link\('abenscreen_element_glosry.htm'\) "Glossary Entry") used to group other screen elements and on which no [user actions](javascript:call_link\('abenuser_action_glosry.htm'\) "Glossary Entry") are possible.
