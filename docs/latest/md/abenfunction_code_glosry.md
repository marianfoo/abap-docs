function code

Sequence of up to 20 characters that can be assigned to specific [control elements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_element_glosry.htm "Glossary Entry") of the [user interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_interface_glosry.htm "Glossary Entry") in [SAP GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_gui_glosry.htm "Glossary Entry"). When selecting one of these control elements, the [dynpro event](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_event_glosry.htm "Glossary Entry") [PAI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpai_glosry.htm "Glossary Entry") is raised and the function code is transferred to the [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") using the [system field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensystem_field_glosry.htm "Glossary Entry") [sy-ucomm](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensystem_fields.htm) or the [OK field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenok_field_glosry.htm "Glossary Entry") of the [dynpro](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_glosry.htm "Glossary Entry").
