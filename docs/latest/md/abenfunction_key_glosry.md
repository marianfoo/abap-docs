function key

[Control element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_element_glosry.htm "Glossary Entry") of the [user interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_interface_glosry.htm "Glossary Entry"). Defines keys or key combinations on an input device. A [function code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_code_glosry.htm "Glossary Entry") can be assigned to a function key in the [function key settings](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_key_setting_glosry.htm "Glossary Entry").
