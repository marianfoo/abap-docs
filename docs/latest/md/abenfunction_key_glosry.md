function key

[Control element](javascript:call_link\('abencontrol_element_glosry.htm'\) "Glossary Entry") of the [user interface](javascript:call_link\('abenuser_interface_glosry.htm'\) "Glossary Entry"). Defines keys or key combinations on an input device. A [function code](javascript:call_link\('abenfunction_code_glosry.htm'\) "Glossary Entry") can be assigned to a function key in the [function key settings](javascript:call_link\('abenfunction_key_setting_glosry.htm'\) "Glossary Entry").
