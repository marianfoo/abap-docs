fuzzy search index

[Index](javascript:call_link\('abenindex_glosry.htm'\) "Glossary Entry") of the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") that database that provides fast error-tolerant text search, where a query returns records even when the search term contains additional or missing characters or other types of spelling errors. The [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") allows fuzzy search indexes to be created for [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry").
