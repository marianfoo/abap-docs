fuzzy search index

[Index](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenindex_glosry.htm "Glossary Entry") of the [SAP HANA database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_database_glosry.htm "Glossary Entry") that database that provides fast error-tolerant text search, where a query returns records even when the search term contains additional or missing characters or other types of spelling errors. The [ABAP Dictionary](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_dictionary_glosry.htm "Glossary Entry") allows fuzzy search indexes to be created for [DDIC database tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenddic_db_table_glosry.htm "Glossary Entry").
