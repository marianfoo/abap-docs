generic ABAP type

[Built-in](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuiltin_data_type_glosry.htm "Glossary Entry") [generic data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengeneric_data_type_glosry.htm "Glossary Entry"). The generic ABAP types are: any, any table, c, clike, csequence, data, decfloat, hashed table, index table, n, numeric, p, simple, sorted table, standard table, table, x, and xsequence.
