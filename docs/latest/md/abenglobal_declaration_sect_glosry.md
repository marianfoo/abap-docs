global declaration part

Area following the opening statement of an [ABAP program](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") in which [interfaces](javascript:call_link\('abenoo_intf_glosry.htm'\) "Glossary Entry"), [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry"), [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") and [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") that are visible in the whole program can be declared.
