graph procedure

A graph procedure is an [ABAP Managed Database Procedure](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_managed_db_proc_glosry.htm "Glossary Entry"). It refers to [graph workspace](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengraph_workspace_glosry.htm "Glossary Entry") and operates on [graph](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengraph_glosry.htm "Glossary Entry") models. It is a read-only procedure that is written in [GraphScript](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengraphscript_glosry.htm "Glossary Entry").
