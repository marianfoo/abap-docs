graph procedure

A graph procedure is an [ABAP Managed Database Procedure](javascript:call_link\('abenabap_managed_db_proc_glosry.htm'\) "Glossary Entry"). It refers to [graph workspace](javascript:call_link\('abengraph_workspace_glosry.htm'\) "Glossary Entry") and operates on [graph](javascript:call_link\('abengraph_glosry.htm'\) "Glossary Entry") models. It is a read-only procedure that is written in [GraphScript](javascript:call_link\('abengraphscript_glosry.htm'\) "Glossary Entry").
