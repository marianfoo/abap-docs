grouping function

[Aggregate function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenaggregate_function_glosry.htm "Glossary Entry") for [GROUPING SETS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapgrouping_sets_clause.htm) in a [query](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenquery_glosry.htm "Glossary Entry"). The result of the grouping function GROUPING indicates whether a specific column was part of the aggregation. The return value of the grouping function is 0 for the column that was not involved in the aggregation or 1 for the column that was involved in the aggregation.
