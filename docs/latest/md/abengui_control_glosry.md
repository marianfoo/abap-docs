GUI control

Component of the [presentation view](javascript:call_link\('abenpresentation_layer_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Examples of GUI controls are the Picture Control, the Text Edit Control, and the Tree Control. GUI controls are installed with the [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry") and are addressed in ABAP using [classes](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") of the [Control Framework](javascript:call_link\('abencontrol_framework_glosry.htm'\) "Glossary Entry").
