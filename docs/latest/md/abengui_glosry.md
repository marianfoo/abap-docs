GUI

Short for Graphical User Interface. Graphical part of the [user interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_interface_glosry.htm "Glossary Entry"). The GUI is displayed in a [window](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwindow_glosry.htm "Glossary Entry").
