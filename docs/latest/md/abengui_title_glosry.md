GUI title

Text that can be displayed in the [title bar](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentitle_bar_glosry.htm "Glossary Entry") of a [GUI window](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengui_window_glosry.htm "Glossary Entry"). A GUI title is set using the statement [SET TITLEBAR](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapset_titlebar_dynpro.htm) and created using the [Menu Painter](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmenu_painter_glosry.htm "Glossary Entry") tool.
