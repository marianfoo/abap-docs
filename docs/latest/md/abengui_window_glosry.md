GUI window

[Window](javascript:call_link\('abenwindow_glosry.htm'\) "Glossary Entry") of the [SAP GUI](javascript:call_link\('abensap_gui_glosry.htm'\) "Glossary Entry"). Consists of a [title bar](javascript:call_link\('abentitle_bar_glosry.htm'\) "Glossary Entry"), [menu bar](javascript:call_link\('abenmenu_bar_glosry.htm'\) "Glossary Entry"), [standard toolbar](javascript:call_link\('abenstandard_toolbar_glosry.htm'\) "Glossary Entry"), [application toolbar](javascript:call_link\('abenapplication_toolbar_glosry.htm'\) "Glossary Entry"), [screen layout](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry"), [status bar](javascript:call_link\('abenstatus_bar_glosry.htm'\) "Glossary Entry"), and any [scroll bars](javascript:call_link\('abenscroll_bar_glosry.htm'\) "Glossary Entry"). Each GUI window is linked with an [ABAP session](javascript:call_link\('abenabap_session_glosry.htm'\) "Glossary Entry"). A maximum of 16 ABAP sessions and therefore GUI windows can be managed by a single [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry").
