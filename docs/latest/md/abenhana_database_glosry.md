SAP HANA database

The SAP HANA database is the in-memory database of [SAP HANA](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_hana_glosry.htm "Glossary Entry"). It is the only database that is supported as the [standard database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstandard_db_glosry.htm "Glossary Entry") of an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry").
