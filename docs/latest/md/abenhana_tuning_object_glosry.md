HANA tuning object

[Tuning object](javascript:call_link\('abentuning_object_glosry.htm'\) "Glossary Entry") that defines technical settings for a [repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") on the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). Currently, there are the following HANA tuning objects:

-   [DDIC dependency rules](javascript:call_link\('abendependency_rule_glosry.htm'\) "Glossary Entry")
-   [DDIC dynamic caches](javascript:call_link\('abendynamic_cache_glosry.htm'\) "Glossary Entry")
