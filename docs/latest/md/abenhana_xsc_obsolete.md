Obsolete Access to Objects in HANA XS

[SAP HANA XSA](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenxsa_glosry.htm "Glossary Entry") is the successor of [SAP HANA XSC](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenxsc_glosry.htm "Glossary Entry"). The access methods designed specifically for XSC objects are not suitable for XSA objects.

The following access methods for XSC objects are obsolete:

-   [DDIC external views](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenexternal_view_glosry.htm "Glossary Entry")
-   [Database procedure proxies](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_proc_proxy_glosry.htm "Glossary Entry")

This is also described in SAP Notes [3116165](https://launchpad.support.sap.com/#/notes/3116165) and [2465027](https://launchpad.support.sap.com/#/notes/2465027).
