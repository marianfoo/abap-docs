header line

Obsolete [work area](javascript:call_link\('abenwork_area_glosry.htm'\) "Glossary Entry") of an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry") whose type is the [line type](javascript:call_link\('abenrow_type_glosry.htm'\) "Glossary Entry") and has the same name as the internal table. When an internal table with a header line is used in an [operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry"), the header line is usually addressed. To force access to the [table body](javascript:call_link\('abentable_body_glosry.htm'\) "Glossary Entry"), square brackets \[\] can be specified after the table name.
