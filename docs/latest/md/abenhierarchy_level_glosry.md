hierarchy level

Level of a [hierarchy node](javascript:call_link\('abenhierarchy_node_glosry.htm'\) "Glossary Entry") in a [hierarchy](javascript:call_link\('abenhierarchy_glosry.htm'\) "Glossary Entry"). In an [SQL hierarchy](javascript:call_link\('abensql_hierarchy_glosry.htm'\) "Glossary Entry"), the [hierarchy column](javascript:call_link\('abenhierarchy_column_glosry.htm'\) "Glossary Entry") [HIERARCHY\_LEVEL](javascript:call_link\('abenddddl_hierarchy.htm'\)) contains the value of the hierarchy level of a hierarchy node.
