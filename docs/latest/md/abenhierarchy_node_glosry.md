hierarchy node

Hierarchy nodes are the objects of a [hierarchy](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhierarchy_glosry.htm "Glossary Entry") linked using [hierarchy edges](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhierarchy_edge_glosry.htm "Glossary Entry"). Hierarchy nodes can be [root nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenroot_node_glosry.htm "Glossary Entry"), [parent nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenparent_node_glosry.htm "Glossary Entry"), [child nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenparent_node_glosry.htm "Glossary Entry"), [leaf nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenleaf_node_glosry.htm "Glossary Entry"), [orphan nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenorphan_node_glosry.htm "Glossary Entry"), [sibling nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensibling_node_glosry.htm "Glossary Entry"), [descendant nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendescendant_node_glosry.htm "Glossary Entry"), and [ancestor nodes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenancestor_node_glosry.htm "Glossary Entry").

-   In an [SQL hierarchy](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_hierarchy_glosry.htm "Glossary Entry"), a hierarchy node is a row of the tabular [SQL data source](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_data_source_glosry.htm "Glossary Entry") and contains special [hierarchy columns](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhierarchy_column_glosry.htm "Glossary Entry") for the [hierarchy attributes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhierarchy_attribute_glosry.htm "Glossary Entry") of the SQL hierarchy.
-   In a [CDS hierarchy](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_hierarchy_glosry.htm "Glossary Entry"), a hierarchy node is a row of the tabular result set but without the [hierarchy columns](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhierarchy_column_glosry.htm "Glossary Entry").
