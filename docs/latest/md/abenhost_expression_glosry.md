host expression

ABAP-[expression](javascript:call_link\('abenexpression_glosry.htm'\) "Glossary Entry") specified in an operand position of an [ABAP SQL](javascript:call_link\('abenabap_sql_host_expressions.htm'\)) statement. Host expressions are always specified in parentheses preceded by the [escape character](javascript:call_link\('abenescape_character_glosry.htm'\) "Glossary Entry") @: @( ... ).
