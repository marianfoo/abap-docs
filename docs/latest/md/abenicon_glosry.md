icon

Graphical element of the [GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengui_glosry.htm "Glossary Entry") for use on [pushbuttons](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpushbutton_glosry.htm "Glossary Entry") and also as a [status icon](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstatus_icon_glosry.htm "Glossary Entry") on [screens](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_glosry.htm "Glossary Entry") or in [lists](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclassic_list_glosry.htm "Glossary Entry"). The program SHOWICON shows a list of possible icons.
