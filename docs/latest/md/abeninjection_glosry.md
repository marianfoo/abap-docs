injection

Test-friendly code located between the statements [TEST-INJECTION](javascript:call_link\('abaptest-injection.htm'\)) and [END-TEST-INJECTION](javascript:call_link\('abapend-test-injection.htm'\)) in a [method](javascript:call_link\('abenmethod_glosry.htm'\) "Glossary Entry") of a [test class](javascript:call_link\('abentest_class_glosry.htm'\) "Glossary Entry"). Injections replace the associated [test seams](javascript:call_link\('abentest_seam_glosry.htm'\) "Glossary Entry") when a [unit test](javascript:call_link\('abenunit_test_glosry.htm'\) "Glossary Entry") is executed.
