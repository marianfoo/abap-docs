injection

Test-friendly code located between the statements [TEST-INJECTION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abaptest-injection.htm) and [END-TEST-INJECTION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapend-test-injection.htm) in a [method](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmethod_glosry.htm "Glossary Entry") of a [test class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_class_glosry.htm "Glossary Entry"). Injections replace the associated [test seams](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_seam_glosry.htm "Glossary Entry") when a [unit test](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenunit_test_glosry.htm "Glossary Entry") is executed.
