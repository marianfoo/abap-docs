inner join

A [join type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenjoin_type_glosry.htm "Glossary Entry") for which each row of the result set of the [join](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenjoin_glosry.htm "Glossary Entry") fulfills the [join conditions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenjoin_condition_glosry.htm "Glossary Entry").
