interface component selector

Character ~. An [interface component](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninterface_component_glosry.htm "Glossary Entry") comp of an [interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoo_intf_glosry.htm "Glossary Entry") intf can be addressed in an implementing [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") or a [compound interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencompound_interface_glosry.htm "Glossary Entry") via intf~comp.
