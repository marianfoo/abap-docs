interface reference variable

[Object reference variable](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_refer_variable_glosry.htm "Glossary Entry") of the [static type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstatic_type_glosry.htm "Glossary Entry") of an [interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoo_intf_glosry.htm "Glossary Entry"). An interface reference variable can be used to access the visible [interface components](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninterface_component_glosry.htm "Glossary Entry") of an object that exists in the corresponding interface. This applies to both dynamic and static access.
