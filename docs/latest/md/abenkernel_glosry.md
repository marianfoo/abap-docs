ABAP kernel

Part of the [ABAP runtime framework](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_runtime_frmwk_glosry.htm "Glossary Entry") which is primarily programmed in C/C++. In particular, [ABAP statements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_statement_glosry.htm "Glossary Entry") are implemented by C functions in the ABAP kernel. When an ABAP statement is executed, the corresponding C functions or the ABAP kernel are called.
