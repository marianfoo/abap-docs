list buffer

Memory area for storing [screen lists](javascript:call_link\('abenscreen_list_glosry.htm'\) "Glossary Entry"). Subdivided into [list levels](javascript:call_link\('abenlist_level_glosry.htm'\) "Glossary Entry"). For each [dynpro sequence](javascript:call_link\('abendynpro_sequence_glosry.htm'\) "Glossary Entry"), the list buffer manages a hierarchy consisting of a [basic list](javascript:call_link\('abenbasic_list_glosry.htm'\) "Glossary Entry") and up to 20 [details lists](javascript:call_link\('abendetails_list_glosry.htm'\) "Glossary Entry").
