LOB handle

An object from one of the [system classes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensystem_class_glosry.htm "Glossary Entry") that support [streaming](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_streaming.htm) and [locators](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlocators.htm) for [LOBs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlob_glosry.htm "Glossary Entry") in [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry"). These classes all implement the [interface](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoo_intf_glosry.htm "Glossary Entry") IF\_ABAP\_DB\_LOB\_HANDLE. See also: [LOB Handle Structure](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlob_handle_structure_glosry.htm "Glossary Entry").
