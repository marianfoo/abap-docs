local data queue

Persistence layer into which data can be placed and read only in the order in which it was stored. The local data queue triggers the no-send scenario of the [qRFC](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenqrfc_glosry.htm "Glossary Entry").
