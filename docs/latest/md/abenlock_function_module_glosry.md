lock function module

Special [function module](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_module_glosry.htm "Glossary Entry") for setting and removing [SAP locks](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_lock_glosry.htm "Glossary Entry"). When creating a [lock object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlock_object_glosry.htm "Glossary Entry"), a function module with the prefix ENQUEUE\_ is created to set the lock, and a function module with the prefix DEQUEUE\_ is created to remove the lock.
