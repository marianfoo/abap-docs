lock object

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") defined in the [ABAP Dictionary](javascript:call_link\('abenabap_dictionary_glosry.htm'\) "Glossary Entry") that is used as the basis for [SAP locks](javascript:call_link\('abensap_lock_glosry.htm'\) "Glossary Entry"). In a lock object [DDIC database tables](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") are specified to which a common [lock](javascript:call_link\('abenlock_glosry.htm'\) "Glossary Entry") is to be applied and which are linked by [foreign key dependencies](javascript:call_link\('abenforeign_key_dependency_glosry.htm'\) "Glossary Entry"). When a lock object is created, two [lock function modules](javascript:call_link\('abenlock_function_module_glosry.htm'\) "Glossary Entry") are automatically generated.
