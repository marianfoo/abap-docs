logical function

[Built-in function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuiltin_function_glosry.htm "Glossary Entry"). Either a [Boolean function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenboole_function_glosry.htm "Glossary Entry") that evaluates a [logical expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogical_expression_glosry.htm "Glossary Entry") or a [predicate function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpredicate_function_glosry.htm "Glossary Entry") that returns a [logical value](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentruth_value_glosry.htm "Glossary Entry").
