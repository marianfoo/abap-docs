logical database

Special [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") that provides other ABAP programs with data from the nodes of a hierarchical tree structure (obsolete). Is either linked to an [executable program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenexecutable_program_glosry.htm "Glossary Entry") in its [program properties](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprogram_property_glosry.htm "Glossary Entry") or is called using [function module](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_module_glosry.htm "Glossary Entry") LDB\_PROCESS. A logical database is made up of a hierarchical structure with nodes, an ABAP [database program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_program_glosry.htm "Glossary Entry") and its own [standard selection screen](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstandard_selscreen_glosry.htm "Glossary Entry"). Logical databases are maintained in the [Logical Database Builder](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogical_database_build_glosry.htm "Glossary Entry").
