Expressions and Functions for Conditions

The flow control of a program is performed using [control structures](javascript:call_link\('abencontrol_structure_glosry.htm'\) "Glossary Entry") that are generally controlled by logical expressions. The following options exist for the formulation and evaluation of conditions:

-   [Logical expressions](javascript:call_link\('abenlogexp.htm'\))
-   [Logical functions](javascript:call_link\('abenlogic_functions.htm'\))

In addition to control structures, logical expressions and functions can also be listed in other statements.
