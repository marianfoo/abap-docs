logon

Time at which an [ABAP user](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_user_glosry.htm "Glossary Entry") logs on to the [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") by specifying the [client ID](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclient_identifier_glosry.htm "Glossary Entry"), [user name](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_name_glosry.htm "Glossary Entry"), [password](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpassword_glosry.htm "Glossary Entry"), and the [logon language](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogon_language_glosry.htm "Glossary Entry"). Each logon starts a [user session](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_session_glosry.htm "Glossary Entry"). Logons are made using the [SAP GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_gui_glosry.htm "Glossary Entry") logon window or using frameworks such as [RFC](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrfc_glosry.htm "Glossary Entry"), [ICF](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenicf_glosry.htm "Glossary Entry"), or [APC](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc_glosry.htm "Glossary Entry").
