mesh path expression

Standalone specification of a [mesh path](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmesh_path_glosry.htm "Glossary Entry") in an [operand position](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoperand_position_glosry.htm "Glossary Entry") for which the same rules apply as for a [table expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_expression_glosry.htm "Glossary Entry").
