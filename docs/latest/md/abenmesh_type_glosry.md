mesh type

[Data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_type_glosry.htm "Glossary Entry") of a [mesh](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmesh_glosry.htm "Glossary Entry"). Special [structured type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstructured_type_glosry.htm "Glossary Entry") that contains only [table types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_type_glosry.htm "Glossary Entry") with structured [line types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrow_type_glosry.htm "Glossary Entry") as [components](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstructure_component_glosry.htm "Glossary Entry") that can be linked using [mesh associations](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmesh_association_glosry.htm "Glossary Entry").
