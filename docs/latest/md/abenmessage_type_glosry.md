message type

Must be specified when the statement [MESSAGE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapmessage.htm) is used to send a [message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenmessage_glosry.htm "Glossary Entry"). It determines the representation of the message in the [SAP GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_gui_glosry.htm "Glossary Entry") and the subsequent program flow. The possible message types are [information message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninformation_message_glosry.htm "Glossary Entry"), [status message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstatus_message_glosry.htm "Glossary Entry"), [error message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenerror_message_glosry.htm "Glossary Entry"), [warning](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwarning_glosry.htm "Glossary Entry"), [termination message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentermination_message_glosry.htm "Glossary Entry"), and [exit message](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenexit_message_glosry.htm "Glossary Entry").
