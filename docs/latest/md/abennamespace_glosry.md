namespace

A namespace is a set of unique names that identify a set of objects. In ABAP, we can distinguish [namespaces for program-internal objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abennaming_conventions.htm) and a namespace for [repository objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry"). The latter can be divided into a [SAP namespace](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_namespace_glosry.htm "Glossary Entry"), a [customer namespace](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencustomer_namespace_glosry.htm "Glossary Entry"), a [partner namespace](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpartner_namespace_glosry.htm "Glossary Entry") and [prefix namespaces](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprefix_name_space_glosry.htm "Glossary Entry").
