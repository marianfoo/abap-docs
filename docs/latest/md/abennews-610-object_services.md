Object Services in ABAP Release 6.10

Object services are low-level services used for object-oriented programming with ABAP Objects, but which are not implemented as language elements. In ABAP release 6.10, persistence services and transaction services are available. For more details, see [ABAP - Object Services](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_object_services.htm).
