Lists in ABAP Release 7.0, EhP2

Modification

Support for the New 12 Hour Format

The new addition [ENVIRONMENT TIME FORMAT](javascript:call_link\('abapwrite_to_options.htm'\)) to the WRITE TO and WRITE statements supports the new [12 hour format](javascript:call_link\('abennews-710-others.htm'\)).

The [predefined time formats](javascript:call_link\('abenwrite_formats.htm'\)) of the statements WRITE TO and WRITE cannot be affected automatically due to downward compatibility.

Hint

This modification was also added to ABAP release 7.0 in SP14.
