ABAP CDS in ABAP Release 7.40, SP05

The new [ABAP CDS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds.htm) is the ABAP-specific implementation of the general [Core Data Services (CDS)](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencore_data_services_glosry.htm "Glossary Entry"). In its first phase, ABAP CDS provides a [DDL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_f1_ddl_syntax.htm) for the definition of

[CDS views](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_v1_views.htm), used together with ABAP SQL to perform reads.
