ABAP Channels in ABAP Release 7.40, SP05

In ABAP release 7.40, two new communication channels were introduced, which can also be connected to each other. ABAP Messaging Channels can be used from [SP02](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abennews-740-abap_channels.htm).

[1\. ABAP Push Channels](#!ABAP_MODIFICATION_1@1@)
[2\. New Variant of WAIT UNTIL](#!ABAP_MODIFICATION_2@2@)

Modification 1   

ABAP Push Channels

[ABAP Push Channels (APC)](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc.htm) enable bidirectional communication between AS ABAP and the internet using the WebSocket protocol.

Modification 2   

New Variant of WAIT UNTIL

The new variant [WAIT FOR PUSH CHANNELS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapwait_apc.htm) waits for APC messages in [ABAP Push Channels (APC)](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenapc.htm).
