Standard Database in ABAP Release 7.53

Modification

Only SAP HANA Database as Standard Database

As of this release, only an [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry") is supported as the [standard database](javascript:call_link\('abenstandard_db_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). For this reason, Open SQL was renamed to ABAP SQL (see [ABAP SQL in ABAP Release 7.53](abennews-753-abap_sql.htm#!ABAP_MODIFICATION_1@1@)). Note: Using a [secondary connection](javascript:call_link\('abensecondary_db_connection_glosry.htm'\) "Glossary Entry") in [ABAP SQL](javascript:call_link\('abenabap_sql_glosry.htm'\) "Glossary Entry"), it is still possible to connect to other databases.
