System Classes in ABAP Release 7.53

Modification

System Classes for RAP Runtime Context Information

The new system class CL\_ABAP\_BEHV\_AUX is available for retrieving RAP runtime context information. Using the GET\_CURRENT\_HANDLER\_KIND method, you get the information about which RAP handler method is currently running.
