Classic Lists in ABAP Release 7.55

Modification

Tooltips for Input Ready Fields

If the profile parameter dynp/enhanced\_system\_feedback has the value TRUE, the addition [QUICKINFO](abapwrite-.htm#!ABAP_ADDITION_4@4@) is not ignored for [checkboxes](abapwrite_list_elements.htm#!ABAP_ALTERNATIVE_1@1@) on classic lists in SAP GUI for Windows any more.
