System Classes in ABAP Release 7.56

Modification

New Method for System Class CL\_ABAP\_BEHV\_AUX

The system class CL\_ABAP\_BEHV\_AUX has a new method available: RAP runtime context information. Using the GET\_CURRENT\_CONTEXT method, you get context information about the current RAP handler and RAP saver method.
