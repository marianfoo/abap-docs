Expressions and Functions in ABAP Release 7.57

Modification

Inline Declarations

The new [declaration operator](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendeclaration_operator_glosry.htm "Glossary Entry") [FINAL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfinal_inline.htm) declares an [immutable variable](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenimmutable_variable_glosry.htm "Glossary Entry") that cannot be assigned another value at other write positions of the same context.
