numeric data type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") whose [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") contain numeric values in platform-dependent encoding. These data types are [integer type](javascript:call_link\('abeninteger_type_glosry.htm'\) "Glossary Entry"), [packed number type](javascript:call_link\('abenpacked_number_type_glosry.htm'\) "Glossary Entry"), [decimal floating pointing number type](javascript:call_link\('abendecfloat_type_glosry.htm'\) "Glossary Entry") and [binary floating pointing number type](javascript:call_link\('abenbinfloat_type_glosry.htm'\) "Glossary Entry"). The corresponding [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [numeric](javascript:call_link\('abenbuilt_in_types_generic.htm'\)).
