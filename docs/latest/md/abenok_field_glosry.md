OK field

Twenty-character [dynpro field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_field_glosry.htm "Glossary Entry") that is not linked to a [screen element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_element_glosry.htm "Glossary Entry"). When a [user action](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_action_glosry.htm "Glossary Entry") is performed on a [control element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_element_glosry.htm "Glossary Entry") that is linked to a [function code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfunction_code_glosry.htm "Glossary Entry"), the function code is placed in the OK field and is transferred to a [data object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry") of the same name during [PAI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpai_glosry.htm "Glossary Entry").
