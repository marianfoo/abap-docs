OK field

Twenty-character [dynpro field](javascript:call_link\('abendynpro_field_glosry.htm'\) "Glossary Entry") that is not linked to a [screen element](javascript:call_link\('abenscreen_element_glosry.htm'\) "Glossary Entry"). When a [user action](javascript:call_link\('abenuser_action_glosry.htm'\) "Glossary Entry") is performed on a [control element](javascript:call_link\('abencontrol_element_glosry.htm'\) "Glossary Entry") that is linked to a [function code](javascript:call_link\('abenfunction_code_glosry.htm'\) "Glossary Entry"), the function code is placed in the OK field and is transferred to a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") of the same name during [PAI](javascript:call_link\('abenpai_glosry.htm'\) "Glossary Entry").
