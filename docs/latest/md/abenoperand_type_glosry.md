operand type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") with which an [operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry") is typed. The operand type can be complete or generic. An [operand](javascript:call_link\('abenoperand_glosry.htm'\) "Glossary Entry") must match the operand type. Depending on the statement, [type conversions](javascript:call_link\('abentype_conversion_glosry.htm'\) "Glossary Entry") or [castings](javascript:call_link\('abencast_casting_glosry.htm'\) "Glossary Entry") take place for incompatible operands. In [inline declarations](javascript:call_link\('abeninline_declaration_glosry.htm'\) "Glossary Entry"), the operand types determine the declared data type. For [constructor expressions](javascript:call_link\('abenconstructor_expression_glosry.htm'\) "Glossary Entry"), the operand type can be used to infer the type of the result.
