password

Sequence of characters that is only known to a specific [ABAP user](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_user_glosry.htm "Glossary Entry") and is required from this user to [log on](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogon_glosry.htm "Glossary Entry") to [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry"). Used together with the [user name](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_name_glosry.htm "Glossary Entry"), the password enables authorized use of an AS ABAP.
