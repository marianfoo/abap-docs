period

Time interval whose limits are defined by the content of the columns of a [temporal SQL hierarchy](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentemporal_hierarchy_glosry.htm "Glossary Entry"). These columns are defined when the [SQL hierarchy](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_hierarchy_glosry.htm "Glossary Entry") is created.
