persistent object

[Object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") in a [persistent class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpersistent_class_glosry.htm "Glossary Entry") whose [attributes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenattribute_glosry.htm "Glossary Entry") are saved as database content after the runtime of an [ABAP program](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry"). Persistent objects are managed by [object services](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_services_glosry.htm "Glossary Entry").
