popup level

Hierarchical level of a [GUI window](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengui_window_glosry.htm "Glossary Entry"). The first GUI window that appears during a [logon](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogon_glosry.htm "Glossary Entry") to [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") has the popup level 0. Higher popup levels are created by stacking modal [dialog boxes](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendialog_box_glosry.htm "Glossary Entry") using the statement [CALL SCREEN](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapcall_screen.htm) or [CALL SELECTION-SCREEN](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapcall_selection_screen.htm). The maximum popup level is 9.
