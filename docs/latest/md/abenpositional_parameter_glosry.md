positional parameter

[Formal parameter](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenformal_parameter_glosry.htm "Glossary Entry") or other parameter to which an [actual parameter](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenactual_parameter_glosry.htm "Glossary Entry") is passed according to its position in the calling statement. See also [keyword parameter](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenkeyword_parameter_glosry.htm "Glossary Entry").
