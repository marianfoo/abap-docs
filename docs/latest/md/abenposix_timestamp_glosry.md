POSIX time stamp

[UTC time stamp](javascript:call_link\('abenutc_timestamp_glosry.htm'\) "Glossary Entry") where each day has 86400 seconds in accordance with the standard [POSIX](https://en.wikipedia.org/wiki/POSIX) 1:2001 and [leap seconds](javascript:call_link\('abenleap_second_glosry.htm'\) "Glossary Entry") are not supported. A time stamp that is created during a leap second 23:59:60 is rounded to a valid time.
