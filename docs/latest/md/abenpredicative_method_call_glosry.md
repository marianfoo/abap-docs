predicative method call

Call of a [functional method](javascript:call_link\('abenfunctional_method_glosry.htm'\) "Glossary Entry") as the only [ABAP operand](javascript:call_link\('abenabap_operand_glosry.htm'\) "Glossary Entry") of a [relational expression](javascript:call_link\('abenrelational_expression_glosry.htm'\) "Glossary Entry"). The result of the relational expression is true if the result of the [functional method call](javascript:call_link\('abenfunctional_method_call_glosry.htm'\) "Glossary Entry") is not initial. See also [predicate method](javascript:call_link\('abenpredicate_method_glosry.htm'\) "Glossary Entry").
