prefix namespace

[Namespace](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abennamespace_glosry.htm "Glossary Entry") for [repository objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry") of SAP departments, SAP partners, and SAP customers that can be requested from SAP. The names of the repository objects of a prefix namespace are introduced with a [namespace prefix](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenname_space_prefix_glosry.htm "Glossary Entry").
