presentation server

Front-end computer belonging to an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") user. The [presentation layer](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenpresentation_layer_glosry.htm "Glossary Entry") on these computers is provided either by installing [SAP GUI](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_gui_glosry.htm "Glossary Entry") or by a web browser.
