primary key

[Table key](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentable_key_glosry.htm "Glossary Entry") that is determined when a [database table](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_table_glosry.htm "Glossary Entry") is defined and which uniquely identifies a table row. In [internal tables](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninternal_table_glosry.htm "Glossary Entry"), there is a [primary table key](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprimary_table_key_glosry.htm "Glossary Entry").
