primary table index

[Table index](javascript:call_link\('abentable_index_glosry.htm'\) "Glossary Entry") of [index tables](javascript:call_link\('abenindex_table_glosry.htm'\) "Glossary Entry") ([standard tables](javascript:call_link\('abenstandard_table_glosry.htm'\) "Glossary Entry") and [sorted tables](javascript:call_link\('abensorted_table_glosry.htm'\) "Glossary Entry")). Every index table has a primary table index, administrated by the [ABAP runtime framework](javascript:call_link\('abenabap_runtime_frmwk_glosry.htm'\) "Glossary Entry"), which contains a unique line number for each table line. The primary table index of an index table is updated each time a line is added or deleted from an internal table.
