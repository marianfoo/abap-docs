spool parameter

Parameters that must be passed to a [spool request](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_request_glosry.htm "Glossary Entry"). [Archiving parameters](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenarchiving_parameter_glosry.htm "Glossary Entry") can be added to spool parameters.
