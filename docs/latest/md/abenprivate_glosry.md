private

Term in [ABAP Objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_objects_glosry.htm "Glossary Entry"). A private [component](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_component_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") can be accessed only by the class itself and its [friends](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfriend_glosry.htm "Glossary Entry"), which also only can create [objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") of a class with private instances. The private [visibility section](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenvisibility_section_glosry.htm "Glossary Entry") of a class is declared in its [declaration part](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendeclaration_part_glosry.htm "Glossary Entry") using the statement [PRIVATE SECTION](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapprivate.htm).
