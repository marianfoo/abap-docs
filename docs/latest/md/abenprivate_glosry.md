private

Term in [ABAP Objects](javascript:call_link\('abenabap_objects_glosry.htm'\) "Glossary Entry"). A private [component](javascript:call_link\('abenclass_component_glosry.htm'\) "Glossary Entry") of a [class](javascript:call_link\('abenclass_glosry.htm'\) "Glossary Entry") can be accessed only by the class itself and its [friends](javascript:call_link\('abenfriend_glosry.htm'\) "Glossary Entry"), which also only can create [objects](javascript:call_link\('abenobject_glosry.htm'\) "Glossary Entry") of a class with private instances. The private [visibility section](javascript:call_link\('abenvisibility_section_glosry.htm'\) "Glossary Entry") of a class is declared in its [declaration part](javascript:call_link\('abendeclaration_part_glosry.htm'\) "Glossary Entry") using the statement [PRIVATE SECTION](javascript:call_link\('abapprivate.htm'\)).
