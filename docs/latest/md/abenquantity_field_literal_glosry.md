quantity field literal

[Literal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_literal_glosry.htm "Glossary Entry") of type [QUAN](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenddic_builtin_types.htm) that is used to store a quantity. Only [typed](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentyped_literal_glosry.htm "Glossary Entry") quantity literals are available.
