queued Remote Function Call

qRFC for short. Obsolete extension of the [transactional remote function call](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentrfc_1_glosry.htm "Glossary Entry") enabling a call sequence to be determined.
