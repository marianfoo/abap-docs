radio button

[Screen element](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_element_glosry.htm "Glossary Entry") that is linked with a [dynpro field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynpro_field_glosry.htm "Glossary Entry") and in which the user can enter the value X or blank. Radio buttons are grouped together in [radio button groups](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenradio_button_group_glosry.htm "Glossary Entry").
