released API

[Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") or part of an repository object that is released for access in [ABAP Cloud](javascript:call_link\('abenabap_cloud_glosry.htm'\) "Glossary Entry"). A released API is based on two classifications:

-   A [release contract](javascript:call_link\('abenrelease_contract_glosry.htm'\) "Glossary Entry") that governs changes to released APIs.
-   Visibility for a [restricted ABAP language version](javascript:call_link\('abenrestricted_version_glosry.htm'\) "Glossary Entry").
