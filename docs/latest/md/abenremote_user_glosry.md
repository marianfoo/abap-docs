remote user

[User](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_user_glosry.htm "Glossary Entry") that is logged on to an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") via [remote function call](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenremote_function_call_glosry.htm "Glossary Entry"). The logon data and the [authorization](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenauthorization_glosry.htm "Glossary Entry") can be declared in the [RFC destination](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrfc_dest_glosry.htm "Glossary Entry").
