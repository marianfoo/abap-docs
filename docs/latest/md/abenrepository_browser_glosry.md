repository browser

Standard layout of the [Object Navigator](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_navigator_glosry.htm "Glossary Entry") for the structured display of [repository objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry") sorted according to their type.
