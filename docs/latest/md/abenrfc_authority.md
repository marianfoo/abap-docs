RFC - Authorizations

If the [profile parameter](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprofile_parameter_glosry.htm "Glossary Entry") auth/rfc\_authority\_check is set to 1, an automatic authorization check is executed when a function module is called by remote call.

More Information   

For detailed information about RFC authorizations, see the documentation [RFC](https://help.sap.com/docs/ABAP_PLATFORM_NEW/753088fc00704d0a80e7fbd6803c8adb/4888068AD9134076E10000000A42189D) in SAP Help Portal.
