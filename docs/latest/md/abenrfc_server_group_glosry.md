RFC server group

Group of multiple [AS instances](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_instance_glosry.htm "Glossary Entry") of an [AS ABAP](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_abap_glosry.htm "Glossary Entry") that are provided to process asynchronous [remote function calls](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenremote_function_call_glosry.htm "Glossary Entry") (RFC) concurrently. RFC server groups are created using [transaction code](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentransaction_glosry.htm "Glossary Entry") RZ12.
