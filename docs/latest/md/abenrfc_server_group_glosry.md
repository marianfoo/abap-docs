RFC server group

Group of multiple [AS instances](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry") of an [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") that are provided to process asynchronous [remote function calls](javascript:call_link\('abenremote_function_call_glosry.htm'\) "Glossary Entry") (RFC) concurrently. RFC server groups are created using [transaction code](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry") RZ12.
