SAP HANA Repository

Repository of objects in the [SAP HANA database](javascript:call_link\('abenhana_database_glosry.htm'\) "Glossary Entry"). More specifically, [XSC](javascript:call_link\('abenxsc_glosry.htm'\) "Glossary Entry") objects are stored in the SAP HANA Repository. [XSA](javascript:call_link\('abenxsa_glosry.htm'\) "Glossary Entry") objects, on the other hand, are not stored in the SAP HANA Repository but are usually deployed using [HDI](javascript:call_link\('abenhdi_glosry.htm'\) "Glossary Entry").
