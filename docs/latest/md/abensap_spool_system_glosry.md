SAP spool system

Process of an [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry") for managing sequential data streams that are stored as [spool lists](javascript:call_link\('abenspool_list_glosry.htm'\) "Glossary Entry"), printed on a printer, or stored using [ArchiveLink](javascript:call_link\('abenarchivelink_glosry.htm'\) "Glossary Entry"). [ABAP programs](javascript:call_link\('abenabap_program_glosry.htm'\) "Glossary Entry") can create [spool requests](javascript:call_link\('abenspool_request_glosry.htm'\) "Glossary Entry") during the creation of lists by including pages of lists as parts of a [spool list](javascript:call_link\('abenspool_list_glosry.htm'\) "Glossary Entry") output.
