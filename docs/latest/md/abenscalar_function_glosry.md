scalar function

In the [SAP HANA database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_database_glosry.htm "Glossary Entry"), a [database function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_function_glosry.htm "Glossary Entry") with a scalar or elementary return value. In [SQLScript](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensql_script_glosry.htm "Glossary Entry"), scalar functions can be used as operands in the same places as elementary variables or columns of database tables or views. A scalar function can be managed as an [AMDP function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenamdp_function_glosry.htm "Glossary Entry") in an [AMDP class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenamdp_class_glosry.htm "Glossary Entry").
