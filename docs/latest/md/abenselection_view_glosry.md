selection view

If [dynamic selections](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendynamic_selection_glosry.htm "Glossary Entry") are required for a node of a [logical database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenlogical_data_base_glosry.htm "Glossary Entry"), a selection view can be used to determine the node [fields](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfield_glosry.htm "Glossary Entry") for which the user can define dynamic selections.
