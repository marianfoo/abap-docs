self-reference

[Object reference variable](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_refer_variable_glosry.htm "Glossary Entry") called [me](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenme.htm) that is predefined as a local [constant](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenconstant_glosry.htm "Glossary Entry") in each [instance method](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninstance_method_glosry.htm "Glossary Entry"). It always references the current [instance](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninstance_glosry.htm "Glossary Entry") in which the method is being executed. The static type of me is the [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") in which the instance method is implemented.
