selection parameter

A component of a [selection screen](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenselection_screen_glosry.htm "Glossary Entry") defined by [PARAMETERS](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapparameters.htm) which is displayed as an [input field](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninput_output_field_glosry.htm "Glossary Entry") and which is represented internally as an [elementary data object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenelementary_data_object_glosry.htm "Glossary Entry").
