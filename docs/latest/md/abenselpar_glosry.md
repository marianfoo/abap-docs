selection parameter

A component of a [selection screen](javascript:call_link\('abenselection_screen_glosry.htm'\) "Glossary Entry") defined by [PARAMETERS](javascript:call_link\('abapparameters.htm'\)) which is displayed as an [input field](javascript:call_link\('abeninput_output_field_glosry.htm'\) "Glossary Entry") and which is represented internally as an [elementary data object](javascript:call_link\('abenelementary_data_object_glosry.htm'\) "Glossary Entry").
