set indicator

[ABAP SQL indicator](javascript:call_link\('abenabap_sql_indicator_glosry.htm'\) "Glossary Entry") in a [work area](javascript:call_link\('abenwork_area_glosry.htm'\) "Glossary Entry") or an [internal table](javascript:call_link\('abeninternal_table_glosry.htm'\) "Glossary Entry"). The components of a set indicator indicate the columns of a [DDIC database table](javascript:call_link\('abenddic_db_table_glosry.htm'\) "Glossary Entry") to be changed using the [UPDATE](javascript:call_link\('abapupdate.htm'\)) statement.
