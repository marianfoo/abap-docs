simple enhancement implementation

Semantically interrelated enhancement of a [repository object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry"). A simple enhancement implementation manages one or more [enhancement implementation elements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_impl_elem_glosry.htm "Glossary Entry"). A simple enhancement implementation can occur independently or as a component of a [composite enhancement implementation](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencomp_enhancement_impl_glosry.htm "Glossary Entry").
