simple enhancement spot

Reference to a set of [enhancement spot element definitions](javascript:call_link\('abenenhancement_spot_def_glosry.htm'\) "Glossary Entry"). A simple enhancement spot can occur independently or as a component of a [composite enhancement spot](javascript:call_link\('abencomp_enhancement_spot_glosry.htm'\) "Glossary Entry").
