single code page system

System with only one [system code page](javascript:call_link\('abensystem_codepage_glosry.htm'\) "Glossary Entry"). Either a [Unicode system](javascript:call_link\('abenunicode_system_glosry.htm'\) "Glossary Entry") with [UTF 16](javascript:call_link\('abenutf16_glosry.htm'\) "Glossary Entry") or a non-Unicode system with a non-Unicode code page. From [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry") [ABAP release](javascript:call_link\('abenabap_release_glosry.htm'\) "Glossary Entry") 7.0, only single code page systems are supported and from ABAP release 7.50 only Unicode systems. The [text environment](javascript:call_link\('abentext_environment_glosry.htm'\) "Glossary Entry") for an [internal session](javascript:call_link\('abeninternal_session_glosry.htm'\) "Glossary Entry") can only be set using this code page in a single code page system. See also [MDMP system](javascript:call_link\('abenmdmp-system_glosry.htm'\) "Glossary Entry").
