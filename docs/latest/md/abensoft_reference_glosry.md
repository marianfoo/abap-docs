soft reference

Reference to an [object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") that does not keep an object alive when the available memory becomes insufficient. The class CL\_ABAP\_SOFT\_REFERENCE is intended for soft references but has not yet been implemented. See also [weak reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenweak_reference_glosry.htm "Glossary Entry").
