spool list level

Hierarchical level of a stacked [spool list](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_list_glosry.htm "Glossary Entry"). A spool list level has exactly one [spool request](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_request_glosry.htm "Glossary Entry") assigned to it.
