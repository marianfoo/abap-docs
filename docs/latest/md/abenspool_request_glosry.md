spool request

Output request to the [SAP spool system](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensap_spool_system_glosry.htm "Glossary Entry"). In a spool request, the pages of a [spool list](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_list_glosry.htm "Glossary Entry") are transferred to the SAP spool system. [Spool parameters](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenprint_parameter_glosry.htm "Glossary Entry") are defined when a spool request is opened and cannot be changed subsequently. A spool request is always assigned to the current [spool list level](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_list_level_glosry.htm "Glossary Entry"). The spool requests of stacked spool list levels are also stacked. A spool request is identified by a 10 digit [spool number](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspool_number_glosry.htm "Glossary Entry").
