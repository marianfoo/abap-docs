stack reference

[Reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenreference_glosry.htm "Glossary Entry") to an [object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenobject_glosry.htm "Glossary Entry") or part of an object in the [stack](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstack_glosry.htm "Glossary Entry"). A stack reference does not keep an object alive. See also [heap reference](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenheap_reference_glosry.htm "Glossary Entry").
