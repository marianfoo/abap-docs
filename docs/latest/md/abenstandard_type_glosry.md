standard type

[Complete data type](javascript:call_link\('abencomplete_data_type_glosry.htm'\) "Glossary Entry") used for [generically typed](javascript:call_link\('abengeneric_typing_glosry.htm'\) "Glossary Entry") [field symbols](javascript:call_link\('abenfield_symbol_glosry.htm'\) "Glossary Entry") or [formal parameters](javascript:call_link\('abenformal_parameter_glosry.htm'\) "Glossary Entry") if their data type is required for compilation, or if a LIKE reference is made to a generic field symbol, to which no memory area is assigned. Different standard types are defined for replacement parameters of optional formal parameters, generically typed field symbols, and for [inline declarations](javascript:call_link\('abeninline_declaration_glosry.htm'\) "Glossary Entry"), where they depend on the current statement.
