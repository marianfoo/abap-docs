start value

Value specified when a [data object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry") is declared. The data object is filled with this value when it is created. The start value is set using the [VALUE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapdata_options.htm) addition of the [DATA](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapdata.htm) statement. If the VALUE addition is not used, the system uses the type-specific [initial value](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninitial_value_glosry.htm "Glossary Entry").
