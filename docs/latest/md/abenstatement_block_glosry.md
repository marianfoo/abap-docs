statement block

Structural unit within [control structures](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencontrol_structure_glosry.htm "Glossary Entry") that consists of one or more successive statements. The statements of a statement block are processed sequentially. Statement blocks can again contain control structures.
