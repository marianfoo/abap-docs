static attribute

[Attribute](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenattribute_glosry.htm "Glossary Entry") of a [class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenclass_glosry.htm "Glossary Entry") declared using [CLASS-DATA](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapclass-data.htm) that is valid independent of the class [instance](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninstance_glosry.htm "Glossary Entry"). The content of the static attributes determines the instance-dependent state of a class.
