character string template

String template for short. A character string template creates a [character string](javascript:call_link\('abencharacter_string_glosry.htm'\) "Glossary Entry") from literal text, embedded expressions, and [control characters](javascript:call_link\('abencontrol_character_glosry.htm'\) "Glossary Entry") in a [string expression](javascript:call_link\('abenstring_expression_glosry.htm'\) "Glossary Entry").
