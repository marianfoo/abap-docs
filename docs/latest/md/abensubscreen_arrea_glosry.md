subscreen area

Area within a [screen layout](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenscreen_glosry.htm "Glossary Entry") in which other screen layouts can be displayed as [subscreens](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abensubscreen_glosry.htm "Glossary Entry").
