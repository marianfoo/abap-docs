Switch Framework

SFW for short. Framework for the externally controlling the visibility of [repository objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry"), their components, or [enhancements](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenenhancement_glosry.htm "Glossary Entry") using [switches](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenswitch_glosry.htm "Glossary Entry").
