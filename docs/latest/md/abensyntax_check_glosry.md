syntax check

Static check performed on [ABAP source code](javascript:call_link\('abenabap_source_code_glosry.htm'\) "Glossary Entry") to check whether the syntax is formally correct. The syntax check is called in the [ABAP Editor](javascript:call_link\('abenabap_editor_glosry.htm'\) "Glossary Entry") via Program → Check → Syntax or via the statement [SYNTAX-CHECK](javascript:call_link\('abapsyntax-check_for_itab.htm'\)). An automatic syntax check that checks the source code while it is entered can be configured in the [ABAP development tools for Eclipse (ADT)](javascript:call_link\('abenadt_glosry.htm'\) "Glossary Entry").
