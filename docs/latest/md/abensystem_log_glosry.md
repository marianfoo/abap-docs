system log

Central storage location for the recording of system errors on an [AS instance](javascript:call_link\('abenas_instance_glosry.htm'\) "Glossary Entry"). It is called via [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SM21.
