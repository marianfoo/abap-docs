table iteration

Sequential evaluation of an [internal table](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeninternal_table_glosry.htm "Glossary Entry"). Either in a [loop](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenloop_glosry.htm "Glossary Entry") introduced using [LOOP AT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abaploop_at_itab_variants.htm) or [PROVIDE](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapprovide.htm) or with the corresponding [variants](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfor_itab.htm) of an [iteration expression](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abeniteration_expression_glosry.htm "Glossary Entry").
