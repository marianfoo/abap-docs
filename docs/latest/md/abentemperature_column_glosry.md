temperature column

Column of a [database table](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendatabase_table_glosry.htm "Glossary Entry") called \_DATAAGING. On an [SAP HANA database](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenhana_database_glosry.htm "Glossary Entry"), the values in this column determine whether [data aging](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_aging_glosry.htm "Glossary Entry") data is current or obsolete.
