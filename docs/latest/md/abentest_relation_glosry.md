test relation

Relation between a [test class](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_class_glosry.htm "Glossary Entry") or a [test method](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_method_glosry.htm "Glossary Entry") and another [repository object](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenrepository_object_glosry.htm "Glossary Entry"). A test relation is defined by the special [ABAP Doc comment](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_doc_comment_glosry.htm "Glossary Entry") ["! @testing](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentest_relations.htm) and allows repository objects to be tested with [ABAP Unit](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_unit_glosry.htm "Glossary Entry") in [ADT](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenadt_glosry.htm "Glossary Entry") that do not support test classes themselves.
