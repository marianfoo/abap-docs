text string literal

[Character literal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencharacter_literal_glosry.htm "Glossary Entry") of the type [text string](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentext_string_type_glosry.htm "Glossary Entry"). Available as [typed](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentyped_literal_glosry.htm "Glossary Entry") or [untyped](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuntyped_literal_glosry.htm "Glossary Entry") [ABAP literal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_literal_glosry.htm "Glossary Entry") and as [typed](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abentyped_literal_glosry.htm "Glossary Entry") [CDS literal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abencds_literal_glosry.htm "Glossary Entry").
