text-like data type

[Data type](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") whose [data objects](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") contain texts. The corresponding [built-in data types](javascript:call_link\('abenbuiltin_data_type_glosry.htm'\) "Glossary Entry") are [c](javascript:call_link\('abenbuiltin_types_character.htm'\)) and [string](javascript:call_link\('abenbuiltin_types_character.htm'\)). The corresponding [generic data type](javascript:call_link\('abengeneric_data_type_glosry.htm'\) "Glossary Entry") is [csequence](javascript:call_link\('abenbuilt_in_types_generic.htm'\)).
