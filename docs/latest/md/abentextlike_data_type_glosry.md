text-like data type

[Data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_type_glosry.htm "Glossary Entry") whose [data objects](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_object_glosry.htm "Glossary Entry") contain texts. The corresponding [built-in data types](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuiltin_data_type_glosry.htm "Glossary Entry") are [c](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuiltin_types_character.htm) and [string](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuiltin_types_character.htm). The corresponding [generic data type](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abengeneric_data_type_glosry.htm "Glossary Entry") is [csequence](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenbuilt_in_types_generic.htm).
