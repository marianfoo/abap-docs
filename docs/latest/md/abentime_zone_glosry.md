time zone

Geographical area in which the same [local time](javascript:call_link\('abenlocal_time_glosry.htm'\) "Glossary Entry") and [local date](javascript:call_link\('abenlocal_date_glosry.htm'\) "Glossary Entry") apply based on a set of [time zone rules](javascript:call_link\('abentime_zone_rules.htm'\)) at a given time.
