Time Stamp Functions (ts\_func)

The following [built-in functions](javascript:call_link\('abenbuiltin_function_glosry.htm'\) "Glossary Entry") for processing time stamps in time stamp fields are available:

-   [utclong\_current](javascript:call_link\('abenutclong_current.htm'\)) creates a current time stamp
-   [utclong\_add](javascript:call_link\('abenutclong_add.htm'\)) adds values to a time stamp
-   [utclong\_diff](javascript:call_link\('abenutclong_diff.htm'\)) determines the difference between two time stamps
