transaction variant

Transaction variants can be used to preassign default values to the input fields of several [dynpros](javascript:call_link\('abendynpro_glosry.htm'\) "Glossary Entry") of a [transaction](javascript:call_link\('abentransaction_glosry.htm'\) "Glossary Entry"), to change the attributes of [screen elements](javascript:call_link\('abenscreen_element_glosry.htm'\) "Glossary Entry"), and to suppress entire [screens](javascript:call_link\('abenscreen_glosry.htm'\) "Glossary Entry"). Transaction variants are maintained and administrated using [transaction code](javascript:call_link\('abentransaction_code_glosry.htm'\) "Glossary Entry") SHD0.
