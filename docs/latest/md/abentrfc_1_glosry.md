transactional Remote Function Call

tRFC for short. Obsolete [remote function call](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenremote_function_call_glosry.htm "Glossary Entry") in which the function called remotely is registered for execution and started using the statement [COMMIT WORK](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapcommit.htm).
