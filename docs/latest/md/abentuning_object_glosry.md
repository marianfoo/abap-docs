tuning object

A [Repository object](javascript:call_link\('abenrepository_object_glosry.htm'\) "Glossary Entry") that defines technical settings for another repository object. A tuning object can be defined and transported separately from the repository object for which it is defined. There are [HANA tuning objects](javascript:call_link\('abenhana_tuning_object_glosry.htm'\) "Glossary Entry") and [ABAP tuning objects](javascript:call_link\('abenabap_tuning_object_glosry.htm'\) "Glossary Entry").
