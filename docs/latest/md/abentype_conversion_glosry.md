type conversion

Converts the content of a [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry") according to the associated [conversion rules](javascript:call_link\('abenconversion_rule_glosry.htm'\) "Glossary Entry"). Type conversions occur implicitly in [assignments](javascript:call_link\('abenassignment_glosry.htm'\) "Glossary Entry") between data objects with different [data types](javascript:call_link\('abendata_type_glosry.htm'\) "Glossary Entry") or can be performed explicitly using the [conversion operator](javascript:call_link\('abenconversion_operator_glosry.htm'\) "Glossary Entry") [CONV](javascript:call_link\('abenconstructor_expression_value.htm'\)).
