user memory

Memory area of the current [AS instance](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenas_instance_glosry.htm "Glossary Entry") to which all [ABAP sessions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_session_glosry.htm "Glossary Entry") of a [user session](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenuser_session_glosry.htm "Glossary Entry") have access. [ABAP programs](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_program_glosry.htm "Glossary Entry") have access to [SPA/GPA parameters](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenspa_gpa_parameter_glosry.htm "Glossary Entry") stored in the user memory. [ABAP daemons](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_daemon_glosry.htm "Glossary Entry") access the subarea [ABAP daemon memory](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_daemon_memory_glosry.htm "Glossary Entry") of the user memory
