user name

Name of an [ABAP user](javascript:call_link\('abenabap_user_glosry.htm'\) "Glossary Entry") that is required for logging on to a [user session](javascript:call_link\('abenuser_session_glosry.htm'\) "Glossary Entry") in the [AS ABAP](javascript:call_link\('abenas_abap_glosry.htm'\) "Glossary Entry"). Together with the [password](javascript:call_link\('abenpassword_glosry.htm'\) "Glossary Entry"), the user name allows authorized use of an AS ABAP.
