UTF

Short for Universal Transformation Format. Different representations of the [Unicode](javascript:call_link\('abenunicode_glosry.htm'\) "Glossary Entry") [character set](javascript:call_link\('abencharacter_set_glosry.htm'\) "Glossary Entry"). All UTF character representations objectively refer to the characters defined in Unicode and can be transformed into each other. The representations differ only in the type of addressing. See [UTF-8](javascript:call_link\('abenutf8_glosry.htm'\) "Glossary Entry"), [UTF-16](javascript:call_link\('abenutf16_glosry.htm'\) "Glossary Entry"), and [UTF-32](javascript:call_link\('abenutf32_glosry.htm'\) "Glossary Entry") plus [UCS-2](javascript:call_link\('abenucs2_glosry.htm'\) "Glossary Entry").
