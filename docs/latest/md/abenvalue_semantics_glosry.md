value semantics

Value semantics means that the referenced data object or the dereferenced [reference](javascript:call_link\('abenreference_glosry.htm'\) "Glossary Entry") is used when accessing a [deep](javascript:call_link\('abendeep_glosry.htm'\) "Glossary Entry") [data object](javascript:call_link\('abendata_object_glosry.htm'\) "Glossary Entry"). In ABAP, value semantics apply when accessing [dynamic data objects](javascript:call_link\('abendynamic_data_object_glosry.htm'\) "Glossary Entry") and [field symbols](javascript:call_link\('abenfield_symbol_glosry.htm'\) "Glossary Entry"). See also [reference semantics](javascript:call_link\('abenreference_semantics_glosry.htm'\) "Glossary Entry").
