window expression

In [ABAP SQL](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_sql_glosry.htm "Glossary Entry"), a window expression uses [OVER](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abapselect_over.htm) to define a subset of the result set of a [query](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwindow_function_glosry.htm "Glossary Entry") and applies a [window function](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwindow_function_glosry.htm "Glossary Entry") to it.
