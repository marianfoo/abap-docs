write position

[Operand position](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoperand_position_glosry.htm "Glossary Entry") in which the [operand](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenabap_operand_glosry.htm "Glossary Entry") is written to. There are [read/write positions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenread_write_position_glosry.htm "Glossary Entry"), where an operand is read-accessed before writing and [write-only positions](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenwrite_only_position_glosry.htm "Glossary Entry") where no read access occurs.
