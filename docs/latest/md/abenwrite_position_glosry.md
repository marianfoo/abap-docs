write position

[Operand position](javascript:call_link\('abenoperand_position_glosry.htm'\) "Glossary Entry") in which the [operand](javascript:call_link\('abenabap_operand_glosry.htm'\) "Glossary Entry") is written to. There are [read/write positions](javascript:call_link\('abenread_write_position_glosry.htm'\) "Glossary Entry"), where an operand is read-accessed before writing and [write-only positions](javascript:call_link\('abenwrite_only_position_glosry.htm'\) "Glossary Entry") where no read access occurs.
