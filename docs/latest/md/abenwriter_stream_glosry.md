writer stream

Also referred to as [output stream](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenoutput_stream_glosry.htm "Glossary Entry"). Either a [data stream](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abendata_stream_glosry.htm "Glossary Entry") that is connected to a [streaming data sink](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenstreaming_data_sink_glosry.htm "Glossary Entry") or a [filter stream](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/abenfilter_stream_glosry.htm "Glossary Entry") that writes data to another stream. The direction of a writer stream always goes toward the data sink. In a writer stream data can only be written.
