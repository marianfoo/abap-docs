SAP HANA XS

Short for SAP HANA extended application services. XS is a framework for developing and managing native applications in [SAP HANA](javascript:call_link\('abensap_hana_glosry.htm'\) "Glossary Entry"). The current version, [XSA](javascript:call_link\('abenxsa_glosry.htm'\) "Glossary Entry"), replaces the previous version, [XSC](javascript:call_link\('abenxsc_glosry.htm'\) "Glossary Entry"). XS objects can be accessed in ABAP.
