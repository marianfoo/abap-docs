dynpro - ENDCHAIN

Syntax

Effect

Closes a processing chain defined in the [dynpro flow logic](javascript:call_link\('abendynpro_flow_logic_glosry.htm'\) "Glossary Entry"). See [CHAIN](javascript:call_link\('dynpchain.htm'\)) for more information.
